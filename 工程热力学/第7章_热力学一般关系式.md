## 引言——为什么需要这一章

能源工程所用工质多为**成分固定、两个自由度**的简单可压缩流体。给定两个独立状态参数 → 整个状态确定 → 其余状态参数原则上都可由此二参数计算出来。本章目的：**找出 z = z(x, y) 函数关系的具体形式**，利用可测参数 (p, v, T, cp) 推算难以直接测量的量 (u, h, s 等)。

周磊课件定位：本章公式**最多可达188个**，是全书数学味最浓的一章，但其威力在于——**只需少数必要的热物性数据（如 cp 和状态方程），就能推算出全面的热物性知识**。

## 7-1 数学基础——二元连续函数的数学特性

### 状态参数 = 点函数

**定义**：状态参数是**点函数**——只取决于状态本身，与路径无关。对简单可压缩物质，任意第三个状态参数 z 必为两个独立变量 (x, y) 的函数：z = z(x, y)，其微分满足全微分条件。

一·三大数学关系式

  1. 全微分条件（欧拉互易关系）

**定义**：若 dz = M dx + N dy 是某二元函数 z(x, y) 的全微分，则混合偏导数相等。

**公式**：

$$\frac{\partial M}{\partial y}\bigg|_x = \frac{\partial N}{\partial x}\bigg|_y \quad\text{即}\quad \frac{\partial^2 z}{\partial x\partial y} = \frac{\partial^2 z}{\partial y\partial x}$$

其中 $M = \left(\frac{\partial z}{\partial x}\right)_y$，$N = \left(\frac{\partial z}{\partial y}\right)_x$。

**推导**：
- （1）由 z = z(x, y)，写出全微分：$dz = \left(\frac{\partial z}{\partial x}\right)_y dx + \left(\frac{\partial z}{\partial y}\right)_x dy = M dx + N dy$ ← 全微分定义
- （2）对 M 求 y 偏导：$\frac{\partial M}{\partial y} = \frac{\partial^2 z}{\partial y\partial x}$；对 N 求 x 偏导：$\frac{\partial N}{\partial x} = \frac{\partial^2 z}{\partial x\partial y}$ ← 混合偏导
- （3）因 z 是状态函数（连续可微）→ 混合偏导与求导次序无关 → $\frac{\partial^2 z}{\partial x\partial y} = \frac{\partial^2 z}{\partial y\partial x}$ ← 欧拉定理
→ 结论：==全微分条件是判断某微分式是否为状态参数全微分的充要条件==。

> [!note] 核心理解
> 物理上：状态参数的值只与状态有关，不依赖于"怎么到达这个状态"——全微分条件就是这一性质的数学表达。数学上：这是微积分中的施瓦茨定理（Clairaut's theorem）在热力学中的应用——对于 C² 类二元函数，混合偏导与次序无关。

  2. 循环关系式

**定义**：三个两两独立的状态参数 x, y, z 之间，偏导数满足循环乘积 = −1。

**公式**：

$$\left(\frac{\partial x}{\partial y}\right)_z \left(\frac{\partial y}{\partial z}\right)_x \left(\frac{\partial z}{\partial x}\right)_y = -1$$

**推导**：
- （1）对 z = z(x, y)，在 z 不变时 dz = 0：$0 = \left(\frac{\partial z}{\partial x}\right)_y dx_z + \left(\frac{\partial z}{\partial y}\right)_x dy_z$ ← 定 z 条件
- （2）移项：$\left(\frac{\partial z}{\partial x}\right)_y dx_z = -\left(\frac{\partial z}{\partial y}\right)_x dy_z$
- （3）两端除以 $dy_z \cdot \left(\frac{\partial z}{\partial x}\right)_y$：$\frac{dx_z}{dy_z} = -\frac{(\partial z/\partial y)_x}{(\partial z/\partial x)_y}$，即 $\left(\frac{\partial x}{\partial y}\right)_z = -\frac{(\partial z/\partial y)_x}{(\partial z/\partial x)_y}$ ← 偏导定义
- （4）利用倒数式 $\left(\frac{\partial z}{\partial x}\right)_y = 1/\left(\frac{\partial x}{\partial z}\right)_y$，代入得循环关系式 ← 倒数关系

→ 用途：==当某个偏导数不易直接测量时，可用循环式转化为另外两个可测偏导数==。

**图**：三个变量 x ↔ y ↔ z 构成循环三角形：

```
    x
   / \
  /   \
 y ←—→ z
每一对均可互求偏导，三者循环乘积恒为 −1
```

  3. 链式关系式

**定义**：当引入第四个变量 w 作为下标时，三个变量的链式偏导乘积 = 1。

**公式**：

$$\left(\frac{\partial x}{\partial y}\right)_w \left(\frac{\partial y}{\partial z}\right)_w \left(\frac{\partial z}{\partial x}\right)_w = 1$$

**推导**：
- （1）设有四个特性量 x, y, z, w，其中任意两个独立。考虑函数 z = z(y, w) 和 y = y(z, w)
- （2）写出全微分：$dz = \left(\frac{\partial z}{\partial y}\right)_w dy + \left(\frac{\partial z}{\partial w}\right)_y dw$；$dy = \left(\frac{\partial y}{\partial z}\right)_w dz + \left(\frac{\partial y}{\partial w}\right)_z dw$ ← 全微分定义
- （3）将 dy 代入 dz 表达式，整理 dz 和 dw 系数
- （4）与 $dz = \left(\frac{\partial z}{\partial x}\right)_w dx + \left(\frac{\partial z}{\partial w}\right)_x dw$ 对比系数
→ 链式式实质是"同一下标下，偏导像普通分数一样可约分"

**推导细节**（教材式 7-3, 7-4，补充完整）：

- （1）$dz = \left(\frac{\partial z}{\partial y}\right)_w dy + \left(\frac{\partial z}{\partial w}\right)_y dw$ ← 全微分
- （2）代入 $dy = \left(\frac{\partial y}{\partial x}\right)_w dx + \left(\frac{\partial y}{\partial w}\right)_x dw$：
  $$dz = \left(\frac{\partial z}{\partial y}\right)_w \left(\frac{\partial y}{\partial x}\right)_w dx + \left[\left(\frac{\partial z}{\partial y}\right)_w \left(\frac{\partial y}{\partial w}\right)_x + \left(\frac{\partial z}{\partial w}\right)_y\right] dw$$
- （3）又 $z = z(x, w)$ 给出 $dz = \left(\frac{\partial z}{\partial x}\right)_w dx + \left(\frac{\partial z}{\partial w}\right)_x dw$
- （4）对比 dx 系数：$\left(\frac{\partial z}{\partial x}\right)_w = \left(\frac{\partial z}{\partial y}\right)_w \left(\frac{\partial y}{\partial x}\right)_w$
- （5）即 $\left(\frac{\partial x}{\partial y}\right)_w \left(\frac{\partial y}{\partial z}\right)_w \left(\frac{\partial z}{\partial x}\right)_w = 1$ ← 链式关系式

> [!tip] 补充——不同下标式
> 当需要将一种下标下的偏导转换为另一种下标时用：
> $$\left(\frac{\partial x}{\partial w}\right)_z = \left(\frac{\partial x}{\partial w}\right)_y + \left(\frac{\partial x}{\partial y}\right)_w \left(\frac{\partial y}{\partial w}\right)_z$$
> 物理意义：x 对 w 的全变化 = w 直接引起的变化 + 通过 y 间接引起的变化。

> [!note] 核心理解
> 物理上：状态参数空间是二维流形——任何第三量必由前两个决定，所以偏导数之间有约束关系。循环式 −1 来自"绕一圈回到原点偏导乘积抵消一次正负号"的拓扑性质；链式 1 来自"同一下标下偏导数可以像分数一样连乘约分"。数学上：这本质是**雅可比行列式的性质**——J(x,y)/J(z,y) · J(y,z)/J(x,z) · J(z,x)/J(y,x) = −1 就是循环式。

**易错**：循环式是 **−1**，链式是 **+1**，最容易记混。记忆诀窍——循环式下标各不相同（x|y, y|z, z|x），乘出来 −1；链式下标统一（全 |w），乘出来 +1。

## 7-2 热系数

**定义**：由可测参数 (p, v, T) 组成的、有明确物理意义的偏导数统称为**热系数**。它们是连接"可测量"与"不可测量"的桥梁。

一·四个基本热系数

  1. 热膨胀系数（体膨胀系数）αp

**定义**：物质在**定压条件**下，比体积随温度的变化率（以当前比体积归一化）。

**公式**：

$$\alpha_p = \frac{1}{v}\left(\frac{\partial v}{\partial T}\right)_p \quad (\text{K}^{-1})$$

**性质**：
- 一般 αp > 0（定压下升温 → 膨胀）
- 例外：水在 0～4°C 之间 αp < 0（反常膨胀），某些合金也有此性质
- 理想气体：由 $pv = R_g T$ → $\left(\frac{\partial v}{\partial T}\right)_p = \frac{R_g}{p} = \frac{v}{T}$ → ==对理想气体，$\alpha_p = 1/T$==

  2. 定温压缩系数 κT

**定义**：物质在**定温条件**下，比体积随压力的变化率（以当前比体积归一化，加负号使值为正）。

**公式**：

$$\kappa_T = -\frac{1}{v}\left(\frac{\partial v}{\partial p}\right)_T \quad (\text{Pa}^{-1})$$

**性质**：
- ==κT 恒为正值==（定温下加压 → 比体积必定减小）——这是物质**稳定存在的必要条件**，否则"压力越高体积越大 → 体积越大压力越高"无法获得平衡
- 理想气体：$pv = R_g T$ → $\left(\frac{\partial v}{\partial p}\right)_T = -\frac{R_g T}{p^2} = -\frac{v}{p}$ → ==对理想气体，$\kappa_T = 1/p$==

  3. 定熵压缩系数 κs

**定义**：物质在**定熵（绝热可逆）条件**下，比体积随压力的变化率。

**公式**：

$$\kappa_s = -\frac{1}{v}\left(\frac{\partial v}{\partial p}\right)_s \quad (\text{Pa}^{-1})$$

**性质**：恒为正值，道理同 κT。

  4. 弹性系数（相对压力系数）γv

**定义**：物质在**定容条件**下，压力随温度的变化率（以当前压力归一化）。

**公式**：

$$\gamma_v = \frac{1}{p}\left(\frac{\partial p}{\partial T}\right)_v \quad (\text{K}^{-1})$$

**性质**：
- 一般 γv > 0；当 αp < 0 时，γv 也为负值
- 理想气体：$\left(\frac{\partial p}{\partial T}\right)_v = \frac{R_g}{v} = \frac{p}{T}$ → ==对理想气体，$\gamma_v = 1/T$==

二·热系数之间的关系

**公式与推导**：利用循环关系式导出：

- （1）对 p, v, T 应用循环式：$\left(\frac{\partial p}{\partial T}\right)_v \left(\frac{\partial T}{\partial v}\right)_p \left(\frac{\partial v}{\partial p}\right)_T = -1$ ← 循环关系式 (7-2)
- （2）代入各热系数定义：
  - $\left(\frac{\partial p}{\partial T}\right)_v = p\gamma_v$
  - $\left(\frac{\partial T}{\partial v}\right)_p = 1/(v\alpha_p)$ ← 倒数式
  - $\left(\frac{\partial v}{\partial p}\right)_T = -v\kappa_T$
- （3）乘积：$(p\gamma_v) \cdot \frac{1}{v\alpha_p} \cdot (-v\kappa_T) = -1$
→ **==$\alpha_p = \gamma_v \cdot \kappa_T \cdot p$==** ← 热系数间的关系式 (7-10)

→ 推论：==γv 必与 αp 同号==（因 κT·p > 0）。

**应用举例**（周磊课件）：

用实验测熵变——利用 Maxwell 关系 $(\partial s/\partial p)_T = -(\partial v/\partial T)_p = -v\alpha_p$：

$$\Delta s_T = \int -\left(\frac{\partial v}{\partial T}\right)_p dp = \int -v\alpha_p\,dp$$

→ 只需测 p-v-T 关系和 αp，即可算熵变，无需直接测熵。

三·范德瓦尔气体的热系数（例 7-1 核心结果）

对范德瓦尔方程 $p = \frac{R_g T}{v-b} - \frac{a}{v^2}$（a, b 为常数）：

- （1）求 $(\partial p/\partial T)_v = R_g/(v-b)$ ← 直接求导
- （2）弹性系数：$\gamma_v = \frac{1}{p}\left(\frac{\partial p}{\partial T}\right)_v = \frac{R_g}{p(v-b)}$
- （3）用循环式求 $(\partial v/\partial T)_p$：$\left(\frac{\partial v}{\partial T}\right)_p = -\left(\frac{\partial p}{\partial T}\right)_v / \left(\frac{\partial p}{\partial v}\right)_T = \frac{R_g v^3(v-b)}{R_g T v^3 - 2a(v-b)^2}$
- （4）进得 αp, κT

> [!note] 核心理解
> 物理上：热系数刻画物质对温度/压力变化的"响应灵敏度"——αp 是"升温膨胀有多快"，κT 是"加压压缩有多难"。理想气体的三个系数都等于 1/T 或 1/p，因为它们全部源于同一状态方程 pv = R_g T。数学上：四个热系数可通过循环式相互导出——**本质上只需知道其中两个 + 状态方程**，其余都能计算。这就是一般关系式的威力。

**易错**：
- αp 是定**压**下 v 对 T 的偏导，不是定温
- κT 定义有**负号**，保证值为正；别漏
- 循环关系式中的 −1 经常在推导热系数关系时用到，记牢

## 7-3 自由能、自由焓、麦克斯韦关系式

### 3.1 特征函数（特性函数）

**定义**：在适当选择独立变量的情况下，只要给出**一个具体的函数式**，系统的全部平衡性质（所有其他状态参数）均可由此导出——这样的函数称为**特征函数**。

- 对简单可压缩系统：两个独立变量即可确定状态
- u = f(s, v) 是特征函数（以 s, v 为独立变量）
- h = h(s, p) 是特征函数（以 s, p 为独立变量）
- f = f(T, v) 是特征函数（以 T, v 为独立变量）
- g = g(T, p) 是特征函数（以 T, p 为独立变量）——**最实用**，因为 T, p 最易测量

> [!tip] 补充
> 并非任意选两个独立变量得出的函数都是特征函数。例如 u = f(T, v) 就不是特征函数——仅凭 u(T, v) 无法导出全部性质（比如无法直接得到 s）。特征函数的独立变量必须是**共轭对**中恰当的组合。

### 3.2 四个吉布斯方程（特征函数的微分式）

一·推导起点——热力学恒等式

- 可逆过程：$Tds = du + pdv$ → **$du = Tds - pdv$** ← (7-13)，熵定义的直接推论

在此基础上，通过勒让德变换引入三个新的组合状态参数：

二·自由能（亥姆霍兹函数）f

**定义**：$f = u - Ts$ ← (7-11)

**物理意义**：f 的减少 = 可逆等温过程的**膨胀功**（f 是可逆等温条件下内能中可转变为功的那部分），因此也称**亥姆霍兹自由能**。

**推导 df**：
- （1）$df = du - Tds - sdT$ ← 微分 f = u − Ts
- （2）代入 $du = Tds - pdv$：$df = (Tds - pdv) - Tds - sdT$
- （3）消去 Tds：**$df = -sdT - pdv$** ← (7-15)

→ f = f(T, v) 是特征函数

三·自由焓（吉布斯函数）g

**定义**：$g = h - Ts = u + pv - Ts$ ← (7-12)

**物理意义**：g 的减少 = 可逆等温过程对外的**技术功**（g 是可逆等温条件下焓中可转变为功的那部分），因此也称**吉布斯自由焓**。

**推导 dg**：
- （1）$dg = dh - Tds - sdT$ ← 微分 g = h − Ts
- （2）先求 dh：由 $h = u + pv$，$dh = du + pdv + vdp = (Tds - pdv) + pdv + vdp = Tds + vdp$ ← (7-14)
- （3）代入：$dg = (Tds + vdp) - Tds - sdT$
- （4）消去 Tds：**$dg = -sdT + vdp$** ← (7-16)

→ g = g(T, p) 是特征函数——**最重要、最常用**

四·总结——四个吉布斯方程

```
┌─────────────────────────────────────────────────┐
│  du =  Tds − pdv     u = u(s, v)    特征函数    │
│  dh =  Tds + vdp     h = h(s, p)    特征函数    │
│  df = −sdT − pdv     f = f(T, v)    特征函数    │
│  dg = −sdT + vdp     g = g(T, p)    特征函数    │
└─────────────────────────────────────────────────┘
```

**记忆诀窍**（周磊方法）：
- 只需记住 $du = Tds - pdv$ 和 $dh = Tds + vdp$
- df 和 dg 把 Tds 换成 −sdT：$df = -sdT - pdv$，$dg = -sdT + vdp$
- 规律：**T 和 s 在同一项里符号相反（T↔s，v↔p）**

### 3.3 八个偏导数——直接从吉布斯方程读出

将每个 dw = M dx + N dy 与全微分 $dw = \left(\frac{\partial w}{\partial x}\right)_y dx + \left(\frac{\partial w}{\partial y}\right)_x dy$ 对比：

```
  函数           对比 dx 系数              对比 dy 系数
  ─────────────────────────────────────────────────
  u(s,v):     T = (∂u/∂s)v             −p = (∂u/∂v)s
  h(s,p):     T = (∂h/∂s)p              v = (∂h/∂p)s
  f(T,v):    −s = (∂f/∂T)v             −p = (∂f/∂v)T
  g(T,p):    −s = (∂g/∂T)p              v = (∂g/∂p)T
```

共 8 个偏导数，可直接从四个吉布斯方程"读"出来。

### 3.4 四个麦克斯韦关系式——本章核心

**定义**：将全微分条件（混合偏导相等）分别作用于四个吉布斯方程，得到的四个关系式。

**来源方框图**（ASCII）：

```
    du = Tds − pdv               dh = Tds + vdp
    u = u(s, v)                  h = h(s, p)
         │                             │
         ▼ 全微分条件                   ▼ 全微分条件
  ∂²u/(∂v∂s) = ∂²u/(∂s∂v)       ∂²h/(∂p∂s) = ∂²h/(∂s∂p)
         │                             │
         ▼                             ▼
    ∂T/∂v|s = −∂p/∂s|v           ∂T/∂p|s = ∂v/∂s|p
    ←—— Maxwell I ——→            ←—— Maxwell II ——→

    df = −sdT − pdv              dg = −sdT + vdp
    f = f(T, v)                  g = g(T, p)
         │                             │
         ▼ 全微分条件                   ▼ 全微分条件
  ∂²f/(∂v∂T) = ∂²f/(∂T∂v)       ∂²g/(∂p∂T) = ∂²g/(∂T∂p)
         │                             │
         ▼                             ▼
    ∂s/∂v|T = ∂p/∂T|v            ∂s/∂p|T = −∂v/∂T|p
    ←—— Maxwell III ——→          ←—— Maxwell IV ——→
```

**四个关系式完整列出**：

$$
\begin{aligned}
\text{I（来自 du）:}\quad &\left(\frac{\partial T}{\partial v}\right)_s = -\left(\frac{\partial p}{\partial s}\right)_v \\[8pt]
\text{II（来自 dh）:}\quad &\left(\frac{\partial T}{\partial p}\right)_s = \left(\frac{\partial v}{\partial s}\right)_p \\[8pt]
\text{III（来自 df）:}\quad &\left(\frac{\partial s}{\partial v}\right)_T = \left(\frac{\partial p}{\partial T}\right)_v \\[8pt]
\text{IV（来自 dg）:}\quad &\left(\frac{\partial s}{\partial p}\right)_T = -\left(\frac{\partial v}{\partial T}\right)_p
\end{aligned}
$$

**推导——以 Maxwell I 为例，分步**：

- （1）由 $du = Tds - pdv$，令 $M = T$，$N = -p$ ← 识别 M, N
- （2）对 M 求 v 偏导（定 s）：$\left(\frac{\partial M}{\partial v}\right)_s = \left(\frac{\partial T}{\partial v}\right)_s$ ← 偏导运算
- （3）对 N 求 s 偏导（定 v）：$\left(\frac{\partial N}{\partial s}\right)_v = \left(\frac{\partial (-p)}{\partial s}\right)_v = -\left(\frac{\partial p}{\partial s}\right)_v$ ← 偏导运算
- （4）全微分条件要求 $\frac{\partial M}{\partial v} = \frac{\partial N}{\partial s}$ → $\left(\frac{\partial T}{\partial v}\right)_s = -\left(\frac{\partial p}{\partial s}\right)_v$ ← Maxwell I

其余三个同理——每个都把对应的吉布斯方程代入全微分条件即可。

> [!note] 核心理解——为什么麦克斯韦关系式重要
> 物理上：四个 Maxwell 关系式的共同特征是**将不可直接测量的熵的偏导数 (∂s/∂v, ∂s/∂p) 转化为可直接测量的 p-v-T 偏导数**。例如 Maxwell III：(∂s/∂v)T = (∂p/∂T)v——左边是"等温下熵如何随体积变化"（无法直接测量），右边是"等容下压力如何随温度变化"（可直接用 p-v-T 实验测！）。这就是麦克斯韦关系式的革命性意义——==把抽象的熵与可操作的 p-v-T 测量联系起来==。
> 数学上：全微分条件 + 吉布斯方程 = Maxwell 关系式。四个吉布斯方程恰好形成两组共轭对：(T,s) 和 (−p,v)。Maxwell 关系式实质是"交换共轭变量对的偏导符号规则"。

**易错**：
- Maxwell I 和 IV 有**负号**，II 和 III 没有——不要搞混
- 下标容易被忽略：Maxwell I 的左边下标是 s，右边下标是 v；Maxwell III 的左边下标是 T，右边下标是 v
- 记忆图案：T↔v 在下标交换（Maxwell I），s↔p 在下标交换（Maxwell IV），两两配对

### 3.5 特性函数 g(T, p) 的威力——全部性质一键导出

以最实用的 g = g(T, p) 为例（教材 7-3 节总结）：

```
  指定独立变量: T, p
  给出特性函数: g = g(T, p)
  ─────────────────────────────────
  v = (∂g/∂p)T            ← 比体积
  s = −(∂g/∂T)p           ← 熵
  h = g + Ts = g − T(∂g/∂T)p       ← 焓
  u = h − pv = g − T(∂g/∂T)p − p(∂g/∂p)T  ← 热力学能
  f = g − pv = g − p(∂g/∂p)T       ← 自由能
```

→ 只需**一个** g(T, p) 的函数形式，全部热力学性质尽在掌握。

## 7-4 熵、热力学能和焓的一般关系式

核心目标：用可测量 (p, v, T, cp, cv) 表达 ds, du, dh。

### 4.1 熵的三个 ds 方程

一·第一 ds 方程——以 T, v 为独立变量

**推导**：
- （1）s = s(T, v)：$ds = \left(\frac{\partial s}{\partial T}\right)_v dT + \left(\frac{\partial s}{\partial v}\right)_T dv$ ← 全微分展开
- （2）$(\partial s/\partial T)_v = c_v/T$ ← 由 $c_v = T(\partial s/\partial T)_v$ 定义得出（比定容热容的严格定义）
- （3）$(\partial s/\partial v)_T = (\partial p/\partial T)_v$ ← ==Maxwell III==
- （4）代入：

$$\boxed{ds = \frac{c_v}{T} dT + \left(\frac{\partial p}{\partial T}\right)_v dv}\quad\text{——第一 ds 方程 (7-25)}$$

**验证——对理想气体**：$p = R_g T/v$ → $(\partial p/\partial T)_v = R_g/v$，代入：
$$ds = \frac{c_v}{T} dT + \frac{R_g}{v} dv \rightarrow s = \int \frac{c_v}{T} dT + R_g \ln v + s_0 \quad\text{（与第3章结论一致）}$$

二·第二 ds 方程——以 T, p 为独立变量

**推导**：
- （1）s = s(T, p)：$ds = \left(\frac{\partial s}{\partial T}\right)_p dT + \left(\frac{\partial s}{\partial p}\right)_T dp$ ← 全微分展开
- （2）$(\partial s/\partial T)_p = c_p/T$ ← 由 $c_p = T(\partial s/\partial T)_p$ 定义
- （3）$(\partial s/\partial p)_T = -(\partial v/\partial T)_p$ ← ==Maxwell IV==
- （4）代入：

$$\boxed{ds = \frac{c_p}{T} dT - \left(\frac{\partial v}{\partial T}\right)_p dp}\quad\text{——第二 ds 方程 (7-26)}$$

三·第三 ds 方程——以 p, v 为独立变量

**推导**：
- （1）s = s(p, v)：$ds = \left(\frac{\partial s}{\partial p}\right)_v dp + \left(\frac{\partial s}{\partial v}\right)_p dv$ ← 全微分展开
- （2）用链式：$(\partial s/\partial p)_v = (\partial s/\partial T)_v (\partial T/\partial p)_v = \frac{c_v}{T} (\partial T/\partial p)_v$ ← 链式 + cv 定义
- （3）同理：$(\partial s/\partial v)_p = (\partial s/\partial T)_p (\partial T/\partial v)_p = \frac{c_p}{T} (\partial T/\partial v)_p$ ← 链式 + cp 定义
- （4）代入：

$$\boxed{ds = \frac{c_v}{T}\left(\frac{\partial T}{\partial p}\right)_v dp + \frac{c_p}{T}\left(\frac{\partial T}{\partial v}\right)_p dv}\quad\text{——第三 ds 方程 (7-27)}$$

> [!tip] 补充——三个 ds 方程的选用策略
> 第一 ds 方程最简单实用（只用 cv 和一个 p-v-T 偏导），通常优先选 T, v 为独立变量。第二 ds 方程好用在知道 cp 和状态方程 v(T, p) 时。第三 ds 方程较复杂，较少直接使用。

### 4.2 热力学能的三个 du 方程

**推导路径**：将 ds 方程代入 $du = Tds - pdv$。

一·第一 du 方程——以 T, v 为独立变量

- （1）将第一 ds 方程代入 du = Tds − pdv：
  $$du = T\left[\frac{c_v}{T} dT + \left(\frac{\partial p}{\partial T}\right)_v dv\right] - pdv$$
- （2）整理：

$$\boxed{du = c_v dT + \left[T\left(\frac{\partial p}{\partial T}\right)_v - p\right] dv}\quad\text{——第一 du 方程 (7-28)}$$

→ ==三个 du 方程中形式最简单、最常用==。计算 du 时优先选 T, v 为独立变量。

二·第二 du 方程——以 T, p 为独立变量

- （1）将第二 ds 方程代入 $du = Tds - pdv$
- （2）还需将 dv 也用 T, p 展开：$dv = \left(\frac{\partial v}{\partial T}\right)_p dT + \left(\frac{\partial v}{\partial p}\right)_T dp$
- （3）代入整理：

$$\boxed{du = \left[c_p - p\left(\frac{\partial v}{\partial T}\right)_p\right] dT - \left[T\left(\frac{\partial v}{\partial T}\right)_p + p\left(\frac{\partial v}{\partial p}\right)_T\right] dp}\quad\text{——第二 du 方程 (7-29)}$$

三·第三 du 方程——以 p, v 为独立变量

- （1）将第三 ds 方程代入 du = Tds − pdv
- （2）整理：

$$\boxed{du = c_v\left(\frac{\partial T}{\partial p}\right)_v dp + \left[c_p\left(\frac{\partial T}{\partial v}\right)_p - p\right] dv}\quad\text{——第三 du 方程 (7-30)}$$

### 4.3 焓的三个 dh 方程

**推导路径**：将 ds 方程代入 $dh = Tds + vdp$（或利用 $h = u + pv$ 转换）。

一·第二 dh 方程——以 T, p 为独立变量（最常用）

- （1）将第二 ds 方程代入 dh = Tds + vdp：
  $$dh = T\left[\frac{c_p}{T} dT - \left(\frac{\partial v}{\partial T}\right)_p dp\right] + vdp$$
- （2）整理：

$$\boxed{dh = c_p dT + \left[v - T\left(\frac{\partial v}{\partial T}\right)_p\right] dp}\quad\text{——第二 dh 方程 (7-32)}$$

→ ==三个 dh 方程中最常用==，因 T, p 最易测量。

二·第一 dh 方程——以 T, v 为独立变量

- （1）将第一 ds 方程代入 dh = Tds + vdp，还需展开 $dp = (\partial p/\partial T)_v dT + (\partial p/\partial v)_T dv$
- （2）整理：

$$\boxed{dh = \left[c_v + v\left(\frac{\partial p}{\partial T}\right)_v\right] dT + \left[T\left(\frac{\partial p}{\partial T}\right)_v + v\left(\frac{\partial p}{\partial v}\right)_T\right] dv}\quad\text{——第一 dh 方程 (7-31)}$$

三·第三 dh 方程——以 p, v 为独立变量

$$\boxed{dh = \left[c_v\left(\frac{\partial T}{\partial p}\right)_v + v\right] dp + c_p\left(\frac{\partial T}{\partial v}\right)_p dv}\quad\text{——第三 dh 方程 (7-33)}$$

> [!note] 核心理解
> 物理上：du/dh 方程告诉我们——内能/焓的变化来自两方面：温度变化（cv dT / cp dT 项）和体积/压力变化（方括号修正项）。对理想气体，修正项为零（因 T(∂p/∂T)v − p = p − p = 0），内能仅是温度的函数；对实际气体，修正项体现**分子间作用力和分子体积**的贡献。数学上：这三个 du/dh 方程本质是换元——把 ds 方程的独立变量组合转换到 du = Tds − pdv 和 dh = Tds + vdp 的表达式中。

**应用举例——范德瓦尔气体 du（例 7-3, 7-4）**：

对范德瓦尔气体，$(\partial p/\partial T)_v = R_g/(v-b)$，代入第一 du 方程：
$$du = c_v dT + \left[T\cdot\frac{R_g}{v-b} - \left(\frac{R_g T}{v-b} - \frac{a}{v^2}\right)\right] dv = c_v dT + \frac{a}{v^2} dv$$

积分得：$u = c_v T - a/v + u_0$ → 定温下 $u_2 - u_1 = a(1/v_1 - 1/v_2)$（教材例 7-4 数值：对氮气 20°C 下定温膨胀，$u_2 - u_1 = 436.0$ J/kg）

## 7-5 比热容和焦耳-汤姆逊系数的一般关系式

### 5.1 比热容的严格定义

一·cv 和 cp 的严格定义式

- （1）$c_v = \left(\frac{\partial u}{\partial T}\right)_v$ ← 工程定义（第3章），等于 $\left(\frac{\partial u}{\partial s}\right)_v \left(\frac{\partial s}{\partial T}\right)_v = T\left(\frac{\partial s}{\partial T}\right)_v$ ← (7-34)，热力学严格定义
- （2）$c_p = \left(\frac{\partial h}{\partial T}\right)_p$ ← 工程定义（第3章），等于 $\left(\frac{\partial h}{\partial s}\right)_p \left(\frac{\partial s}{\partial T}\right)_p = T\left(\frac{\partial s}{\partial T}\right)_p$ ← (7-35)，热力学严格定义

> [!tip] 补充——为什么需要"严格定义"
> 第3章的 cv = (∂u/∂T)v 和 cp = (∂h/∂T)p 在无摩擦条件下才等价于 T(∂s/∂T)。本章的 cv = T(∂s/∂T)v 是**由物质平衡性质确定**的严格定义式，不论过程是否可逆都成立。

### 5.2 比热容差 cp − cv

**公式与推导**：

- （1）由定义：$c_p - c_v = T\left[\left(\frac{\partial s}{\partial T}\right)_p - \left(\frac{\partial s}{\partial T}\right)_v\right]$ ← (7-34)(7-35)
- （2）利用不同下标式：$(\partial s/\partial T)_p = (\partial s/\partial T)_v + (\partial s/\partial v)_T (\partial v/\partial T)_p$
- （3）代入：$c_p - c_v = T(\partial s/\partial v)_T (\partial v/\partial T)_p$
- （4）用 ==Maxwell III==：$(\partial s/\partial v)_T = (\partial p/\partial T)_v$，得第一形式：

$$\boxed{c_p - c_v = T\left(\frac{\partial p}{\partial T}\right)_v \left(\frac{\partial v}{\partial T}\right)_p}\quad\text{—— (7-36)}$$

- （5）用循环式变换：$(\partial p/\partial T)_v = -(\partial v/\partial T)_p / (\partial v/\partial p)_T$，得第二形式：

$$\boxed{c_p - c_v = -T\frac{(\partial v/\partial T)_p^2}{(\partial v/\partial p)_T}}\quad\text{—— (7-37)}$$

- （6）用热系数表达：$c_p - c_v = \frac{Tv\alpha_p^2}{\kappa_T}$ ← 因 $(\partial v/\partial T)_p = v\alpha_p$，$-(\partial v/\partial p)_T = v\kappa_T$

> [!note] 核心理解
> 物理上：==cp − cv > 0 恒成立==。因为 κT > 0（物质稳定条件），而 (∂v/∂T)p² ≥ 0 → cp ≥ cv 永远成立。定压下升温时气体膨胀对外做功、需额外吸热 → cp 天然大于 cv。数学上：cp − cv 本质上是**状态方程的二阶信息**——只要知道 p-v-T 关系和其中任一个比热容，另一个即可算出（实际中先测 cp，再算 cv）。

**验证——理想气体（迈耶公式）**：
- $pv = R_g T$ → $(\partial v/\partial T)_p = R_g/p$，$(\partial p/\partial T)_v = R_g/v$
- 代入 (7-36)：$c_p - c_v = T \cdot \frac{R_g}{v} \cdot \frac{R_g}{p} = T \cdot \frac{R_g^2}{R_g T} = R_g$
- ==$c_p - c_v = R_g$——迈耶公式得证==

### 5.3 比热容比 γ = cp / cv

**公式与推导**：

- （1）$\gamma = \frac{c_p}{c_v} = \frac{(\partial s/\partial T)_p}{(\partial s/\partial T)_v}$ ← 定义代入
- （2）用循环式：$\left(\frac{\partial s}{\partial T}\right)_p = -\left(\frac{\partial p}{\partial T}\right)_s \left(\frac{\partial s}{\partial p}\right)_T$，代入得：

$$\boxed{\gamma = \frac{c_p}{c_v} = -\left(\frac{\partial p}{\partial v}\right)_s \left(\frac{\partial v}{\partial p}\right)_T = \frac{\kappa_T}{\kappa_s}}\quad\text{—— (7-38)}$$

- （3）由此：$\kappa_s = \kappa_T / \gamma$ → 定熵压缩系数可由定温压缩系数和比热容比求出

**声速联系**（周磊课件补充）：

声速 $a = \sqrt{(\partial p/\partial \rho)_s} = \sqrt{-v^2(\partial p/\partial v)_s} = \sqrt{\frac{1}{\rho \kappa_s}}$

代入比热容比：$$\boxed{\gamma = -\frac{v^2}{a^2}\left(\frac{\partial p}{\partial v}\right)_T}\quad\text{——通过测声速 + 状态方程可得 γ}$$

### 5.4 压力对比热容 cp 的影响

**问题**：已知低压（理想气体）下的 cp0(T)，如何在任意压力 p 下求 cp(T, p)？

**推导**：

- （1）由第二 ds 方程 $ds = \frac{c_p}{T} dT - \left(\frac{\partial v}{\partial T}\right)_p dp$，应用全微分条件（交叉偏导相等）：
  $$\left[\frac{\partial(c_p/T)}{\partial p}\right]_T = -\left[\frac{\partial}{\partial T}\left(\frac{\partial v}{\partial T}\right)_p\right]_p = -\left(\frac{\partial^2 v}{\partial T^2}\right)_p$$
- （2）整理：

$$\boxed{\left(\frac{\partial c_p}{\partial p}\right)_T = -T\left(\frac{\partial^2 v}{\partial T^2}\right)_p}\quad\text{—— (7-40)}$$

- （3）积分——从零压到任意压力 p（定温）：

$$\boxed{c_p(T, p) = c_{p0}(T) - T\int_0^p \left(\frac{\partial^2 v}{\partial T^2}\right)_p dp}\quad\text{—— (7-41)}$$

→ 利用状态方程 v(T, p) 的**二阶偏导数**可计算出定温下 cp 随压力的变化。

> [!note] 核心理解
> 物理上：压力对 cp 的影响源于分子间作用力——高压下分子间距小、相互作用强，改变温度需要的能量与理想气体不同。数学上：需要状态方程的二阶导数 (∂²v/∂T²)p——这要求状态方程足够精确，否则二阶导误差放大。

### 5.5 焦耳-汤姆逊系数 μJ

**定义**：绝热节流的微分温度效应——==在焓不变条件下，压力变化引起的温度变化率==。

$$\boxed{\mu_J = \left(\frac{\partial T}{\partial p}\right)_h}\quad\text{—— (7-42)}$$

**物理分类**：
- μJ > 0：降压 → 降温（**正效应**，大多数气体常温下）
- μJ < 0：降压 → 升温（**负效应**，如氢气常温下）
- μJ = 0：降压 → 温度不变（**零效应**，如理想气体）

**公式推导**：

- （1）由第二 dh 方程：$dh = c_p dT + \left[v - T\left(\frac{\partial v}{\partial T}\right)_p\right] dp$ ← (7-32)
- （2）绝热节流 → dh = 0（焓不变），代入：
  $$0 = c_p dT + \left[v - T\left(\frac{\partial v}{\partial T}\right)_p\right] dp$$
- （3）解出 dT/dp：

$$\boxed{\mu_J = \left(\frac{\partial T}{\partial p}\right)_h = \frac{1}{c_p}\left[T\left(\frac{\partial v}{\partial T}\right)_p - v\right]}\quad\text{—— (7-43)}$$

- （4）用热膨胀系数 $\alpha_p = \frac{1}{v}(\partial v/\partial T)_p$ 表达：

$$\boxed{\mu_J = \frac{v}{c_p}(T\alpha_p - 1)}$$

**验证——理想气体**：
- $\alpha_p = 1/T$ → $T\alpha_p - 1 = 0$ → ==μJ = 0，理想气体绝热节流后温度不变== ← 符合焦耳定律

**转换温度**：μJ = 0 的条件是 $T\alpha_p = 1$，即 $T(\partial v/\partial T)_p = v$。满足该条件的温度称为**转换温度**——低于此温度 μJ > 0（正效应），高于此温度 μJ < 0（负效应）。

> [!note] 核心理解
> 物理上：焦-汤效应是实际气体偏离理想气体的直接标志。μJ 的正负取决于膨胀时**分子间势能的变化 vs 压力做功**的竞争——降温（正效应）意味着分子间引力做负功消耗动能；升温（负效应）意味着斥力主导。数学上：μJ 完全由 cp 和状态方程一阶导 (∂v/∂T)p 决定——==只需知道状态方程和 cp，就能判断节流后是升温还是降温==。这是热力学一般关系式在实际工程中（如液化气体、制冷循环）的重要应用。

**例 7-5 相关**——对满足 $p(v-b) = R_g T$（b 为正常数）的气体：
- （1）$v = R_g T/p + b$ → $(\partial v/\partial T)_p = R_g/p = (v-b)/T$
- （2）代入 (7-43)：$\mu_J = \frac{1}{c_p}[T \cdot \frac{v-b}{T} - v] = -\frac{b}{c_p} < 0$
- （3）→ 此类气体绝热节流后**温度升高**（负效应）

## 速记卡 / 一句话总结

- ==全微分条件 ∂M/∂y = ∂N/∂x 是判断 dz=Mdx+Ndy 是否为状态参数全微分的充要条件==
- ==循环式乘积 = −1，链式乘积 = +1==（最易混，用心记）
- 四个热系数 αp, κT, κs, γv 全部可由 p-v-T 测量得到，且通过循环式相互关联：**αp = γv·κT·p**
- ==四个吉布斯方程只需记住 du = Tds − pdv，其余通过勒让德变换推出==：dh = Tds+vdp, df = −sdT−pdv, dg = −sdT+vdp
- ==Maxwell III 和 IV 最实用==：$(\partial s/\partial v)_T = (\partial p/\partial T)_v$ 和 $(\partial s/\partial p)_T = -(\partial v/\partial T)_p$——把不可测的熵偏导转化为可测的 p-v-T 偏导
- 三个 ds 方程给出熵变的计算途径：第一 ds（T,v）最简，第二 ds（T,p）最实用
- ==du = cv dT + [T(∂p/∂T)v − p] dv==（第一 du 方程，最常用）；对理想气体，修正项为零 → u = u(T)
- ==cp − cv = T(∂p/∂T)v(∂v/∂T)p = Tvαp²/κT > 0==恒成立；理想气体退化到迈耶公式 cp − cv = Rg
- 压力对 cp 的影响需状态方程**二阶导数**：$(\partial c_p/\partial p)_T = -T(\partial^2 v/\partial T^2)_p$
- ==焦-汤系数 μJ = (v/cp)(Tαp − 1)==，正/负/零效应由 Tαp 与 1 的大小关系决定；理想气体 μJ = 0
