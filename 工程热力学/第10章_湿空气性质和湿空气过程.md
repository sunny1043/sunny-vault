## 10-1 湿空气和干空气

一·基本概念
    1.湿空气：含有水蒸气的空气
        - 大气中的空气都或多或少含水蒸气 → 日常遇到的都是湿空气
        - 水蒸气含量不大时可近似按干空气处理；但干燥、空调、蒸发冷却等必须按湿空气考虑
    2.干空气：完全不含水蒸气的空气
        - 干空气 = N₂ + O₂ + Ar + CO₂ + 微量其他气体，成分相对稳定
        - 标准化干空气：O₂（摩尔成分 **0.2095**，M=**32.000**）、N₂（**0.7809**，M=**28.016**）、Ar（**0.0093**，M=**39.944**）、CO₂（**0.0003**，M=**44.010**）
    3.湿空气 = 干空气 + 水蒸气，按理想混合气体处理
        - 理由：水蒸气的分压力通常很低 → 比体积大 → 可按理想气体计算
        - 整个湿空气亦按理想气体处理

二·道尔顿分压定律
    1.湿空气总压力 = 水蒸气分压力 + 干空气分压力
        $$p = p_v + p_{DA}$$
        - $p_v$：水蒸气分压力
        - $p_{DA}$：干空气分压力（下标 DA = Dry Air）
        - 没有特意压缩或抽空时，湿空气压力一般就是当时当地大气压力
    2.按理想气体处理 → 各组分均可分别用 $pV = mRT$ 描述

三·未饱和空气与饱和空气
    1.未饱和空气
        - 条件：$p_v < p_{sv}(t)$，即水蒸气分压力 < 该温度对应的饱和压力
        - 水蒸气处于过热状态（如图中状态 a）
        - 特性：==具有吸湿能力==——还能容纳更多水蒸气
    2.饱和空气
        - 条件：$p_v = p_{sv}(t)$（如图中状态 b）
        - 特性：==不再具有吸湿能力==，再加入水蒸气就会凝结出水珠

> [!note] 核心理解
> 物理上：湿空气的「饱和度」取决于水蒸气分压力与该温度下饱和压力的差距——差距越大，空气越"干"，吸水能力越强。这和海绵的比喻类似：海绵（空气）里水越少（分压低），还能吸的水越多。
> 数学上：核心关系是 $p_v \lesseqgtr p_{sv}(t)$。$p_{sv}(t)$ 由温度唯一确定（查饱和水蒸气表），因此判断未饱和/饱和只需比较 $p_v$ 和 $p_{sv}(t)$。

**易错/边界**：
- 干空气标准化时**忽略了微量其他气体**，对工程计算影响可忽略
- 当 $t > t_{sat}(p)$（即温度超过湿空气总压对应的饱和温度）时，$p_{sv}(t) > p$，此时 $p_v$ 最大只能达到 $p$（干空气分压已为零），相对湿度定义需修正（见 10-2 节）


## 10-2 绝对湿度和相对湿度

### 绝对湿度

一·定义
    1.绝对湿度 $\rho_v$：单位体积（**1 m³**）湿空气中所含水蒸气的质量
        $$\rho_v = \frac{m_v}{V} \quad \text{kg/m³}$$
        - 本质就是湿空气中水蒸气的密度
    2.饱和空气的绝对湿度 $\rho_{sv}$：
        $$\rho_{sv} = \rho_{v,max} = \frac{1}{v''} \quad \text{kg/m³}$$
        - $v''$：该温度下干饱和水蒸气的比体积

二·性质与缺陷
    1.绝对湿度 ==不能完全说明湿空气的干燥程度和吸湿能力==
        - 例：同样 $\rho_v = 0.009\ \text{kg/m³}$：
        - 温度 **20°C** 时 $\rho_{sv}$ ≈ **0.0173 kg/m³** → $p_v < p_{sv}$，还比较干燥，有吸湿能力（如冬季室内开暖气感到干燥）
        - 温度 **10°C** 时 $\rho_{sv}$ ≈ **0.0094 kg/m³** → 接近饱和，感到阴冷潮湿
        - 温度再低 → 水珠凝结
    2.原因：绝对湿度只看水蒸气"含量"，没看该温度下"容量"有多大

### 相对湿度

一·定义
    1.相对湿度 $\varphi$：绝对湿度与同温度下饱和空气绝对湿度的比值
        $$\varphi = \frac{\rho_v}{\rho_{v,max}} = \frac{\rho_v}{\rho_{sv}}$$
        - 也叫**饱和度**，表示湿空气离开饱和状态的远近程度
        - $\varphi = 0$：完全干空气；$\varphi = 1$（100%）：饱和空气

二·公式及推导——用分压比表示相对湿度
    1.由理想气体状态方程：
        $$\rho_v = \frac{p_v}{R_{g,v}T}, \quad \rho_{sv} = \frac{p_{sv}}{R_{g,v}T}$$
        - ← 理想气体状态方程 $pV = mR_gT$，密度 $\rho = m/V = p/(R_gT)$
        - 同温度 T 下分母相同
    2.代入定义式：
        $$\varphi = \frac{p_v/(R_{g,v}T)}{p_{sv}/(R_{g,v}T)} = \frac{p_v}{p_{sv}}$$
        - → $\varphi = \dfrac{p_v}{p_{sv}}$：（**当** $p_{sv}(t) \le p$ **时成立**）
    3.该式表明：==相对湿度 = 水蒸气实际分压力 / 同温下可能达到的最大分压力==

三·边界情况——当 $p_{sv}(t) > p$ 时（高温/低压）
    1.条件：温度超过湿空气总压所对应的饱和温度 → $p_{sv}(t) > p$
    2.此时水蒸气最大分压只能达到湿空气总压 $p$（干空气分压已为零）
        $$\varphi = \frac{p_v}{p_{v,max}} = \frac{p_v}{p}$$
        - ← 因为在 $t > t_{sat}(p)$ 时，$p_v$ 的上限不再是 $p_{sv}(t)$ 而是湿空气总压 $p$
    3.两种情况的统一：
        - $p_{sv}(t) \le p$ 时：$\varphi = p_v / p_{sv}$
        - $p_{sv}(t) > p$ 时：$\varphi = p_v / p$

![[图_相对湿度两种定义域.excalidraw]] <!-- 📌 待补图 -->
> **图注**：以温度为横轴，标出 $p_{sv}(t)$ 曲线和 $p$（常数水平线）。交点对应 $t = t_{sat}(p)$。左侧 $T < T_{sat}$ 用 $\varphi = p_v/p_{sv}$；右侧 $T > T_{sat}$ 用 $\varphi = p_v/p$。

> [!note] 核心理解
> 物理上：相对湿度回答的是"这团空气离'喝饱'还有多远"。$\varphi = 60\%$ 意思是水蒸气分压已达到该温度下饱和值的 60%，还剩 40% 的吸水能力。
> 数学上：$\varphi = p_v / \max(p_{sv}(t), p_v^{\text{可达到}})$。关键在于分母——在低温段分母是 $p_{sv}(t)$，高温段分母受总压 $p$ 限制切换为 $p$。这个"分段函数"的本质是水蒸气的分压不能超过湿空气总压。

**易错/边界**：
- $\rho_v$ 和 $p_v$ ==都不易直接测量==，工程上通过露点计或干湿球温度计间接测定（见 10-3 节）
- 相对湿度减小 ≠ 空气含水分减少——温度升高时相对湿度下降，但含湿量可能不变（见课堂测验判断题）


## 10-3 露点温度和湿球温度

一·为什么要测这两个温度
    1.式 $\varphi = p_v / p_{sv}$ 中 $p_{sv}$ 可由温度查饱和水蒸气表得到，但 $p_v$（或 $\rho_v$）==不易直接测量==
    2.→ 通过**露点计**或**干湿球温度计**间接测定

### 露点温度 $t_d$

一·定义
    1.露点温度 $t_d$：在总压 $p$ 和水蒸气分压 $p_v$ 都不变的条件下，降温至湿空气达到饱和时的温度
        - 即：定压降温（$p$ = const，$p_v$ = const，$p_{DA}$ 随之调整），到 $\varphi = 1$ 的那一刻

二·原理与公式
    1.含湿量不变（降温不增减水蒸气）→ 分压 $p_v$ 不变
    2.温度下降 → $p_{sv}(t)$ 下降 → 当 $p_{sv}(t_d) = p_v$ 时达到饱和
    3.∴ 露点温度满足：
        $$p_v = p_{sv}(t_d)$$
        - → 测出 $t_d$，查饱和水蒸气表得 $p_{sv}(t_d)$ = 原湿空气中的 $p_v$
    4.相对湿度据此计算：
        $$\varphi = \frac{p_{sv}(t_d)}{p_{sv}(t)}$$
        - ← 分子：露点温度对应的饱和压力 = 实际 $p_v$；分母：干球温度对应的饱和压力

三·饱和空气的露点
        $$t_d = t, \quad \varphi = 1$$
        - 饱和空气一降温就结露，露点等于干球温度

四·测量方法——露点计
    1.金属容器内装易挥发液体（乙醚），手捏橡皮球送气使乙醚挥发吸热 → 容器降温
    2.镀铬表面开始失去光泽（出现微小露珠）时 → 温度计读数 = $t_d$
    3.缺陷：==不容易准确判定何时开始凝露，误差较大==
    4.在 T-s 图上：过程 a→c，定压降温至饱和线

> **图注**：T-s 图上定压线 a→c。a 点在过热区（未饱和），沿 p=const 降温，在 c 点与饱和蒸汽线相交。c 点温度 = 露点温度 $t_d$。注意 $p_v$ 不变，所以 $p_{sv}(t_d) = p_v$。

### 湿球温度 $t_w$

一·定义
    1.湿球温度 $t_w$：用湿纱布包裹温度计温包，水分蒸发吸热使温包降温，当外界传入热量 = 蒸发所需热量时，维持不变的温度
        - 本质：==蒸发冷却的动态平衡温度==

二·测量方法——干湿球温度计
    1.两支普通温度计：一支干球（直接接触空气），一支湿球（温包用湿纱布包裹）
    2.若空气未饱和 → 湿纱布表面水分蒸发吸热 → 湿球温度下降
    3.当 $Q_{传入} = Q_{蒸发}$ 时 → 温度稳定 → 读数为 $t_w$
    4.相对湿度越小 → 蒸发越快 → $t_w$ 比 $t$（干球温度）低得越多

三·三个温度的大小关系

| 条件 | 关系 |
|------|------|
| 未饱和空气 ($\varphi < 1$) | ==$t_d < t_w < t$== |
| 饱和空气 ($\varphi = 1$) | ==$t_d = t_w = t$== |

    - T-s 图上：$t$ 在最右（熵最大），$t_d$ 在最左（沿定压线降温至饱和），$t_w$ 在中间

四·注意事项
    1.湿球温度读数与掠过湿球的风速有关
        - 风速 > **2 m/s** 时读数变化很小 → 应以通风式干湿球温度计读数为准
        - 风速为零时读数偏高
    2.湿球温度**并非严格意义上的状态参数**（见 10-6 节），而是一个经验数值

![[图_干湿球温度计_Ts.excalidraw]] <!-- 📌 待补图 -->
> **图注**：T-s 图上标注三个温度位置。未饱和空气（$t$ 点）沿等焓线向右下方交饱和线得 $t_w$，沿定压线向左交饱和线得 $t_d$。明显 $t_d < t_w < t$。

> [!note] 核心理解
> 物理上：露点是"被动结露"——你不给空气加水，光降温它就自己吐水了，吐水那一刻的温度就是露点。湿球温度是"主动蒸发"——纱布上的水拼命蒸发吸热，把自己和环境都冷却下来，直到蒸发所需热量刚好由空气对流补上。
> 数学上：露点 = $p_v$ 对应的饱和温度（$p_v = p_{sv}(t_d)$），是分压的单值函数。湿球温度涉及传热传质耦合，不是纯热力学状态参数。

**易错/边界**：
- $p_v$ 不变 ≠ 含湿量不变（含湿量是相对于干空气质量，干空气分压变化时含湿量也变）——但在定总压降温过程中 $p_v$ 不变 ⇔ 含湿量不变
- 湿球温度受风速影响，课本和图表以通风式（风速 > 2 m/s）为准


## 10-4 含湿量、焓和焓湿图

一·为什么引入含湿量
    1.空调和干燥过程中：加湿/去湿 → 水蒸气质量变化；但干空气质量不变
    2.→ 以 **1 kg 干空气**为计算基准最方便

### 含湿量 $d$

一·定义
    1.含湿量 $d$：每 **kg 干空气**所夹带的水蒸气质量（以克计）
        $$d = 1000 \times \frac{m_v}{m_{DA}} = 1000 \times \frac{\rho_v}{\rho_{DA}} \quad \text{g/kg(DA)}$$
        - 本质：水蒸气密度与干空气密度之比 × 1000
        - 单位：g/kg(DA)，DA 表示干空气

二·公式及推导——$d$ 与 $\varphi$ 的关系
    1.由理想气体状态方程：
        $$\rho_v = \frac{p_v}{R_{g,v}T}, \quad \rho_{DA} = \frac{p_{DA}}{R_{g,DA}T}$$
        - ← $p = \rho R_g T$ 的密度形式
    2.比值：
        $$\frac{\rho_v}{\rho_{DA}} = \frac{p_v/R_{g,v}}{p_{DA}/R_{g,DA}} = \frac{p_v M_v}{p_{DA} M_{DA}}$$
        - ← $R_g = R/M$，$R$ 为通用气体常数，$M$ 为摩尔质量
        - $M_v$ = **18.016 g/mol**，$M_{DA}$ = **28.965 g/mol**
    3.代入 $d$ 定义：
        $$d = 1000 \times \frac{p_v \times 18.016}{p_{DA} \times 28.965} = 621.99 \times \frac{p_v}{p_{DA}}$$
    4.用 $p_{DA} = p - p_v$ 及 $p_v = \varphi p_{sv}$ 代入：
        $$d = 621.99 \times \frac{\varphi p_{sv}}{p - \varphi p_{sv}} \quad \text{g/kg(DA)}$$
    5.反解 $p_v$：
        $$p_v = \frac{p \cdot d}{621.99 + d}$$
        - → 当 $p$ = const 时，==$p_v$ 与 $d$ 有单值对应关系==

三·常用简化式（课件经验公式）：
    $$d = 0.622 \times \frac{p_v}{p - p_v} \quad \text{kg/kg(A)}$$
    - 这里 $d$ 单位是 kg/kg(干空气)，系数 **0.622** = **18.016/28.965**

![[图_含湿量与分压关系.excalidraw]] <!-- 📌 待补图 -->
> **图注**：给定总压 $p$ 下 $d$ 与 $p_v$ 的关系曲线。单调递增，$p_v \to p$ 时 $d \to \infty$（全为水蒸气，无干空气）。

> [!note] 核心理解
> 物理上：含湿量相当于"混合物中水蒸气和干空气的配比"，单位是 g 水/kg 干空气。选择干空气做分母因为干空气质量在加湿/去湿过程不变，比用湿空气总质量方便得多。
> 数学上：$d = 622 \cdot p_v / (p - p_v)$——分母 $(p - p_v)$ 是干空气的分压，$p_v$ 增加时分子涨、分母缩，所以 $d$ 对 $p_v$ 非常敏感（非线性增长）。

### 湿空气的焓 $H$

一·定义
    1.湿空气的焓 $H$：以 **1 kg 干空气**为基准
        $$H = h_{DA} + 0.001d \cdot h_v \quad \text{kJ/kg(DA)}$$
        - $h_{DA}$：**1 kg** 干空气的焓
        - $0.001d$：换算为 kg 水蒸气 / kg 干空气（因为 $d$ 单位是 g/kg）
        - $h_v$：**1 kg** 水蒸气的焓

二·公式推导——工程计算式
    1.取 0°C 为基准：
        - $h_{DA} = c_{p,DA} \cdot t$
        - $h_v = r_0 + c_{p,v} \cdot t$，$r_0$ = **2501 kJ/kg**（0°C 水的汽化潜热）
    2.比热容与温度的关系（线性近似）：
        $$\{c_{p,DA}\}_{\text{kJ/(kg·K)}} = 1.002 + 0.00010\{t\}_{°C}$$
        $$\{c_{p,v}\}_{\text{kJ/(kg·K)}} = 1.850 + 0.00042\{t\}_{°C}$$
    3.代入得完整式：
        $$\{H\}_{\text{kJ/kg(DA)}} = 1.002\{t\}_{°C} + 0.00005\{t\}_{°C}^2 + 0.001d\left(2501 + 1.850\{t\}_{°C} + 0.00021\{t\}_{°C}^2\right)$$
    4.==常用简化经验公式==（课件）：
        $$h = 1.005t + d(2501 + 1.86t) \quad \text{kJ/kg(A)}$$
        - （$d$ 此时为 kg/kg(A) 单位）
        - 示例：$t$ = **30°C**，$d$ = **0.0162 kg/kg(A)** → $h$ = **71.68 kJ/kg(A)**

### 焓湿图（$H$-$d$ 图）

一·基本结构
    1.纵坐标：焓 $H$（kJ/kg(DA)），横坐标：含湿量 $d$（g/kg(DA)）
    2.两坐标轴夹角取 **135°**（而非 90°），使图面曲线展开、清晰可读
    3.图中绘制的定值线族：
        - **等焓线**：$H$ = const，倾斜直线（因夹角 135° 而呈斜向）
        - **等含湿量线**：$d$ = const，垂直竖线
        - **等温线**：$t$ = const，接近直线但略有弯曲（$H = 1.005t + d(2501+1.86t)$，对固定 $t$，$H$ 与 $d$ 线性关系）
        - **等相对湿度线**：$\varphi$ = const，从原点发散的一组曲线
            - ==$\varphi = 1$ 的线称为饱和空气线==，其上每点代表该温度下的饱和空气
            - 饱和空气线上方（$\varphi < 1$）：未饱和空气
        - **水蒸气分压线**：$p_v = f(d)$，标于图的上方

二·h-d 图查法——已知 $t$ 和 $\varphi$
    1.找 $t$ = const 线与 $\varphi$ = const 线的交点 → 状态点
    2.从该点：
        - 沿垂直线（$d$ = const）向下：与饱和线交点 → $t_d$（露点），与 $p_v$ 线交点 → $p_v$
        - 沿等焓线向右下方：与饱和线交点 → $t_w$（湿球温度）
        - 向横轴读数 → $d$，向纵轴读数 → $H$

三·示例（课本）：$p$ = **0.1 MPa**，$t$ = **30°C**，$\varphi$ = **80%**
    - 查图得：$d$ = **22 g/kg(DA)**，$H$ = **86 kJ/kg(DA)**
    - $t_d$ = **26°C**，$t_w$ = **27°C**，$p_v$ = **3.4 kPa**

![[图_焓湿图示意.excalidraw]] <!-- 📌 待补图 -->
> **图注**：H-d 图的坐标系（135° 夹角）。标注等温线（斜向右上方）、等相对湿度线（发散曲线，$\varphi = 1$ 为饱和线）、等焓线（斜线）、等含湿量线（竖线）。在图上标注状态点 a，从 a 点沿垂线到饱和线得露点 c，沿等焓线到饱和线得湿球温度点。

> [!note] 核心理解
> 物理上：h-d 图是把 4 个参数（$H, d, t, \varphi$）投影到二维平面上，给定任意两个就能读出其余全部。$\varphi = 100\%$ 线像"天花板"——再往上没意义（过饱和会结露）。定含湿量垂线穿到天花板的交点给出露点，定焓线穿到天花板的交点给出湿球温度。
> 数学上：夹角 135° 使 $H$ 和 $d$ 不完全正交，这样等焓线和等温线在图面上角度拉开，避免各路曲线挤成一团无法读数。

**易错/边界**：
- 常见错误：把 $d = 0.622 \cdot p_v/(p-p_v)$ 中的 **0.622** 和精确系数 **621.99** 混用，前者 $d$ 是 kg/kg 单位，后者是 g/kg 单位
- 课件和课本的焓公式略有不同（课本含 $t^2$ 项修正），精度要求不高时用经验公式 $h = 1.005t + d(2501+1.86t)$ 即可
- h-d 图是针对**特定压力**绘制的（如附图 6 对应 $p$ = **0.1 MPa**），不同压力要用不同图或通用焓湿图（见 10-5）
- 等湿球温度线 ≈ 等焓线（温度不高时误差很小），所以有时直接用等焓线代替


## 10-5 比相对湿度和通用焓湿图

一·问题引出
    1.常规 h-d 图针对**指定压力**（如 **0.1 MPa**）绘制 → 换一个压力就得换图
    2.能否一张图适用于不同压力？→ ==通用焓湿图==

二·理论基础
    1.湿空气可作理想气体处理 → 温度、含湿量相同时，焓和饱和蒸汽压相同
    2.但相对湿度 $\varphi = p_v / p_{sv}$ 中 $p_v$ 与总压 $p$ 有关：
        $$p_v = \frac{p \cdot d}{621.99 + d}$$
        - ← $d$ 相同但 $p$ 不同时，$p_v$ 也不同
    3.引入**比分压力**（水蒸气分压与总压之比）：
        $$\frac{p_v}{p} = \frac{d}{621.99 + d}$$
        - → ==含湿量 $d$ 相同时，比分压力 $p_v/p$ 相同，与总压无关==

三·比相对湿度 $\psi$
    1.定义：
        $$\psi = \frac{\varphi}{p} = \frac{p_v/p}{p_{sv}} = \frac{p_v}{p \cdot p_{sv}}$$
        - 单位：(**0.1 MPa**)⁻¹（取 0.1 MPa 为单位压力）
    2.含义：单位压力下的相对湿度
    3.关键性质：==具有相同温度和相同含湿量但总压不同的湿空气，其比相对湿度 $\psi$ 相同==
    4.对 $p$ = **0.1 MPa**：$\psi$ 与 $\varphi$ 数值相同（单位不同）

四·通用焓湿图（附图 7）
    1.以 $t$, $d$, $p_v/p$, $\psi$ 为基本参数 → 不再依赖具体压力
    2.使用转换：
        - 给定实际压力 $p$，$\varphi = \psi \times p$
        - $\psi = 1$(0.1MPa)⁻¹ 的线对应 $p$ = 0.1 MPa 时的饱和线
        - 对 $p$ = **0.2 MPa**：饱和线变为 $\psi = 0.5$(0.1MPa)⁻¹ 的线（$\varphi = \psi p = 1$）
        - 对 $p$ = **0.05 MPa**：饱和线变为 $\psi = 2$(0.1MPa)⁻¹ 的线
    3.定律：$\varphi_{\text{任意}} = \psi \times p$，一张通用图 = 无数张普通图

五·定湿球温度线处理
    1.定湿球温度线在通用图中是**平行直线**（有确定斜率），但位置随压力而变
        - 附图 7 中画出的位置是 **0.1 MPa** 下的实际位置
    2.温度不高时，定湿球温度线 ≈ 等焓线，可直接用等焓线近似

六·高温修正（$t > t_{sat}(p)$ 时）
    1.当干球温度所对应的饱和蒸汽压超过总压时，按 $\varphi = p_v/p_{sv}$ 算出的值需修正
        $$\varphi_{\text{正确}} = \varphi_{\text{计算}} \times \frac{p}{p_{sv}}$$
        - ← 因为此时最大 $p_v$ = $p$（不是 $p_{sv}$）
    2.在通用图中，当 $t > t_{sat}(p)$ 时，比分压力 $p_v/p$ 就是相对湿度本身

![[图_通用焓湿图三条饱和线.excalidraw]] <!-- 📌 待补图 -->
> **图注**：通用焓湿图中标注三条不同压力下的饱和线。$p$ = **0.05 MPa** 时饱和线为 $\psi = 2$；$p$ = **0.1 MPa** 时饱和线为 $\psi = 1$；$p$ = **0.2 MPa** 时饱和线为 $\psi = 0.5$。三条等湿球温度线（30°C）平行移位。

> [!note] 核心理解
> 物理上：比相对湿度的思路是把"总压效应"从相对湿度里剥离出来——因为 $\varphi$ = (水蒸气分压)/(饱和压力)，而水蒸气分压随总压正比变化（含湿量不变时），所以 $\varphi/p$ 是个不变量。
> 数学上：$\psi = \varphi/p$ 是"归一化"操作。$p_v/p$ 只取决于 $d$（比分压力 = $d/(622+d)$），与 $p$ 无关；而 $p_{sv}$ 只取决于 $t$，也与 $p$ 无关。所以 $\psi$ 仅由 $t$ 和 $d$ 决定，一张图覆盖所有压力。

**易错/边界**：
- 高温条件下（$t > t_{sat}(p)$）需要对从通用图读出的 $\varphi$ 进行修正
- 定湿球温度线严格来说不是等焓线，温度较高时偏差不可忽略，但课本指出低温时近似足够


## 10-6 绝热饱和温度

一·湿球温度的问题
    1.湿球温度 $t_w$ 是水-空气系统的经验数值，==并非严格的状态参数==
    2.真正与湿空气有关的状态参数是**绝热饱和温度** $t_{as}$

二·绝热饱和器原理
    1.装置：绝热良好的容器，内置多层结构保证水与空气充分接触
    2.流程：
        ```
        未饱和空气 →（自下而上）→ 与喷淋水充分接触 → 饱和空气流出
        补充水（温度 = t_as）← 循环泵 ← 容器底部
        ```
    3.空气将热量传给水使水蒸发 → 空气被加湿冷却 → 出口为饱和空气
    4.出口测得的温度 = 绝热饱和温度 $t_{as}$

三·能量平衡方程
    1.每流过 **1 kg 干空气**，忽略泵功：
        $$H_1 + \left(\frac{d_{as} - d_1}{1000}\right)h_w = H_{as}$$
        - $H_1$：进口湿空气焓
        - $H_{as}$：出口饱和空气焓
        - $h_w$：补充水的焓（补充水温度 = $t_{as}$）
    2.由于 $\dfrac{d_{as} - d_1}{1000} \cdot h_w$ 通常很小（特别是温度较低时），可忽略
        → ==$H_1 \approx H_{as}$==，即绝热饱和过程近似等焓过程

四·与湿球温度的关系
    1.==$t_{as} \approx t_w$==，经验表明二者非常接近
    2.因 $t_w$ 易于测量，工程上常用 $t_w$ 代替 $t_{as}$
    3.→ 在 h-d 图中用"沿等焓线到饱和线"的方法确定 $t_w$ 的理论依据即源于此

五·确定性
    1.对给定的湿空气进口状态，$t_{as}$ 有完全确定的值 → $t_{as}$ 是**状态参数**
    2.$t_w$ 则不是——它依赖于水-空气系统的传热传质特性

![[图_绝热饱和器.excalidraw]] <!-- 📌 待补图 -->
> **图注**：绝热饱和器示意图。绝热壁内多层结构，空气自下而上，水自上而下喷淋。标注进口参数（$t_1, d_1, H_1$）和出口参数（$t_{as}, d_{as}, H_{as}$）。补充水温度 = $t_{as}$。

> [!note] 核心理解
> 物理上：绝热饱和过程是让空气在绝热容器里"尽情喝水喝到饱"，出口温度就是 $t_{as}$。由于没有外界热交换，空气"降自己的温、蒸发别人的水"——自身显热减少 = 水蒸发潜热增加，总焓近似不变。这就像湿球温度计的放大版：湿球纱布是一小片，绝热饱和器是一个大容器。
> 数学上：$t_{as}$ 由进口空气状态唯一确定 → 是状态函数；$t_w$ 不仅取决于空气状态还取决于对流传热传质特性 → 不是纯热力学状态参数。但因为数值接近，工程上混用。

**易错/边界**：
- $t_{as} = t_w$ 只是近似，高温/高湿时偏差增大
- 绝热饱和器中的泵功可忽略，这是 $H_1 \approx H_{as}$ 成立的前提之一


## 10-7 湿空气过程——焓湿图的应用

> **本节所有过程均假设不做技术功，压力基本不变。**

一·加热（或冷却）过程
    1.特点：含湿量不变（不加湿不去湿）
        $$d = \text{const}, \quad \Delta d = d_2 - d_1 = 0$$
    2.h-d 图上路径：==沿定含湿量线垂直向上（加热）或向下（冷却）==
    3.状态变化（加热）：
        - 温度升高：$\Delta t = t_2 - t_1 > 0$
        - 焓增加：$\Delta H = H_2 - H_1 > 0$
        - 相对湿度下降：$\Delta \varphi = \varphi_2 - \varphi_1 < 0$
    4.加热量：
        $$Q = \Delta H = H_2 - H_1 \quad \text{kJ/kg(DA)}$$
    5.应用：空调加热、烘干前预热

    ```
    H ↑
      |        2(H₂,t₂,φ₂<φ₁)
      |       /
      |      /  ← 定d线（垂直线）
      |     /
      |    1(H₁,t₁,φ₁)
      +-------------------→ d
    ```
    > 加热过程 h-d 图：1→2 沿 $d$ = const 向上

二·绝热加湿过程
    1.条件：==压力基本不变 + 与外界基本绝热==
    2.机理：空气将热量传给水 → 水蒸发 → 水蒸气进入空气
        - 忽略被蒸发液态水的焓值 → ==$H = \text{const}$==
    3.h-d 图上路径：==沿等焓线向右下方（降温增湿方向）==
    4.状态变化：
        - $\Delta H = H_3 - H_2 = 0$
        - $\Delta t = t_3 - t_2 < 0$（温度降低）
        - $\Delta \varphi = \varphi_3 - \varphi_2 > 0$（相对湿度增加）
        - $\Delta d = d_3 - d_2 > 0$（含湿量增加）
    5.水分增加量：$\Delta m_w = \Delta d \times m_{DA}$
    6.应用：烘干过程（物品干燥 = 空气加湿）、蒸发冷却

    ```
    H ↑
      |   2        .
      |    \        \  ← 等H线
      |     \        3(H₃=H₂,t₃<t₂,φ₃>φ₂)
      |      \
      +-------------------→ d
    ```
    > 绝热加湿过程 h-d 图：2→3 沿 $H$ = const 向右下

三·绝热混合过程
    1.条件：两股（多股）==压力相同==的气流绝热混合
    2.设：$\dot{m}_1$（流量1），$d_1, H_1$；$\dot{m}_2$（流量2），$d_2, H_2$；混合后 $\dot{m}_3, d_3, H_3$
    3.守恒方程：
        - 干空气质量守恒：$\dot{m}_1 + \dot{m}_2 = \dot{m}_3$
        - 水蒸气质量守恒：$\dot{m}_1 d_1 + \dot{m}_2 d_2 = \dot{m}_3 d_3$
        - 能量守恒：$\dot{m}_1 H_1 + \dot{m}_2 H_2 = \dot{m}_3 H_3$
    4.图解——杠杆规则：
        $$\frac{\dot{m}_1}{\dot{m}_2} = \frac{d_3 - d_2}{d_1 - d_3} = \frac{H_3 - H_2}{H_1 - H_3}$$
        - → ==混合后状态点 3 落在状态 1 和 2 的连线上==
        - → 线段比 = 流量反比：$\dfrac{\overline{32}}{\overline{31}} = \dfrac{\dot{m}_1}{\dot{m}_2}$
    5.应用：空调中混合新风与回风、干燥中混合热空气与环境空气

    ```
    H ↑
      |   1
      |    \
      |     \  3（混合点，在线段12上）
      |      \
      |       2
      +-------------------→ d
        ← d₁  d₃  d₂ →
    ```
    > 绝热混合过程 h-d 图：1 和 2 的连线，3 在线上按流量比例内分

四·绝热节流过程
    1.特点：节流前后 ==$H$ 不变、$d$ 不变、比相对湿度 $\psi$ 不变==
    2.变化项（随总压降低按比例减小）：
        - $\varphi \downarrow$（$\varphi = \psi \times p$）
        - $p_v \downarrow$（$p_v = (p_v/p) \times p$）
        - $t_w \downarrow$
    3.通用 h-d 图中：节流前后为==同一点==，但该点对不同压力代表不同的 $\varphi$、$p_v$、$t_w$
    4.示例（课本）：湿空气 $p_1$ = **0.2 MPa**，$t$ = **30°C**，$\varphi_1$ = **0.6**
        - $\psi = 0.6/0.2 = 0.3$ (0.1MPa)⁻¹
        - 节流至 $p_2$ = **0.1 MPa** → $\varphi_2 = 0.3 \times 0.1 = 0.3$
        - 节流至 $p_3$ = **0.05 MPa** → $\varphi_3 = 0.3 \times 0.05 = 0.15$
        - $H$、$d$、$\psi$ 三者不变

![[图_湿空气四大过程_hd图.excalidraw]] <!-- 📌 待补图 -->
> **图注**：h-d 图上一次画出四个过程的典型路径。(1)加热：1→2 沿 $d$=const 向上；(2)绝热加湿：2→3 沿 $H$=const 向右下；(3)绝热混合：状态1和状态2连线上取点3；(4)冷却去湿：先沿 $d$=const 向下到饱和线，再沿饱和线向左下（降温+凝水）。

五·典型工程案例——烘干过程（加热 + 绝热加湿组合）

    流程路径（h-d 图上）：
    ```
    冷空气(1)——加热器——热空气(2)——干燥箱（绝热加湿）——废气(3)
      定d ↑            定H ↘（降温增湿）
    ```

    1.阶段 1→2（加热器）：$d_1 = d_2$，$H_2 > H_1$
        - 加入热量：$Q = H_2 - H_1$
    2.阶段 2→3（干燥箱）：$H_3 = H_2$，$d_3 > d_2$
        - 吸收水分：$\Delta m_w = m_{DA}(d_3 - d_2)$
    3.若"先加热后加湿"（分段操作，非边加热边加湿）：
        - 加热到温度更高（h-d 图上沿 $d$=const 走得更远），再加湿时沿等焓线落到目标状态

六·典型工程案例——冷却去湿（空调）
    1.流程路径：空气先在 $d$ = const 下降温至露点 → 继续冷却时沿饱和线（$\varphi = 1$）降温+凝水 → 再加热到目标温度
    2.析出水分 = $d_{\text{初}} - d_{\text{末}}$（g/kg(DA)）
    3.课本例题 [例 10-4]：$t_1$ = **30°C**，$\varphi_1$ = **80%** → 冷却至 **10°C** → 加热至 **20°C**
        - 露点 $t_d$ = **26°C**
        - 析出水分 = **22 - 7.7** = **14.3 g/kg(DA)**
        - 最终 $\varphi_4$ = **52%**

> [!note] 核心理解
> 物理上：h-d 图上四个基本过程涵盖了空调和干燥的全部操作。加热 = 竖直向上走（多含热量少含相对湿度）；冷却 = 竖直向下走（到饱和线后沿饱和线凝水）；加湿 = 沿等焓线斜向右下（用自身热量蒸发外来水）；混合 = 两点连线取内分点。所有工程问题都是这四种基本过程的组合。
> 数学上：$d$ = const（竖直）、$H$ = const（斜线）、连线内分点（杠杆规则）构成 h-d 图上的三种"轨迹"，混合问题简化为线性插值 $\dot{m}_1/\dot{m}_2 = \overline{32}/\overline{31}$。

**易错/边界**：
- 冷却到露点以下才有去湿效果，只冷却不到露点只是降温
- 绝热加湿中"焓不变"的前提是忽略被蒸发液态水的焓值——水本身也带焓，但量太小
- 绝热混合要求两股气流压力相同，压力不同需先统一压力
- 有显著压力降的过程 = 原过程 + 附加绝热节流，按比相对湿度方法处理


## 速记卡 / 一句话总结

- ==湿空气 = 干空气 + 水蒸气==，两者均按理想气体处理，$p = p_v + p_{DA}$
- ==未饱和空气：$p_v < p_{sv}(t)$==（具有吸湿能力）；==饱和空气：$p_v = p_{sv}(t)$==（不吸湿）
- ==相对湿度 $\varphi = p_v / p_{sv}$==（$p_{sv} \le p$ 时）；当 $p_{sv} > p$ 时用 $\varphi = p_v/p$
- ==含湿量 $d = 621.99 \times \varphi p_{sv} / (p - \varphi p_{sv})$ g/kg(DA)==，工程简化：$d = 0.622 p_v/(p-p_v)$ kg/kg
- ==露点 $t_d$==：定压降温至饱和的温度，$p_v = p_{sv}(t_d)$——查表反推 $p_v$
- ==湿球温度 $t_w$==：蒸发冷却平衡温度，$\varphi < 1$ 时 $t_d < t_w < t$；$\varphi = 1$ 时三者相等
- ==焓 $H$==（kJ/kg(DA)）= $1.005t + d(2501 + 1.86t)$（$d$ 为 kg/kg），h-d 图以 135° 夹角展开
- ==比相对湿度 $\psi = \varphi/p$==，含湿量相同时 $\psi$ 不随总压变 → 一张通用图适用任意压力
