# 第3章 气体的热力性质和热力过程 ★★★

#engineering-thermodynamics #ideal-gas

## 3-1 理想气体与实际气体

**定义**: 理想气体是满足以下两个简化假定的气体模型:
1. 分子本身**不具有体积**（分子是几何点）
2. 分子之间**没有作用力**

**物理直觉**: 气体分子间距离很大时（低压、高温），分子自身的体积和相互间微弱的引力都可以忽略。就像空旷操场上的几个乒乓球——球的体积相对于操场可以忽略，球与球之间也几乎没有碰撞之外的相互作用。反之，高压、低温下分子挤在一起，就要考虑实际气体效应了。

工程中空气、燃气、烟气在压力不很高时一般按理想气体处理。

## 3-2 理想气体状态方程

$$\\boxed{pv = R_g T}$$

摩尔形式: $pV_m = RT$，其中 $R = 8.3145$ J/(mol·K) — 通用气体常数。

**推导思路**: 从分子运动论出发。气体压力是大量分子碰撞容器壁的宏观表现。
1. 分子运动论: $p = \\frac{2}{3}n \\cdot \\frac{m\\bar{c}^2}{2}$ (压力 ∝ 分子平均动能密度)
2. 温度定义: $\\frac{m\\bar{c}^2}{2} = \\frac{3}{2}kT$
3. 代入消去分子量: $p = nkT = \\frac{N}{V}kT$
4. 引入 $R_g = kN_A/m$: $pv = R_g T$

**气体常数**: $R_g = R/M$。例如空气 M≈0.028965 kg/mol → $R_{g,air} = 287.05$ J/(kg·K)

## 3-3 理想混合气体

**道尔顿分压定律**: 混合气体总压力等于各组分分压力之和。

$$p = \\sum p_i, \\quad p_i = x_i p$$

其中 $x_i = n_i/n_{mix}$ 为摩尔分数（对理想气体等于体积分数）。

**物理直觉**: 假设一个容器里有氧气和氮气。每种气体的分子都在独立地碰撞壁面。器壁感受到的总压力 = 氧分子碰撞贡献的压力 + 氮分子碰撞贡献的压力。

**平均摩尔质量**: $M_{mix} = \\sum x_i M_i$
**混合气体常数**: $R_{g,mix} = R/M_{mix}$

## 3-4 比热容

**定义**: 比热容是单位质量物质温度升高1K所需的热量。但为什么有两种比热容？

$$c_v = \\left(\\frac{\\partial u}{\\partial T}\\right)_v, \\quad c_p = \\left(\\frac{\\partial h}{\\partial T}\\right)_p$$

**物理直觉**: 定容加热时，气体体积不变，所有热量都用于增加内能（温度升高）。定压加热时，气体受热膨胀对外做功，需要额外吸热来补偿这部分功 → $c_p$ > $c_v$。

### 迈耶公式 (Mayer's Formula)

$$\\boxed{c_p - c_v = R_g}$$

**推导**: $h = u + pv$。对于理想气体 $pv = R_g T$。
$$\\frac{dh}{dT} = \\frac{du}{dT} + R_g \\Rightarrow c_p = c_v + R_g$$

### 热容比

$$\\gamma = \\frac{c_p}{c_v} = \\frac{C_{p,m}}{C_{v,m}}$$

| 气体类型 | γ 近似值 | 例子 |
|----------|:--------:|------|
| 单原子 | 1.67 | He, Ar |
| 双原子 | 1.40 | O₂, N₂, 空气 |
| 多原子 | ~1.30 | CO₂, CH₄ |

> [!tip] 补充内容
> 为什么单原子γ最大？单原子只有3个移动自由度，全部能量都用于移动。双原子有额外的转动自由度，同样温升需要更多能量，比热容更大，差值 $c_p-c_v=R_g$ 固定 → γ更小。

## 3-5 理想气体的热力学能与焓

理想气体的热力学能仅是温度的函数（无分子间位能）:

$$u = u(T), \\quad h = u(T) + R_g T = h(T)$$

因此，对**任何过程**（不限于定容/定压）:

$$du = c_v dT, \\quad dh = c_p dT$$

## 3-6 理想气体的熵变

**T-v 形式**: $\\displaystyle s_2 - s_1 = c_v \\ln\\frac{T_2}{T_1} + R_g \\ln\\frac{v_2}{v_1}$

**T-p 形式**: $\\displaystyle s_2 - s_1 = c_p \\ln\\frac{T_2}{T_1} - R_g \\ln\\frac{p_2}{p_1}$

> [!warning] 注意符号
> dv 项是 **+R_g**（体积增大→熵增），dp 项是 **−R_g**（压力增大→熵减）

## 3-7 多变过程与五种基本过程

**多变过程通式**: $pv^n = C$

### 五种基本过程全表

| 过程 | n | 过程方程 | p-v-T 关系 | 膨胀功 w | 热量 q |
|------|:-:|---------|------------|-----------|--------|
| **定容** | ∞ | v=C | $p_2/p_1 = T_2/T_1$ | **0** | $c_v\\Delta T$ |
| **定压** | 0 | p=C | $v_2/v_1 = T_2/T_1$ | $p(v_2-v_1)$ | $c_p\\Delta T$ |
| **定温** | 1 | pv=C | $p_1/p_2 = v_2/v_1$ | $R_g T\\ln(v_2/v_1)$ | = w |
| **定熵** | k | $pv^k$=C | $T_2/T_1 = (v_1/v_2)^{k-1}$ | $c_v(T_1-T_2)$ | **0** |
| **多变** | n | $pv^n$=C | $T_2/T_1 = (v_1/v_2)^{n-1}$ | $\\frac{R_g(T_1-T_2)}{n-1}$ | $c_n\\Delta T$ |

### 多变过程功的计算

**膨胀功**: $w = \\frac{R_g T_1}{n-1}\\left[1 - \\left(\\frac{p_2}{p_1}\\right)^{\\frac{n-1}{n}}\\right] = \\frac{R_g}{n-1}(T_1 - T_2)$

**技术功**: $w_t = n \\cdot w$

**多变比热容**: $c_n = c_v \\cdot \\frac{n-k}{n-1}$

### 定熵过程（绝热可逆）专项

$$\\frac{T_2}{T_1} = \\left(\\frac{v_1}{v_2}\\right)^{k-1} = \\left(\\frac{p_2}{p_1}\\right)^{\\frac{k-1}{k}}$$

膨胀功: $w = c_v(T_1 - T_2) = \\frac{R_g T_1}{k-1}\\left[1 - \\left(\\frac{p_2}{p_1}\\right)^{\\frac{k-1}{k}}\\right]$

## 3-8 绝热自由膨胀与绝热节流

**绝热自由膨胀**（向真空）: w=0, q=0 → Δu=0 → 理想气体T不变, Δs>0（不可逆）

**绝热节流**（阀门/孔板）: h₁=h₂（焓不变）, 理想气体T不变

## 例题

### [例3-1] 气体常数计算

**已知**: 氮气 M=0.028016 kg/mol, R=8.31451 J/(mol·K)

**求**: 氮气的气体常数 R_g

**解**: $R_{g,N_2} = \\frac{8.31451}{0.028016} = 296.78$ J/(kg·K)

### [例3-2] 空气平均摩尔质量

**已知**: 空气体积分数 $x_{N_2}=0.78026, x_{O_2}=0.21, x_{CO_2}=0.0003, x_{H_2}=0.00014, x_{Ar}=0.0093$

**求**: 空气的平均摩尔质量和气体常数

**解**:
$$M_{air} = \\sum x_i M_i = 0.78026\\times0.028016 + 0.21\\times0.032 + 0.0003\\times0.044011 + 0.00014\\times0.002016 + 0.0093\\times0.039948$$
$$= 0.028965 \\text{ kg/mol}$$

$$R_{g,air} = \\frac{R}{M_{air}} = \\frac{8.31451}{0.028965} = 287.05 \\text{ J/(kg·K)}$$

> [!tip] 解题技巧
> 理想气体中摩尔分数=体积分数。平时取 $R_{g,air} \\approx 0.287$ kJ/(kg·K)。
