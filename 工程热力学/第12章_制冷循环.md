## 一· 逆向循环基本原理

一· 制冷机与热泵

1. **制冷机**：使热量从低温物体→高温物体的装置（循环方向与动力循环相反）
2. **热泵**：与制冷机热力学原理完全相同，只是使用目的不同——供热 vs 制冷
3. 工质称为**制冷剂**（制冷装置中的工质）

二· 制冷系数（COP）

1. 定义：ε = 制冷量/净耗功 = q₂/w₀（收获/消耗）

2. ==逆向卡诺循环的制冷系数==
   - ε_c = T_c / (T_h − T_c) = 1 / (T_h/T_c − 1)
   - 推导（分步）：
     （1）逆向卡诺循环：T_h等温放热 + 绝热膨胀 + T_c等温吸热 + 绝热压缩 ← 卡诺循环反向
     （2）吸热量 q₂ = T_c × Δs（低温等温过程） ← T-s图上矩形面积
     （3）放热量 q₁ = T_h × Δs（高温等温过程） ← T-s图上矩形面积
     （4）净功 w₀ = q₁ − q₂ = (T_h − T_c) × Δs ← 能量守恒
     （5）ε_c = q₂/w₀ = T_c × Δs / [(T_h − T_c) × Δs] = **T_c/(T_h − T_c)** ← 代入化简

3. 关键推论
   - T_h固定时，T_c越低 → ε越小（深冷时ε常<1）
   - 例：T_h=300K时，T_c<150K → ε_c<1
   - ==制冷系数可以大于1，也可以小于1==

三· 供热系数

1. ζ = 供热量/净耗功 = q₁/w₀
2. 逆向卡诺循环：ζ_c = T_h/(T_h − T_c) → ==供热系数必定 > 1==

> [!note] 核心理解
> 物理上：制冷循环是"自费搬运热量"——花一份功把热量从冷处搬到热处。温度跨度越大搬运越费劲，制冷系数越低。数学上：ε = T_c/(T_h−T_c) 展示了温差惩罚——温差翻倍，COP减半。热泵之所以 COP>1 是因为它搬运的热量比消耗的功多（功也变成了热）。

## 二· 空气压缩制冷循环

一· 基本循环（1→2→3→4→1）

1. 装置：压气机 + 冷却器 + 膨胀机 + 冷库
2. T-s图过程
   - 1→2：压气机中绝热压缩（T↑, p↑）
   - 2→3：冷却器中定压冷却至接近大气温度
   - 3→4：膨胀机中绝热膨胀（T↓↓, p↓）→ 温度降至冷库温度以下
   - 4→1：冷库中定压吸热（制冷）

3. 制冷系数（定比热容理想气体）
   - ε = 1 / (π^{(κ−1)/κ} − 1)，其中 π = p₂/p₁（增压比）
   - ==增压比越大 → ε越小==

4. 两难困境
   - 增压比大：制冷系数低
   - 增压比小：每千克空气制冷量小（空气比定压热容本来就小）
   - 且增压比过小 → 实际制冷系数反而降低（设备不可逆损失占比增大）
   - 存在==最佳增压比==（例题：约 **3.45**）

二· 回热式空气制冷循环

1. 措施：增加回热器 → 降低增压比同时保持制冷量
2. T-s图变化
   - 无回热：1→2→3→4→1（增压比大）
   - 有回热：1→2_r→3_r→4_r→1_r（增压比小，理论制冷系数不变但实际改善）
3. 效果
   - 理论制冷系数不变（因制冷量和净功同比例变化）
   - 但增压比降低 → 压气机和膨胀机工作压力范围缩小 → 机器轻便
   - ==实际制冷系数提高==（因回热使不可逆损失分布更优）

4. 应用：深度冷冻、气体液化

## 三· 蒸气压缩制冷循环

一· ==最常用的制冷循环==

1. 工质：低沸点制冷剂（氨、氟利昂等），利用相变吸/放热
2. T-s图过程
   - 1→2：压气机中绝热压缩（过热蒸气 → 高压高温蒸气）
   - 2→3：冷凝器中定压放热→凝结（蒸气→液体）
   - 3→4：节流阀中节流降压（不可逆，h₃=h₄，液体→湿蒸气）
   - 4→1：蒸发器中定压定温吸热→蒸发（湿蒸气→饱和蒸气→过热蒸气）

3. 制冷系数
   - ε = q₂/w₀ = (h₁ − h₄)/(h₂ − h₁)
   - q₂ = h₁ − h₄（蒸发器吸热量 = 制冷量）
   - w₀ = h₂ − h₁（压气机耗功）

二· 与空气压缩制冷的对比

1. 优势
   - 利用**相变潜热**（大）→ 制冷量大 → 同样制冷量下设备体积小
   - 接近逆向卡诺循环（蒸发和冷凝都是定温过程）

2. 改进措施
   - （1）**过冷**（3→3'）：冷凝液进一步冷却 → 增大制冷量（q₂↑）
   - （2）**过热**（1→1'）：蒸发器出口蒸气过热 → 保护压气机（避免液击），但压气机耗功也增加
   - （3）回热循环：用过冷液体的热量来过热蒸气

> [!tip] 补充
> 制冷剂选择要点：①临界温度高于冷凝温度；②蒸发压力大于大气压（防止空气漏入）；③冷凝压力不过高；④气化潜热大；⑤化学稳定、无毒、不燃、不破坏臭氧层（ODP=0）、低GWP。

## 四· 其他制冷方式

一· 吸收式制冷

1. 原理：用**热能**代替压缩功驱动制冷循环
   - 利用制冷剂-吸收剂工质对（如氨-水、溴化锂-水）
   - 发生器加热→制冷剂蒸气逸出→冷凝→节流→蒸发制冷→吸收器中被吸收剂吸收→泵回发生器

2. 优势：可利用低品位余热/太阳能驱动，运动部件少

二· 热泵应用

- ε（制冷）+ 1 = ζ（供热）→ 供热系数 = 制冷系数 + 1
- 从环境（空气/水/土壤）中"免费"取热
- ==热泵的供热系数通常>1，节能效果显著==（比电加热省 2~4 倍）

## 速记卡 / 一句话总结

- ==逆向卡诺循环：ε_c = T_c/(T_h−T_c)，温差越大COP越低，深冷可<1==
- 供热系数 ζ = ε + 1，必定 > 1
- 空气压缩制冷：ε = 1/(π^{(κ−1)/κ}−1)，增压比越大COP越低，存在最优值（~3.45）
- 回热降低增压比 → 实际COP提高，用于深度冷冻/气体液化
- ==蒸气压缩制冷：利用相变潜热，最常用==，ε = (h₁−h₄)/(h₂−h₁)
- 过冷→增制冷量；过热→护压气机
- 吸收式制冷：用热驱动，可利用余热/太阳能
- 热泵比电加热节能 2~4 倍
