## 一· 热能的利用

1. 能源现状
   - 自然界主要能源：风能、水能、太阳能、地热能、燃料化学能、原子能
   - 目前主力仍是**矿物燃料**（石油、煤、天然气）的化学能
   - 化石燃料两大问题：①储量递减；②燃烧烟气 → 各国转向清洁能源（核能、风能、太阳能、地热能、海洋能、生物质能）

2. 热能的特殊地位
   - 风能（空气动能）、水能（水位能）→ 直接提供机械能
   - 其他能源 → 往往只能直接/间接（燃烧、核反应）提供**热能**
   - 热能 → 直接用于冶炼、分馏、加热、蒸煮、烘干、采暖
   - ==更大量的热能 → 通过热机转变为机械能 → 进一步转电能==

3. 热机类型
   - 蒸汽轮机、内燃机、燃气轮机、喷气发动机
   - **热能只能部分地转变为机械能**（第二定律约束 → 第4章）

> [!note] 核心理解
> 物理上：人类获取的能源中，除风水直接给机械能外，几乎都先变成热能，再转机械能——热能是**能源转换的中枢环节**。数学上：研究热能→机械能的转换效率上限和实现途径，就是热力学的根本任务。

## 二· 热力学发展简史

1. 18世纪中叶：**瓦特发明蒸汽机**
   - 第一次实现大规模热能 → 机械能转换
   - 推动欧洲工业革命，激发热现象研究

2. 18世纪末：热素说（错误理论）
   - 核心观点：热是一种**无质量、不生不灭**的物质"热素"，物体冷热取决于含热素多少
   - 致命缺陷：无法解释**摩擦生热**
   - **1798年伦福德**：造炮时炮筒和切屑都高温，并无热素流入 → 热必与运动有关

3. 热力学第一定律的确立
   - **1842年 迈耶**：热是能量形式，可与机械能相互转换，总量不变
   - **1849年 焦耳**：多种实验方法测定热功当量（实验结果 **4.154 J/cal**，现代值 **4.1868 J/cal**）
   - **1847年 亥姆霍兹**：提出普遍能量守恒原理，从牛顿力学框架推广到热、电、磁乃至生命过程
   - → 热力学第一定律取代热素说

4. 热力学第二定律的确立
   - **1824年 卡诺**：研究提高蒸汽机效率 → 热机必须在不同温度热源间工作（有温差才有动力），效率取决于高低温热源温度（类比水轮机的高低位落差）
   - 卡诺缺陷：仍信热素说，结论正确但对热能本质理解错误——**他只是猜到了第二定律**
   - **1850年 克劳修斯**：正式提出 ==**"不可能将热量由低温物体传送到高温物体而不引起其他变化"**== → 以此正确论证卡诺定理
   - **1851年 开尔文**：等价表述——"不可能从单一热源吸取热量使之完全变为有用功而不产生其他影响"

5. 热力学第三定律
   - **1912年 能斯特**：绝对零度不可能达到；在绝对零度下，处于热力学平衡的系统的熵值为零

> [!tip] 补充
> 热力学三大定律的简洁表述：
> - 第一定律：能量守恒——**"你不能赢"**（能量不会凭空产生）
> - 第二定律：熵增原理——**"你甚至不能打平"**（总有耗散）
> - 第三定律：绝对零度不可达——**"你无法退出游戏"**

## 三· 工程热力学的研究对象、内容与方法

一· 研究对象

1. **热力学**：研究能量（特别是热能）性质及其转换规律的科学
2. **工程热力学**：着重研究**热能和机械能的转换规律** → 核心任务：从理论上阐明提高热机效率的途径

二· 为什么工质是气体

1. 热能 → 机械能 需要：热机 + **工质**（载能物质）
2. 工质要求：①良好膨胀性（有效做功）、②良好流动性（连续进排气）
3. 同时满足两者的：**气体**（空气、水蒸气等）——固体不行（无流动性）、液体不行（膨胀性差）
4. 例外：蒸汽工质时涉及液态（相变）

三· 三大组成部分

1. **理论基础**：热力学第一定律 + 热力学第二定律
2. **工质热力性质**：常用工质（水蒸气、空气、燃气、湿空气等）的温度、压力、比体积、热力学能、焓、熵、比热容
3. **热力过程与循环**：结合定律 + 工质性质，分析热功转换过程，阐明提高效率的途径

四· 研究方法：宏观唯象方法

1. **工程热力学 = 热学的宏观理论**
   - 依据两个基本定律，运用严密逻辑推理
   - 研究宏观性质和宏观现象
   - **不涉及微观结构和微观粒子运动**

2. 对比：**统计物理学 = 热学的微观理论**
   - 从微观结构出发，依据微观粒子力学规律
   - 用概率理论和统计平均方法

3. 二者的互补关系
   - 热力学 → 利用微观理论成果（工质热物理性质）、检验微观理论正确性
   - 热力学缺陷：不涉及微观结构 → **无法获得物质具体性质**（只能测量，不能计算）
   - 统计物理学缺陷：推导复杂，须对物质结构模型做简化假设 → 与实际情况常有差异

4. 工程热力学的方法特征：**抽象、概括、理想化、简化**
   - 略去细节、抽出共性、抓主要矛盾
   - 这种科学抽象不仅不脱离实际，反而**更深刻地反映事物本质**

> [!note] 核心理解
> 物理上：宏观热力学像"黑箱"——不管里面粒子怎么动，只看进出口的能量账；统计物理像"开箱"——从粒子行为算宏观性质。二者各有盲区，互补使用。数学上：热力学用全微分/循环积分（状态函数）推导，不依赖任何物质模型假设 → 结论普遍可靠；统计物理则依赖微正则/正则系综假设 → 有模型风险。

## 四· 常用计量单位

一· 单位制背景

1. 国际单位制（SI）：世界各国逐步统一采用
2. 我国：**1984年**颁布法定计量单位，以SI为基础
3. 工程热力学涉及的**5个基本单位**：长度（m）、质量（kg）、时间（s）、热力学温度（K）、物质的量（mol）

二· 关键换算

1. 力：**1 kgf = 9.80665 N**（标准重力加速度 g₀ = **9.80665 m/s²**）
2. 压力：1 Pa = 1 N/m²，工程常用 **MPa**（= 10⁶ Pa）
   - 1 atm = 760 mmHg = 1.03323 at = 1.01325 bar = **0.101325 MPa**
   - 1 at = 1 kgf/cm² = 735.559 mmHg = 0.96784 atm = **0.0980665 MPa**
3. 工程单位制特点：以长度、力、时间为基本量，质量是导出量

> [!tip] 补充
> 工程热力学压力必须用**绝对压力**计算——表压力和真空度都是相对于大气压力的差值，实际运算要还原成绝对压力。这是最常见的考试失分点。

## 速记卡 / 一句话总结

- ==热力学三大任务：能量守恒（一）→ 能量品质/方向（二）→ 绝对零度不可达（三）==
- 工程热力学三大块：**基本定律 + 工质性质 + 热力过程循环**
- 工质选气体：膨胀性好 + 流动性好
- 宏观方法 = 黑箱，不涉微观 → 结论普遍可靠但算不出具体物性
- 热功当量现代值：**4.1868 J/cal**
- 压力必用绝对压力，表压/真空度需换算
- 周磊版：32学时，及格率≥90%，第二定律是难点
- 课程主线链：热能↔机械能 → 定律 → 过程循环 → 提高效率
