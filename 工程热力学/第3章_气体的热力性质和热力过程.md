## 一· 实际气体与理想气体

1. 气体的分子运动本质
   - 气体分子：==永不停息的紊乱热运动==——这就是热的本质
   - 气体比体积大 → 分子间距大 → 分子力弱、约束小
   - 分子特征：**微粒性** + **热动性** + **有质量** + **有体积** + **有作用力**

2. 两类气体模型
   - **实际气体**：分子本身具有体积 + 分子间有作用力
     - 当比体积不是很大、工程计算必须考虑分子本身体积和分子间作用力时
     - 典型：水蒸气、制冷剂蒸气（压力不很低时）
   - **理想气体**：分子本身不具有体积 + 分子间没有作用力
     - = 由大量相互间没有作用力的质点组成的可压缩流体
     - ==本质是理想化的简化模型==，但分析计算简单且精度不低

3. 模型选择原则
   - 取决于：①气体所处状态、②计算精度要求
   - 大比体积（**高温、低压**）→ 视为理想气体
   - 工程常用气体（**空气、燃气、烟气、湿空气**等）→ 压力不很高时按理想气体处理，精度满意
   - 实际气体当比体积 → ∞ 时 → 趋近理想气体（分子力消失、分子体积可忽略）

> [!note] 核心理解
> 物理上：理想气体 = "刚性小球"的极限——球之间互不碰撞（无分子力）、球本身退化为质点（无体积）。真实气体越稀薄越接近此极限。数学上：实际气体需 van der Waals 等状态方程修正，而理想气体直接用简单的 **pv = RgT**，使热力过程分析大幅简化。

## 二· 理想气体状态方程与摩尔气体常数

### 理想气体状态方程

**定义**：理想气体在平衡状态下，压力 p、比体积 v 和温度 T 之间满足：

$$pv = R_g T$$

或写为：

$$pV = mR_g T$$

式中 **R_g** 为气体常数，单位：J/(kg·K)（我国法定计量单位）。

**性质 / 条件**：
- 适用于理想气体（无分子力、无本身体积）
- 同一种气体 R_g 为不变常数；不同气体 R_g 不同
- 是工程热力学计算最基本的方程——==本章几乎所有公式都从此导出==

**推导**（来自分子运动论 + 理想气体假定）：
- （1）分子运动论给出：$p = \frac{2}{3}n\frac{m\bar{c}^2}{2}$ ← n 为分子浓度，mc²/2 为分子平均移动能
- （2）引入热力学温度 $T$（$T \propto m\bar{c}^2/2$），结合 $n = N_A/v$
- （3）推导得出 $pv = kN_AT = R_gT$ ← k 为玻尔兹曼常数

> [!tip] 补充
> 玻尔兹曼常数 k = 1.380658 × 10⁻²³ J/K，阿伏加德罗常数 N_A = 6.0221367 × 10²³ mol⁻¹

### 摩尔气体常数（通用气体常数）

**定义**：取 1 mol 气体时，状态方程写为：

$$pV_m = RT$$

式中：M 为摩尔质量（g/mol 或 kg/mol）、V_m 为 1 mol 体积（摩尔体积）、R 为摩尔气体常数。

**关键结论**：==对不同气体，R 是同一数值==

证明：
- （1）$R_g = kN_A \propto 1/M$ ← 气体常数与摩尔质量成反比
- （2）$R = MR_g = kMN_A/M = kN_A$ ← M 消去，R 与 M 无关
- （3）数值：**R = 8.31451 J/(mol·K)** ← 普适常数
- （4）工程单位制：R = **0.847844 kgf·m/(mol·K)**

**实用关系**：
$$R_g = \frac{R}{M}$$

- 例：N₂ 的 M = **0.028016 kg/mol** → R_g,N₂ = 8.31451 / 0.028016 = **296.777 J/(kg·K)**

**标准摩尔体积**（1 atm, 0°C = 273.15 K）：
$$V_{m,std} = \frac{RT_{std}}{p_{std}} = \frac{8.31451 \times 273.15}{101325} = 0.0224141 \text{ m}^3\text{/mol}$$

> [!note] 核心理解
> 物理上：R_g 反映"单位质量气体升温 1K 能膨胀做多少功"——分子越轻（M 越小），R_g 越大，同温同压下比体积越大。数学上：R = k·N_A 说明摩尔气体常数 = 两个基本物理常数的乘积，是**普适自然常数**，不依赖任何物质属性。

## 三· 理想混合气体

### 1. 混合气体成分的表示

一· 绝对成分与相对成分

1. **绝对成分**：用各组分的绝对量表示
   - 质量 $m_i$（kg）、物质的量 $n_i$（kmol）、标准状况体积 $V_{i,std}$（m³）

2. **相对成分**：各分量与总量之比 ← 更常用
   - （1）**质量分数** $w_i = m_i / m_{mix}$，$\sum w_i = 1$
   - （2）**摩尔分数** $x_i = n_i / n_{mix}$，$\sum x_i = 1$
   - （3）**体积分数** $\varphi_i = V_{i,std} / V_{mix,std}$

3. **==摩尔分数 = 体积分数==**（理想混合气体）
   - 因为同温同压下 $V_{i,std} \propto n_i$，且标准摩尔体积相同（**0.0224141 m³/mol**）

4. 质量分数 ↔ 摩尔分数换算：
   $$x_i = \frac{w_i/M_i}{\sum_{i=1}^n w_i/M_i}$$

   $$w_i = \frac{M_i x_i}{\sum_{i=1}^n M_i x_i}$$

二· 平均摩尔质量与气体常数

1. **平均摩尔质量** $M_{mix}$：
   $$M_{mix} = \sum_{i=1}^n M_i x_i = \frac{1}{\sum_{i=1}^n w_i / M_i}$$

   - 含义：各组成气体摩尔质量与摩尔分数乘积的总和
   - 也等于：各质量分数与其摩尔质量比值总和的倒数

2. **混合气体气体常数**：
   $$R_{g,mix} = \frac{R}{M_{mix}}$$

三· 道尔顿分压定律

1. **定律**：理想混合气体的总压力 = 各组成气体分压力之和
   $$p_{mix} = \sum_{i=1}^n p_i$$

2. **分压力定义**：各组分单独存在，且具有与混合气体**相同的 T 和 V** 时施加的压力

3. 分压力计算：
   $$\frac{p_i}{p_{mix}} = \frac{n_i}{n_{mix}} = x_i \quad \Rightarrow \quad p_i = p_{mix} x_i$$

> [!note] 核心理解
> 物理上：理想气体分子间无作用力 → 各组分独立碰撞器壁 → 总碰撞效果 = 各自效果之和 → 总压 = 分压和。数学上：道尔顿定律本质是**线性可加性**——理想气体的压力是分子数密度的线性函数，无交叉项。

**基准案例**（空气）：
- 体积分数：$\varphi_{N_2}=78.026\%,\; \varphi_{O_2}=21.000\%,\; \varphi_{CO_2}=0.030\%$
- → 平均摩尔质量 $M_{air} \approx$ **0.028965 kg/mol**
- → 气体常数 $R_{g,air} \approx$ **287.05 J/(kg·K)**

## 四· 气体的热力性质

### 一· 比热容

**定义**：单位质量物质在无摩擦内平衡的特定过程（x）中，单位温度变化时吸收或放出的热量。

$$c_x = \left( \frac{\delta q}{\delta T} \right)_x$$

- 单位：SI —— **J/(kg·K)**；工程 —— **kcal/(kg·K)**
- 一般 $c = f(T, p)$ ——比热容随物质、过程、状态而异

### 二· 定容比热 c_v 与定压比热 c_p

**定义**：
$$c_v = \left( \frac{\delta q}{\delta T} \right)_v = \left( \frac{\partial u}{\partial T} \right)_v$$

$$c_p = \left( \frac{\delta q}{\delta T} \right)_p = \left( \frac{\partial h}{\partial T} \right)_p$$

- c_v：定容过程中单位温度变化对应的**热力学能变化**
- c_p：定压过程中单位温度变化对应的**焓变化**

**推导依据**（来自热力学第一定律对无摩擦内平衡过程）：
- （1）$\delta q = du + p dv$ ← 第一定律闭口系
- （2）$\delta q = dh - v dp$ ← 第一定律开口系（技术功形式）
- （3）定容时 $dv = 0$ → $\delta q_v = du$ → $c_v = (\partial u/\partial T)_v$
- （4）定压时 $dp = 0$ → $\delta q_p = dh$ → $c_p = (\partial h/\partial T)_p$

### 三· 理想气体的热力学能、焓与迈耶公式

**关键性质**：
- ==理想气体的热力学能仅仅是温度的函数==：$u = u(T)$ ← 分子只有动能，无分子力位能
- ==理想气体的焓也仅仅是温度的函数==：$h = u + pv = u(T) + R_g T = h(T)$

**推论**：对于理想气体，**无论什么过程**：
$$du = c_{v0} dT \quad \quad dh = c_{p0} dT$$

**迈耶公式**（==理想气体核心公式之一==）：

（1）$h = u + pv$ ← 焓定义
（2）微分：$dh = du + d(pv)$
（3）代入理想气体：$c_{p0} dT = c_{v0} dT + R_g dT$ ← $pv = R_g T$
（4）消去 dT：

$$c_{p0} - c_{v0} = R_g$$

摩尔形式：$$C_{p0,m} - C_{v0,m} = R = 8.31451 \text{ J/(mol·K)}$$

物理意义：定压加热时需要额外做膨胀功（$R_g$），所以 $c_p > c_v$

### 四· 热容比 γ

$$\gamma_0 = \frac{c_{p0}}{c_{v0}}$$

由迈耶公式可得：
$$c_{v0} = \frac{R_g}{\gamma_0 - 1}, \quad c_{p0} = \frac{\gamma_0 R_g}{\gamma_0 - 1}$$

- 单原子气体：γ ≈ **5/3**（c_v 和 c_p 基本为定值）
- 双原子气体（常温）：γ ≈ **1.4**（如空气，常取 c_p = **1.005**，c_v = **0.718 kJ/(kg·K)**）
- ==温度越高，γ 越偏离定值，必须用经验公式==

### 五· 比热容的温度依赖

理想气体的 $c_{p0}$ 可表示为温度三次多项式（经验式）：

$$c_{p0} = a_0 + a_1 T + a_2 T^2 + a_3 T^3$$

$$c_{v0} = (a_0 - R_g) + a_1 T + a_2 T^2 + a_3 T^3$$

不同气体有不同 $a_0, a_1, a_2, a_3$（查附表 2）。

热量积分计算（定压过程）：
$$q_p = a_0(T_2 - T_1) + \frac{a_1}{2}(T_2^2 - T_1^2) + \frac{a_2}{3}(T_2^3 - T_1^3) + \frac{a_3}{4}(T_2^4 - T_1^4)$$

**简便方法**：利用平均比热容表（附表 3、4）
$$q_p = \bar{c}_{p0}\big|_0^{t_2} t_2 - \bar{c}_{p0}\big|_0^{t_1} t_1$$

> [!tip] 补充
> 常温下可按**定比热容**简化计算（取 25°C = 298K 的值），但温度范围大时误差显著——温度越高，变比热容与定比热容结果的偏差越大。

### 六· 理想气体的熵

**推导**：
- （1）$ds = \frac{du + p dv}{T}$ ← 热力学基本关系
- （2）代入理想气体 $du = c_{v0} dT$，$p/T = R_g/v$
- （3）$ds = \frac{c_{v0}}{T} dT + \frac{R_g}{v} dv$
- （4）积分：$s = \int \frac{c_{v0}}{T} dT + R_g \ln v + C_1$

或利用焓的形式：
$$s = \int \frac{c_{p0}}{T} dT - R_g \ln p + C_2$$

**定比热容时的简化形式**：
$$s = c_{v0} \ln T + R_g \ln v + C_1'$$
$$s = c_{p0} \ln T - R_g \ln p + C_2'$$

> [!note] 核心理解
> 物理上：==理想气体的 u 和 h 只取决于 T，但 s 不仅取决于 T，还取决于 v 或 p==——因为熵不仅反映分子动能（温熵贡献），还反映"空间分布的无序度"（R_g·ln v 项）。数学上：R_g·ln v 项中的对数 —— 当气体自由膨胀到更大体积时，分子位置的不确定性增大 → 熵增。

## 五· 四种基本热力过程

> 本节过程均指**无摩擦内平衡过程**。

### 概览

四种过程分别保持**比体积 v、压力 p、温度 T、比熵 s** 不变。

| 过程 | 特征常数 | 过程方程（理想气体） | p-v 图形态 | T-s 图形态 |
|------|----------|---------------------|-----------|-----------|
| 定容 | v = const | $p/T = R_g/v$ = 常数 | 垂直线 | 指数曲线 |
| 定压 | p = const | $v/T = R_g/p$ = 常数 | 水平线 | 指数曲线（比定容线平坦） |
| 定温 | T = const | $pv = R_g T$ = 常数 | 等边双曲线 | 水平线 |
| 定熵 | s = const | $pv^\gamma$ = 常数 | 高次双曲线 | 垂直线 |

### 1. 定容过程（v = const）

**定义**：热力系保持比体积不变的情况下进行的吸热或放热过程（如：爆米花机加热）。

**状态参数变化**（理想气体）：$$\frac{p}{T} = \text{常数} \quad \Rightarrow \quad \frac{p_2}{p_1} = \frac{T_2}{T_1}$$

**T-s 关系**（定比热容）：$s = c_{v0} \ln T + C_1'$，斜率 $\frac{\partial T}{\partial s}\big|_v = \frac{T}{c_{v0}}$

```
p ↑                         T ↑
  |   2                        |    2
  |   |  (吸热)                |   /
  |   |                        |  /  (斜率 = T/cv₀)
  |   1                        | /
  |                            |/1
  +--------→ v               +--------→ s
  定容过程 p-v                定容过程 T-s
```

**功与热量**（无摩擦）：
- （1）膨胀功：$w_v = \int p\,dv = 0$ ← $dv = 0$
- （2）技术功：$w_{t,v} = -\int v\,dp = v(p_1 - p_2)$
- （3）热量：$q_v = \int_{1}^{2} c_v dT = u_2 - u_1$ ← $w_v = 0$

==定容过程热量 = 热力学能增量==

### 2. 定压过程（p = const）

**定义**：热力系保持压力不变的情况下进行的吸热或放热过程（如：锅炉中的燃烧过程）。

**状态参数变化**（理想气体）：$$\frac{v}{T} = \text{常数} \quad \Rightarrow \quad \frac{v_2}{v_1} = \frac{T_2}{T_1}$$

**T-s 关系**（定比热容）：$s = c_{p0} \ln T + C_2'$，斜率 $\frac{\partial T}{\partial s}\big|_p = \frac{T}{c_{p0}}$

- 由于 $c_{p0} > c_{v0}$ → 定压线斜率 < 定容线斜率 → 定压线比定容线**更平坦**

**功与热量**（无摩擦）：
- （1）膨胀功：$w_p = \int p\,dv = p(v_2 - v_1)$
- （2）技术功：$w_{t,p} = -\int v\,dp = 0$ ← $dp = 0$
- （3）热量：$q_p = \int_{1}^{2} c_p dT = h_2 - h_1$

==定压过程热量 = 焓增量==

### 3. 定温过程（T = const）

**定义**：热力系在温度保持不变的情况下进行的膨胀（吸热）或压缩（放热）过程（如：冷凝器、蒸发器）。

**状态参数变化**（理想气体）：$$pv = R_g T = \text{常数} \quad \Rightarrow \quad \frac{p_2}{p_1} = \frac{v_1}{v_2}$$

- p-v 图中为**等边双曲线**

**功与热量**（无摩擦）：

膨胀功：
$$w_T = \int_{1}^{2} p\,dv = \int_{1}^{2} \frac{R_g T}{v} dv = R_g T \ln\frac{v_2}{v_1}$$

技术功：
$$w_{t,T} = -\int_{1}^{2} v\,dp = R_g T \ln\frac{p_1}{p_2}$$

- （1）由于 $v_2/v_1 = p_1/p_2$ → ==$w_T = w_{t,T}$==（技术功 = 膨胀功）
- （2）原因：进气功与排气功恰好抵消

热量：$$q_T = \int_{1}^{2} T\,ds = T(s_2 - s_1)$$

熵变：$$s_2 - s_1 = R_g \ln\frac{v_2}{v_1} = R_g \ln\frac{p_1}{p_2}$$

==无论有无摩擦，理想气体定温过程始终 $q_T = w_T = w_{t,T}$==

### 4. 定熵过程（s = const，即无摩擦绝热过程）

**定义**：热力系保持比熵不变的情况下进行的膨胀或压缩过程（如：蒸汽轮机、压气机）。

**过程方程**（定比热容理想气体）：$$pv^{\gamma_0} = \text{常数}$$

三个等价形式：
- （1）$pv^{\gamma_0}$ = 常数
- （2）$Tv^{\gamma_0-1}$ = 常数
- （3）$\frac{T}{p^{(\gamma_0-1)/\gamma_0}}$ = 常数

状态参数关系：
$$\frac{T_2}{T_1} = \left(\frac{v_1}{v_2}\right)^{\gamma_0-1} = \left(\frac{p_2}{p_1}\right)^{\frac{\gamma_0-1}{\gamma_0}}$$

**p-v/T-s 图特征**：
- p-v 图：高次双曲线（γ₀ > 1），比定温线**更陡**
- T-s 图：垂直线

> - 定熵过程：p₂/p₁ = p_{r2}/p_{r1}、v₂/v₁ = v_{r2}/v_{r1}
> - 功：w_s = u₁ − u₂、w_{t,s} = h₁ − h₂
> - 这种方法比定比热容结果**更精确**，计算更简捷。

> [!note] 核心理解（四过程对比）
> 物理上：理想气体四个基本过程覆盖了"约束-响应"的四种极限——约束体积（定容）、约束压力（定压）、约束温度（定温）、约束熵/绝热（定熵）。定熵过程中没有热量交换，膨胀全靠消耗自身热力学能 → 温度必然下降（膨胀时）或上升（压缩时）。数学上：四过程的状态方程可统一为 pvⁿ = 常数，n = ∞（定容）、n = 0（定压）、n = 1（定温）、n = γ（定熵）——这直接引出多变过程概念。

## 六· 多变过程

### 定义与物理意义

**定义**：凡是状态变化规律符合 $pv^n$ = 常数的内平衡过程，称为**多变过程**，n 为**多变指数**。

- n 理论上可取 $-\infty$ 到 $+\infty$ 之间的任意实数
- 每一个 n 值对应一种不同的状态变化规律
- ==多变过程是四种基本过程的统一推广==

**四种基本过程作为特例**：
| n | 对应过程 | 验证 |
|---|---------|------|
| n = ±∞ | 定容过程 | $p^{1/n}v = const$ → $p^0 v = v = const$ |
| n = 0 | 定压过程 | $pv^0 = p = const$ |
| n = 1 | 定温过程 | $pv = R_g T = const$ |
| n = γ₀ | 定熵过程 | $pv^{\gamma_0} = const$ |

### 状态参数关系

将定熵过程中的 γ₀ 全部替换为 n：
$$\frac{T_2}{T_1} = \left(\frac{v_1}{v_2}\right)^{n-1} = \left(\frac{p_2}{p_1}\right)^{\frac{n-1}{n}}$$

### p-v 图与 T-s 图中的分布规律

```
p-v 图中多变过程簇（n 递增方向 = 顺时针）：
   n = -∞ (定容)
     ↻ 顺时针递增
   n = 0  (定压)
     ↻
   n = 1  (定温)
     ↻
   n = γ  (定熵)
     ↻
   n = +∞ (定容)
```

- ==p-v 图中：从定容线出发，n 按**顺时针**方向递增==
- ==T-s 图中：从定容线开始，n 按**顺时针**方向递增==
- n = 0、±∞、-1 时 → 直线；$0 < n < +\infty$ 时 → 双曲线（不同方次）；负 n 区间 → 抛物线

### 对数坐标系中的直线化

将 $pv^n$ = 常数 取对数：
$$\lg p = -n\lg v + \text{常数}$$

==在 $\lg p$ - $\lg v$ 坐标系中，所有多变过程都是直线==，斜率 = **-n**

应用：将任意不规则过程在对数坐标中近似为若干直线段 → 用多个多变过程分段近似。

### 功的计算

膨胀功：
$$w_n = \frac{1}{n-1}R_g T_1 \left[1 - \left(\frac{p_2}{p_1}\right)^{\frac{n-1}{n}}\right] = \frac{1}{n-1}R_g(T_1 - T_2)$$

技术功：
$$w_{t,n} = n w_n = \frac{n}{n-1}R_g(T_1 - T_2)$$

### 比多变热容 c_n

**定义**：$c_n = \left( \frac{\delta q_n}{\partial T} \right)_n = T \left( \frac{\partial s}{\partial T} \right)_n$

**与多变指数的关系**：$$c_n = \frac{n c_{v0} - c_{p0}}{n-1}$$

$$n = \frac{c_n - c_{p0}}{c_n - c_{v0}}$$

**热量计算**（定比热容）：$$q_n = c_n(T_2 - T_1) = \frac{n c_{v0} - c_{p0}}{n-1}(T_2 - T_1)$$

- n = γ₀ 时 $c_n = 0$ → 绝热无热交换
- n = 1 时 $c_n = \infty$ → 定温过程中热量"无限大"（维持温度不变需持续换热）
- n = 0 时 $c_n = c_{p0}$ → 定压过程
- n = ±∞ 时 $c_n = c_{v0}$ → 定容过程

> [!note] 核心理解
> 物理上：多变指数 n 刻画了"在膨胀/压缩过程中，热量输入与输出对状态变化的调节程度"。n = 1（定温）→ 全程传热维持温度；n = γ（定熵）→ 完全绝热、温度随膨胀/压缩变化最剧烈。实际工程过程中 1 < n < γ 或 γ < n < ∞ 最常见——代表有摩擦的绝热过程，摩擦热相当于"注入过程"使温度变化偏离定熵情况。数学上：n = d(lg p)/d(lg v) 是对数空间曲线的瞬时斜率，提供了一种对任意不规则过程做分段线性近似的方法。

## 七· 无功过程与绝热过程

> 按热力系与外界能量交换情况定义的过程。"**传热过程不做功（w_t = 0）**"和"**做功过程传热难（q ≈ 0）**"→ 给分析带来很大便利。

### 1. 不做膨胀功的过程

**定义**：闭口热力系经历状态变化时 $\delta w = 0$。

**有/无摩擦的差异**：
- 无摩擦：$\delta w = p dv = 0$ → $p>0$ 故 $dv = 0$ → **==定容过程==**
- 有摩擦：$p dv = \delta w + \delta w_L > 0$ → $dv > 0$ → **比体积增大过程**（如：气体向真空自由膨胀）

**热量**：$\delta q = du + \delta w = du$ → $q = u_2 - u_1$

- ==无论有无摩擦、内部是否平衡，只要不做膨胀功，热量 = 热力学能变化==
- 理想气体：$q = \int c_{v0} dT$；定比热容：$q = c_{v0}(T_2 - T_1)$

> [!warning] 易混淆
> 不做膨胀功的过程**≠**定容过程——有摩擦时比体积会增大（不做膨胀功只是说功没传出去，不是体积没变）。定容过程也只在无摩擦时才不做膨胀功。

### 2. 不做技术功的过程

**定义**：热力系在稳定流动过程中 $\delta w_t = 0$（如：各种换热设备——锅炉、冷凝器、加热器等）。

**有/无摩擦的差异**：
- 无摩擦：$\delta w_t = -v dp = 0$ → $v > 0$ 故 $dp = 0$ → **==定压过程==**
- 有摩擦：$-v dp = \delta w_t + \delta w_L > 0$ → $dp < 0$ → **压力降落过程**（如流体在管道中流动）

**热量**：$\delta q = dh + \delta w_t = dh$ → $q = h_2 - h_1$

- ==无论有无摩擦、压力是否降落，只要不做技术功，热量 = 焓变化==
- 理想气体：$q = \int c_{p0} dT$；定比热容：$q = c_{p0}(T_2 - T_1)$

> [!tip] 补充（工程要点）
> 在各种换热设备中，尽管存在摩擦阻力导致压力有所降落，但==用 Δh 计算换热量是严格正确的==（因为过程本质是不做技术功的过程），而**不是**近似按定压过程处理的简化方法。这常常是概念题考点。

### 3. 绝热过程

**定义**：热力系和外界无热量交换，$\delta q = 0$。

**有/无摩擦的差异**：
- 无摩擦：$T ds = du + p dv = \delta q = 0$ → $ds = 0$ → **==定熵过程==**
- 有摩擦：$T ds = du + p dv = \delta q + \delta q_g > 0$ → $ds > 0$ → **熵增过程**

**功的计算**（==适用于任何工质，有无摩擦都成立==）：
$$w = u_1 - u_2$$
$$w_t = h_1 - h_2$$

- 理想气体：$w = -\int c_{v0} dT$、$w_t = -\int c_{p0} dT$
- 定比热容：$w = c_{v0}(T_1 - T_2)$、$w_t = c_{p0}(T_1 - T_2)$

**有摩擦绝热过程的近似处理**：
- 实际动力机械中有摩擦的绝热膨胀/压缩近似遵守 $pv^n$ = 常数
- 绝热膨胀时：$n < \gamma_0$（摩擦热使气体膨胀"不够冷"）
- 绝热压缩时：$n > \gamma_0$（摩擦热积累使气体更热）
- ==n 偏离 γ₀ 的程度反映了摩擦的大小==

功的计算（有摩擦绝热、多变规律近似）：
$$w = \frac{1}{n-1}R_g T_1 \left[1 - \left(\frac{p_2}{p_1}\right)^{\frac{n-1}{n}}\right]$$

$$w_t = \frac{n}{n-1}R_g T_1 \left[1 - \left(\frac{p_2}{p_1}\right)^{\frac{n-1}{n}}\right]$$

> [!note] 核心理解
> 物理上：三者揭示了功和热的"对偶"——不做膨胀功时热量全变为内能（q = Δu）；不做技术功时热量全变为焓（q = Δh）；绝热时内能/焓的减少全变为功（w = -Δu、w_t = -Δh）。这种"能量单一流向"使工程分析变得简洁。数学上：绝热过程的 w = u₁ - u₂ 是最干净的能量守恒结论——不依赖过程路径、不依赖工质模型、不依赖可逆性，是真正的**状态函数的差**。

## 八· 绝热自由膨胀与绝热节流

### 1. 绝热自由膨胀过程

**定义**：气体在绝热条件下向真空进行的不做膨胀功的膨胀过程（如：隔板抽开，气体从一侧冲向真空另一侧）。

**过程特征**：
- （1）$w = 0$（不做膨胀功）+ $q = 0$（绝热）
- （2）→ $Δu = 0$ → $u_2 = u_1$ ==热力学能不变==

**理想气体**：$u = u(T)$ 只取决于温度 → ==$T_2 = T_1$，温度不变==

**实际气体**（焦耳效应）：
- 自由膨胀后比体积 ↑ → 分子力位能 ↑（需要消耗动能）→ 动能 ↓ → ==$T_2 < T_1$，温度降低==
- 这是**存在内摩擦的绝热过程** → $s_2 > s_1$（熵增）

### 2. 绝热节流过程

**定义**：流体在管道中流经阀门、孔板等缩口，因局部阻力导致压力显著降落的流动过程。

**过程特征**：
- （1）有内摩擦的不做技术功过程 → $p_2 < p_1$（压力必降）
- （2）绝热：$q = 0$
- （3）不做技术功的绝热稳定流动 → ==$h_2 = h_1$，焓不变==

**理想气体**：$h = h(T)$ 只取决于温度 → ==$T_2 = T_1$，温度不变==
- 熵增：$s_2 > s_1$（内摩擦）

**实际气体**——焦耳-汤姆逊效应（焦-汤效应）：
- ==节流后温度可能降低、不变或升高==
- 取决于焦-汤系数 $\mu_{JT} = (\partial T/\partial p)_h$ 的正负
- 冷效应（$ΔT < 0$）→ 可用于**制冷**

**工程应用**：
- 调节压力和流量（阀门节流）
- 节流制冷（冷效应）
- 节流孔板测流量（压差 → 流量）
- 轴封（多次节流降低泄漏）

> [!note] 核心理解
> 物理上：绝热自由膨胀 = "失控的膨胀"——体积变大但不对外做功（冲向真空），热量也进不来，所以能量只能内部重新分配；理想气体温度不变，实际气体因"拉开分子距离"消耗动能而降温。绝热节流 = "强迫流过窄口"——压力降了但焓不变，是典型的不做技术功过程。数学上：自由膨胀 u = const + 绝热 = 热力学能守恒；节流 h = const + 绝热 = 焓守恒（来自稳定流动能量方程 h₁ + c₁²/2 + gz₁ = h₂ + c₂²/2 + gz₂ 中忽略动能和位能变化）。

## 九· 混合过程

### 1. 定容混合过程

**场景**：刚性容器内，n 种气体抽去隔板充分混合。

**基本关系**：
- （1）质量守恒：$m = \sum m_i$
- （2）体积守恒：$V = \sum V_i$
- （3）不做膨胀功（W = 0）

**绝热定容混合（Q = 0）**：
- ==$U = \sum U_i$，热力学能不变==
- 定比热容理想气体：$m c_{v0} T = \sum m_i c_{v0,i} T_i$

**混合温度**：
$$T = \frac{\sum_{i=1}^n m_i c_{v0,i} T_i}{\sum_{i=1}^n m_i c_{v0,i}}$$

**混合压力**：$p = \frac{m R_g T}{V}$，其中 $M_{mix} = 1/\sum(w_i/M_i)$

**熵增**：
$$\Delta S = \sum_{i=1}^n m_i \left( c_{v0,i} \ln\frac{T}{T_i} + R_{g,i} \ln\frac{V}{V_i} \right)$$

- 同种气体混合：温度相同时 $\Delta S = m R_g \ln(V/V_{\text{原}}) > 0$（不可逆）

### 2. 流动混合过程

**场景**：n 股不同气体流入混合室，充分混合后流出。

**基本关系**：
- 质量流量守恒：$q_m = \sum q_{mi}$
- ==不做技术功的绝热过程==（$W_t = 0, Q = 0$）

**混合后温度**：
$$T = \frac{\sum_{i=1}^n q_{mi} c_{p0,i} T_i}{\sum_{i=1}^n q_{mi} c_{p0,i}}$$

**各组分分压力**：$p_i' = p \cdot x_i = p \cdot \frac{w_i}{M_i} / \sum\frac{w_j}{M_j}$

**熵增率**：
$$\Delta\dot{S} = \sum_{i=1}^n q_{mi} \left( c_{p0,i} \ln\frac{T}{T_i} - R_{g,i} \ln\frac{p_i'}{p_i} \right)$$

> [!note] 核心理解
> 物理上：混合过程本质是"不同状态气体的不可逆趋同"——温度、压力、成分差异通过分子扩散/碰撞抹平 → 熵增。定容混合是闭口系（W=0 → ΔU 决定温度），流动混合是开口系（W_t=0 → ΔH 决定温度）——这是两种混合用不同热容（c_v vs c_p）算温度的根因。数学上：熵增公式中 ln(V/V_i) 和 ln(T/T_i) 项分别来自**体积随机性**和**热随机性**——两者都 > 0（混合使系统更无序）。

## 十· 充气与放气过程

> 非稳定流动过程的两个典型。容器中气体状态随时间变化，但任意瞬时可认为空间均匀。

### 1. 充气过程

**场景**：稳定气源（$p_0, T_0, h_0$）向容积 V 不变的刚性容器充气。

**基本能量方程**（取容器内气体为热力系）：
$$Q = (m_2 u_2 - m_1 u_1) - (m_2 - m_1) h_0$$

**（A）绝热充气（Q = 0）**——快速充气或容器绝热良好：
$$m_2 u_2 = m_1 u_1 + (m_2 - m_1) h_0$$

==充气后容器内热力学能 = 原有气体热力学能 + 充入气体焓==

同种定比热容理想气体：
$$T_2 = \frac{p_2 \gamma_0 T_0 T_1}{(p_2 - p_1) T_1 + p_1 \gamma_0 T_0}$$

**特殊情形——向真空中绝热充气（$p_1 = 0$ 或 $m_1 = 0$）**：
$$T_2 = \gamma_0 T_0$$

==向真空绝热充气后温度变为原来的 γ₀ 倍！==
- 例：空气（γ₀ = 1.4）、$T_0$ = 300K → $T_2$ = **420K (147°C)**
- 原因：充气时的推动功（$p_0 v_0$）转变为气体的热力学能

**（B）定温充气**——缓慢充气或容器传热良好（$T_2 = T_1$）：
$$Q = (p_2 - p_1)V \cdot \frac{T_1 - \gamma_0 T_0}{T_1(\gamma_0 - 1)}$$

- 通常 $T_1 < \gamma_0 T_0$ → Q 为负值 → 容器**向外界放热**

### 2. 放气过程

**场景**：容器内高压气体向外界排出，容积 V 不变。

**基本能量方程**：
$$Q = m_2 u_2 - m_1 u_1 - \int_{m_1}^{m_2} h\,dm$$

**（A）绝热放气（Q = 0）**——快速放气：
- 通常可认为是一个定熵膨胀过程（超出 V 的部分从容器排出）
- 温度和压力关系：$\frac{T_2}{T_1} = \left( \frac{p_2}{p_1} \right)^{\frac{\gamma_0-1}{\gamma_0}}$ ← 容器中剩余气体遵循定熵规律
- 剩余质量：$m_2 = m_1 \left( \frac{p_2}{p_1} \right)^{1/\gamma_0}$
- 放出质量：$-Δm = m_1 - m_2 = m_1 \left[ 1 - \left( \frac{p_2}{p_1} \right)^{1/\gamma_0} \right]$

**（B）定温放气（$T_2 = T_1$）**：
$$Q = (p_1 - p_2)V$$

==定温放气吸热量只取决于容积和压力降==，与气体温度、比热容、气体常数均无关。

> [!note] 核心理解
> 物理上：充气过程的核心是"外来气体的焓如何转化为容器内气体的热力学能"——绝热充气最极端的例子是向真空充气，温度飙升 γ₀ 倍，因为推动功全部变成热力学能。放气过程本质是"容器内气体膨胀后将多余部分排出"——剩余气体按定熵规律降温降压。数学上：充气/放气是非稳态开口系能量方程 dE/dt = Q̇ − Ẇ + Σṁ_in·h_in − Σṁ_out·h_out 的积分结果，是热力学第一定律在**变质量系统**的经典应用。

## 速记卡 / 一句话总结

- ==理想气体 $pv = R_g T$，$R_g = R/M$，$R = 8.31451$ J/(mol·K)（普适常数）==
- ==迈耶公式 $c_{p0} - c_{v0} = R_g$：定压加热需要额外做膨胀功==
- 理想气体：$u = u(T)$，$h = h(T)$——只取决于温度；但 $s$ 还取决于 $v$ 或 $p$
- 定比热容理想气体：空气 $c_p$ ≈ **1.005**、$c_v$ ≈ **0.718 kJ/(kg·K)**、γ₀ ≈ **1.4**
- 四过程统一于 $pv^n$ = 常数：n = ±∞（定容）、n = 0（定压）、n = 1（定温）、n = γ₀（定熵）
- ==p-v / T-s 图中 n 按**顺时针**方向递增==
- 定熵膨胀功 $w_s = \frac{R_g T_1}{\gamma_0-1}[1 - (p_2/p_1)^{(\gamma_0-1)/\gamma_0}]$，技术功 = **γ₀ ×** 膨胀功
- 多变过程热量 $q_n = c_n(T_2 - T_1)$，$c_n = \frac{n c_{v0} - c_{p0}}{n-1}$
- 不做膨胀功 → $q = \Delta u$；不做技术功 → $q = \Delta h$；绝热 → $w = -\Delta u$、$w_t = -\Delta h$
- 绝热自由膨胀（理想气体）：$T_2 = T_1$；绝热节流（理想气体）：$T_2 = T_1$、$h_2 = h_1$
- ==向真空绝热充气（定比热容理想气体）：$T_2 = \gamma_0 T_0$==（温度飙升！原因是推动功→热力学能）
- 定温放气吸热 $Q = (p_1 - p_2)V$——与温度和比热容均无关
