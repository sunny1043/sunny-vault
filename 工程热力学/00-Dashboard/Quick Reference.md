# Quick Reference — 工程热力学公式速查

#dashboard #quick-reference #thermodynamics

> 记法约定：$R_g$=气体常数 $=R/M$ · $k=c_p/c_v$=绝热指数 · $\varepsilon$=压缩比 · $\pi$=增压比 · 下标 0 多指环境/初态

## 第一定律 → [[第2章_热力学第一定律]]
| 公式 | 说明 |
|------|------|
| $q=\Delta u+w$ | 闭口系（单位质量） |
| $Q=\Delta U+W$ | 闭口系（总量） |
| $q=\Delta h+w_s$ | 开口系稳定流动 |
| $w=\int p\,dv$ | 容积变化功（膨胀功） |
| $w_s=-\int v\,dp$ | 技术功（轴功） |
| $h=u+pv$ | 焓的定义 |

## 气体性质与过程 → [[第3章_气体的热力性质和热力过程]]
| 公式 | 说明 |
|------|------|
| $pv=R_gT$ , $pV=mR_gT$ | 理想气体状态方程 |
| $R_g=R/M$ | 气体常数与通用常数 |
| $c_p-c_v=R_g$ | 迈耶公式 |
| $k=c_p/c_v$ | 绝热指数 |
| $pv^n=C$ | 多变过程（$n$=多变指数） |
| $n=0,1,k,\infty$ | →定压/定温/绝热/定容 |

## 第二定律 · 熵 · 㶲 → [[第4章_热力学第二定律]]
| 公式 | 说明 |
|------|------|
| $\eta_t=1-T_2/T_1$ | 卡诺循环热效率（上限） |
| $ds=\delta q_{rev}/T$ | 熵的定义 |
| $\Delta S_{iso}\ge 0$ | 孤立系熵增原理（=0 可逆） |
| $\oint \delta q/T\le 0$ | 克劳修斯积分（不等式取不可逆） |
| $e_{x,Q}=(1-T_0/T)\,\delta Q$ | 热量㶲（可用能） |
| $\Pi=T_0\,\Delta S_{iso}$ | 㶲损 = 环境温度 × 孤立系熵增 |

## 流动与压缩 → [[第5章_气体的流动和压缩]]
| 公式 | 说明 |
|------|------|
| $\nu_{cr}=\left(\tfrac{2}{k+1}\right)^{\frac{k}{k-1}}$ | 喷管临界压力比 |
| $Ma=c/a$ , $a=\sqrt{kR_gT}$ | 马赫数 / 当地声速 |
| 渐缩喷管 $Ma\le1$ | 缩放(拉伐尔)喷管才能 $Ma>1$ |
| 多级压缩+中间冷却 | 各级增压比相等最省功 |

## 气体动力循环 → [[第6章_气体动力循环]]
| 公式 | 说明 |
|------|------|
| $\eta_{Otto}=1-\dfrac{1}{\varepsilon^{k-1}}$ | 定容加热（奥托）循环 |
| $\eta_{Brayton}=1-\dfrac{1}{\pi^{(k-1)/k}}$ | 燃气轮机（布雷顿）循环 |
| 狄塞尔/萨巴德 | 含预胀比 $\rho$，效率介于两者间 |
| 比较准则 | 同压缩比 Otto>Sabathe>Diesel |

## 蒸汽动力循环 → [[第9章_水蒸气性质和蒸汽动力循环]]
| 公式 | 说明 |
|------|------|
| $\eta_t=1-\dfrac{h_2-h_3}{h_1-h_0}$ | 朗肯循环热效率（含泵功） |
| $\eta_t\approx\dfrac{h_1-h_2}{h_1-h_3}$ | 忽略泵功的近似 |
| 提高 $\eta_t$ | ↑初温初压、↓背压、再热、回热 |
| 干度 $x=\dfrac{m_v}{m_v+m_l}$ | 湿蒸汽状态 |

## 一般关系式 → [[第7章_热力学一般关系式]]
| 公式 | 说明 |
|------|------|
| $\left(\frac{\partial T}{\partial v}\right)_s=-\left(\frac{\partial p}{\partial s}\right)_v$ | 麦克斯韦关系式（4 个之一） |
| $\mu_J=\left(\frac{\partial T}{\partial p}\right)_h$ | 焦耳-汤姆逊系数 |

## 湿空气 → [[第10章_湿空气性质和湿空气过程]]
