# 第2章 热力学第一定律 ★★★

#engineering-thermodynamics #first-law

## 2-1 实质：能量不会凭空产生或消失

**定义**: 热力学第一定律是**能量守恒与转换定律**在热现象中的体现——热能和机械能在转移与转换时，能量的**总量必定守恒**。

**物理直觉**: 给一个系统加热，这些热量去哪了？要么变成系统"内部储存的能量"（热力学能增加），要么变成系统对外做的功。不可能凭空消失，也不可能凭空产生。这就是第一定律的精髓——**能量会计学**。

### 一般热力系能量方程

考虑一个有质量流进流出的开口系，系统总能量 $E = U + E_k + E_p$:

$$\\delta Q = dE + (e_2\\delta m_2 - e_1\\delta m_1) + \\delta W_{tot}$$

其中 $e = u + c^2/2 + gz$ (每千克流体携带的总能量)。

## 2-2 闭口系能量方程

**适用场景**: 密封容器内的气体膨胀/压缩过程。

**条件**: 无物质交换 ($\\delta m_1 = \\delta m_2 = 0$)，宏观动能和位能不变。

$$\\boxed{q = \\Delta u + w}$$

微分形式: $\\delta q = du + \\delta w$

**推导**: 
一般表达式 $\\delta Q = dE + (e_2\\delta m_2 - e_1\\delta m_1) + \\delta W_{tot}$ 中：
- 闭口系 → $\\delta m_1 = \\delta m_2 = 0$
- 无KE/PE变化 → $dE = dU$
- 只有膨胀功 → $\\delta W_{tot} = \\delta W$

→ $\\delta Q = dU + \\delta W$，除以质量得 $\\delta q = du + \\delta w$

## 2-3 开口系与稳定流动能量方程

### 为什么需要开口系？

实际工程设备（汽轮机、压气机、锅炉）中工质不断流进流出——这就是开口系。

### 稳定流动能量方程

**条件**: 流道中各处参数不随时间变化。

$$q = (h_2 - h_1) + \\frac{c_2^2 - c_1^2}{2} + g(z_2 - z_1) + w_s$$

忽略动能和位能变化:

$$\\boxed{q = \\Delta h + w_s}$$

**物理直觉**: 稳定流动中，加给流体的热量去哪了？一部分使流体的焓增加（温度升高），一部分变成轴功输出。这与闭口系 ($q = \\Delta u + w$) 的区别在于：闭口系用**内能u**，开口系用**焓h**——因为流体进/出系统时连带做了推动功 $pv$。

## 2-4 三种功的关系

| 功的类型 | 公式 | 适用 | p-v图含义 |
|----------|------|------|-----------|
| 膨胀功 w | $\\int p dv$ | 闭口系 | 曲线与v轴围成面积 |
| 技术功 $w_t$ | $-\\int v dp$ | 开口系 | 曲线与p轴围成面积 |
| 轴功 $w_s$ | $w_t - \\Delta(c^2/2) - g\\Delta z$ | 涡轮输出 | |

**关系**: $w_t = w - (p_2v_2 - p_1v_1)$

## 2-5 功和热量在坐标图中的表示

**无摩擦内平衡过程**:
- 膨胀功: p-v 图中过程曲线与 v 轴围成的面积
- 技术功: p-v 图中过程曲线与 p 轴围成的面积  
- 热量: T-s 图中过程曲线下的面积

```
  p                  T
  │ 1                 │   1
  │  ╲   w            │    ╲
  │   ╲ (面积)        │     ╲  q = ∫Tds
  │    ╲              │      ╲ (面积)
  │     2             │       2
  └────── v           └──────── s
```

## 2-6 循环的净功 = 净热量

$$\\oint \\delta q = \\oint \\delta w$$

**物理直觉**: 工质绕一圈回到原位，内能没变——净收入的热量必然全部转化为净输出的功。这无论有无摩擦都成立。

## 例题

### [例2-1] 水泵输水到50m高塔

**题目**: 向50m高水塔送水。(1)理论上最少消耗多少功？(2)若水泵效率70%，实际消耗多少？

**已知**: $\\Delta z = 50$m, $g = 9.807$ m/s²

**解**:
理论功（技术功公式）: 
$$w_t = -\\int v dp = g\\Delta z = 9.807 \\times 50 = 490.4 \\text{ J/kg} = 0.4904 \\text{ kJ/kg}$$

实际功: 
$$w_{act} = \\frac{w_t}{\\eta} = \\frac{0.4904}{0.70} = 0.7006 \\text{ kJ/kg}$$

多消耗的功 ($0.2102$ kJ/kg) 变成热（功损→热产），使水温微升。

> [!tip] 解题技巧
> 水的比体积近似不变时，$w_t = v\\Delta p = g\\Delta z$。不要死记——从 $-\\int vdp$ 推导。
