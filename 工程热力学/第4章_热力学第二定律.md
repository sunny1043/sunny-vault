## 4-1 热力学第二定律的任务

一· 过程的方向性

- 热力学第一定律：能量总量守恒，创造能量（第一类永动机）不可能，消灭能量也办不到。
- 但：==不违反第一定律的过程，不一定都能实现==。经验事实：
    - 1. 烧红锻件在空气中自发冷却 → 热量从锻件→空气；反过来（已冷的锻件从空气中收回热量重新灼热），不违反第一定律但**不会自发实现**。
    - 2. 飞轮因摩擦减速 → 动能→热能散失到空气；反过来（空气将热能→动能还给飞轮），不违反第一定律但**不会自发实现**。
    - 3. 高压氧气瓶向低压大气漏气；反过来（空气自动向高压瓶充气）**不会自发实现**。
    - → 这些反例共同说明：过程有方向性。

二· 自发过程 vs 非自发过程

1. **自发过程**：不必借助外界作用，靠系统内部势差自发进行
    - 热量从高温→低温
    - 机械能→热能（摩擦生热）
    - 气体自发膨胀（高压→低压）
    - 核心：都是 ==势差驱动==，使系统趋于平衡。

2. **非自发过程**：必须借助外界作用才能进行的反向过程
    - 热量从低温→高温（需制冷机耗功）
    - 热能→机械能（需热机，且只能部分转换）
    - 气体压缩（需压气机耗功）

3. **非自发过程的补偿条件**（全书关键思想）：
    - ==一·个·非·自·发·过·程·的·进·行，必须有另外的自发过程来推动== → 即"以自发过程作为代价/补偿条件"。
    - 制冷装置：热量从低温→高温（非自发），代价 = 制冷机耗功→热（自发）。
    - 热机：部分高温热能→机械能（非自发），代价 = 另一部分高温热能→低温大气（自发）。
    - 压气机：气体被压缩（非自发），代价 = 消耗机械能→热能（自发）。

> [!note] 核心理解
> 物理上：非自发过程违背"势差驱动"的自然倾向，必须靠自发过程（释放势差）来"付费"。数学上：总熵增 ≥ 0——自发过程熵增 > 0，非自发过程熵减 < 0，但配对的补偿过程使孤立系总熵增 ≥ 0。这是第二定律的深层结构。

三· 热力学第二定律的任务（三个关键词）

1. **方向**：判断过程自发进行的方向
    - 第一定律只管能量"数量"，不能说明方向；第二定律管"能不能发生"。
    - 周磊比喻：第一定律 = 会计（Accountant），只管账平不平；第二定律 = 老板（Boss），决定这事能不能干。

2. **条件**：过程得以进行的条件
    - 自发过程条件 → 热力系内部存在某种势差（温差/压差/浓度差）。
    - 非自发过程条件 → 必须有外界作用（另一自发过程推动）。
    - （补：周磊概括——一切非自发过程的进行中必须有自发过程的时时伴随；一切自发过程的进行前必须有非自发过程的历史积累。）

3. **限度**：能量转换的理论最大值
    - 第一定律解决了能量数量的守恒性 → 第二定律解决能量转换的差异性和有限性。
    - ==热机的热效率在一定条件下有理论上的最大值==（即卡诺效率 $1 - T_2/T_1$），实际不可超越。

> [!question] 自发过程都是不可逆的吗？
> 课后思考题第1问即涉此问题。自发过程一定不可逆（因为存在有限势差），但不可逆过程不一定都是自发的（如外界驱动的不可逆压缩）。详见4-2节。

## 4-2 可逆过程和不可逆过程

一· 不可逆性的两大根源

1. **机械运动摩擦（力不平衡）**
    - 凡产生相对运动的接触部分（包括流体各相邻层），摩擦不可避免。
    - 造成的影响：一部分机械能不可逆复地变成热能（功→热，能量耗散）。
    - 膨胀/压缩过程均如此——见式（2-17）。

2. **传热温差（热不平衡）**
    - 实际过程有热量交换时，热量总是从高温→低温。
    - 当一个热力系完成"吸热→返回放热"的往返：
        - 吸热时：外界温度 $T_A > T_{系}$
        - 放热时：外界温度 $T_B < T_{系}$
    - 一次往返后，系统恢复原状态，但给外界留下不能消除的影响：一部分热能不可逆地从高温→低温转移了。

二· 不可逆过程的定义与特性

**定义**：过程进行后，如果使热力系沿原路线反向进行并恢复原状态，==将会给外界留下不能消除的影响==——这样的过程称为不可逆过程。

**关键理解——"不可消除"的含义**：
- 不是说无法消除系统中某影响——借助外界帮助，可以消除任何影响。
- 但消除该影响的同时，给外界留下了新的（往往更大的）影响。
- → 要使系统和外界**最终都完全恢复初始状态**是不可能的。

**结论**：==一切实际的过程都是不可逆过程。==

三· 可逆过程的定义与条件

**定义**：过程进行后，如果使热力系沿原过程路线反向进行并恢复到原状态，==将不会给外界留下任何影响==。

**三个必要条件**（缺一不可）：
1. 热力系内部原来处于**平衡状态**（内平衡）
2. 作机械运动时热力系和外界保持**力平衡** → 运动无摩擦
3. 传热时热力系和外界保持**热平衡** → 传热无温差

> [!note] 一句话概括
> ==可逆过程 = 运动无摩擦 + 传热无温差 + 内平衡过程==

四· 可逆过程的本质理解

- 可逆过程**实际上不能进行**：没有温差不能传热；完全避免摩擦不能有机械运动。
- 可逆过程是**极限概念**：不可逆过程中不平衡因素无限趋小时的极限。
    - 无限小温差下传热
    - 摩擦无限微弱下作机械运动
- 周磊比喻：人在高温炎热的夏天感到闷热（无温差则无热交换→无散热感）；水平管道中如无克服阻力的压降则无法流动（无摩擦→无法驱动的矛盾）。

五· 可逆过程概念的重要意义

1. 给出了能量利用（转换和传递）的**理想情况和最高标准**。
2. 用理想情况的极限值衡量实际过程（循环）的完善程度。
3. 分析实际过程改善潜力的大小和改进方向。
4. **简化和方便了热工分析与计算**——这是工程上最重要的实用价值：先按可逆算理论值，再用经验系数修正。

六· 不可逆过程的工程处理

- 精确分析计算不可逆过程往往比较困难（系统内外都可能存在不同程度的力不平衡和热不平衡）。
- 工程做法：==先对假想的可逆过程进行分析计算，必要时用经验系数修正==。
- 很多实际过程比较接近可逆过程，因此这种近似方法在工程上可行且有效。

## 4-3 状态参数熵 —— 熵的导出证明

> [!tip] 补充
> 熵（Entropy）由克劳修斯于**1865年**提出，玻耳兹曼于**1877年**给出统计解释 $S = k \ln \Omega$（$\Omega$ = 微观状态数）。熵 = 系统无序度的量度。本章重点不在统计力学，而在**克劳修斯熵的热力学导出**——证明任意物质都存在状态参数"熵"。

一· 问题提出

- 第3章已证明理想气体存在状态参数熵（见式3-59~3-62）。
- 问题：==任意物质是否也存在状态参数熵？==
- 对任意气体，从可逆过程定义出发，$dS' = (dU' + p'dV')/T' = \delta Q'/T'$ 是成立的，但 $\oint \delta Q'/T'$ 是否等于零？这有待证明。

二· 严家騄证明法（本书采用的证明）

核心思路：借助一个**已证实存在熵的理想气体**作为"探针"，通过可逆过程的思想实验，证明任意气体也存在状态参数熵。

**装置设置**（思想实验）：

```
┌─────────────────────────────────────────────┐
│ 完全绝热气缸                                  │
│  ┌──────────┬──────────┐                      │
│  │ 理想气体 │ 任意气体  │                      │
│  │          │          │                      │
│  │ ←活塞   │ 完全导热 │  活塞→               │
│  │          │  刚性隔板 │                      │
│  └──────────┴──────────┘                      │
└─────────────────────────────────────────────┘
```

- 条件：气缸内壁完全绝热；活塞内壁完全绝热；活塞-气缸间无摩擦无泄漏；隔板完全导热且刚性。
- 初始状态：理想气体和任意气体各自平衡，且温度相同 $T = T'$。

**证明逻辑链（分步推导）**：

（1）对理想气体（已确证存在熵 $dS$）：
$$dS = \frac{dU + p dV}{T} = \frac{\delta Q}{T}$$
对任意气体，类似定义：
$$dS' = \frac{dU' + p' dV'}{T'} = \frac{\delta Q'}{T'}$$
← 基于可逆过程定义；$dS'$ 是否真的是状态参数的全微分 → 等价于证明 $\oint dS' = 0$。

（2）由完全绝热气缸 → 热量只能在理想气体和任意气体间传递：
$$\delta Q = -\delta Q'$$
← 两者热量时刻相等、符号相反。

（3）由完全导热隔板 + 过程足够缓慢 → 两者时刻处于热平衡：
$$T = T'$$
← 温度始终一致。

（4）设想任意气体**缓慢地进行任意一个循环**。对任意气体应用热力学第一定律：
$$\oint \delta Q' = \oint dU' + \oint \delta W'$$
← 热一：循环净热量 = 循环净功。

（5）热力学能 $U'$ 是任意气体的状态参数 → 循环积分为零：
$$\oint dU' = 0$$
代入（4）得：
$$\oint \delta Q' = \oint \delta W'$$
← 循环净热量 = 循环净功 [即式（2-23）的推广]。

（6）再看理想气体。任意气体进行一个循环时，理想气体相应进行一个过程。对理想气体应用热一：
$$\oint \delta Q = \Delta U + \oint \delta W$$
← 注意：理想气体的过程不一定是一个循环。

（7）任意气体完成循环后温度恢复原值 → 由 $T = T'$，理想气体温度也恢复原值。理想气体 $U$ 只是温度的函数 → $\Delta U = 0$。

代入（6）得：
$$\oint \delta Q = \oint \delta W$$
← 理想气体的净热量 = 净功。

（8）由（2）$\delta Q = -\delta Q'$，结合（5）和（7），可得：
$$\oint \delta W = -\oint \delta W'$$
← 两者的功相等、符号相反。

（9）对**整个装置**而言（理想气体+任意气体看作一个热力系）：
- 总传热量：$\oint \delta Q + \oint \delta Q' = 0$（热量内部抵消）
- 总做功量：$\oint \delta W + \oint \delta W' = 0$（功内部抵消）
- 任意气体完成了一个循环，回到原态。
- 理想气体：温度恢复（$U$ 恢复），但仅凭温度恢复**不足以**判断体积是否恢复。

（10）关键推理——利用可逆过程的特性：
整个装置中进行的是**可逆过程**（原来平衡、传热无温差、运动无摩擦）。如果任意气体循环结束后，只剩理想气体体积改变了（膨胀或压缩），即一个可逆过程进行后的唯一结果是气体"自发"膨胀/压缩 → 这违反可逆过程的特性。==因此理想气体体积也必须未变。==

（11）温度+体积（+质量）均未变 → 理想气体也完成了一个循环，回到了原状态。

（12）对理想气体应用熵是状态参数的已知结论：
$$\oint dS = 0$$
即：
$$\oint \frac{\delta Q}{T} = 0$$

（13）由（2）$\delta Q = -\delta Q'$ 和（3）$T = T'$：
$$\oint \frac{\delta Q'}{T'} = -\oint \frac{\delta Q}{T} = 0$$

（14）==结论==：$\oint dS' = \oint \frac{\delta Q'}{T'} = 0$，即 $dS'$ 的循环积分为零，说明 $dS'$ 是全微分——==任意气体也存在状态参数熵==。

（15）论证未涉及任意气体的任何具体特性 → 可推广：==任何物质都存在状态参数熵==。

（16）进一步推广：以上论证针对热和机械两个自由度的简单可压缩物质，但结论可推广到**任意自由度**的任何物质。

> [!note] 核心理解
> 物理上：这个证明的巧妙之处在于"借力"——用一个已知存在熵的系统（理想气体）去"检测"未知系统。通过可逆过程的对称性（无摩擦、无温差、返回无痕迹），迫使任意气体也必须与理想气体同步恢复原态。数学上：核心是证明 $\oint \delta Q/T = 0$（全微分条件），等价于证明热温商的循环积分为零。一旦证明，$dS = \delta Q_{rev}/T$ 就是一个全微分，$S$ 就是状态参数。

三· 玻耳兹曼熵公式（背景）

**定义**：$S = k \ln \Omega$
- $k$：玻耳兹曼常数（普朗克1900年引入）
- $\Omega$：宏观态对应的微观状态数

**物理意义**：熵是系统**微观无序度**的量度。$\Omega$ 越大 → 无序度越高 → $S$ 越大。自发过程总是向 $\Omega$ 增大的方向进行 → 数学上等价于 $S$ 增大。

> [!tip] 补充：熵概念的延展
> 周磊课件提到熵在多个领域的推广——
> - 熵与时间：热力学箭头（熵增方向）、心理学箭头（只能记住过去）、宇宙学箭头（宇宙膨胀），三者方向一致。
> - 熵与生命：薛定谔1943年提出"生命以负熵为生"——生物通过从环境吸收负熵来对抗自身的熵增。
> - 熵与信息：香农信息熵——信息 = 负熵，熵增 = 信息减少。
> - 熵与社会："碳达峰"和"碳中和"本质上是在减缓人类活动引起的熵增速率。

## 4-4 热力学第二定律的表达式 —— 熵方程

一· 熵流和熵产——两个核心概念

**定义**：
1. **熵流** $\delta S_f$：热力系与外界交换热量而导致的熵的流动量
   $$\delta S_f = \frac{\delta Q}{T}$$
   - $\delta S_f$ 可正（吸热）、可负（放热）、可为零（绝热）。
   - 纯粹由**传热**引起。

2. **熵产** $\delta S_g$：热力系内部不可逆因素（摩擦生热产 + 内部不等温传热）造成的熵的**产生量**
   $$\delta S_g = \frac{\delta Q_g}{T}$$
   - ==$\delta S_g \geq 0$==：可逆过程 $= 0$，不可逆过程 $> 0$。
   - ==熵产恒不为负==——这是第二定律的核心数学表达。

**熵方程基本形式**（对内部平衡的闭口系）：
$$dS = \frac{\delta Q}{T} + \frac{\delta Q_g}{T} = \delta S_f + \delta S_g \qquad \text{(4-1)}$$

（1）从熵的定义式出发：$dS = \frac{dU + p dV}{T}$
（2）将 $dU + pdV = \delta Q + \delta Q_g$ 代入（$\delta Q_g$ 是热产，即摩擦功转化的热量）
（3）得 $dS = \frac{\delta Q}{T} + \frac{\delta Q_g}{T} = \delta S_f + \delta S_g$

> [!note] 核心理解
> 物理上：熵的变化有两个来源——外界"注入"的熵流（可正可负），以及内部自发生成的熵产（只增不减）。数学上：$dS$ 是全微分（状态参数的变化），但 $\delta S_f$ 和 $\delta S_g$ 都是过程量（不是全微分），它们的和恰好等于全微分 $dS$。

二· 内部不平衡闭口系的熵方程

热力系内部温度不均匀时，内部也会发生**不等温传热**，产生的熵产 $\delta S_g^{Qi}$：

假设热力系由温度各自均匀的两部分 $A$（温度 $T_A$）和 $B$（温度 $T_B$）组成，$T_A > T_B$：

（1）$A$ 向 $B$ 传递 $\delta Q_i$ 的热量，总内部传热量代数和为零：
$$\delta Q_A + \delta Q_B = -\delta Q_i + \delta Q_i = 0$$

（2）内部熵流的代数和却**总大于零**：
$$\delta S_{fA} + \delta S_{fB} = \frac{-\delta Q_i}{T_A} + \frac{\delta Q_i}{T_B} = \delta Q_i\left(\frac{1}{T_B} - \frac{1}{T_A}\right) > 0$$
← 因为 $T_A > T_B$。

这就是内部不等温传热引起的熵产：
$$\delta S_g^{Qi} = \delta Q_i\left(\frac{1}{T_B} - \frac{1}{T_A}\right) > 0$$

推广到一般情况（积分形式）：
$$\delta S_f = \int_{(A)} \frac{\delta(\delta Q)}{T}$$
$$\delta S_g = \int_{(V)} \frac{\delta(\delta Q_i + \delta Q_g)}{T}$$
$$dS = \delta S_f + \delta S_g$$

三· 开口系的熵方程

考虑物质流入/流出带来的熵输运：
$$dS = \delta S_f + \delta S_g + s_1 \delta m_1 - s_2 \delta m_2 \qquad \text{(4-9)}$$
$$\Delta S = S_f + S_g + \int_{(\tau)} (s_1 \delta m_1 - s_2 \delta m_2) \qquad \text{(4-10)}$$

**文字表达**：（流入热力系的熵的总和）+（热力系的熵产）-（从热力系流出的熵的总和）=（热力系总熵的增量）

四· 孤立系熵增原理（全书最重要结论）

对**闭口绝热系**：$\delta S_f = 0$（无熵流），则
$$dS = \delta S_g \geq 0 \qquad \text{(4-13)}$$
$$\Delta S = S_g \geq 0 \qquad \text{(4-14)}$$

对**孤立系**（闭口 + 绝热 + 与外界无任何相互作用）：
$$dS_{iso} = \delta S_{g,iso} \geq 0 \qquad \text{(4-15)}$$
$$\Delta S_{iso} = S_{g,iso} \geq 0 \qquad \text{(4-16)}$$

**==孤立系的熵只会增加，不会减少！==** 不等号对不可逆过程，等号对可逆过程。

五· 稳定流动开口系

对稳定流动开口系（$dS = 0$，$\delta m_1 = \delta m_2 = \delta m$），取流过 **1kg** 流体的时段：
$$s_f + s_g + (s_1 - s_2) = 0$$
即：
$$s_2 - s_1 = s_f + s_g \qquad \text{(4-17)}$$

若过程**绝热**（$s_f = 0$）：
$$s_2 - s_1 = s_g \geq 0 \qquad \text{(4-18)}$$
- 不等号对应不可逆绝热稳定流动过程
- 等号对应可逆绝热（定熵）稳定流动过程
- ==绝热稳流系的出口熵不小于入口熵==

六· 熵方程的核心——熵产

熵流可有可无（只是配角），==熵产才是熵方程的主角==——熵产是热力学第二定律的实质内容。

**熵产两个来源**（周磊课件总结）：
1. **有限势差推动的输运过程**：热量传递（温差）、质量传递（浓度差）、动量传递（速度差）——势差越大，过程越不可逆。
2. **能量转换中的耗散效应**：热功转换中摩擦→热产→熵产；电光转换中焦耳效应→熵产；功电转换中发电机发热→熵产。

> [!note] 核心理解
> 物理上：熵产 = 不可逆性的定量度量。有熵产就有能量品质的下降（贬值）。熵本身守恒吗？不！能量守恒，但熵不守恒——熵会在能量转换和转移过程中**自·发·地·产·生**出来。数学上：$dS_{iso} = \delta S_g \geq 0$，孤立系的熵单调不减——这是热力学第二定律最简洁、最普适的数学表达，也是整个热力学的基石。

七· 易错点（课堂测验高频陷阱）

- ❌"任何过程，熵只增不减" → ✗（闭口系可以通过放热减少熵）
- ❌"可逆与不可逆路径到同一终点，不可逆的 $\Delta S$ > 可逆的 $\Delta S$" → ✗（$\Delta S$ 是状态量，只要初终态相同，$\Delta S$ 相同）
- ❌"不可逆循环 $\Delta S > 0$" → ✗（循环 $\Delta S = 0$，熵是状态参数）
- ❌"不可逆过程 $\Delta S$ 永远大于可逆过程 $\Delta S$" → ✗（取决于初终态是否相同）

八· 教材例题核心数据

**【例4-1】** 电加热 20kg 水 20°C→80°C，再与 40kg 20°C 凉水混合：
- 混合后温度：**40°C**（313.15K）
- 电加热熵产：**15.593 kJ/K**
- 混合熵产：**0.987 kJ/K**
- 总熵产：**16.580 kJ/K**

**【例4-2】** 热空气-水换热器：
- 空气流量：**3819 kg/h**
- 总熵产：**435.5 kJ/(K·h)**

## 4-5 热力学第二定律各种表述的等效性

一· 三种经典表述

1. **克劳修斯表述（1850）**："不可能将热量由低温物体传送到高温物体而不引起其他变化。"
2. **开尔文-普朗克表述（1851）**："不可能制造出从单一热源吸热而使之全部转变为功的循环发动机。" → 即**第二类永动机不可能制成**。
3. **孤立系熵增原理**："自然界中的一切实际过程总是自发地、不可逆地朝着使孤立系熵增加的方向进行。"

二· 等效性证明——从熵增原理出发

**证明1 → 克劳修斯表述**：

假设存在不等温传热而不产生其他变化的制冷机（热量 $Q$ 从 $T_2$ 传向 $T_1$，$T_1 > T_2$）：

$$\Delta S_{iso} = \Delta S_H + \Delta S_L + \Delta S_{机} = \frac{Q}{T_1} + \frac{-Q}{T_2} + 0 = Q\left(\frac{1}{T_1} - \frac{1}{T_2}\right) < 0$$

← 因为 $T_1 > T_2$，与孤立系熵增原理（$\Delta S_{iso} \geq 0$）矛盾。→ 克劳修斯表述成立。

**证明2 → 开尔文-普朗克表述**：

假设存在从单一热源 $T_0$ 吸热 $Q_0$ 全部变为功 $W_0$ 的循环发动机（$Q_0 = W_0$）：

$$\Delta S_{iso} = \Delta S_{热源} + \Delta S_{热机} = \frac{-Q_0}{T_0} + 0 < 0$$

与孤立系熵增原理矛盾。→ 开尔文-普朗克表述成立。

**三种表述的逻辑关系**：任一种表述成立 → 另两种也必然成立。孤立系熵增原理是最概括、最便于应用的表述——因为只要将参与过程的全部物体纳入孤立系，就可以直接应用。

## 4-6 卡诺定理和卡诺循环

一· 卡诺定理

**内容**：工作在两个恒温热源（$T_1$、$T_2$）之间的循环，不管采用什么工质：
- ==可逆循环：热效率均为 $\eta_t = 1 - T_2/T_1$==
- ==不可逆循环：热效率恒小于 $1 - T_2/T_1$==

**利用孤立系熵增原理证明**：

（1）设热机从热源 $T_1$ 吸热 $Q_1$，向冷源 $T_2$ 放热 $Q_2$，输出功 $W_0 = Q_1 - Q_2$。

（2）热效率：
$$\eta_t = \frac{W_0}{Q_1} = 1 - \frac{Q_2}{Q_1} \qquad \text{(4-20)}$$

（3）一个循环后工质回原态，孤立系（热源+冷源+热机）熵变：
$$\Delta S_{iso} = \frac{-Q_1}{T_1} + \frac{Q_2}{T_2} + 0 \geq 0$$
→
$$\frac{Q_2}{T_2} \geq \frac{Q_1}{T_1} \quad \Rightarrow \quad \frac{Q_2}{Q_1} \geq \frac{T_2}{T_1}$$

（4）代入热效率：
$$\eta_t = 1 - \frac{Q_2}{Q_1} \leq 1 - \frac{T_2}{T_1} \qquad \text{(4-22)}$$

- 等号对可逆循环；不等号对不可逆循环。卡诺定理得证。

二· 卡诺循环

**（简单）卡诺循环**由四个可逆过程组成：

```
      T ↑
        │    a ______ b      ← 可逆定温吸热（T = T₁）
    T₁  │     ╱      ╲
        │    ╱        ╲     ← 可逆绝热（定熵）膨胀 b→c
        │   ╱          ╲
    T₂  │  d ————————— c    ← 可逆定温放热（T = T₂）
        │  ↑              ↑
        │  可逆绝热压缩   可逆绝热膨胀
        └────────────────────→ s
```

- 过程 a→b：**可逆定温吸热**（膨胀），$T = T_1$
- 过程 b→c：**可逆绝热膨胀**（定熵），$T_1 \to T_2$
- 过程 c→d：**可逆定温放热**（压缩），$T = T_2$
- 过程 d→a：**可逆绝热压缩**（定熵），$T_2 \to T_1$

卡诺循环热效率：
$$\eta_{t,C} = \frac{W_{0C}}{Q_{1C}} = 1 - \frac{Q_{2C}}{Q_{1C}} = 1 - \frac{T_2(S_b - S_a)}{T_1(S_b - S_a)} = 1 - \frac{T_2}{T_1}$$

**回热卡诺循环**（在温熵图中两绝热过程平行）：
$$\eta'_{t,C} = 1 - \frac{T_2}{T_1}$$

三· 任意内平衡循环 vs 卡诺循环

对任意内平衡循环 abcda：
- 平均吸热温度：$T_{m1} = Q_1/\Delta S < T_1$
- 平均放热温度：$T_{m2} = Q_2/\Delta S > T_2$
- 热效率：
$$\eta_t = 1 - \frac{T_{m2}}{T_{m1}} < 1 - \frac{T_2}{T_1} = \eta_{t,C}$$

==工作在 $T_{m1}$、$T_{m2}$ 之间的卡诺循环 ABCDA 称为循环 abcda 的**等效卡诺循环**。==

四· 卡诺定理对热机的原则指导

1. ==不能期望热机的热效率达到 **100%**==。$\eta_{t,C} = 100\%$ 要求 $T_2 = 0\text{K}$ 或 $T_1 = \infty$——绝对零度达不到，无限高温不可能。
2. ==在温度范围 $T_1$～$T_2$ 之间，任何实际热机的热效率都小于 $1 - T_2/T_1$==。最高热效率只能接近，实际达不到。
3. ==不能靠单一热源供热使热机循环工作==（$T_1 = T_2$ 时 $\eta_{t,C} = 0$）。
4. ==提高热机热效率的根本途径：提高平均吸热温度 + 降低平均放热温度。==

> [!note] 核心理解
> 物理上：卡诺效率 $1 - T_2/T_1$ 揭示了"温度差"才是热功转换的驱动力。$T_2/T_1$ 越小→效率越高。冷源温度不可能低于环境温度（除非有额外的制冷耗功），热源温度受材料限制——这就是热机效率存在理论上限的物理根源。数学上：卡诺效率来自卡诺循环在 T-s 图上的矩形面积比——吸热面积 $T_1\Delta S$，放热面积 $T_2\Delta S$，净功面积 $(T_1 - T_2)\Delta S$，效率 = 净功/吸热。

## 4-7 克劳修斯积分式

一· 数学表达式

$$\oint \frac{\delta Q}{T'} \leq 0 \qquad \text{(4-26)}$$

- $T'$：外界温度（不是系统温度！）
- 等号对应**可逆循环**；不等号对应**不可逆循环**。

**文字描述**：任何闭口热力系，进行一个循环后，它和外界交换的微元热量与参与该微元换热过程时外界温度的比值（商）的循环积分，不可能大于零——只能小于零（不可逆循环），或等于零（可逆循环）。

二· 从熵方程推导克劳修斯积分式

（1）对闭口系：$dS = \delta S_f + \delta S_g$

（2）熵产：$\delta S_g = \int_{(V)} \frac{\delta(\delta Q_i + \delta Q_g)}{T} \geq 0$

（3）熵流：$\delta S_f = \int_{(A)} \frac{\delta(\delta Q)}{T} \geq \int_{(A)} \frac{\delta(\delta Q)}{T'}$
← 吸热时 $T' \geq T$（外界温度>系统温度），放热时 $T' \leq T$，无论哪种情况，不等式均成立。

（4）代入得：
$$dS \geq \frac{\delta Q}{T'}$$
对过程积分：
$$S_2 - S_1 \geq \int_1^2 \frac{\delta Q}{T'} \qquad \text{(4-27)}$$

（5）对循环（$S_2 = S_1$）：
$$\oint \frac{\delta Q}{T'} \leq 0$$
等号对可逆循环，不等号对不可逆循环。得证。

三· 物理意义——熵 vs 热温商

- **熵**（$S$）：==状态量==，工质的状态参数之一。
- **热温商**（$\int \delta Q/T'$）：==过程量==，是一个数学表达式。

两者的关系：
- 过程**可逆**：熵变 = 热温商 → $S_2 - S_1 = \int_1^2 \delta Q/T'$
- 过程**不可逆**：熵变 > 热温商 → $S_2 - S_1 > \int_1^2 \delta Q/T'$

**应用**：
1. 判断闭口系中进行的循环或过程是否可逆、是否可能。
2. 利用热温商计算不可逆过程的熵变（因为熵是状态量，不可逆过程的熵变 = 可逆过程的熵变 = 热温商积分）。

> [!warning] 注意
> 热温商里用的温度是**外界（环境）温度 $T'$**，不是系统温度 $T$。这是克劳修斯积分式和熵定义式的关键区别。

四· 例题核心数据

**【例4-3】** 三台热机（$T_1 = 1000\text{K}$，$T_2 = 300\text{K}$），每从热源吸热 100kJ：
- A 热机放热 50kJ → $\oint \delta Q/T = -0.0667 \text{ kJ/K} < 0$ → 不可逆，可实现。
- B 热机放热 30kJ → $\oint \delta Q/T = 0$ → 可逆，理论上可实现。
- C 热机放热 20kJ → $\oint \delta Q/T > 0$ → 违反第二定律，不可能实现。

## 4-8 热量的可用能及其不可逆损失

一· 能量品质与可用性

- **机械能、电能**：具有完全的可用性，==全部是可用能==。
- **热能**：不具有完全的可用性，即使通过可逆循环也只能==部分转变==为机械能。
- 环境（大气、海水）：温度基本恒定 $T_0$，蕴含巨大热能的"热库"，但这些温度一致的热能==无法用来做功==，全是**废热**。
- 只有当热能温度**高于（或低于）环境温度 $T_0$** 时，其中才包含可用能。

二· 热量㶲（热量可用能）

**定义**：热量 $Q$ 中理论上可通过可逆循环转变为功的最大部分，称为热量㶲 $E_{x,Q}$。

**公式及推导**：

（1）供热源在温度范围 $T_a \sim T_b$ 间提供热量 $Q$（在 T-s 图上为面积 abcda）。

（2）通过可逆循环（如卡诺循环 12341），最大可转变的功：
$$W_{max} = Q - T_0(S_a - S_b) = \int_b^a \delta Q - T_0 \int_b^a \frac{\delta Q}{T}$$

（3）引入卡诺因子 $(1 - T_0/T)$：
$$E_{x,Q} = W_{max} = \int_b^a \left(1 - \frac{T_0}{T}\right) \delta Q \qquad \text{(4-29)}$$

（4）不能转变的废热：$Q - W_{max} = T_0(S_a - S_b)$ → 排给大气。

**关键点**：$(1 - T_0/T)$ 是卡诺效率的形式——热量中可用能的相对含量取决于供热温度 $T$ 与环境温度 $T_0$ 的比值。==温度越高，可用能比例越高。==

三· 不可逆损失 = 㶲损

任何不可逆因素（摩擦、温差传热）都会使实际做功量 < $W_{max}$，减少的部分即**㶲损** $E_L$。

普遍关系式：
$$E_L = W_{max} - W_{实际}$$

**证明 $E_L = T_0 S_g$**：

（1）取孤立系（供热源+热机+环境），热机工质完成循环后熵不变。

（2）孤立系熵增：
$$\Delta S_{iso} = \Delta S_{源} + \Delta S_{机} + \Delta S_{amb} = -\Delta S_{源} + 0 + \frac{Q + E_L}{T_0} = S_g$$

（3）由此：
$$E_L = T_0 \Delta S_{iso} = T_0 S_g \qquad \text{(4-30)}$$

==任何不可逆因素造成的可用能损失 = 孤立系熵增（熵产）× 环境温度。==

四· 各环节的不可逆损失

```
循环中的各不可逆损失叠加：
  E_L = E_{L1}（吸热温差）+ E_{L2}（膨胀摩擦）+ E_{L3}（放热温差）+ E_{L4}（压缩摩擦）
实际可用功 = W_max - ΣE_L
废热 = Q - W_max + ΣE_L
```

> [!note] 核心理解
> 物理上：能量的"数量"守恒，但"品质"贬值。机械能→热能→环境热，是一路"降级"的过程。㶲损 $E_L$ 衡量的是能量降级的程度——原本能做的功，因不可逆因素变成了废热。数学上：$E_L = T_0 S_g$ 建立了"功损"和"熵产"之间的桥梁——熵产是原因，功损是结果，$T_0$ 是换算系数。

## 4-9 流动工质的㶲和㶲损

一· 流动工质的㶲（$e_x$）

**定义**：在环境参数 $T_0$、$p_0$ 下，流动工质从给定状态可逆地变化到与环境平衡的状态时，对外做出的**最大技术功**。

**公式及推导**（从状态 A 经可逆绝热→B，再经可逆定温→0）：

（1）路径：$A \xrightarrow{\text{可逆绝热}} B \xrightarrow{\text{可逆定温 } T_0} 0$

（2）最大技术功：
$$w_{t,max} = w_{t,AB} + w_{t,B0}$$

（3）展开：
$$= [q_{AB} - (h_B - h_A)] + [q_{B0} - (h_0 - h_B)]$$

（4）$q_{AB} = 0$（绝热），$q_{B0} = T_0(s_0 - s_B)$（定温可逆）：
$$= [0 - (h_B - h)] + [T_0(s_0 - s) - (h_0 - h_B)]$$

（5）化简得比㶲（$e_x$）：
$$e_x = (h - h_0) - T_0(s - s_0) \qquad \text{(4-31)}$$
单位为 **kJ/kg**。

对任意质量工质：
$$E_x = (H - H_0) - T_0(S - S_0) \qquad \text{(4-32)}$$
单位为 **kJ**。

**性质**：
- ==$e_x$ 是状态参数==（$h_0$、$s_0$ 仅取决于 $T_0$、$p_0$，环境确定后为定值）。
- 表示单位质量流动工质在给定状态下具有的做功能力。
- 做功能力的条件：大气是唯一热源，从给定状态**可逆**变化到与环境参数相同时全部发挥出来。

二· 㶲降和㶲损

流动工质从状态 1→状态 2 的㶲降，理论上等于对外界做出的技术功：
$$W_{t,the} = E_{x1} - E_{x2} = (H_1 - H_2) - T_0(S_1 - S_2) \qquad \text{(4-35)}$$

实际技术功总是小于㶲降（不可逆性），差值即**㶲损**：
$$E_L = W_{t,the} - W_t = (H_1 - H_2) - T_0(S_1 - S_2) - W_t \qquad \text{(4-36)}$$

**推导出 $E_L = T_0 S_g$**：

（1）根据热一律：$H_2 - H_1 + W_t = Q$

（2）大气是唯一热源：$Q_{大气} = -Q$

（3）代入：$E_L = -Q + T_0(S_2 - S_1) = Q_{大气} + T_0 \Delta S_{工质} = T_0 (\Delta S_{大气} + \Delta S_{工质})$

（4）==$E_L = T_0 \Delta S_{iso} = T_0 S_g$== ——与式（4-30）完全相同！

三· 㶲在 h-s 图中的表示

在焓熵图(h-s图)中，$e_x$ 可表示为任意状态点到"环境直线"的**垂直距离**。

环境直线：过环境状态点 0（$h_0$, $s_0$, $p_0$）的 $p_0$ 定压线的切线。
- 斜率：$\tan \alpha = (\partial h/\partial s)_{p_0} = T_0$（由 $Tds = dh - vdp$ 得）。

> [!note] 核心理解
> 物理上：流动工质的㶲 = 焓的可用部分 - 必须付出的"熵税"。$(h - h_0)$ 是工质相对于环境的能量优势，$T_0(s - s_0)$ 是这部分能量中因与环境熵差而"注定"要排给环境的废热。数学上：$e_x$ 的形式与 Gibbs 自由能（$g = h - Ts$）相似，只是参考态取环境状态。区别在于自由能 $g$ 用于判断化学反应方向（封闭系），$e_x$ 用于计算做功能力（开口系）。

四· 教材例题核心数据

**【例4-4】** 压缩空气管道输送（1.2MPa, 320K → 0.8MPa, 298K），流量 **0.5 kg/s**：
- 可用能损失 $E_L$：**63,451 kJ/h**

## 4-10 工质的㶲和㶲损

一· 工质㶲（闭口系㶲，也称热力学能㶲）

**定义**：对一定的环境而言，工质在某一状态下所具有的热力学能中理论上可以转化为可用能的部分——即在从该状态可逆过渡到与环境参数相同的过程中对外做出的**最大可用功**。

**公式及推导**（从初始状态 $(T, p)$ 经可逆绝热→$(T_0, p')$，再经可逆定温→$(T_0, p_0)$）：

（1）与流动工质㶲类似，但闭口系考虑的是**可用功**（膨胀功 - 排开大气的无用功 $p_0(V_0 - V)$）：
$$W_{av,max} = W_{AB} + W_{B0} - p_0(V_0 - V)$$

（2）展开：
$$= [Q_{AB} - (U_B - U_A)] + [Q_{B0} - (U_0 - U_B)] - p_0(V_0 - V)$$

（3）$Q_{AB} = 0$，$Q_{B0} = T_0(S_0 - S)$：
$$= [0 - (U_B - U)] + [T_0(S_0 - S) - (U_0 - U_B)] - p_0(V_0 - V)$$

（4）化简：
$$E_{x,U} = (U - U_0) - T_0(S - S_0) + p_0(V - V_0) \qquad \text{(4-38)}$$

对单位质量工质：
$$e_{x,U} = (u - u_0) - T_0(s - s_0) + p_0(v - v_0) \qquad \text{(4-39)}$$

**性质**：
- ==$E_{x,U}$ 是状态量==。
- 包含三项：热力学能差 - 环境温度×熵差 + 环境压力×容积差。
- $p_0(V - V_0)$ 项：闭口系膨胀时必须**推开大气**做功 → 这部分功无法利用，被排除。

二· 工质㶲降与㶲损

工质从状态 1→2 的㶲降，理论上应等于对外做出的可用功：
$$W_{av,the} = E_{x,U1} - E_{x,U2} = (U_1 - U_2) - T_0(S_1 - S_2) + p_0(V_1 - V_2) \qquad \text{(4-40)}$$

实际可用功 < 㶲降，差值即㶲损：
$$E_L = T_0 \Delta S_{iso} = T_0 S_g \qquad \text{(4-41)}$$
——与前述式（4-30）、（4-37）完全一致！

三· 教材例题核心数据

**【例4-5】** 压缩空气汽车（0.2 m³，表压 1.5MPa，环境 0.1MPa, 20°C）：
- 比㶲：$e_x = 338.67 \text{ kJ/kg}$
- 空气质量：**35.88 kg**
- 总㶲：**12,151 kJ**
- 4PS功率下最多行驶：**1h 9min**（理论值）

> [!warning] 实际行驶时间会显著减少——各种不可逆损失使可用能最终都变成废热排给环境。

## 4-11 关于㶲损的讨论及㶲方程

一· 㶲损的普遍表达式

三种情况（热量㶲、流动工质㶲、工质㶲）的㶲损计算公式**完全相同**：
$$E_L = T_0 \Delta S_{iso} = T_0 S_g$$

==这个公式是普遍成立的——任何㶲损（可用能的不可逆损失）都等于孤立系熵增（熵产）与环境温度的乘积。==

二· 㶲损 ≠ 功损

重要区分：
- **功损** $W_l = Q_g = \int T dS_g$（摩擦功转化为热产，热产的温度可能高于环境温度）
- **㶲损** $E_L = T_0 S_g$（热产中还能做功的部分被"扣除"）

当热产温度 $T_{avg} > T_0$ 时，$E_L < W_l$（部分热产仍有做功能力）。
当 $T_{avg} = T_0$ 时，$E_L = W_l$（全部是废热）。

→ 能量数量上守恒，"能量损失"实质上是**能量质量上的损失**——可用能 → 废热。

三· 㶲方程（可用能平衡方程）

**文字表达**：
（输入热力系的可用能的总和）-（热力系输出的可用能的总和）-（热力系内部可用能的不可逆损失）=（热力系的可用能的增量）

**一般表达式**（对时间积分）：
$$\begin{aligned} E_{x,Q} = \Delta E_{x,U} + \Delta E_k + \Delta E_p &+ \int_{(\tau)}\Big[\big(e_{x,j} + \frac{c_j^2}{2} + gz_j\big)q_{m,j} \\ &- \big(e_{x,i} + \frac{c_i^2}{2} + gz_i\big)q_{m,i}\Big] d\tau + W_{av} + E_L \qquad \text{(4-43)} \end{aligned}$$

**简化形式（稳定流动，1kg流体）**：
$$e_{x,Q} = (e_{x2} - e_{x1}) + \frac{c_2^2 - c_1^2}{2} + g(z_2 - z_1) + w_{sh} + e_L \qquad \text{(4-44)}$$

**几种特殊情况的简化**：
- 涡轮机械（绝热，忽略动能位能变化）：$w_{sh} = (e_{x1} - e_{x2}) - e_L$ → 代入展开后还原为 $w_{sh} = h_1 - h_2$。
- 换热器（无轴功）：$e_{x,Q} = (e_{x2} - e_{x1}) + e_L$ → 展开后还原为 $q = h_2 - h_1$。

四· 三方程的相容性

能量方程、熵方程、㶲方程虽然表达形式不同，但只要是正确的，就必定==相通、相容==。它们共同确定热力过程中各状态量和过程量之间的变化关系。

## 4-12 热力学第二定律对工程实践的指导意义

一· 对热机的理论指导

（详见4-6节末四条原则）：
- 热效率不能达100%
- 任何实际热机效率 < 卡诺效率
- 至少需要两个热源
- 提高效率的根本途径：==提高平均吸热温度 + 降低平均放热温度==

二· 预测过程方向，判断平衡状态

- 简单过程：直接判断（高温→低温传热，进行到温度相等为止 → 热平衡）。
- 复杂过程（如化学反应）：==通过计算孤立系的熵变来判断== —— $\Delta S_{iso} > 0$ 则过程自发进行；$\Delta S_{iso} = 0$ 则达平衡。
- （补：亥姆霍兹自由能判据——定温定容系；吉布斯自由焓判据——定温定压系。）

三· 指导节约能源

核心原则：==减少一切实际过程的不可逆损失==。

具体策略：
- **能量梯次利用**：高温热能发电 → 中温热能供工业 → 低温热能供暖。
- **多能互补优化**：将需要冷却的设备放出的热量，在需要加热的设备中利用。
- 错误做法：用电炉取暖（功→热，最大的能量品质浪费）；用高温热能直接供暖。
- 正确做法：利用低温热能（地热、热机排气余热、工业余热）供暖。

四· 避免违背热力学第二定律

- 勿重复制造永动机的尝试（第二类永动机违反第二定律）。
- 复杂方案要透过现象看本质，用熵增原理做最终判别。

> [!note] 核心理解
> 物理上：第二定律不仅是理论工具，更是工程实践的"指南针"——告诉你哪些过程可以发生、能量利用的上限在哪里、如何减少浪费。数学上：每次实际过程的 $\Delta S_{iso} > 0$ 都意味着可用能的永久损失——这部分损失无法通过任何手段回收。节能的本质 = 减少不必要的熵产。

## 速记卡 / 一句话总结

- ==孤立系熵增原理==：$dS_{iso} \geq 0$，不可逆 >0，可逆 =0——第二定律最简洁数学表达。
- ==熵方程==：$dS = \delta S_f + \delta S_g$（熵变 = 熵流 + 熵产），熵流可正可负，熵产恒≥0。
- ==熵产是第二定律的实质==：有熵产就有不可逆损失（能量品质下降）。
- ==可逆过程三条件==：内平衡 + 运动无摩擦（力平衡）+ 传热无温差（热平衡）。
- ==卡诺效率==：$\eta_{t,C} = 1 - T_2/T_1$（仅取决于热源/冷源温度），所有可逆热机效率相等。
- ==卡诺循环组成==：可逆定温吸热 → 可逆绝热膨胀 → 可逆定温放热 → 可逆绝热压缩。
- ==克劳修斯积分式==：$\oint \delta Q/T' \leq 0$（等号可逆，不等号不可逆），$T'$ 是**外界温度**。
- ==㶲（可用能）==：能量中可转变为功的最大部分。$e_x = (h-h_0) - T_0(s-s_0)$（流动工质）；$E_{x,U} = (U-U_0) - T_0(S-S_0) + p_0(V-V_0)$（闭口工质）。
- ==㶲损普遍式==：$E_L = T_0 \Delta S_{iso} = T_0 S_g$——任何不可逆可用能损失 = 熵产 × 环境温度。
- ==提高热效率的根本途径==：提高平均吸热温度 + 降低平均放热温度（能量梯次利用）。
