## §5-1 一元稳定流动的基本方程

### 前置概念

**定义**：
- **一元流动**：流动的一切参数仅沿**一个方向**（直线或弯曲流道的轴线）有显著变化，在其他两个方向上变化极小。
- **稳定流动**：流道中任意指定空间的一切参数**不随时间而变**。

> [!note] 核心理解
> 物理上：一元+稳定 = 把3D瞬态问题降维成1D稳态——这是工程分析的经典简化策略。只要管道截面变化不太剧烈、流动不太分离，此假设基本成立。数学上：偏微分 → 常微分，参数只随流向坐标 x 变化。

---

### 一· 六大基本方程

#### 1. 连续方程（质量守恒）

**定义与推导**：
- 体积流量 $q_V = q_m v = Ac$
- 故 $q_m = \frac{Ac}{v}$

（1）对截面 1：$q_{m1} = \frac{A_1 c_1}{v_1} = \text{常数}$
（2）对截面 2：$q_{m2} = \frac{A_2 c_2}{v_2} = \text{常数}$
（3）对任意截面 i：$q_{mi} = \frac{A_i c_i}{v_i} = \text{常数}$

→ 稳定流动质量守恒：

$$q_{m1} = q_{m2} = \cdots = q_m = \frac{Ac}{v} = \text{常数} \qquad (5-1)$$

**性质 / 条件**：
- 适用于**任何**一元稳定流动
- 不管什么流体（理想/实际气体、液体）
- 不管过程可逆或不可逆
- ⚠️ ==质量流量是不变的常数，但容积流量 $q_V$ 不是常数==（因为 v 沿流向变化）

---

#### 2. 能量方程（能量守恒）

**定义**：稳定流动能量方程的一般形式（第2章已得）：

$$q = h_2 - h_1 + \frac{1}{2}(c_2^2 - c_1^2) + g(z_2 - z_1) + w_{sh}$$

**喷管/扩压管的简化条件**：
- （1）$w_{sh} = 0$（无轴功）
- （2）$q \approx 0$（气体和外界基本上绝热——流速极快，来不及换热）
- （3）$g(z_2 - z_1) \approx 0$（重力位能变化可忽略）

→ 简化形式：

$$\frac{1}{2}(c_2^2 - c_1^2) = h_1 - h_2 \qquad (5-2)$$

或写成：$h_1 + \frac{c_1^2}{2} = h_2 + \frac{c_2^2}{2} = \text{常数}$

> [!note] 核心理解
> 物理上：绝能（绝热 + 绝功）过程中，==工质的焓 + 动能 = 不变常数==——焓降全部转为动能增量。喷管加速的物理本质就是焓→动能的转换。数学上：这是 Bernoulli 方程的热力学推广（不是机械能守恒，而是总能量守恒）。

**性质 / 条件**：
- 适用于任何工质的绝热稳定流动
- 不管可逆或不可逆
- 是**流速计算的基本公式**

---

#### 3. 动量方程（牛顿第二定律）

**定义与推导**：取微元柱体（截面积 A，长度 dx），作用力分析：

（1）压力合力：$pA - (p+dp)A = -A\,dp$ ← 净压力
（2）粘性阻力：$dF_f$ ← 柱体侧面摩擦力
（3）冲量 = 动量变化：$[pA - (p+dp)A - dF_f]d\tau = dm\,dc$ ← 牛顿第二定律
（4）$dm = \frac{A\,dx}{v}$ ← 质量 = 体积 / 比体积
（5）代入整理得：$\frac{1}{2}dc^2 = -v\,dp - v\frac{dF_f}{A} = -v\,dp - \delta w_L$

**无摩擦简化**（$dF_f = 0$）：

$$\frac{1}{2}dc^2 = -v\,dp \qquad (5-4)$$

积分形式：

$$\frac{1}{2}(c_2^2 - c_1^2) = -\int_1^2 v\,dp \qquad (5-5)$$

> [!note] 核心理解
> 物理上：式(5-4)表明 ==流速和压力呈反向变化==——气体流速越快，压力越低。飞机几十吨能飞起来靠的就是这个原理：机翼上方流速快→压力低，下方流速慢→压力高→产生升力。数学上：−∫ v dp 正是技术功——无摩擦时，气体膨胀获得的动能恰好等于气体膨胀作出的技术功。

---

#### 4. 状态方程

**定义**：$F(p, v, T) = 0$

- 实际气体：p、v、T 函数关系复杂，常利用现成图表
- 理想气体：$pv = R_g T \qquad (5-6)$（最简单形式）

---

#### 5. 过程方程（定熵过程）

**定义**：本章只讨论绝热流动。不考虑摩擦 → 定熵流动。

$$pv^\kappa = \text{const} \qquad (5-7)$$

- $\kappa$ 称为**定熵指数**（绝热指数）
- 对定比热容理想气体：$\kappa = \gamma_0$（比热容比）

---

#### 6. 声速方程

**声速定义**：微小扰动在连续介质中产生的压力波的传播速度。

- 扰动很小 → 内摩擦很小 → 可逆
- 传播很快 → 来不及散热 → 绝热
- → ==声音传播是定熵过程==

**推导**：

（1）声速基本式：$c_s = \sqrt{\frac{K}{\rho}}$，其中 $K = \frac{dp}{d\rho/\rho}$ 为体积弹性模量
（2）对气体，因定熵：$c_s = \sqrt{\left(\frac{\partial p}{\partial \rho}\right)_s}$
（3）由 $pv^\kappa = \text{const}$：$\left(\frac{\partial p}{\partial v}\right)_s = -\kappa \frac{p}{v}$
（4）$\rho = 1/v$，$d\rho = -dv/v^2$，代入：

$$c_s = \sqrt{\kappa p v} \qquad (5-8)$$

（5）对理想气体（$pv = R_g T$）：

$$c_s = \sqrt{\kappa R_g T} \qquad (5-9)$$

**关键结论**：
- ==声音在理想气体中的传播速度与绝对温度的平方根成正比==
- 温度越高，声速越大
- 例：空气 **300 K**, $\kappa$=**1.4** → $c_s \approx$ **347 m/s**；**650 K**, $\kappa$=**1.37** → $c_s \approx$ **506 m/s**

> [!note] 核心理解
> 物理上：声速是"信息"在介质中传播的速度——压力扰动靠分子碰撞传递，温度越高 → 分子运动越快 → 信号传得越快。数学上：cₛ = √(κ R_g T) 表明声速是状态的函数，不是常数——这就是"当地声速"概念的必要性：喷管中 T 沿流向下降，声速也沿流向下降。

**易错**：
- 声速**不是固定数值**——同一介质中温度不同、声速不同
- "当地声速"：气体实际所处状态下的声速

---

### 二· 本节的逻辑骨架

```
连续方程 (质量守恒)
    │
能量方程 (总能量守恒) ──→ 焓降→动能
    │
动量方程 (牛顿定律)  ──→ p↓ ↔ c↑ 反向变化
    │
状态方程 (pv=R_gT)
    │
过程方程 (pv^κ=const，定熵)
    │
声速方程 (c_s = √κR_gT)
```

后面各节的流动讨论，==本质上就是这六个方程在具体条件下的联立应用==。

## §5-2 喷管中气流参数变化和喷管截面变化的关系

### 一· 喷管的定义与特点

**定义**：喷管是利用**压力降落**使流体**加速**的管道。

- 流速极高（每秒几百米），喷管长度有限（几厘米到几十厘米）
- → 气流停留时间极短 → 和外界换热极少 → **可认为绝热**

> [!note] 核心理解
> 物理上：喷管 = "压力势能 → 动能"的转换器。要控制气流按规律加速，就必须设计出相应形状的喷管——截面变化规律决定了流动变化规律。

---

### 二· 截面变化与流速变化的关系（核心推导）

**出发点**：连续方程 $q_m = \frac{Ac}{v} = \text{const}$

（1）取对数：$\ln A + \ln c - \ln v = \ln q_m = \text{const}$
（2）微分：$\frac{dA}{A} + \frac{dc}{c} - \frac{dv}{v} = 0$

$$\frac{dA}{A} = \frac{dv}{v} - \frac{dc}{c} \qquad (5-10)$$

> 喷管截面的增加率 = 气体比体积的增加率 − 流速的增加率

**问题是**：喷管中 dv/v 和 dc/c 都是正的（压力下降 → v↑、c↑），但 dv/v 什么时候小于 dc/c、什么时候大于 dc/c？这决定了 dA 的正负。

---

### 三· 引入马赫数 Ma——解开 dv/v 和 dc/c 的关系

**定义**：

$$\text{Ma} = \frac{c}{c_s} \qquad (5-11)$$

马赫数 = 流速与当地声速之比。

**推导 dv/v 与 dc/c 的关系**（无摩擦流动）：

（1）动量方程：$c\,dc = -v\,dp$ ← (5-4)
（2）过程方程 $pv^\kappa = \text{const}$ 微分：$\kappa p\,dv + v\,dp = 0$，即 $dp = -\kappa p\frac{dv}{v}$
（3）(2) 代入 (1)：$c\,dc = -v(-\kappa p\frac{dv}{v}) = \kappa p\,dv$
→ $cdc = \kappa p\,dv$
（4）除以 $c^2$，利用 $c_s^2 = \kappa p v$：

$$\frac{dv}{v} = \frac{c^2}{\kappa p v}\frac{dc}{c} = \frac{c^2}{c_s^2}\frac{dc}{c} = \text{Ma}^2 \frac{dc}{c} \qquad (5-12)$$

（5）将 (5-12) 代入 (5-10)：

$$\frac{dA}{A} = (\text{Ma}^2 - 1)\frac{dc}{c} \qquad (5-13)$$

> [!note] 核心理解 — **这是本章最重要的公式**
> 式(5-13) 一举统一了亚声速和超声速的行为差异，物理本质在于：
> - Ma < 1 时（亚声速）：比体积增长率 < 流速增长率 → 气体"变密"跟不上"变快" → 要加速必须**收窄**管道
> - Ma > 1 时（超声速）：比体积增长率 > 流速增长率 → 气体膨胀得比加速更快 → 要加速必须**放宽**管道
> - Ma = 1 时（声速）：dA/A = 0，截面取极值（最小截面 = **喉部**）

---

### 四· Ma 对喷管形状的决定

| Ma 范围 | dc > 0（加速） | dA 符号 | 喷管形状 |
|:---|:---|:---|:---|
| Ma < 1（亚声速） | dc/c 为正 | dA/A < 0 | ==渐缩喷管== |
| Ma = 1（声速） | dc/c 为正 | dA/A = 0 | 最小截面（喉部） |
| Ma > 1（超声速） | dc/c 为正 | dA/A > 0 | ==渐放喷管== |

**如果气体从亚声速加速到超声速**：先渐缩 → 在喉部达到声速 → 再渐放
→ 形成 ==**缩放喷管**（拉伐尔喷管，Laval Nozzle）==

> [!tip] 补充
> 拉伐尔喷管由瑞典工程师 Gustaf de Laval 于 19 世纪末发明，最早用于蒸汽轮机。火箭发动机的尾喷管就是典型的拉伐尔喷管——喉部达到声速，扩张段继续加速到超声速（Ma 可到 3~5）。

**关键认知**：
- 在喷管中流速不断增加时，==声速是不断下降的==（T↓ → $c_s = \sqrt{\kappa R_g T}$↓）
- → 流速 c↑ 而声速 $c_s$↓ → 两者必然在某一截面**相遇**（c = $c_s$）
- 该截面即**喉部**（最小截面），此时的流速称为**临界流速** $c_c$

---

## §5-3 气体流经喷管的流速和流量

### 一· 滞止参数

**定义**：把气体流速为零时（或按定熵压缩折算到流速为零时）的各种参数称为**滞止参数**。

- 标记：上标 `*`，如 $p^*$、$T^*$、$h^*$、$v^*$
- 物理含义：设想气体在扩压管中定熵压缩到流速为零 → 得到的参数即滞止参数

> [!note] 核心理解
> 物理上：滞止状态 = "把动能全部回收为焓"的理想极限状态。滞止参数为流动过程提供了一个统一的参考基准（就像电势的"地"）。

---

### 二· 流速计算

#### 1. 通用公式（基于能量方程）

由能量方程 $h^* = h + \frac{c^2}{2}$：

$$c = \sqrt{2(h^* - h)} \qquad (5-14)$$

- 适用于**任何工质、任何绝热流动**（可逆或不可逆）
- 只要知道滞止焓降 $h^* - h$ 即可算流速

#### 2. 定比热容理想气体

$$c = \sqrt{2c_{p0}(T^* - T)} \qquad (5-15)$$

#### 3. 无摩擦定熵流动的流速公式（基于动量方程 + 过程方程）

**推导**：

（1）由动量方程积分：$c = \sqrt{2\left(-\int_{p^*}^p v\,dp\right)}$ ← 以滞止状态为起点
（2）过程方程 $pv^\kappa = \text{const}$ → $v = p^{*1/\kappa}v^* p^{-1/\kappa}$
（3）代入积分：$-\int_{p^*}^p v\,dp = -p^{*1/\kappa}v^*\int_{p^*}^p p^{-1/\kappa}dp$
$= -\frac{\kappa}{\kappa-1}p^{*1/\kappa}v^*\left[p^{(\kappa-1)/\kappa} - p^{*(\kappa-1)/\kappa}\right]$
$= \frac{\kappa}{\kappa-1}p^*v^*\left[1 - \left(\frac{p}{p^*}\right)^{(\kappa-1)/\kappa}\right]$
（4）代入 (1)，利用 $c_s^* = \sqrt{\kappa p^*v^*}$：

$$c = c_s^*\sqrt{\frac{2}{\kappa-1}\left[1 - \left(\frac{p}{p^*}\right)^{\frac{\kappa-1}{\kappa}}\right]} \qquad (5-16)$$

或对定比热容理想气体：

$$c = \sqrt{\frac{2\kappa}{\kappa-1}R_g T^*\left[1 - \left(\frac{p}{p^*}\right)^{\frac{\kappa-1}{\kappa}}\right]} \qquad (5-17)$$

---

### 三· 临界流速与临界压力比（核心）

**临界状态定义**：Ma = 1，即 c = $c_s$。此时压力为**临界压力 $p_c$**，流速为**临界流速 $c_c$**。

**推导临界压力比 $\beta_c = p_c/p^*$**：

（1）由定义：$c_c = c_{s,c} = \sqrt{\kappa p_c v_c}$ ← 临界流速 = 当地声速
（2）由式(5-16)在临界截面：$c_c = c_s^*\sqrt{\frac{2}{\kappa-1}\left[1 - \left(\frac{p_c}{p^*}\right)^{\frac{\kappa-1}{\kappa}}\right]}$
（3）又 $c_s^* = \sqrt{\kappa p^* v^*}$，且 $\frac{c_c}{c_s^*} = \frac{\sqrt{\kappa p_c v_c}}{\sqrt{\kappa p^* v^*}}$

经过代数变换（过程方程 $v_c/v^* = (p^*/p_c)^{1/\kappa}$）：

$$\left(\frac{p_c}{p^*}\right)^{\frac{\kappa-1}{\kappa}}\left(1 + \frac{2}{\kappa-1}\right) = \frac{2}{\kappa-1}$$

$$\beta_c = \frac{p_c}{p^*} = \left(\frac{2}{\kappa+1}\right)^{\frac{\kappa}{\kappa-1}} \qquad (5-18)$$

**各种气体的临界压力比**：

| 气体类型 | $\kappa$ | $\beta_c$ |
|:---|:---:|:---:|
| 单原子气体 | **1.67** | **≈0.487** |
| 双原子气体（空气） | **1.40** | **≈0.528** |
| 多原子气体 | **1.30** | **≈0.546** |
| 过热水蒸气 | **1.30** | **≈0.546** |
| 饱和水蒸气 | **1.135** | **≈0.577** |

> 大致概念：==各种气体从滞止加速到临界流速，压力大约降低一半==。

**临界流速**：

$$c_c = c_s^*\sqrt{\frac{2}{\kappa+1}} = \sqrt{\frac{2\kappa}{\kappa+1}R_g T^*} \qquad (5-20)$$

- 临界流速**仅取决于滞止温度** $T^*$ 和气体种类（$\kappa$、$R_g$）
- 对定比热容理想气体：$c_c = \sqrt{\kappa R_g T_c}$（在临界截面上）

> [!note] 核心理解
> 物理上：β_c ≈ 0.5 意味着——要让气流从静止加速到声速，压力只需要降到原来的一半左右。而从声速继续加速到高超声速，压力需要进一步大幅下降（式5-16中 p/p* 趋近于零时 c 才趋于最大值）。数学上：β_c 仅取决于 κ，是气体的固有属性——κ 越大（分子越"硬"），β_c 越小。

---

### 四· 流量与最大流量

**流量计算**：按最小截面（喉部）参数计算：

$$q_m = \frac{A_{min}}{v^*}c_s^*\sqrt{\frac{2}{\kappa-1}\left[\left(\frac{p_{th}}{p^*}\right)^{\frac{2}{\kappa}} - \left(\frac{p_{th}}{p^*}\right)^{\frac{\kappa+1}{\kappa}}\right]} \qquad (5-21)$$

**最大流量条件**：

当 $p_{th}/p^*$ 使括号内表达式取极大值时，流量最大。

令 $\beta = p_{th}/p^*$，求 $f(\beta) = \beta^{2/\kappa} - \beta^{(\kappa+1)/\kappa}$ 的极值：

（1）$\frac{df}{d\beta} = \frac{2}{\kappa}\beta^{\frac{2}{\kappa}-1} - \frac{\kappa+1}{\kappa}\beta^{\frac{1}{\kappa}} = 0$
（2）$\beta^{\frac{2}{\kappa}-1}\left[\frac{2}{\kappa} - \frac{\kappa+1}{\kappa}\beta^{1-\frac{1}{\kappa}}\right] = 0$
（3）非零解：$\beta = \left(\frac{2}{\kappa+1}\right)^{\frac{\kappa}{\kappa-1}} = \beta_c$

→ ==当喉部达到临界状态（Ma=1，$p_{th}=p_c$）时，流量达到最大值==

**最大流量**：

$$q_{m,max} = \frac{A_{min}}{v^*}c_s^*\left(\frac{2}{\kappa+1}\right)^{\frac{\kappa+1}{2(\kappa-1)}} \qquad (5-22)$$

**壅塞（Choking）的物理本质**：

> [!note] 核心理解 — 壅塞现象
> 物理图像：==喉部达到声速后，下游的压力信号（以声速传播）无法逆流传到喉部上游==——因为喉部流速 = 声速，"信息"在原地踏步。因此，背压再降低也无法改变喉部上游的流动，流量被"锁死"在最大值。
>
> 类比：就像高速公路上车流达到最大通行能力——收费站后面排再多车，通过收费站的车辆数也不会增加。
>
> 数学上：当 p_th/p* = β_c 时，流量取全局最大值。p_th/p* 减小 → 喉部 Ma 无法超过 1（对渐缩喷管而言）→ 流量不再增加。

---

### 五· 扩压管

**定义**：利用**流速降低**使气体**增压**的流道。

- 扩压管 = 喷管的逆过程
- 气流进行的是绝热压缩过程

**参数变化对比**：

| 参数 | 喷管（加速降压） | 扩压管（减速增压） |
|:---|:---|:---|
| dc | > 0 | < 0 |
| dp | < 0 | > 0 |
| dh | < 0 | > 0 |
| dT | < 0 | > 0 |
| dv | > 0 | < 0 |

> [!tip] 补充
> ⚠️ 熵的变化不可简单取反：无摩擦时两者 ds=0；有摩擦时两者 ds>0（都是不可逆性使熵增加）。熵的方向由不可逆性决定，不由流向决定。

**滞止参数的计算**（按定熵压缩）：

滞止焓：$h^* = h + \frac{c^2}{2} \qquad (5-23)$

滞止温度（定比热容理想气体）：$T^* = T + \frac{c^2}{2c_{p0}} \qquad (5-24)$

滞止压力（定熵过程）：$p^* = p\left(1 + \frac{c^2}{2c_{p0}T}\right)^{\frac{\kappa}{\kappa-1}} \qquad (5-25)$

滞止比体积：$v^* = \frac{R_g T^*}{p^*} \qquad (5-26)$

## §5-4 喷管背压变化时的流动状况

### 一· 渐缩喷管

**条件**：滞止参数 $p^*$、$T^*$ 不变，背压 $p_B$ 从 $p^*$ 逐渐下降。

| 背压范围 | 出口压力 | 出口流速 | 流量 |
|:---|:---|:---|:---|
| $p_B = p^*$ | $=p^*$ | 0 | 0 |
| $p_c < p_B < p^*$ | $=p_B$ | 亚声速，随 $p_B\downarrow$ 而 ↑ | 随 $p_B\downarrow$ 而 ↑ |
| $p_B = p_c$ | $=p_c$ | ===声速（临界流速，最大出口流速）== | ==最大== |
| $p_B < p_c$（直至真空） | ==锁定为 $p_c$== | ==锁定为 $c_c$== | ==锁定为最大值（壅塞）== |

> [!note] 核心理解
> 物理本质：渐缩喷管的出口最高只能达到声速（Ma=1）。因为渐缩段 dA<0，由式(5-13)，在 Ma<1 时 dc>0 对应 dA<0，但一旦 Ma=1，dA=0 就需要喉部——渐缩喷管没有喉部之后的扩张段，无法继续加速到超声速。背压降到 p_c 以下后，气流在出口处**膨胀不足**，在喷管外继续膨胀降压到 p_B（自由射流膨胀）。

---

### 二· 缩放喷管（拉伐尔喷管）

![[图_缩放喷管背压变化工况.excalidraw]] <!-- 📌 待补图 -->

> **图注**：缩放喷管在不同背压下的流动状态。分为四个特征区域：亚声速全流、喉部临界+激波、过膨胀、设计工况/欠膨胀。

**背压从 $p^*$ 下降到真空的各阶段**：

1. **$p_B = p^*$**：无流动，$c=0$，$q_m=0$

2. **$p_B$ 开始下降，喉部未达临界**：缩放喷管像**文丘里管**一样工作——渐缩段加速降压、渐放段**减速增压**（按扩压管工作），出口 $p=p_B$。随 $p_B$↓，c↑、$q_m$↑。

3. **喉部达到临界，$p_B$ 在一定范围**：喉部锁为 Ma=1，$q_m$ 锁定最大值。渐放段前段达到超声速 → 在某个截面产生**激波** → 流速从超声速急剧降到亚声速 → 激波后按扩压管减速增压 → 出口 $p=p_B$

4. **$p_B$ 等于设计出口压力**：==一切正常，出口 $p=p_B$，全流超声速，$q_m$ 最大==

5. **$p_B$ 降到设计值以下（直至真空）**：喷管内部不再变化，出口保持设计工况。气流在出口处**膨胀不足**，在喷管外继续膨胀 → 压力降到 $p_B$

**激波（Shock Wave）**：
- 发生在超声速气流受到压缩扰动的截面上
- 流速从超声速**急剧**降到亚声速，压力**突然**升高
- **绝非定熵过程**——不可逆性极强（熵增很大）
- 激波使喷管效率显著降低、工作不稳定 → ==应尽量避免==
- 激波后温度骤降 → 水汽冷凝 → 可见雾化锥（如战斗机突破音障时的锥形云）

> [!tip] 补充
> 普朗特-迈耶（Prandtl-Meyer）理论详细讨论了激波前后的参数关系。激波厚度仅为分子平均自由程量级（~10⁻⁷ m），宏观上可视为**间断面**。

## §5-5 喷管中有摩擦的绝热流动过程

### 一· 基本设定

- 定比热容理想气体（$\kappa = \gamma_0$）
- 有摩擦的绝热流动遵守**多变过程**规律：$pv^n = \text{const}$（$n < \kappa$）

### 二· 关键结论

#### 1. 有摩擦时喉部流速为亚声速

由截面变化关系（类似5-2节推导，但用多变指数 n 替代 $\kappa$）：

$$\frac{dA}{A} = \left(\frac{\kappa-1}{n-1}\text{Ma}^2 - 1\right)\frac{dc}{c} \qquad (5-33)$$

当 dA=0（喉部）时，$dc \neq 0$，故 $\text{Ma} = \sqrt{\frac{n-1}{\kappa-1}}$，而 $n < \kappa$ → ==$\text{Ma} < 1$==

→ 有摩擦时，喉部流速是**亚声速**的，Ma=1 的临界截面后移到渐放段。

#### 2. 临界压力比减小

$$\beta_c = \frac{p_c}{p^*} = \left(\frac{2}{n+1}\right)^{\frac{n}{n-1}} \qquad (5-37)$$

因为 $n < \kappa$，有摩擦时的 $\beta_c$ 比无摩擦时**小**。

#### 3. 速度系数和喷管效率

**速度系数** $\varphi$：
- 定义：相同参数条件下，实际出口流速与定熵膨胀理论流速之比

$$\varphi = \frac{c_{2,act}}{c_{2,s}} \qquad (5-35)$$

**喷管效率** $\eta_N$：实际出口动能与定熵膨胀可获得动能之比

$$\eta_N = \frac{c_{2,act}^2}{c_{2,s}^2} = \varphi^2 \qquad (5-36)$$

#### 4. 摩擦损失的物理表现

- 式(5-14) $c = \sqrt{2(h^* - h)}$ 对有摩擦和无摩擦都适用
- 摩擦损失表现为：同样压力降下，**实际出口焓 $h_2$ 比定熵膨胀时大**（摩擦热使气体升温），因而 $(h^* - h_2)$ 变小 → 出口流速变小

## §5-6 活塞式压气机的压气过程

### 一· 压气机概述

**定义**：消耗机械功使气体从较低压力提升到较高压力的设备。

- 活塞式（往复运动）← 本节
- 叶轮式（旋转运动）← §5-7
- 鼓风机、引风机、真空泵 → 热力学本质相同，只是工作压力范围不同

> [!note] 核心理解
> 物理上：压气机和喷管是"镜像"关系——喷管是用压力降换取速度（膨胀），压气机是用功换取压力升（压缩）。两者的热力学方程结构对称，参数变化方向相反。

---

### 二· 单级活塞式压气机工作过程

**三个过程**组成的循环：

```
进气(4→1)   压缩(1→2)   排气(2→3)
  │             │            │
p₁           p₁→p₂        p₂
V: 0→V₁      V₁→V₂       V₂→0
功: p₁V₁入     ∫pdV耗      p₂V₂耗
```

**比功（每 kg 气体消耗的功）**：

$$w_c = -\int_1^2 v\,dp \qquad (5-46)$$

（习惯上取压气机耗功为正）

---

### 三· 三种压缩方式的功耗比较

| 压缩方式 | 过程方程 | 功耗 | 排序 |
|:---|:---|:---|:---|
| 定温压缩（T=const） | $pv = \text{const}$ | $w_{c,T} = R_g T_1 \ln \pi$ | ==最少== |
| 多变压缩 | $pv^n = \text{const}$ | $w_{c,n} = \frac{n}{n-1}R_g T_1(\pi^{\frac{n-1}{n}}-1)$ | 居中 |
| 定熵压缩 | $pv^\kappa = \text{const}$ | $w_{c,s} = \frac{\kappa}{\kappa-1}R_g T_1(\pi^{\frac{\kappa-1}{\kappa}}-1)$ | ==最多== |

其中 $\pi = p_2/p_1$ 为**增压比**。

> [!note] 核心理解
> 物理上：定温压缩耗功最少——因为压缩过程中不断散热，气体温度不升高，比体积较小，同样的压力升高需要的体积变化小 → 功小。绝热压缩时气体升温最严重 → 比体积最大 → 耗功最多。这就是为什么压气机要用水套冷却：让压缩过程**逼近定温**以省功。

**工程实际**：
- 气体在气缸中停留时间很短 → 散热不充分
- 压缩过程介于定温和绝热之间，接近**多变过程**（$1 < n < \kappa$）
- 冷却好 → n 靠近 1（接近定温）；冷却差 → n 靠近 $\kappa$（接近绝热）

---

### 四· 余隙容积的影响

**定义**：
- **余隙容积 $V_c$**：活塞上止点与气缸顶端之间的空隙（为安装阀门和避免碰撞）
- **活塞排量 $V_h$**：活塞一个行程扫过的容积
- **有效容积 $V_e$**：实际吸入的新鲜气体体积
- **余隙比**：$V_c/V_h$（一般 **3%~8%**）

**容积效率**：

$$\eta_V = \frac{V_e}{V_h} = 1 - \frac{V_c}{V_h}\left(\pi^{\frac{1}{n}} - 1\right) \qquad (5-51)$$

**影响因素**：
- （1）余隙比 $\frac{V_c}{V_h}$↓ → $\eta_V$↑
- （2）增压比 $\pi$↑ → $\eta_V$↓
- （3）多变指数 n↑ → $\eta_V$↑

**极限情况**：当 $\pi$ 过高时 → $\eta_V = 0$ → 压气机无法输出压缩气体

> [!note] 核心理解
> 物理上：余隙容积中的高压气体在吸气前必须先膨胀到进气压力——膨胀时占据的容积就是"浪费"的吸气容积。增压比越高，余隙气体膨胀得越多 → 有效吸气容积越小。

> [!tip] 补充
> 在理论上，余隙容积**不影响耗功**（余隙气体膨胀做的功 = 压缩它消耗的功——如果两过程可逆、n相同）。但实际上：不可逆损失存在 → 余隙气体膨胀回收的功 < 压缩消耗的功 → 余隙容积既降产量也降效率。单级增压比一般不超过 **10**。

---

### 五· 多级活塞式压气机

**目的**：
- （1）达到更高压力（单级增压比有限）
- （2）==省功==（中间冷却使压缩趋近定温）
- （3）避免压缩终温过高（影响润滑→活塞环卡死）

**理论分析假定**：
- （1）各级多变指数 n 相同，无摩擦
- （2）级间无压力损失：$p_{i+1,in} = p_{i,out}$
- （3）中间冷却充分：$T_{i+1,in} = T_{1,in}$

**最优增压比**（两级情形）：

（1）两级总耗功：$w_c = \frac{n}{n-1}R_g T_1\left[\left(\frac{p_2}{p_1}\right)^{\frac{n-1}{n}} + \left(\frac{p_4}{p_2}\right)^{\frac{n-1}{n}} - 2\right]$
（2）求 $\frac{dw_c}{dp_2} = 0$：

$$\frac{p_2}{p_1} = \frac{p_4}{p_2} \quad \Rightarrow \quad \pi_1 = \pi_2 = \sqrt{\frac{p_4}{p_1}} \qquad (5-52)$$

→ ==各级增压比相等时，总耗功最少==

**推广到 m 级**：

各级增压比：$\pi = \left(\frac{p_{max}}{p_{min}}\right)^{\frac{1}{m}}$

总耗功：$w_{c,n} = m\frac{n}{n-1}R_g T_1\left(\pi^{\frac{n-1}{n}}-1\right) \qquad (5-55)$

---

### 六· 活塞式压气机的效率

三种效率定义（分子为理论耗功，分母为实际耗功）：

- **绝热效率**：$\eta_{c,s} = \frac{w_{c,s}}{w_{c(act)}} \qquad (5-56)$
- **多变效率**：$\eta_{c,n} = \frac{w_{c,n}}{w_{c(act)}} \qquad (5-57)$
- **定温效率**：$\eta_{c,T} = \frac{w_{c,T}}{w_{c(act)}} \qquad (5-58)$

> 效率 = 理论最小功 / 实际功，保证效率 ≤ 1

---

## §5-7 叶轮式压气机的压气过程

### 一· 叶轮式压气机的特点

| 特点 | 说明 |
|:---|:---|
| 工作连续 | 气体不断流入、压缩、流出 |
| 过程接近绝热 | 大量气体快速流过 → 每 kg 散热极少 |
| 无往复部件 | 运行平稳、高转速、轻小 |
| 适用场景 | ==大流量==压气设备 |

**两种主要型式**：
- **离心式**：叶轮使气体离心加速 → 扩压管减速增压 → 多级串联
- **轴流式**：转子工作叶片 + 定子导向叶片 → 逐级提速升压

> [!tip] 补充
> 轴流式压气机每一级转子叶片本质上是排"机翼"——气流绕叶片流动产生升力（环量），多排叶片接力压缩。

---

### 二· 耗功计算

能量方程（不计进出口动能和位能变化）：

$$w_c = h_2 - h_1 \qquad (5-59)$$

定比热容理想气体：$w_c = c_{p0}(T_2 - T_1) \qquad (5-60)$

可逆定熵压缩：$w_{c,s} = \frac{\kappa}{\kappa-1}R_g T_1\left(\pi^{\frac{\kappa-1}{\kappa}}-1\right) \qquad (5-62)$

---

### 三· 叶轮式压气机的效率

实际不可逆绝热压缩 ≈ 多变过程（$n > \kappa$）：

实际耗功：$w_{c(act)} = \frac{\gamma_0}{\gamma_0-1}R_g T_1\left(\pi^{\frac{n-1}{n}}-1\right) \qquad (5-63)$

**绝热效率**：

$$\eta_{c,s} = \frac{w_{c,s}}{w_{c(act)}} = \frac{\pi^{\frac{\kappa-1}{\kappa}}-1}{\pi^{\frac{n-1}{n}}-1} \qquad (5-64)$$

**多变效率**（= 同一多变过程可逆功 / 不可逆功）：

$$\eta_{c,n} = \frac{n(\kappa-1)}{\kappa(n-1)} \qquad (5-65)$$

**效率关系**：$\eta_{c,n} > \eta_{c,s}$（因为多变效率对比的是同一多变过程，绝热效率对比的是绝热过程）

> [!note] 核心理解
> 物理上：绝热效率把实际过程与"可逆绝热"比——两者终点温度不同；多变效率把实际过程与同一多变指数的"可逆多变"比——终点温度相同。从热力学角度看，多变效率更合理（同一过程的比较），但工程习惯用绝热效率。

**与活塞式的对比**：
- 活塞式：小流量、高压比
- 叶轮式：大流量、运行平稳、可高转速

## §5-8 引射器的工作过程

### 一· 引射器的结构与原理

**结构**：

```
高压流体(p₁) → [喷管] → 高速低压 → [混合室] ← 低压流体(p₂)
                                        │
                                        ▼
                              [扩压管] → 中压(p₃) 流出
```

**工作原理**：
- （1）高压流体（$p_1$）经喷管降压加速 → 高速射流
- （2）高速射流**引射**低压流体（$p_2$）进入混合室
- （3）两股流体在混合室中混合达到中等流速（前者减速、后者加速）
- （4）混合流体经扩压管**升压**到 $p_3$（$p_2 < p_3 < p_1$）

**能量交换本质**：==以高压流体的压力降为代价，换取低压流体的压力升==

> [!note] 核心理解
> 物理上：引射器 = "喷管 + 混合室 + 扩压管"的串联——喷管把高压的"势能"转为"动能"，混合室通过动量交换把动能传给低压流体，扩压管把混合后的"动能"再转回"势能"。整个过程实现了"压力搬运"。

---

### 二· 热力学分析

对外界而言仍是**无技术功的绝热过程**：

$$q_{m3}h_3 = q_{m1}h_1 + q_{m2}h_2 \qquad (5-67)$$

即：$h_3 = g_1 h_1 + g_2 h_2$（其中 $g_i = q_{mi}/q_{m3}$ 为质量分数）

对同种定比热容理想气体：$T_3 = g_1 T_1 + g_2 T_2 \qquad (5-68)$

**熵产**（每 kg 流出气体）：

$$s_g = c_{p0}\ln\frac{T_3}{T_1^{g_1}T_2^{g_2}} - R_g\ln\frac{p_3}{p_1^{g_1}p_2^{g_2}} \qquad (5-69)$$

**效率（㶲效率）**：

$$\eta_{ex} = \frac{g_2[(h_3 - h_2) - T_0(s_3 - s_2)]}{g_1[(h_1 - h_3) - T_0(s_1 - s_3)]} \qquad (5-70)$$

= 低压流体获得的㶲 / 高压流体消耗的㶲

**特点**：
- 混合过程中不可逆损失很大 → ==效率一般很低==
- 但结构简单、无运动部件、工作可靠
- 应用场景：维持真空抽气、高低压蒸汽混合制中压蒸汽（比节流降压经济）

---

## 速记卡 / 一句话总结

- **连续性方程**：$q_m = Ac/v = \text{const}$——质量流量不变，但容积流量沿程变化
- **能量方程**：$h + c^2/2 = \text{const}$——焓降 = 动能增量（绝能绝功流）
- **动量方程**：$c\,dc = -v\,dp$——流速和压力反向变化（无摩擦）
- **马赫数**：$\text{Ma} = c/c_s$——区分亚/跨/超声速的**唯一判据**
- **截面变化定律**：$dA/A = (\text{Ma}^2-1)dc/c$——亚声速加速收窄，超声速加速放宽
- **临界压力比**：$\beta_c = [2/(\kappa+1)]^{\kappa/(\kappa-1)}$，空气 **0.528**——流速从零到声速，压力约降一半
- **壅塞本质**：==喉部 Ma=1 时，下游扰动无法逆流上传→流量锁死==
- **缩放喷管（拉伐尔）**：渐缩→喉部（Ma=1）→渐放（Ma>1），火箭发动机的标准配置
- **激波**：超声速→亚声速的剧烈不可逆过渡，应尽量避免
- **压气机三种压缩**：定温耗功最少 < 多变 < 绝热最多——故要水套冷却
- **多级压缩最优**：==各级增压比相等时总耗功最少==，$\pi_{stage} = (p_{max}/p_{min})^{1/m}$
- **活塞式 vs 叶轮式**：活塞式小流量高压比，叶轮式大流量连续运行
- **引射器**：高压换低压——结构简单，但不可逆损失大、效率低
- **有摩擦喷管**：喉部 Ma<1（亚声速），临界截面后移到渐放段，$\beta_c$ 比无摩擦时小
