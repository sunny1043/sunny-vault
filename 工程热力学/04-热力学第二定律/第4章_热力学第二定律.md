# 第4章 热力学第二定律 ★★★

#engineering-thermodynamics #second-law

## 4-1 为什么需要第二定律？

第一定律告诉我们**能量不会消失**，但它不能解释：
- 为什么热量**自发**地从高温流向低温，从不反向？
- 为什么摩擦产生的热不能**完全**变回机械功？
- 为什么热机的效率有**上限**？

**物理直觉**: 第一定律是"能量会计学"——保证账目平衡。第二定律是"能量质量管理学"——虽然能量总量不变，但它的**品位**（做功能力）在每次转换中**必然下降**。

## 4-2 两种经典表述

| 表述 | 内容 | 否定什么 |
|------|------|----------|
| **开尔文** | 不可能从单一热源吸热使之完全变为有用功而不引起其他变化 | 第二类永动机 |
| **克劳修斯** | 热量不可能自发地、不付代价地从低温传到高温 | 自发逆向传热 |

> [!tip] 关键理解
> 两种表述等价——违背其一必违背另一。就像说"水不能自发从低处流向高处"和"不能只靠一个热源发电"是同一个物理规律的两种说法。

## 4-3 卡诺循环 (Carnot Cycle)

卡诺循环是工作于两个热源之间的**最理想**的可逆循环。

### 四个过程
```
  T
  │ 1 ──── 2  定温吸热 T₁ (高温)
  │  ╲    ╱
  │   ╲  ╱   定熵膨胀 (对外做功)
  │    ╲╱     定熵压缩 (消耗功)
  │    ╱╲
  │   ╱  ╲
  │  ╱    ╲
  │ 4 ──── 3  定温放热 T₂ (低温)
  └──────────── s
```

### 卡诺循环热效率——理论上限

$$\\boxed{\\eta_C = 1 - \\frac{T_2}{T_1}}$$

**推导思路**: 效率 = 净功/吸热量。定温过程 q=T·Δs，两个定熵过程 q=0。
- 吸热: $q_1 = T_1(s_2 - s_1)$
- 放热: $q_2 = T_2(s_3 - s_4) = T_2(s_2 - s_1)$ (因 s₃=s₂, s₄=s₁)
- 净功: $w = q_1 - q_2$
- $\\eta = w/q_1 = 1 - T_2/T_1$

**关键结论**: 卡诺效率仅取决于高低温热源的温度，与工质无关。

### 卡诺定理
1. 工作于**相同两个热源**间的所有可逆热机，效率相等
2. 可逆热机的效率**大于**不可逆热机

→ 任意热机效率 $\\eta \\leq 1 - T_2/T_1$

## 4-4 克劳修斯不等式与熵

### 克劳修斯不等式

$$\\oint \\frac{\\delta q}{T} \\leq 0$$

- = 0: 可逆循环
- < 0: 不可逆循环

### 熵的定义

由克劳修斯不等式，对可逆过程 $\\oint \\frac{\\delta q_{rev}}{T} = 0$，被积函数是全微分，定义为熵:

$$dS = \\frac{\\delta q_{rev}}{T}$$

**物理直觉**: 熵是"不可逆性"的度量。可逆过程熵不变；不可逆过程（摩擦、温差传热、自由膨胀）导致熵增。熵越高，系统的"做功能力"越低。

### 闭口系熵方程

$$dS = \\frac{\\delta q}{T} + dS_g, \\quad dS_g \\geq 0$$

- $dS_g$ = 熵产（不可逆性的度量，永不<0）
- $\\delta q/T$ = 熵流（由于传热进入系统）

## 4-5 孤立系熵增原理

$$\\boxed{\\Delta S_{isolated} \\geq 0}$$

这是判断过程**方向和不可逆性**的核心判据:
- >0: 不可逆（自发方向）
- =0: 可逆
- <0: **不可能**

## 4-6 㶲 (Exergy) 与 㶲损失

### 㶲 — "可用能"

**定义**: 系统在给定环境 $(T_0, p_0)$ 下，通过可逆过程所能做出的**最大有用功**。

**物理直觉**: 㶲是能量的"可用部分"。1kJ电能=1kJ㶲（100%可做功）；1kJ 50°C热水的热量<0.1kJ㶲（大部分不能用）。能量守恒=数量不变；㶲守恒=品质不变——但㶲不守恒，它总是在减少。

### 流动㶲

$$e_{x,f} = (h - h_0) - T_0(s - s_0)$$

### 㶲损失 — Gouy-Stodola 定理

$$\\boxed{I = T_0 \\cdot S_{g,total}}$$

**物理直觉**: 熵产 $S_g$ 越多 → 㶲损失越大。摩擦、温差传热、混合——所有这些不可逆过程都产生熵，都"烧掉"可用能。

## 例题

### [例4-1] 卡诺效率计算

某热机工作于800K和300K之间，求最大理论效率。

$$\\eta_C = 1 - \\frac{300}{800} = 1 - 0.375 = 62.5\\%$$

### [例4-2] 㶲损失

某不可逆过程总熵产0.5 kJ/K，环境温度300K，求㶲损失。

$$I = T_0 \\cdot S_{g,total} = 300 \\times 0.5 = 150 \\text{ kJ}$$

### [例4-3] 气化熵变

1kg水在100°C定温气化，$r=2257$ kJ/kg。求熵变。

$$\\Delta s = \\frac{r}{T} = \\frac{2257}{373.15} = 6.048 \\text{ kJ/(kg·K)}$$

## Common Misconceptions

> [!warning] 误区: "熵只增不减"
> **错误**。是**孤立系**的熵只增不减。开放系统可以通过向外界排出热量（负熵流）来减少自身熵。冰箱内的熵在减少，但冰箱+环境的**总熵**在增加。
