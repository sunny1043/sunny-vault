# 第5章 气体的流动和压缩 ★★

#engineering-thermodynamics #gas-flow

## 5-1 基本概念

### 声速 — 信息在气体中传播的速度

$$a = \sqrt{k R_g T}$$

**物理直觉**: 声速是微小扰动在气体中传播的速度。气体的温度越高，分子运动越快，扰动传播越快。对空气 (k=1.4, R_g=287): a≈20√T (m/s)。常温300K下约347 m/s。

### 马赫数 — 流速与声速的比值

$$M = \frac{c}{a}$$

| M范围 | 流动状态 |
|:-----:|----------|
| <1 | 亚声速 |
| =1 | 声速（临界） |
| >1 | 超声速 |

### 连续性方程 — 质量守恒

$$\dot{m} = \frac{Ac}{v} = C$$

## 5-2 变截面方程 — 喷管设计的核心

$$\boxed{\frac{dA}{A} = (M^2 - 1)\frac{dc}{c}}$$

**推导**: 对连续性方程 $\dot{m}=Ac/v=C$ 取对数微分: $dA/A + dc/c - dv/v = 0$。结合能量方程 $cdc = -vdp$ 和声速定义，消去 dv 和 dp 得到。

**物理直觉 — 三条规则**:

| M范围 | dA/dc符号 | 加速条件 | 喷管形状 |
|:-----:|:--------:|----------|----------|
| M<1 | (-) | 截面减小→加速 | **渐缩** |
| M=1 | 0 | 喉部 | 临界截面 |
| M>1 | (+) | 截面增大→加速 | **渐扩** |

> [!important] 拉法尔喷管: 亚声速→超声速必须先用收缩段加速到M=1，再通过扩张段继续加速。**渐缩喷管出口最高只能达到M=1**。

## 5-3 压气机

### 三种压缩过程耗功比较

$$W_{iso} < W_{poly} < W_{isen}$$

| 过程 | 特征 | 耗功 |
|------|------|:---:|
| 定温 | 充分冷却 | 最小 |
| 多变 | 有限冷却 | 中间 |
| 定熵 | 绝热无摩擦 | 最大 |

### 多级压缩 + 中间冷却

好处: 1) 压缩功接近定温→省功 2) 降低排气温度

## 🔗 工程案例：AL-31F 涡扇发动机压气机

> **关联专题**：[[AL-31F_压气机/AL31F_压气机文献综述|AL-31F 压气机文献综述]]

### 技术参数映射

| 热力学概念 | AL-31F 实际参数 | 说明 |
|-----------|----------------|------|
| 总增压比 $π_c$ | **23:1** | 4级LPC(π≈3.5) × 9级HPC(π≈6.6) |
| 多变压缩 | 级间放气旁路 | 中间机匣涵道放气(涵道比0.57)，模拟中间冷却效果 |
| 多级压缩 | 13级轴流 (4LPC+9HPC) | 双转子转速独立，自动调节级间匹配 |
| 进气条件 | $T_1≈288K$ (标准天) | 可调IGV根据进气温度自动调节角度 |
| 绝热效率 $η_{ad}$ | HPC中间级≈0.87 (CIAM 试验) | 三维弯刀形静子+镰刀形转子叶片 |

### 防喘体系中的热力学原理

AL-31F 的三重防喘手段都可以用热力学解释：

1. **可调 IGV/VSV**：改变进口预旋角→调整攻角→控制叶栅通道内的流动分离（本质是控制多变指数 n 的偏离程度）
2. **级间放气**：在 LPC 出口向旁路泄放部分流量→降低后级负荷→等效于在压气机中间设置了一个"放气冷却级"
3. **双转子**：高/低压转子通过转速比自适应调节转速，自动匹配各级的流量系数 $φ = C_a/U$

> [!tip] 与第5章理论的对应
> - 变截面方程 $dA/A = (M^2-1)dc/c$ → 压气机通道扩张减速（扩压器原理）
> - 多级压缩省功 → AL-31F 级间放气降低后级负荷
> - 绝热效率 → CIAM 实测镰刀型叶片提升效率1-2%

## 例题

### [例] 声速计算
空气 k=1.4, R_g=0.287 kJ/(kg·K), T=300K。求声速。

$$a = \sqrt{1.4 \times 0.287 \times 10^3 \times 300} = \sqrt{120540} \approx 347.2 \text{ m/s}$$

### [例] 马赫数
喷管中流速500 m/s，当地声速400 m/s。求M并判断流态。

$$M = 500/400 = 1.25 > 1 \rightarrow \text{超声速}$$
