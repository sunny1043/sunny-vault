## 第一章：绪论

- 一·叶轮机械总览
  - 1. 定义
    - （1）以连续旋转叶片为本体，使能量在流体工质与轴动力之间相互转换的动力机械
  - 2. 与往复活塞式机械的区别
    - （1）与环境贯通 → 通流能力更强、功率更大
  - 3. 类型
    - （1）原动机：燃气轮机、蒸汽轮机、风力机、水轮机
    - （2）工作机：鼓风机、水泵

- 二·燃气轮机（Gas Turbine）
  - 1. 定义
    - （1）以连续流动气体为工质，带动叶轮高速旋转，将燃料化学能转变为机械功的**内燃式**旋转热力发动机
  - 2. 三大核心部件（简单循环）
    - （1）压气机（Compressor）：吸入空气 → 逐级压缩 → 增压 + 升温
    - （2）燃烧室（Combustor）：压缩空气 + 燃料燃烧 → **高温高压燃气**
    - （3）涡轮（Turbine）：燃气膨胀做功 → 推动涡轮转子 → 带动压气机 + 输出功
  - 3. 工作循环（四个连续过程）
    - 压缩 → 加热 → 膨胀 → 排气（大气冷却）
    - 流程示意：
      ```
      大气空气 → [压气机] → 高压空气 → [燃烧室] + 燃料 → 高温燃气 → [涡轮]
              ↑                                                          │
              └───────────────── 带动压气机 ─────────────────────────────┘
                                                                      ↓ 余功 → 输出
      ```
  - 4. 能量分配
    - （1）涡轮发出机械功中约 **1/2～2/3** 带动压气机
    - （2）约 **1/3** 驱动发电机 / 产生推力
    - （3）起动需外界起动机带动，待涡轮功 > 压气机消耗功后脱扣
  - 5. 两个决定性参数
    - （1）**燃气初温**（涡轮第一级导叶前平均滞止温度）↑ → 效率↑
    - （2）**压缩比**（压气机出口 / 进口压力比）↑ → 效率↑
    - （3）提高初温 + 相应提高压比 → 效率显著提高
  - > [!note] 核心理解
    > 物理上：燃气轮机是 Brayton 循环的工程实现——等熵压缩（压气机）+ 等压加热（燃烧室）+ 等熵膨胀（涡轮）+ 等压放热（大气）。功率密度高是因为工质连续流动，不像活塞式受限于间歇过程。
    > 数学上：效率 η = 1 − 1/π^(γ−1)/γ（π = 压比），提高初温和压比是提效的两条根本路径。但初温受材料耐热性限制（需涡轮冷却），压比受级数/气动稳定性限制——**热力学上限与工程约束之间的博弈**定义了燃机的进化阶梯。

- 三·重型燃机分级（按燃气温度）
  - 1. E 级：约 **1200℃**（早期主流）
  - 2. F 级：约 **1400℃**（==国内外在役主流==）
  - 3. G/H 级：约 **1500℃**（先进）
  - 4. J 级：约 **1600℃**（最先进）
  - 5. **2024 进展**：我国自主研制 **300MW** F 级重型燃机在上海临港首次点火成功，技术指标与国际主流相当

- 四·蒸汽轮机对比
  - 1. 定义
    - （1）高温高压蒸汽穿过固定喷嘴加速 → 冲击叶片转动 → 对外做功
  - 2. 与燃气轮机本质区别
    - （1）燃气轮机：工质由自身燃烧产生（内燃）→ 启动快、功率密度高
    - （2）蒸汽轮机：工质由外部锅炉提供（外燃）→ 单机功率更大、燃料更灵活
  - 3. 按蒸汽参数分级
    - （1）低压：< **1.47MPa** → 中压 **1.96–3.92MPa** → 高压 **5.88–9.8MPa**
    - （2）超高压 **11.77–13.93MPa** → 亚临界 **15.69–17.65MPa** → 超临界 > **22.15MPa**

- 五·活塞式发动机（四冲程）
  - 1. 汽油机（点燃式）：吸气 → 压缩 → 做功（火花塞点燃）→ 排气
  - 2. 柴油机（压燃式）：吸气 → 压缩（空气升温至燃油自燃点）→ 做功（喷油压燃）→ 排气
  - 3. 功率密度界限
    - （1）< **200kW** 宜用活塞式 > **200kW** 宜用燃气涡轮
    - （2）例：**11MW** 船用柴油机重 **250 吨**；同功率 CFM56 涡扇 < **3 吨**

- 六·喷气发动机类型
  - 1. 基本原理：牛顿第三定律——加速空气向后 → 反作用力推动飞机
  - 2. 两种推力方式
    - （1）螺旋桨：大量空气以较低速度向后滑流
    - （2）喷气：高速燃气射流（涡喷喷速 ~**610m/s** ≈ **2253km/h**）
  - 3. 各类型特征对比
    - （1）冲压喷气（Ramjet）
      - 原理：无旋转件，扩散涵道减速 → 燃烧 → 喷管加速
      - 特点：结构简单、无压气机
      - 局限：需外力加速至 Ma ≥ **3–4** 才能工作，不能独立启动
    - （2）超燃冲压（Scramjet）
      - 原理：超声速气流中直接燃烧，**无旋转部件**
      - 优势：速度范围 Ma **6–25**（液氢燃料），推重比高
      - 挑战：需火箭助推启动，燃烧室 > **3000K** 需主动冷却
    - （3）涡轮喷气（Turbojet）
      - 原理：压气机 + 燃烧室 + 涡轮，按工作循环连续运行
      - 特点：低速也能产生大推力；油耗较高
    - （4）涡轮风扇（Turbofan）——==现代民航主力：推力大、噪声小、油耗低==
      - 原理：核心机驱动风扇，气流分内涵 / 外涵
    - （5）涡轮螺旋桨（Turboprop）
      - 原理：核心机 + 减速齿轮 + 螺旋桨
      - 特点：螺旋桨拉力为主，喷气推力仅占约 **1/9**
    - （6）涡轮轴（Turboshaft）
      - 原理：自由涡轮输出轴功，带动直升机旋翼
      - 用于：直升机 / 垂直起落飞机
  - 4. 核心机概念（燃气发生器）
    - （1）高压压气机 + 燃烧室 + 高压涡轮 = 能量平衡单元（涡轮功 = 压气机消耗功）
    - （2）配不同下游部件 → 不同发动机类型

## 第二章：发动机结构与性能

- 一·各类燃气涡轮发动机工作原理
  - 1. 混合排气双涵道涡扇发动机气流路径

    ```
    进气道 → [风扇] ─┬→ 内涵 → [高压压气机] → [燃烧室] → [涡轮] ─┐
                      │                                              │
                      └→ 外涵（风扇后离轴较远）──────────────────────┘
                                                                  ↓
                                                         [尾喷管] 混合 → 高速排气
    ```
  - 2. 发动机发展历程（半个多世纪）
    - （1）推力：**2–3 kN** → **570 kN**（↑ ≈ **200 倍**）
    - （2）耗油率：> **0.1 kg/(N·h)** → **0.035 kg/(N·h)**（↓ ≈ **2/3**）
    - （3）寿命：**几十小时** → **2–3 万小时**（↑ ≈ **千倍**）
    - （4）推重比：< **1** → > **10**
  - 3. 推重比对比
    - （1）定义：发动机推力与重力之比
    - （2）各代发动机水平
      - 涡喷：**3.5–4.0**
      - 加力涡喷：**5.0–6.0**
      - 加力小涵道比涡扇：≥ **8.0**
      - 高性能加力涡扇：**12–15**
      - 升力发动机：**16+**（目标 **20–24**）
      - 冲压发动机（Ma 2–3）：≈ **20**
      - 液体火箭（不含推进剂）：**70–100**
    - （3）代表机型
      - 中国 WS10 ≈ **8**，WS15 ≈ **10**
      - F119（F-22）≈ **10**，F135（F-35）≈ **10–12**

- 二·轴流式压气机结构特点
  - 1. 结构剖面示意

    ```
    机匣 ──── 径向间隙 ────
          R1   S1   R2   S2   R3   S3
          │    │    │    │    │    │
    轮毂 ─┴────┴────┴────┴────┴────┴─
          转子 静子 转子 静子 转子 静子
           ←—————— 级 ——————→
    ```
  - 2. 四个结构特征
    - （1）旋转工作轮叶排 + 静止整流器叶排**交错排列**
    - （2）叶排由几何参数相同的叶片沿圆周向均匀排列
    - （3）工作轮叶排与整流器叶排之间存在**轴向间隙**
    - （4）工作轮叶排与机匣之间存在**径向间隙**（漏气损失根源）
  - 3. 叶片表面为**空间曲面**——由高性能要求决定

- 三·气体流动的空间三维性
  - 1. 径向流动
    - （1）轮毂 / 机匣曲率大 → 近轮毂径向分速 > **0**，近机匣径向分速 < **0**
    - （2）叶片扭曲表面 → 叶片力存在径向分力
  - 2. 周向变化
    - （1）相邻叶片通道内气流参数沿周向不均匀——这是叶片对气流做功的必要条件
  - 3. 轴向变化
    - （1）压力沿轴向逐步升高；现代发动机要求在更短轴向长度内实现更大增压

- 四·黏性、可压缩性与非定常性
  - 1. 黏性
    - （1）压气机内是**逆压力梯度**流动（压力不断上升）→ 黏性起决定作用
  - 2. 可压缩性
    - （1）跨音级转子尖部相对马赫数 > **1.0**，部分 > **1.6**
  - 3. 非定常性
    - （1）转子：通道参数周向不均匀 + 转动 → 绝对坐标系中周期性变化
    - （2）静子：上游转子出口流场周期性变化 → 内部流动也非定常
  - 4. 复杂流动现象
    - （1）间隙涡、角涡、通道涡、激波-附面层干扰、动静叶相互干涉

- 五·轴流式压气机工作原理
  - 1. **级**的定义
    - （1）级 = 一排转子叶片 + 一排整流器（静子）叶片 —— 压气机基本增压单元
    - （2）单级增压比仅 **1.2–1.5**
    - （3）现代发动机总增压比 **25–50** → 需多级（例：压比 **8**，平均级压比 **1.26** → 需 **9** 级）
  - 2. 工作过程
    - （1）工作轮叶片对气流做功 → 气流获得**轮缘功 Lu** → 压力↑ + 绝对动能↑
    - （2）整流器中气体减速 → 动能 → 静压提高 + 改变流向
  - 3. **轮缘功 Lu 的推导**（伯努利方程在工作轮的平衡关系）
    - 从能量守恒出发：
      - （1）工作轮对气流做功 Lu = 气流流入系统的总能量变化（稳定流动能量方程）
        $$h_2 + \frac{C_2^2}{2} = h_1 + \frac{C_1^2}{2} + Lu - L_f$$ ← 开口系稳流能量方程
        其中 h 为比焓，C 为绝对速度，L_f 为流动损失
      - （2）可逆微元过程的热力学关系：$dh = dP / \rho$ （熵不变假设）
        沿流道积分：$h_2 - h_1 = \int_1^2 \frac{dP}{\rho}$ （可逆压缩）
      - （3）代入能量方程：
        $$Lu = \int_1^2 \frac{dP}{\rho} + \frac{C_2^2 - C_1^2}{2} + L_{f,R}$$ （L_{f,R} 为实际损失的功）
      - （4）物理含义：Lu 用于三部分
        - $\int dP/\rho$ —— 压缩功（提高压力 P₂ > P₁）
        - $(C_2^2 - C_1^2)/2$ —— 动能增量
        - $L_{f,R}$ —— 克服流动损失
  - > [!note] 核心理解
    > 物理上：压气机本质是**逆向涡轮**——涡轮从燃气中提取功，压气机向气流注入功来提高压力。每一级转子叶片相当于一排"机翼"，通过旋转产生升力（对气流的反作用力）来推挤空气向后并增压。
    > 数学上：轮缘功 Lu 的表达式本质上是**开口系稳流能量方程在压气机中的特例**，三个分量（压缩功、动能、损失）分别对应热力学第一定律的焓变、动能变化和不可逆耗散项。

- 六·基元级速度三角形
  - 1. 基元级定义
    - （1）在压气机叶片某一半径处，沿圆周方向取极薄一层气流——将三维问题降维至二维
  - 2. 速度三角形的构成
    - ![[图_基元级速度三角形.excalidraw]]   <!-- 📌 待补图 -->
      > **图注**：基元级速度三角形。需画出 C₁（进口绝对速度）、W₁（进口相对速度）、U（轮缘速度/圆周速度）的矢量合成关系，标注气流角 α₁；以及出口 C₂、W₂、U 的合成关系，标注转折角 Δβ。关键尺寸标注轴向分速 C₁ₐ 和周向分速 C₁ᵤ。
  - 3. 为什么速度三角形极其重要（4 个原因）
    - （1）压气机增压靠速度变化反映（压力和速度可相互转化）
    - （2）流动损失与速度大小和方向密切相关
    - （3）U↑ 和 C₁ₐ↑ 都增大加功量，但也导致 W₁ 超音速 → 损失剧增
    - （4）速度三角形**集中反映**基元级流动过程及参数数量关系
  - 4. 决定速度三角形的 4 个速度分量
    - （1）**C₁ₐ**（轴向分速）
      - 与流量 / 迎风面积直接相关
      - 例：十级压气机 C₁ₐ=**120m/s**，九级 C₁ₐ=**202m/s**
    - （2）**C₁ᵤ**（切向分速 = 预旋）
      - 正预旋（与 U 同向）、反预旋（与 U 反向）
    - （3）**U**（轮缘速度 / 圆周速度）
      - U↑ → 轮缘功 Lu↑
    - （4）**ΔWᵤ**（扭速 = W₁ᵤ – W₂ᵤ）
      - 与 U 共同决定加功量大小
  - 5. **速度分量关系式的推导**
    - 从速度三角形的几何关系出发（参见速度三角形图）：
      - （1）工作轮进口：绝对速度 C₁ 分解为轴向 C₁ₐ 和周向 C₁ᵤ
        相对速度 W₁ = C₁（矢量差）- U（轮缘速度）
        轴向分量：$W_{1a} = C_{1a}$ （轴向不受叶轮旋转影响）
        周向分量：$W_{1u} = C_{1u} - U$ （旋转坐标系变换）
        → $W_1 = \sqrt{W_{1a}^2 + W_{1u}^2} = \sqrt{C_{1a}^2 + (U - C_{1u})^2}$ （勾股定理）
      - （2）工作轮出口：扭速 $\Delta W_u = W_{1u} - W_{2u}$，且 $\Delta W_u = \Delta C_u = C_{2u} - C_{1u}$
        $C_{2u} = C_{1u} + \Delta C_u$ （周向速度变化）
        $C_2 = \sqrt{C_{2a}^2 + C_{2u}^2} = \sqrt{C_{2a}^2 + (C_{1u} + \Delta C_u)^2}$ （勾股定理）
      - （3）轮缘功 Lu 与速度分量的关系（欧拉透平方程）：
        $$Lu = U \cdot \Delta C_u = U (C_{2u} - C_{1u}) = U \cdot \Delta W_u$$ （动量和能量传递）
      - （4）约束条件：C₁ₐ、U、ΔCᵤ 三者不能无限增大，否则 W₁ 或 C₂ 接近音速 → 激波 → 效率骤降
  - 6. 预旋的调和作用
    - （1）U、C₁ₐ 不变 → 正预旋降低 W₁（例：Mw₁ 从 **0.78** 降至 **0.65**）
    - （2）W₁、C₁ₐ 不变 → 正预旋允许更高 U → 更高 Lu
    - （3）W₁、U 不变 → 正预旋允许更大 C₁ₐ → 增大流量 / 减小迎风面积
  - > [!note] 核心理解
    > 物理上：速度三角形是压气机的"经济仪表盘"——4 个速度分量（C₁ₐ、C₁ᵤ、U、ΔWᵤ）相互掣肘，设计本质上是在**迎风面积（C₁ₐ）** → **级加功量（U、ΔWᵤ）** → **流动损失（马赫数控制）** 三者之间做多目标权衡。预旋（C₁ᵤ）提供了一个额外的自由度，在不改变其余三个量的前提下调和矛盾。
    > 数学上：速度分量之间的关系由三角恒等式 W₁ = √(C₁ₐ² + (U − C₁ᵤ)²) 和欧拉透平方程 Lu = U·ΔWᵤ 约束。给定 Lu 需求后，4 分量只剩 2 个自由度——**二维流道设计问题被降为一维参数空间**。正是这个强约束让预旋成为唯一灵活的调节手段。

- 七·叶型与叶栅的几何参数
  - 1. 叶型主要几何参数（6 个）
    - （1）**弦长 b**——前后缘连线长度
    - （2）**最大厚度 C_max**——叶背到叶盆最大垂直距离
    - （3）最大厚度相对位置——距前缘距离 / 弦长
    - （4）**最大挠度 f_max**——中弧线到弦线最大垂直距离
    - （5）最大挠度相对位置——距前缘距离 / 弦长
    - （6）前缘 / 后缘半径 r₁、r₂
  - 2. 叶栅位置参数
    - （1）**安装角 γ**——弦线与额线（切线方向）的夹角
    - （2）**栅距 t**——相邻叶型对应点沿额线方向距离
    - （3）**稠度 τ = b/t**——稠度↑ → 扩压能力↑但摩擦损失↑
      - （补）稠度是"矛盾的集中体现"：扩压 vs 摩擦的权衡
  - 3. 关键气动力参数
    - （1）**攻角 i = β₁ – β₁k**——进气角与叶型安装角之差
    - （2）i 是叶栅性能最敏感的调节参数

- 八·平面叶栅中的流动过程
  - 1. 亚音叶栅绕流（M₁ ≈ **0.6**）

    ```
    前缘 A'
      ├── 叶背（吸力面）：小圆曲率大→先加速→最大速度→减速至尾缘
      │                   （逆压力梯度大→附面层厚→易分离）
      └── 叶盆（压力面）：马赫数变化平缓→压力高
    ```
    - （1）叶背速度 > 叶盆速度 → 静压相反 → 叶背 = 吸力面，叶盆 = 压力面
  - 2. 临界马赫数 M_cr
    - （1）叶背上某点最大速度首次达到当地音速时的来流 M₁
  - 3. 超临界流动（M₁ > M_cr）

    ```
    来流 M₁ → 前缘加速 → 叶背 D 点达音速 → 超音速膨胀 → E 点激波 → 亚音 → 尾缘
                                          （局部超音区）  （波阻损失↑）
    ```
    - （1）例：M₁=**0.8** 时，某亚音叶栅激波前马赫数可达 **1.29**
    - （2）激波 → 激波-附面层干扰 → 附面层分离 → 损失剧增
  - 4. 流动损失（5 类叶型损失 ω_f）
    - （1）附面层摩擦损失——黏性应力导致内能增加
    - （2）附面层分离损失——正压力梯度 + 激波-附面层干扰
    - （3）尾迹损失——上下表面附面层在尾缘汇合 → 涡流耗散动能
    - （4）掺混损失——主流区与尾迹区速度梯度 → 黏性混合消能
    - （5）波阻损失——激波 → 总压下降
    - （6）尾迹不均匀度在叶栅下游约 **1/4 弦长**内大部分消失，完全消失需**几倍弦长**
  - > [!note] 核心理解
    > 物理上：压气机叶栅的本质是**在逆压力梯度下做扩压流动**——与喷管（顺压梯度加速）完全相反。逆压力梯度 = 附面层天然不稳定、极易分离，这也是压气机设计远比涡轮困难的根本原因。
    > 数学上：叶栅的扩压能力由**稠度 τ = b/t** 和**转折角 Δβ** 共同决定，这是"流动转向量（Δβ）与流动通道长度（τ）"对逆压梯度的权衡问题。5 类损失均可归为**黏性耗散**（附面层摩擦/分离/尾迹/掺混）和**激波耗散**（波阻）两类——前者受 Re 数控制，后者受 M₁ 控制。

- 九·平面叶栅风洞试验
  - 1. 为什么要做实验
    - （1）理论计算无法充分反映激波-附面层干扰、分离等真实流动
    - （2）工程上叶栅实验仍是设计数据的主要来源
  - 2. 进口马赫数 M₁ 的影响
    - （1）M₁ = **0.3**：全流场亚音，叶背吸力峰未达音速
    - （2）M₁ > M_cr：局部超音区出现
    - （3）M₁↑ → 超音区扩大 → 激波增强 → 损失↑ → 低损失攻角范围↓
  - 3. 雷诺数 & 紊流度的影响
    - （1）低 Re → 紊流度↑ → 损失↓
    - （2）紊流度 ≤ **3%** → 转折角↑ + 损失↓；> **3%** → 损失↑（耗散增大）
    - （3）低 Re + 高 M₁ → 大紊流度使失速临界攻角变大

- 十·压气机叶片叶型
  - 1. 叶型构造（传统方法）
    - （1）中弧线（骨架）+ 原始叶型（厚度分布）→ 完整叶型
  - 2. 中弧线形式及适用马赫数
    - （1）**抛物线形**——低马赫数
    - （2）**双圆弧形**——高亚音速（Ma **0.7–1.1**），可用至 Ma **1.3**
    - （3）多圆弧 / 直线+圆弧 / 直线+双曲线——Ma > **1.2**
    - （4）**S 形中弧线**（前缘凹形）——Ma **1.5–1.6**，通过膨胀波 + 压缩波控制激波强度
  - 3. 常用原始叶型
    - （1）**C4**（英国 NGTE）、**NACA 0010**（美国）、**BC-6**（苏联）
      - 前缘厚，低损失攻角范围大，适合亚音
    - （2）**双圆弧叶型**——可用到 Ma **1.3**，广泛用于 Ma **0.7–1.1**

- 十一·级的基本理论——二次流损失
  - 1. 二次流定义：非主流方向的三维流动现象
  - 2. 组成
    - （1）环壁附面层及与主流的混合
    - （2）通道旋涡
    - （3）叶片附面层潜移
    - （4）**径向间隙中的倒流 / 潜流 + 间隙旋涡**

      ```
      叶盆（高压）──→ 径向间隙潜流 ──→ 叶背（低压）
                          ↓
                      间隙旋涡（端壁运动时）
      ```
  - 3. 径向间隙控制
    - （1）一般限制在叶片高度的 **0.5%–1%**
    - （2）**间隙主动控制技术**（现代高性能民机）
      - 起飞 / 爬升 → 压气机后热气加热机匣 → 避免碰磨
      - 巡航 → 风扇后冷气冷却机匣 → 收缩保持小间隙 → 级效率↑

- 十二·多级轴流式压气机
  - 1. 为什么需要多级
    - （1）单级增压比受限：亚音 ≈ **1.3**，超跨音 ≈ **1.5**，单级风扇可达 **2.0**
    - （2）总增压比需求：**10–50**（现代大涵道比涡扇 **25–50**）
  - 2. 效率研究方向
    - （1）先进气动热力设计与计算分析
    - （2）径向间隙主动控制技术
    - （3）新叶型 + 叶片三维气动设计技术
    - （4）小展弦比无凸台风扇叶片
    - （5）减少环壁附面层和二次流损失

## 第三章：转子叶片强度

- 一·转子叶片受力分析
  - 1. 四种载荷
    - （1）**离心力**（质量 × ω²r）
      - 质量 **0.7kg** 的叶片在 **11000 r/min** 时，离心力达 **70000N** ≈ 自重 **1 万倍**
    - （2）**横向气体力**（气动弯矩）
      - 压气机或涡轮叶片都处于大流量、高流速气流中
    - （3）**热负荷**
      - 涡轮叶片表面温度 ~**1700℃**
      - 压气机末几级 **400–550℃**
      - 温度分布不均匀 → 热应力；起停过程更严重 → 热机械疲劳
    - （4）**振动负荷**——气流扰动激起叶片振动 → 交变弯曲 / 扭转应力
  - 2. 工程简化策略
    - （1）方案 / 初步设计：只计主要应力成分 → 代数和 → 安全系数
    - （2）忽略的应力成分通过强度储备覆盖

- 二·计算模型与假设
  - 1. 悬臂梁模型（根部完全固装）
    - （1）叶片视为根部固装悬臂梁，忽略变形
    - （2）仅计离心力拉伸 / 弯曲应力 + 气体力弯曲应力，不计扭转 / 热 / 振动应力
    - （3）设扭转中心、压力中心、形心**三者重合** → 力均作用于形心
  - 2. 坐标系
    - （1）**总体 X-Y-Z**
      - OX = 旋转轴（顺排气方向）
      - OZ = 通过叶根形心指向叶尖
      - OY = OX 绕 OZ 顺转 90°
    - （2）**局部 x-y-z**
      - 各截面形心处，各轴平行对应总体轴

- 三·计算状态的选取
  - 1. 选取逻辑
    - （1）离心力 → 取最大转速 n_max
    - （2）气体力 → 兼顾最大和最小两种极端，使离心弯矩与气动弯矩相互抵消
  - 2. 四种典型状态
    - （1）**A 状态**（设计 / 地面试车）：H=**0**，V=**0**，n=n_max
    - （2）**B2 状态**（低空高速 / 最大气动）：H=**0**，V=V_max，n=n_max，T=**313K**
    - （3）**C 状态**（高空低速 / 最小气动）：H=H_max，V=V_min，n=n_巡航
    - （4）**D/B1 状态**（最大热负荷）：T₄=T_max，n=n_max（限制使用时间）

- 四·离心拉伸应力计算
  - 1. **离心拉伸应力公式推导（范本）**
    - **目标**：求任意截面 Zⱼ 的离心拉伸应力 σ_c。
    - （1）取叶高 Z 处微元段 dZ，其质量：
      $$dm = \rho A(Z) \, dZ$$ （密度 ρ × 微元体积 A(Z)dZ）
    - （2）该微元产生的离心力：
      $$dP = dm \cdot \omega^2 R(Z) = \rho \omega^2 R(Z) A(Z) \, dZ$$ （离心力定义 F = m·ω²r）
    - （3）Zⱼ 截面承受其**以上全部**微元的离心力之和（叶尖自由、叶根固支）：
      $$P(Z_j) = \int_{Z_j}^{H} \rho \omega^2 R(Z) A(Z) \, dZ$$ （沿叶高从 Zⱼ 积分到叶尖 H）
    - （4）除以该截面面积 A(Zⱼ) 得平均拉应力：
      $$\sigma_c(Z_j) = \frac{P(Z_j)}{A(Z_j)} = \frac{1}{A(Z_j)} \int_{Z_j}^{H} \rho \omega^2 R(Z) A(Z) \, dZ$$ （应力 = 力 / 面积）
    - （5）**推论**：Zⱼ 越小（越靠叶根）积分限越长 → **叶根应力最大（危险截面）**；叶尖 Zⱼ→H 积分→0 → σ_c = 0
  - 2. 数值积分法
    - （1）将叶片沿叶高分 **n** 段（一般 ≥ **5 段**）
    - （2）从叶尖到叶根逐段累加
    - （3）分段越多越精确
  - 3. 应力分布规律

    ```
    应力 σ_c
      ↑
      │         等截面叶片
      │           ╱
      │          ╱
      │         ╱
      │        ╱
      │       ╱
      │      ╱
      │     ╱
      │    ╱
      │   ╱
      │  ╱
      │ ╱
      └──────────────────→ 叶高 Z
      叶尖(0)          叶根(max)
    ```
    - （1）Z 越小（越靠近叶根）→ 应力越大
    - （2）**叶根截面应力最大**（危险截面）
    - （3）叶尖处离心拉应力为零
    - （4）等截面叶片应力集中在叶根，材料利用率低
  - 4. 叶根平均离心拉伸应力

    $$\bar{\sigma}_{c,root} \propto S \cdot n_{max}^2$$

    - （1）S = 流道截面环形面积
    - （2）长叶片（一级风扇）→ 流道面积大 → 需限制转速
  - > [!note] 核心理解
    > 物理上：离心力本质是"质量 × 向心加速度"，叶片从叶尖到叶根逐渐承受更多质量产生的力，所以越靠根应力越大，这与悬臂梁根部弯矩最大的直觉一致。涡轮/风扇叶片往往做成**根部粗、尖部薄**的变截面形状，就是为了在叶根多承担质量的同时不浪费材料。
    > 数学上：离心拉伸应力公式是**质量累积积分**的经典应用——从自由端（叶尖 σ=0）向固支端（叶根 σ_max）积分质量力，除以截面积即得逐点应力分布，这是材料力学中"截面法"在旋转结构中的直接体现。

- 五·弯曲应力计算
  - 1. 气体力弯矩
    - （1）基本原理：气流流过叶栅时动量变化 → 叶片对气流的力 → 反作用力作用于叶片
    - （2）近似估算法：忽略 p_x、p_y 沿叶高变化，取平均半径恒定值 → 均布载荷悬臂梁
    - （3）数值积分法：将叶片分 **n** 段 → 每段气流参数取平均值 → 逐段计算力 → 逐截面累加力矩
  - 2. 离心力弯矩
    - （1）产生原因：各截面形心连线不与 Z 轴重合 → 离心力产生附加弯矩
    - （2）计算方法与气体力弯矩数值积分法相同
    - （3）近似处理：接近等截面的压气机叶片可取整叶身作为一段估算
  - > [!note] 核心理解
    > 物理上：叶片设计核心技巧：让**离心力弯矩与气体力弯矩相互抵消**——这正是叶片截面沿叶高扭转、形心连线特意弯曲的原因。两者相反相消时净弯曲应力很小。这个思想是叶片"弯-扭-掠"三维设计的力学根源。
    > 数学上：总弯矩 M_total = M_gas + M_cf，设计目标是通过调整叶片几何使 M_gas ≈ -M_cf 即相互抵消。这本质上是一个**多约束优化问题**——气动性能要求 M_gas 分布不可随意改动，只能通过形心轨迹（即叶片弯扭）调控 M_cf，使之尽量与 M_gas 反向等值。

- 六·影响叶片强度的因素
  - 1. 扭转应力
    - （1）扭转中心与形心 / 压力中心不重合时产生
    - （2）简化计算中假设三者重合以忽略
  - 2. 热应力
    - （1）压气机叶片：较薄，温差小 → 问题不大
    - （2）涡轮叶片：高温（~**1700℃**）+ 厚度差大 + 内部冷却通道 → **热应力显著**
    - （3）热疲劳：每起停一次 → 一次交变热应力 → 裂纹产生
    - （4）缓解措施
      - 减小叶片厚度差（排气边缘不过薄）
      - 选用高导热材料
      - 冷却方法：空心冷却、冲击冷却、对流通道、前缘气膜冷却、尾缘开缝冷却
  - 3. 蠕变
    - （1）定义：应力 < 弹性极限时，应变随时间延长而增加
    - （2）两种表现
      - **蠕滑**：变形逐渐增大
      - **松弛**：应力逐渐减小
    - （3）危险点：最大总应力点应力值随时间减小，但附近应变松弛变大
  - 4. 柔性叶片弯曲变形
    - （1）抗弯刚度大（厚叶片）→ 变形小，可忽略
    - （2）抗弯刚度小（薄叶型、大展弦比风扇/压气机叶片）→ 弹性变形显著
    - （3）**自然补偿**（补）：变形导致离心力径向分量产生附加弯矩 → 对气体力弯矩补偿更好
      - 忽略变形偏于安全但不利于减重
      - 高性能发动机追求推重比 → 必须用柔性叶片方法精细计算

## 第四章：轮盘强度

- 一·概述与破损形式
  - 1. 轮盘是燃气涡轮关键旋转件，承载叶片离心力
    - 破裂失效 → 非包容性严重事故
  - 2. 四种破损形式
    - （1）外缘榫头断裂
    - （2）外缘径向裂纹（尤其在销孔处）
    - （3）材料内部缺陷（松孔/夹杂）→ 盘中心破裂
    - （4）高温蠕变（局部颈缩）→ 盘外径增大 → 叶尖碰磨
  - 3. 轮盘结构设计准则（7 项）
    - （1）足够应力储备
    - （2）限制有害变形
    - （3）防止轮盘破裂
    - （4）足够低周疲劳寿命
    - （5）防止有害振动（共振/颤振）
    - （6）足够裂纹扩展寿命
    - （7）防止轮盘屈曲

- 二·主要负荷
  - 1. 叶片质量离心力 + 轮盘自身离心力
    - （1）离心力 ∝ ρ·ω²·r，转速越高应力越大
  - 2. 热负荷（径向温差引起）
    - （1）压气机盘：热负荷一般可忽略；后几级温差大时不可忽略
    - （2）涡轮盘：全盘温差明显，非稳定工况更严重
  - 3. 其他负荷（常规计算中暂不考虑）
    - （1）叶片传来的气动力（轴向/周向）
    - （2）机动飞行的陀螺力矩
    - （3）振动动负荷
    - （4）装配应力 / 变形不协调附加应力
  - 4. 设计特点：追求高推重比 → 强度裕度适当、应力水平高、材料利用率优

- 三·应力计算——轴对称薄盘模型
  - 1. 基本假设
    - （1）**轴对称**——几何、载荷、温度场均轴对称
    - （2）**平面应力**——盘外径/厚度 > **4**，按"薄盘"处理，应力沿轴向不变
    - （3）**弹性假设**——材料处于弹性状态
  - 2. 弹性力学方程组（圆柱坐标系 r-θ）

    ```
    微元体（r × dθ × dr × h）：
      ┌──────┐
      │σ_r   │──→ σ_r + dσ_r
      │      │     周向应力 σ_θ（两侧面）
      └──────┘
           重心处离心力 dC = ρ·ω²·r·dV
    ```
    - （1）**平衡微分方程**：
      $$\frac{d\sigma_r}{dr} + \frac{\sigma_r - \sigma_\theta}{r} + \rho \omega^2 r = 0$$
    - （2）**几何方程**：
      $$\varepsilon_r = \frac{du}{dr}, \quad \varepsilon_\theta = \frac{u}{r}$$
    - （3）**物理方程**（热弹性）：
      $$\varepsilon_r = \frac{1}{E}(\sigma_r - \nu \sigma_\theta) + \alpha t$$
      $$\varepsilon_\theta = \frac{1}{E}(\sigma_\theta - \nu \sigma_r) + \alpha t$$
  - 3. 位移法求解
    - （1）代入消去 → 以位移 u 表示的微分方程
      $$\frac{d^2 u}{dr^2} + \frac{1}{r}\frac{du}{dr} - \frac{u}{r^2} + \frac{1-\nu^2}{E}\rho \omega^2 r + \alpha(1+\nu)\frac{dt}{dr} = 0$$
    - （2）解出 u → 代入几何/物理方程 → σ_r、σ_θ

- 四·等厚轮盘应力分析
  - 1. 等厚盘一般解
    - （1）应力 = 边界条件部分 + 离心应力部分 + 热应力部分（**线性叠加**）
    $$\sigma_r = K_1 + \frac{K_2}{r^2} - \frac{3+\nu}{8}\rho \omega^2 r^2$$
    $$\sigma_\theta = K_1 - \frac{K_2}{r^2} - \frac{1+3\nu}{8}\rho \omega^2 r^2$$
  - 2. 等温实心等厚盘（无温度梯度）
    - （1）边界：r=0 处应力有限，r=r_a 处 σ_r = σ_a（叶片离心力等效径向应力）
    - （2）特征：**盘心处 σ_r = σ_θ 最大**——危险点在盘心
    - （3）转速↑ → 盘心应力↑ → 限制最大转速

> [!note] 核心理解
> 物理上：轮盘像一个高速旋转的"离心机盘"——外缘拉叶片，自身质量产生径向向外飞散的倾向。材料被"径向拉伸、周向胀开"，所以 σ_r 和 σ_θ 同时存在，盘心处二者相等且最大。
> 数学上：轮盘应力是轴对称弹性力学问题的经典解，三个方程（平衡＋几何＋物理）消元后归结为一个二阶 ODE，边界条件确定积分常数。离心项和热应力项线性叠加，使设计可以独立分析转速和温度的影响。

## 第五章：叶片振动

- 一·概述
  - 1. 叶片振动是最常见的损坏原因——统计表明叶片故障绝大部分由振动引起
    - 振动交变载荷 + 离心静载 → 疲劳裂纹 → 瞬断 → 碎片飞出
  - 2. 振动原因三类型
    - （1）**强迫振动**：尾迹激振、旋转失速/喘振气流激振
    - （2）**自激振动**：颤振
    - （3）其他随机振动
    - → （1）（2）危险性最大
  - 3. 三个关键研究因素
    - （1）自振频率（固有频率）——范围数十 Hz ~ 万余 Hz，低频最危险
    - （2）振型——各部位振幅相对关系
    - （3）振动应力——疲劳寿命依据
  - 4. 疲劳断口特征
    - （1）**疲劳源**（贝壳花纹中心）
    - （2）**疲劳扩展区**（光滑贝壳纹）
    - （3）**瞬时断裂区**（粗糙断面）

- 二·等截面叶片（梁）弯曲振动
  - 1. 梁弯曲振动微分方程
    - （1）微元段简谐运动：$y(x,t) = y_0(x)\cos(pt)$
      - p = 自振角频率，y₀(x) = 振型函数（弹性线）
    - （2）力和力矩平衡 → 偏微分方程
      $$EJ\frac{\partial^4 y}{\partial x^4} + \rho A \frac{\partial^2 y}{\partial t^2} = 0$$
    - （3）分离变量 → 常微分方程
      $$\frac{d^4 y_0}{dx^4} - k^4 y_0 = 0, \quad k^4 = \frac{\rho A p^2}{EJ}$$
    - （4）通解：$y_0 = A\sin kx + B\cos kx + C\sinh kx + D\cosh kx$

  - 2. 悬臂梁固有频率与振型（转子叶片模型）
    - （1）边界条件：固装端 y₀=0、y₀'=0；自由端 y₀''=0、y₀'''=0
    - （2）**频率方程**：$\cos kl \cdot \cosh kl + 1 = 0$
    - （3）特征值：
      $$k_1 l = 1.875, \quad k_2 l = 4.694, \quad k_3 l = 7.855$$
    - （4）固有频率：
      $$p_n = (k_n l)^2 \sqrt{\frac{EJ}{\rho A l^4}}$$
    - （5）**频率比**：p₁ : p₂ : p₃ = **1 : 6.27 : 17.55**

  - 3. 双简支梁固有频率（静子叶片模型）
    - 频率方程：$\sin kl = 0$ → $k_n l = n\pi$
    - 基频：$p_1 = \pi^2 \sqrt{EJ/(\rho A l^4)}$

  - 4. 固有频率影响因素
    - （1）**材料影响很小**：钢/铝/钛合金的 E/ρ 值相差不超过 **2%**
    - （2）**约束显著**：自由端→铰支→固支，自由度↓ → 频率↑
    - （3）**温度↑ → E↓ → 频率↓**：最大影响约 **19%**（钛合金最大）

- 三·实际叶片频率影响因素
  - 1. **扭向影响**（宽厚比大时显著）
    - 实际叶片扭曲 → 双向弯曲耦合振动，出现新频率和新振型
    - 压气机叶片宽厚比大 → 扭角影响大
  - 2. 离心力影响
    - 旋转 → 离心力场增加刚性 → **动频 > 静频**
    $$f_d = \sqrt{f_0^2 + B n^2}$$
    - B = 离心力影响系数（与边界和阶次有关）
  - 3. 温度影响
    - 温度↑ → 弹性模量 E↓ → 频率↓
    $$\frac{f_t}{f_0} = \sqrt{\frac{E_t}{E_0}}$$

> [!note] 核心理解
> 物理上：叶片振动是一个"分布质量-弹性"系统，类似一根弹簧上串着无数小质量。每阶振型对应一种特定的"弹-惯"能量分布形态。基频最危险因为振幅最大、能量最集中。
> 数学上：从梁的弯曲振动偏微分方程出发，分离变量后化为四阶常微分方程的特征值问题——频率由特征值 $k_n l$ 决定，振型由三角函数和双曲函数组合描述。边界条件决定特征值，这就是为什么约束类型对频率影响如此之大。

## 第六章：轮盘振动

- 一·圆盘振动形式
  - 1. **第一类——伞形振动**（节圆型）
    - 节线为同心圆，中心固定 → 一阶/二阶伞形
    - 一般不易导致盘损坏
  - 2. **第二类——扇形振动**（节径型）
    - 节线为径向直线（节径），记为 0/1、0/2
    - ==在实际发动机中最容易引起轮盘损坏==
  - 3. **第三类——复合振动**
    - 伞形+扇形组合，自振频率高 → 危险性较小

- 二·薄等厚圆盘自振频率
  - 1. 薄板弯曲振动方程（极坐标）
    - 横向位移 w(r,θ,t) 满足：
    $$D\nabla^4 w + \rho h \frac{\partial^2 w}{\partial t^2} = 0$$
    - 弯曲刚度：$D = Eh^3/[12(1-\nu^2)]$
    - 二维拉普拉斯算子：$\nabla^2 = \frac{\partial^2}{\partial r^2} + \frac{1}{r}\frac{\partial}{\partial r} + \frac{1}{r^2}\frac{\partial^2}{\partial \theta^2}$
  - 2. 固有频率公式
    - 分离变量：w = R(r)cos(mθ)cos(pt)，m = 节径数
    - 径向控制方程为贝塞尔方程：
    $$\frac{d^2R}{dr^2} + \frac{1}{r}\frac{dR}{dr} + (k^2 - \frac{m^2}{r^2})R = 0$$
    - 通解为**m阶贝塞尔函数**组合
    - 固有频率：
    $$p_{mn} = \frac{\lambda_{mn}}{r_0^2}\sqrt{\frac{D}{\rho h}}$$
    - λₘₙ 取决于边界条件 (m=节径数, n=节圆数)

- 三·影响盘自振频率的四大因素
  - （1）**盘半径** r₀：$p \propto 1/r_0^2$
  - （2）**厚度** h：$p \propto \sqrt{h^2} = h$（厚盘频率高）
  - （3）**材料**：$p \propto \sqrt{E/\rho}$，与叶片类似
  - （4）**支撑方式**：自由度↓ → 频率↑
  - （5）**叶片耦合**：盘+叶片耦合振动，出现新频率和新振型
  - （6）**离心力**：转速↑ → 动频↑（离心力增加盘刚性）
  - （7）**温度**：E↓ → 频率↓；径向温差 → 热应力影响

- 四·动波与临界转速
  - 1. 行波振动
    - 扇形振动的节线（动波）以角速度 ±p/m 在盘上旋转
    - 盘上质点每通过一个动波振动一次 → 动波转一周质点振动 m 次
  - 2. 驻波与临界
    - 旋转盘中，向前传播的动波与盘同向、向后传播的反向
    - 当向后传播的动波在绝对坐标系中静止（ω = p/m）→ **共振临界**
    - 这是盘片振动设计与转速避开的关键依据

> [!note] 核心理解
> 物理上：轮盘振动比叶片更复杂——它是"二维板"而非"一维梁"，存在节圆和节径两个维度的振型。扇形振动（节径型）最危险，因为发动机的静子周向不均匀流场正好激励节径型共振。
> 数学上：薄板振动偏微分方程在极坐标下分离变量后，径向部分归结为**贝塞尔方程**——这是旋转圆盘结构的数学特征。频率由无量纲特征值 λₘₙ 决定，该值由边界条件（盘缘是否自由/固支）查表取得。

## 第七章：转子振动与平衡

- 一·概述
  - 1. 转子不平衡 → 不平衡离心力 → 通过支承传到静子 → 整机/飞机振动
  - 2. **临界转速**——转子的固有特性
    - 转速达到某特定值时 → 挠度明显增大
    - 超过后 → 挠度又明显减小
    - 本质：ω_cr 与转子弯曲振动固有频率 p 在数值上一致

- 二·单盘转子临界转速
  - 1. 模型：轴中央装薄盘，两端铰支，忽略轴质量，垂直放置（避重力）

    ```
    ════[○]════      盘质量 m，偏心距 e
    ▲              ▲  轴刚性系数 c
    支座           支座
    ```

  - 2. 稳态平衡方程
    - （1）离心力：$F_c = m(y+e)\omega^2$
    - （2）弹性恢复力：$F_e = c y$
    - （3）两力平衡：
      $$m(y+e)\omega^2 = c y \quad → \quad y = \frac{m e \omega^2}{c - m \omega^2} = \frac{e}{(\omega_{cr}/\omega)^2 - 1}$$
    - （4）**临界转速**：$\omega_{cr} = \sqrt{c/m}$

  - 3. 三种工作状态
    - （1）$\omega < \omega_{cr}$（**亚临界**）：y/e > 0，挠度与偏心同向
      - 质心在轴挠曲线外侧，转速↑ → 挠度↑
    - （2）$\omega = \omega_{cr}$（**临界**）：挠度→∞（无阻尼理想情况）
    - （3）$\omega > \omega_{cr}$（**超临界**）：y/e < 0，挠度与偏心反向
      - 质心在轴挠曲线内侧 → 自动对中 → 挠度趋于 e
      - ==超临界时质心自动落到旋转轴线上（自定心效应）==

- 三·轴质量的影响
  - 1. 光轴（无盘）的临界转速
    - 微元段离心力作为分布载荷 → 与梁弯曲振动方程同形
    - 等截面双简支轴特征值：$k_n l = n\pi$
    - 临界转速公式：
    $$\omega_{cr,1} = \pi^2 \sqrt{\frac{EJ}{\rho A l^4}}$$
  - 2. 轴质量不可忽略时，将光轴临界转速折算成中央集中质量的修正

- 四·工程意义
  - 1. 工作转速应避开临界转速附近（通常 ±**20%**）
  - 2. 提高穿越临界转速的速率 → 减小峰值挠度
  - 3. 动平衡减小偏心距 e → 减小同转速下的稳态挠度
  - y/e 约束条件决定不准停留的转速范围

> [!note] 核心理解
> 物理上：当转速接近系统固有频率时，即使微小的不平衡也会激起巨大的振动位移——这是"共振"在旋转机械中的体现。有趣的是越过临界后反而趋于平稳（自定心效应），这是旋转机械区别于一般振动系统的特有现象。
> 数学上：单盘转子模型的挠度公式 y = e/[(ω_cr/ω)²-1] 是典型的**受迫振动幅频响应**——不平衡质量偏心 e 是激振力源，临界转速 ω_cr 是系统的固有频率。公式分母过零点对应无限大响应（无阻尼时），工程上通过阻尼和快速穿越来控制峰值。

## 第八章：燃气轮机性能指标

- 一·推力性能指标
  - 1. **推力 F**
    $$F = \dot{m}_a(v_e - v_i) + (p_e - p_a)A_e$$
    - 动量推力 + 压力推力；典型量级数十~数百 kN
  - 2. **单位推力 F_s = F / \dot{m}_a**：衡量空气利用效率，**600–750 N·s/kg**
  - 3. **推重比 F/W**：**6–10+**，现代军用已超 **10**
  - 4. **单位迎面推力 F/A_max**：**8000–10000 daN/m²**，影响安装阻力
  - 5. 有效推力 F_ef = F - 附加阻力 - 波阻 - 摩擦阻力

- 二·经济性能指标
  - 1. **耗油率 TSFC** = $\dot{m}_f/F$
    - 涡喷：**0.8–1.0 kg/(daN·h)**；涡扇：**0.5–0.6**
  - 2. **热效率 η_th** = W_net / Q_in（**30%–50%**）
  - 3. **推进效率 η_p** = 2 / (1 + v_e/v_0)（**50%–80%**）
    - v_e → v_0 时最高，涡扇更优
  - 4. **总效率 η_o = η_th × η_p**（**20%–30%**）
  - 5. 关键权衡：提高 v_e → 单位推力↑但推进效率↓ → 涡扇用大涵道比折中

- 三·热力学循环指标（布雷顿循环）
  - 1. 理想循环效率：$\eta_{th} = 1 - \pi^{-(\gamma-1)/\gamma}$
  - 2. **功比**：涡轮功 / 压气机功 ≈ 2–3
  - 3. 实际循环修正：考虑部件效率、压力损失、燃烧不完全

- 四·部件级性能指标
  - 1. **压气机效率 η_c**：**85%–90%**
  - 2. **涡轮效率 η_t**：**87%–92%**
  - 3. **燃烧室效率 η_b**：**98%–99.5%**
  - 4. **总压恢复系数 σ**：燃烧室 **0.95–0.97**，进气道 **0.95–0.98**
  - 5. **涵道比 BPR**（涡扇）：**5–12**（民用大涵道），**0.3–1**（军用加力）

- 五·关键气动参数
  - 1. **增压比 π**：**15–50**（现代），取决级数 × 级压比
  - 2. **涡轮前温度 T₄**：**1500–2000K**（现代）
  - 3. **排气速度 v_e**：**300–600 m/s**（涡扇），**500–1000 m/s**（涡喷）
  - 4. **单位迎风面积流量**：**150–200 kg/(s·m²)**

## 速记卡 / 一句话总结

- 燃气轮机 = 压气机 + 燃烧室 + 涡轮（Brayton 循环），涡轮流输出约 **1/3** 为净功
- 燃机效率由**燃气初温**和**压缩比**双参数决定；F 级燃机温度约 **1400℃**（在役主流）
- 级 = 转子（工作轮）+ 静子（整流器），单级压比 **1.2–1.5**，总压比需 **25–50**（10+ 级）
- 速度三角形 4 参数（C₁ₐ、C₁ᵤ、U、ΔWᵤ）相互制约，设计是多目标权衡
- 预旋可调和速度三角形：正预旋在 U、C₁ₐ 不变时降低 W₁（例 Mw₁ **0.78→0.65**）
- 压气机本质是**逆压力梯度扩压**→ 附面层易分离 → 设计核心是与分离做斗争
- 叶型损失 5 类：附面层摩擦、分离、尾迹、掺混、波阻
- 叶片强度 4 大载荷：离心力（**1 万倍自重**）、气体力、热负荷（涡轮 ~**1700℃**）、振动
- 离心拉伸应力叶根最大，∝ S·n²；长叶片（风扇）需限制转速
- 叶片设计的力学精髓：**离心弯矩与气动弯矩相互抵消** → 弯-扭-掠三维设计
- 径向间隙控制在叶片高度的 **0.5%–1%**；间隙主动控制可提效
- 推重比：F119（F-22）≈ **10**，F135（F-35）≈ **10–12**，升力发动机目标 **20–24**
- 轮盘强度：盘心 σ_r = σ_θ 最大，应力由离心力 + 热应力线性叠加
- 叶片振动：悬臂梁频率比 p₁:p₂:p₃ = **1:6.27:17.55**，材料影响<**2%**，温度影响达**19%**
- 轮盘振动：扇形振动（节径型）最危险，频率 p ∝ **1/r²**（半径影响最大）
- 转子临界：超临界状态自定心效应，工作转速应避开临界 ±**20%**
- 性能指标：涡扇 TSFC **0.5–0.6**、热效率 **30–50%**、推重比 **6–10+**
- 部件效率：压气机 **85–90%**、涡轮 **87–92%**、燃烧室 **98–99.5%**
