## 一·调压回路

一·**定义**：用溢流阀将系统压力限定在设定值，使系统在恒定（或多级）压力下工作。==最基本的回路，几乎所有液压系统都有==。

二·**单级调压**（最常见）：溢流阀（先导式）并联在泵出口。泵输出压力 = 溢流阀设定压力（定量泵系统）或溢流阀作安全阀（变量泵系统）。

三·**多级调压**（快慢进切换）：多个溢流阀 + 电磁换向阀组合——根据工况切换不同设定压力的溢流阀接入：
  - 工进（高压）→ 高压溢流阀
  - 快进/辅助（低压）→ 低压溢流阀
  - 应用：注塑机（锁模高压→注射中压→顶出低压）

  ```
  多级调压示意：
  定量泵 ──┬──→ 电磁换向阀 ──→ 低压溢流阀（p₁）
          │                  (通电时接入低压阀)
          └──→ 主溢流阀（p₂，高压）
  ```

## 二·减压回路

一·**定义**：将某个支路的压力降低（低于主系统压力）并保持稳定。用**减压阀串联**在支路入口实现。

二·**典型应用**：机床夹紧油路（夹紧缸需低于主系统的工作压力，以免夹坏工件）；控制油路（先导控制需 ~ **0.5~2 MPa** 稳定低压源）

三·**回路结构**：
  ```
  主系统高压 ─→ 减压阀（设定 p₂） ─→ 支路负载
  （p₁ > p₂）              │
                          └→ 单向阀（防止倒流）
  ```

四·**注意事项**：
  - 减压阀后的支路负载不应对主系统压力产生反冲影响——减压阀出口必须有**单向阀**（防止主系统失压时支路油回流）。
  - 减压回路中不能长时间保持"减压后"状态而不取油（减压阀先导油会微量泄漏 → 出口压力逐渐升高 → 需定期泄压）。

## 三·增压回路

一·**定义**：用低压源通过增压器（增压缸）将某局部压力放大到高于主系统压力——以解决主泵压力不够满足局部超高压需求的问题。

二·**增压器原理**：基于帕斯卡力平衡——两端面积比即增压比。
  ```
  低压区 p₁, A₁      高压区 p₂, A₂
  ┌───────────┬──────┬──────────┐
  │           │活塞/ │           │
  │  p₁·A₁    │柱塞  │  p₂·A₂   │
  │           │      │           │
  └───────────┴──────┴──────────┘
  力平衡：p₁·A₁ = p₂·A₂ → p₂ = p₁·(A₁/A₂)
  ```

三·**结论**：增压比 = $A_1/A_2$。若 $A_1/A_2=3$，则可得到 **3 倍**主系统压力的超高压输出（如主压 10 MPa → 增压器输出 30 MPa）。

四·**应用**：液压机在最后压制阶段的增压（低压快进→高压增压慢压）；注塑机超高压保持（锁模力远大于注射力）。

## 四·卸荷回路

一·**定义**：当执行元件不工作时，让泵的压力油以**极低压**（接近零压）回油箱，减少功率损失和发热。==节能是卸荷回路的第一目标==。

二·**常用卸荷方式**：

  1.**用三位换向阀中位卸荷**（最简单）：
    - 如 M 型中位数机能（P 通 T，A/B 锁紧）——泵卸荷，执行元件锁紧
    - 如 H 型（P-A-B-T 全通）——泵卸荷、执行元件浮动
    - 注意：中位卸荷时卸荷压力 = 回油管路背压（约 **0.2~0.5 MPa**），不是零

  2.**用卸荷阀（带二位二通电磁阀）**：在泵出口并联一个二位二通阀 → 工作结束→通电→泵直接回油箱→卸荷

  3.**用先导式溢流阀远程卸荷**：在先导式溢流阀远程口 K 接小电磁阀 → 电磁阀通电→K 接油箱→溢流阀全开卸荷（≈泵排空）→ 卸荷压力低（约 **0.1~0.2 MPa**）

  4.**蓄能器—卸荷回路**：当蓄能器充到足以维持压力→压力继电器发信号→溢流阀卸荷→泵停机→蓄能器维持系统压力→压力低时重新启动泵

  > [!tip] 补充
  > 卸荷压力越小越好（理想 0），但实际管路背压+阀阻力决定最低卸荷压。M 型中位卸荷约 **0.5 MPa**，溢流远控卸荷可低至 **~0.1 MPa**。工程机械中"卸荷空载"是降低燃油消耗的最有效措施之一。

## 五·保压回路

一·**定义**：液压缸需要在一段时间内保持一定压力（如液压机保压压制、夹紧缸保持夹紧力），即使泵已停止或已卸荷——保压回路负责维持压力不下降。

二·**常用保压方式**：

  1.**蓄能器保压**（最实用、最常用）：
    - 系统压力正常时给蓄能器充油 → 蓄能器储存压力能
    - 泵卸荷/停止后 → 蓄能器压力油通过单向阀维持系统压力 → 补偿泄漏 → 长时间保压
    - 应用：夹紧回路（机床夹具断电不掉）、液压机保压（慢放）

  2.**液控单向阀（液压锁）保压**：在液压缸两腔油路中各装液控单向阀——液控单向阀在无控制油时紧锁油路 → 缸内压力因容积密闭而不降（理论上滴水不漏——但实际有油液压缩+温度变化造成的缓慢泄压）。

  3.**补油泵保压**：用一个极小流量的小泵在系统保压期间持续补油（补偿泄漏）+ 溢流阀限压。精度高但持续耗电。

---

## 速记卡

- 调压回路：溢流阀（单级）、多个溢流阀+换向阀（多级）
- 减压回路：减压阀串联支路，降低局部压力；必须加单向阀
- 增压回路：增压器（帕斯卡力平衡 $p₂ = p₁·A₁/A₂$）——局部超高压
- 卸荷回路：M 型中位、先导溢流远程卸荷、蓄能器卸荷——节能核心手段
- 保压回路：蓄能器、液压锁、补油泵——各有利弊（蓄能器最通用）
- ==卸荷≈不干活时少耗能，保压≈干活时维持住压力==——这是回路设计的两大节能目标
