## 一·液压缸结构组成

一·**整体组成**：液压缸（以单杆双作用缸为例）由以下主要零部件组成：

  ```
  进油口A                             进油口B
    │                                    │
    ▼                                    ▼
  ┌──────┬─────────────┬──────┐
  │端盖   │  缸筒       │ 端盖  │
  │      │   ┌─────┐   │      │
  │  A───┤── │活塞 │─ ├───B  │   ← A/B油口
  │      │   │     │   │      │
  │      │   └──┬──┘   │      │
  │      │      │ 活塞杆│      │
  │      └──────┼──────┘      │
  ├──────┤密封   │密封 ├────┤
  │缓冲   │      │      │缓冲 │
  └──────┴──────┴──────┴──────┘
  ```

二·**主要部件功能**：
  1.**缸筒**：承受油液压力（最重的结构件）——材质要求强度高、内表面粗糙度低、耐磨损。
  2.**活塞**：钢（+耐磨环）或铸铁、铝合金——通过密封圈与缸筒密封；将液体压力转化为推力。
  3.**活塞杆**：钢、表面镀铬防锈——传递力的输出端；靠导向套导向。
  4.**端盖**：连接缸筒两端、含油口/密封/缓冲机构。
  5.**密封装置**：活塞密封（内泄） + 活塞杆密封（外泄） + 静密封（端盖/法兰）。
  6.**缓冲装置**：行程末端减速防冲击。
  7.**排气装置**：排除缸内积存空气。

## 二·缓冲装置

一·**为什么要缓冲**：当活塞高速运动撞到端盖时，若不减速 → 巨大冲击 + 噪声 + 密封破坏。缓冲装置在行程末端（约 **5~20 mm**）通过节流增大背压实现减速。

二·**常见缓冲形式**：
  1.**节流缓冲**（最常用）：活塞尾部有缓冲柱塞/节流槽/可调锥面。当缓冲柱塞进入端盖缓冲孔时，回油必须经过逐渐缩小的节流间隙排出 → 背压升高 → 活塞减速。
  2.**可调缓冲**：在缓冲腔外侧并联一个可调节流阀（调节锥阀开度）---用户可在外部调整缓冲强度（适应不同工况）。
  3.**缓冲圈/液压阻尼**：柱塞进入后，排油通道突然变细，形成油垫阻尼。

三·**特点**：
  - 缓冲能力取决于缓冲腔长度、节流孔尺度、系统压力
  - 不可调缓冲受制于出厂设定（不能适应负载变化）

## 三·排气装置

一·**为什么需要排气**：液压系统中（新装/维修/油液更换后）溶解在液压油中的空气在高压下析出并在缸顶部聚集 → 空气的可压缩性会导致爬行、响应延迟和噪声。

二·**排气装置形式**：
  1.**排气螺塞/单向排气阀**：在缸的两端最高点设置排气口。在活塞低速往返运动中，旋松排气阀→气体排出→气泡逸出（直至有油液连续流出）→拧紧。
  2.**自动排气阀**：浮子式或阀芯式——气多的时候排、有油就关（自动免维护）。

## 四·液压缸设计计算步骤

  > [!tip] 补充
  > 以下为工程标准设计流程（按 JB/T 10205-2014 等标准），课程总结只列了大纲，这里展开。

一·**第 1 步：按工作压力确定缸筒内径 $D$**
  1.已知最大推力需求 $F_{max}$ 和系统工作压力 $p$；
  2.取作用在无杆腔（伸出工况）的推力：
  $$ F = p \cdot A_1 = p \cdot \frac{\pi}{4} D^2 $$
  3.考虑摩擦损失和安全系数（取 $\eta_{缸} \approx 0.9\sim 0.95$）：
  $$ D = \sqrt{\frac{4 F_{max}}{\pi p \eta_{缸}}} $$
  4.圆整为标准缸径（GB/T 2348 标准系列：40, 50, 63, 80, 90, 100, 110, 125, 140, 160, 180, 200, 220, 250 mm 等）。

二·**第 2 步：按速比确定活塞杆直径 $d$**
  1.需求速比 $\phi = v_2/v_1$（缩回速度/伸出速度）
  $$ \phi = \frac{A_1}{A_2} = \frac{D^2}{D^2 - d^2} \quad \Rightarrow \quad d = D \sqrt{\frac{\phi-1}{\phi}} $$
  2.经验取值：工作压力 ≤10 MPa 时 $d \approx (0.5\sim 0.55)D$；高压（16~32 MPa）时 $d \approx (0.6\sim 0.7)D$（受压稳定性要求）。
  3.圆整为系列标准杆径。

三·**第 3 步：验算缸筒壁厚 $\delta$**
  1.按薄壁圆筒（当 $\delta/D \le 0.1$）：
  $$ \delta \ge \frac{p_{max} D}{2 [\sigma]} $$
  其中 $[\sigma]$ 为缸筒许用应力（材料屈服强度 ÷ 安全系数 $n \approx 2\sim 4$）
  2.若 $\delta/D > 0.1$ 需用厚壁圆筒公式（Lamé 公式），课程中少用、了解即可。

四·**第 4 步：活塞杆强度与稳定性校核**
  1.**强度校核**（拉/压应力）：
  $$ \sigma = \frac{F_{max}}{A_{杆}} = \frac{F_{max}}{\frac{\pi}{4} d^2} \le [\sigma] $$
  2.**稳定校核（弯曲——仅对长杆受压）**：当 $s/d > 10$ 时（杆长度 ≥ 10倍杆径），需按欧拉公式（或经验 Johnson 公式）校核抗压临界失稳：
  $$ F_{cr} = \frac{\pi^2 EI}{(\mu s)^2} $$
  其中 $I$ = 杆截面惯性矩（$I = \pi d^4/64$），$\mu$ 为长度系数（两头铰接=1, 一头固定=0.7, 两端固定=0.5），$E$ = 弹性模量（钢 ~ 2.1×10¹¹ Pa）。
  要求：$F_{cr} / F_{max} \ge n_{cr}$（稳定安全系数 ~ 2~4）。

  > [!note] 核心理解
  > 设计计算本质上就是**由出力求缸径 → 由速比求杆径 → 由压力求壁厚 → 校核杆不失稳**的四步走。反映的物理关系是：力→面积→直径；面积差→速比；壁厚承压→材料强度；长杆受压→欧拉弯曲。

## 速记卡

- 液压缸结构：缸筒、活塞、杆、端盖、密封、缓冲、排气
- 缓冲：行程末节流回油 → 背压升高 → 减速；可调缓冲更灵活
- 排气：缸顶排气阀排空气体 → 防止爬行/噪声/响应延迟
- 设计四步：出力求缸径 D → 速比求杆径 d → 压力求壁厚 δ → 校核杆稳定性
- 杆径经验：低压 0.5D，高压 0.6~0.7D
- 弯曲校核：$s/d>10$ 时必做，用欧拉公式
