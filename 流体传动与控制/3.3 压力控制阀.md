## 一·溢流阀（Relief Valve）

一·**定义**：当系统压力达到预设值时，阀口打开使部分油液溢回油箱，从而限制（或维持）系统压力不超过设定值。溢流阀**并联**于液压系统中（旁接在泵出口与油箱之间）。

二·**工作原理**（共通过程）：系统压力作用在阀芯底面 → 当液压力 > 弹簧预紧力时 → 阀芯抬起 → 阀口打开 → 溢流。

---

## 二·直动式溢流阀 vs 先导式溢流阀

| 项目 | 直动式溢流阀 | 先导式溢流阀 |
|------|-----------|-------------|
| **结构** | 弹簧直接顶住阀芯；阀芯受力 = 弹簧力 - 液压推力 | 先导阀（小锥阀调压）+ 主阀（大滑阀/锥阀受控）；先导阀设定压力、主阀执行溢流 |
| **动态简化** | 溢流口开度 ∝ 压力升高 | 主阀弹簧很柔（仅提供复位力），压力由先导锥阀弹簧决定 |
| **优点** | 结构简单、响应快（无先导延迟） | ==稳定性好==（先导阻尼减振）、开启压力—流量特性平坦（≈ 恒压性好）、可远程调压 |
| **缺点** | 高压时弹簧力需很大（尺寸大）、压力—流量特性陡（开启压力随流量增大显著上升） | 响应比直动慢、结构复杂（多了先导级） |
| **适用** | **低压小流量**（≤ ~6.3 MPa、≤ 40 L/min） | **高压大流量**（≥ ~6.3 MPa，可达 35+ MPa） |
| **压力范围** | 通常 ≤ ~**6.3~8 MPa** | 可达 **35 MPa** 以上 |

  > [!note] 核心理解
  > 直动式就是一根弹簧顶着阀芯——压力高了就推开弹簧溢流。但高压下弹簧必须很硬（F = pA），调压精度低、体积大。先导式用一个小锥阀做"司令"来调压，主阀做大流量"士兵"——司令的弹簧软（调压准），士兵只干开/关/稳的苦活。所以先导式的高压性能远优于直动式。

## 三·溢流阀的四种典型应用

  1.**安全保护（常闭）**：系统正常运行时溢流阀关闭；压力超过安全值时瞬间打开——作为"安全阀"防止超压事故。
  2.**稳压溢流（常开）**：在**定量泵系统**中，泵输出流量 > 执行元件所需 → 多余油液通过溢流阀持续溢回油箱（保持系统压力基本恒定）。==常开——这是最常见的角色==。
  3.**远程调压**：在先导式溢流阀的远程控口 K 之间接一个小型远程调压阀（直动式），即可在远端设定比本阀调节弹簧更低的压力值——实现远程减压或分级调压。
  4.**多级压力控制**：通过电磁换向阀切换接通不同调压阀→改变系统压力等级（如快速=低压、工进=高压）。

## 四·减压阀

一·**定义**：==减压阀将入口高压降为稳定的出口低压==，供给某支路使用。它**串联**在支路中（与负载串联），控制**出口压力**为设定值。

二·**与溢流阀的本质差异**：

| 对比项 | 溢流阀 | 减压阀 |
|-------|-------|--------|
| 连接方式 | **并联**在系统（泵出口到油箱） | **串联**在支路（上游到下游负载） |
| 控制对象 | ==控制入口压力==（进口处保压） | ==控制出口压力==（出口处定压） |
| 常态阀口 | 常闭（正常不溢流，超过才开） | 常开（正常全通，超过才关小） |
| 泄油方式 | 溢流油直接回油箱 | 有单独泄油口 L 回油箱（先导油通过 L 回） |
| 弹簧腔作用 | 弹簧腔与出口相道（阀芯上下压力差决定开度） | 弹簧腔与出口相道（出口压力反馈至弹簧腔与设定平衡） |

三·**工作原理**：
  （1）出口压力 p₂ 经内部通道进入阀芯底部弹簧腔。
  （2）当 p₂ < 设定值：弹簧力 > 液压反力 → 阀口全开 → 压降小。
  （3）当 p₂ 上升到设定值：液压反力 = 弹簧力 → 阀口关到某开度维持平衡 → 保持 p₂ = 设定值。
  （4）负载变大 → p₂ 短暂上升 → 阀口进一步关小（节流损失增大） → p₂ 回落至设定值；负载变小 → 阀口开大 → p₂ 回升。

四·**应用**：机床液压系统的安全工作油路、夹紧油路、润滑油路等——需要比主系统低的稳定压力。

## 五·顺序阀

一·**定义**：利用液压系统的**压力**变化来控制多个执行元件的**顺序动作**。当进口（或控制口）压力达到设定值时，阀口突然打开，接通下游油路。

二·**工作原理**：
  （1）直控顺序阀：压力从进口直接引到阀芯底部 → 当系统压力上升到设定值 → 阀芯被推起 → 阀口开启 → 油液通过去驱动下一个执行元件的动作。
  （2）远控顺序阀：控制口 K 接外部控制油路——可由别处压力信号触发本阀开启（与系统主油路压力无关）。

三·**典型应用——"A 动作完 B 再动"**：
  机床多缸工作时——①液压缸 A 先前进 → ②压力上升（A 到位顶死）→ ③顺序阀打开 → ④油液进入液压缸 B → ⑤B 开始动作。

---

## 六·压力继电器

一·**定义**：将液压力（液压信号）转换为电信号（开关量）的转换元件。当系统压力达到调定值时，压力继电器内部的微动开关接通或断开，输出电信号。

二·**应用**：液压—电气联锁。
  - 压力到达 → 发信号 → 切换电磁阀 → 进入下一工况
  - 压力过低 → 发信号 → 启动补油泵 / 报警
  - 如组合机床动力滑台"死挡铁停留"工况：压力继电器检测到加压压力到 → 发信号 → 开始延时 → 延时到 → 切换电磁阀 → 快退。

---

## 七·溢流阀 vs 减压阀 vs 顺序阀对比

| 项目 | 溢流阀 | 减压阀 | 顺序阀 |
|------|-------|--------|-------|
| 连接方式 | ==并联==（主路到油箱） | ==串联==（支路中） | ==串联==（两执行元件间） |
| 控制压力位置 | 入口 | 出口 | 入口 |
| 常态阀口 | 常闭 | 常开 | 常闭 |
| 泄油方式 | 内泄（出口直接接油箱） | 外泄（L 口单独回油箱） | 外泄（L 口） |
| 主要用途 | 安全 + 稳压 | 降压 + 定压 | 顺序动作 |

---

## 速记卡

- 溢流阀：并联、常闭（安全）或常开（定量泵稳压），先导式适合高压大流量
- 减压阀：串联、常开、控出口定压，用于夹紧/润滑等需要比主系统更低压力
- 顺序阀：串联、常闭、达到设定压力才打开——控制多缸顺序动作
- 压力继电器：压力→电信号（液电联锁开关）
- ==远程调压 == ：在先导式溢流阀 K 口接远程调压阀，可实现在远端设定不同于主阀的更低压力
