```
      ↓

```

## 三元件主线（一条逻辑链）
```
动力元件（泵）→ 工作介质（油）→ 控制元件（阀）→ 执行元件（缸/马达）→ 负载
   Ch2              Ch5            Ch3              Ch4
   机械能→油压能    能量载体        控方向/压力/流量   油压能→机械能
```
- 泵的容积变化制造压力油 → 经管路 + 阀分配控制 → 推缸/马达出力。
- 基本回路（Ch6）就是把「泵 + 阀 + 缸」按目标功能串成最小可控单元。
- 典型系统（Ch7）= 多个基本回路的工程组合，复习时先认回路再认整机。
