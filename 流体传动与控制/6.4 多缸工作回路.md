## 一·顺序动作回路

一·**定义**：在多个执行元件（通常是液压缸）依次动作的场景中——"A 先到底→B 再动 →C 再动"——用压力控制或行程控制让多个缸按制定顺序动作。

二·**实现方式**：

  1.**压力控制（顺序阀）—最常见**：
    - 在每个缸的进油路上设置顺序阀（或液压单向阀），调定开启压力
    - 工作过程：缸①前进到底→压力上升→达到顺序阀设定值→顺序阀打开→油液进入缸②→缸②动作
    - 特点：简单、无需外部信号、工作可靠
    - 缺限：顺序阀的设定压力必须高于前一缸完成动作所需最高压力的 **1.2~1.5 倍**（否则会提前开启——顺序紊乱）

  ```
  顺序阀控制顺序动作示意：
  泵 ─→ 换向阀① ─→ 缸①  A   → A 到位顶死→压力上升
                    │
                    ├→ 顺序阀①（设定 p₁）──→ 缸② B
                    │     （p₂升到 p₁→阀打开→B动）
                    └→ 顺序阀②（设定 p₂）──→ 缸③ C
  ```

  > [!tip] 补充
  > 顺序阀设定压力时需要估算缸①工作过程中的最高压力（克服摩擦+负载+管路损失）。如果设定值太低→缸①还没到位顺序阀就被打开了→B 和 A 同时动，然后顺序紊乱。所以**取 1.2~1.5 倍安全系数**。

  2.**行程控制（行程阀/电气行程开关）**：
    - 在缸① 行程端部装行程阀/接近开关 → 缸① 到位撞到行程阀 → 行程阀换向 → 油液进入缸② → 缸②动作
    - 特点：顺序与压力无关（与负载无关、可靠性高）
    - 应用：组合机床中常用（动力滑台快进到位→挡块压下行程阀→转工进）

## 二·同步回路

一·**定义**：使两个（或多个）液压缸在运动过程中保持相等的位移速度（位置同步），或保持等力（力同步）。

二·**实现方案**：

  1.**机械同步**（刚性连接——最简单可靠）：
    - 两缸的活塞杆通过刚性梁/齿轮齿条连接在一起——机械强制同步
    - 特点：结构简单、无需液压调整；但偏载时可能产生附加应力、结构庞大
    - 应用：两边工作台各一个缸、二者用龙门横梁刚性连接

  2.**串联缸同步**：
    - 缸① 的无杆腔与缸② 的有杆腔直接串联——缸①排油量 = 缸②进油量→两腔有效面积相等时流量相等→速度相等
    - 条件：$A_{1,无杆} = A_{2,有杆}$
    - 特点：液压硬同步、不需控制阀；但两缸须相互匹配、受泄漏/空气影响时会累积误差
    - 应用：小型双缸举升、闸门同步

  3.**分流集流阀同步**：
    - 在两个缸的进油路上装分流集流阀——确保进入两缸的流量**比例**恒定（1:1 或其它比例）
    - 特点：精度可达 **2~5%**（偏差比率），结构紧凑、无需外部信号
    - 缺限：低速/偏载大时精度下降（摩擦力干扰）；分流集流阀有压损

  4.**电液伺服/比例同步**（最高精度）：
    - 在每个缸进油路上装比例阀/伺服阀 + 位置/速度传感器 + 闭环控制器
    - 控制器按位置偏差实时调节各缸的阀开度→补偿偏差→**同步精度 < 0.1~0.5 mm**
    - 特点：精度最高、响应快、灵活（可编程）；但成本高（传感器+控制器+比例阀）
    - 应用：大型压机多缸压制（同步举模）、航空航天风洞多缸同步

## 三·互不干扰回路

一·**定义**：在具有多个执行元件的系统中，当一个支路的负载突然变化（如外载消失→速度突增、或外载突增→压力上升），不应干扰其他支路的动作。

二·**常见干扰原因**：
  - 定量泵 + 单溢流阀系统中，多个换向阀同时工作——如果一个支路负载突然很轻（如缸运动到空载阶段）→该支路速增、大量油被分流→其他支路油量不足→速度突然下降。——这就是"相互干扰"。

三·**解决方案**：

  1.**各支路分别用调速阀/节流阀**——调速阀的速度刚性高 → 单个支路的负载变化不会显著改变其流量 → 对其他支路影响小

  2.**双泵分别供油**（最彻底）
    - 大流量泵（低压）供快速移动的支路；小流量泵（高压）供工进/加压支路；
    - 两支路液压回路互相独立 → 负载变化完全不相互干扰
    - 应用：组合机床——快进（大泵限压式变量泵）、工进（小泵高压）、工进时大泵也向工进路（通过单向阀补油）→ 效率高、不互扰

  3.**蓄能器隔离**：
    - 蓄能器作为压力源供某个支路——当主系统压力波动时，蓄能器充/放油抵消波动→该支路压力稳定
    - 原理：蓄能器提供"软缓冲"——瞬时压力升高 → 蓄能器吸收 → 升幅降低；压力瞬降 → 蓄能器放油 → 降幅减缓

## 速记卡

- 顺序动作：顺序阀（压力控制，+1.2~1.5 倍系数）vs 行程阀/电开关（位置控制，不受负载影响）
- 同步：串联缸（液压硬同步）、分流集流阀（精度2~5%）、闭环伺服/比例（<0.1~0.5 mm 最高精度）
- 互不干扰：双泵独立供油最彻底、调速阀限流有效、蓄能器隔离补充
- ==顺序阀控制的瓶颈在设定值的可靠性（压力估算错误=顺序紊乱）==；行程阀的位置控制比压力控制可靠
- ==双泵供油=组合机床的标准方案——效率最高又互不干扰==
