## 一·液压油的类型

  1.**矿物油型**（最广泛，占 ~**85~90%**）
    - 由石油精馏 + 添加剂调配而成
    - 根据是否加抗磨添加剂 → 又分为 HH（无添加剂）、HL（加抗氧化防锈）、HM（加抗磨——==液压行业最主流==）、HV（高 VI）
    - 成本低、性能可调性好；唯一缺点是不抗燃

  2.**乳化型（抗燃液）**
    - **水包油（O/W）**：水相连续相、油滴分散 ≈ **95% 水 + 5% 油**——几乎不燃但润滑性很差
    - **油包水（W/O）**：油相连续相、水滴分散 ≈ **60% 油 + 40% 水**——抗燃性 + 润滑性平衡
    - **水-乙二醇**：水 + 乙二醇增稠 + 添加剂——抗燃性好，使用广泛（尤其在压铸机/熔炉附近）

  3.**合成型（特殊工况）**
    - 磷酸酯：难燃 + 自熄 + 极高润滑性——大量用于航空液压系统（飞机/导弹）
    - 脂肪酸酯：环保可降解；适用于环境敏感领域（林业/农业/船舶）
    - 硅油：宽温范围（-50~200°C）但润滑性差、昂贵——用于特殊高低温

  > [!question] 选矿物油还是抗燃液？关键是失火风险。冶金、压铸、矿山等高温明火环境必须用抗燃液（W/O 或水—乙二醇）；普通机床、工程机械一般矿物油。

## 二·液压油的主要性能指标

  1.**合适的黏度** + **良好的黏温特性（VI 高）**
    - 见 1.2 节详细：$\nu = \frac{\mu}{\rho}$，VI ≈ 80~120（矿物油）
    - 如黏度过低 → 泄漏大 → 效率低 / 润滑失效；过高 → 流动阻力大 → 泵吸空 / 压力损失大

  2.**良好的润滑性和抗磨性**
    - 液压油就是泵/马达/阀的"润滑油"——在间隙中形成油膜防止金属直接接触
    - 抗磨添加剂（ZnDTP/ZDDP 等）在金属表面形成反应膜——提高抗胶合/抗磨能力
    - **重要**：柱塞泵/叶片泵的侧向力大、对油膜抗磨能力要求高

  3.**抗氧化安定性**
    - 液压油长期在高温、高压下循环 → 与氧反应产生酸性物质、油泥、漆膜
    - 抗氧化剂（酚类/胺类）减缓这一过程 → 延长油液使用寿命

  4.**抗剪切安定性**
    - 油液高速通过泵/阀节流口 → 高剪切力 → 聚合增黏剂分子链被打断 → 黏度下降
    - 抗剪切差的油→长期使用黏度逐渐下降→泄漏增大→需要提前换油

  5.**抗乳化性**
    - 液压系统难免混入水分（冷却器泄漏/油箱冷凝）→ 油水乳化（白色乳浊液）
    - 乳化会降低润滑性 + 产生沉积 + 加速氧化 + 促进生锈
    - 抗乳化剂帮助油水分离 → 水分沉底、油液恢复透明

  6.**抗泡沫性**
    - 油液搅动/回油冲击产生气泡 → 可压缩性增大 → 泵噪声、执行元件爬行/响应延迟
    - 消泡剂（硅油类）快速破泡 → 气泡上升至液面（油箱）后破裂消散

  7.**防锈防腐性**
    - 钢铁零部件遇水/氧气 → 生锈；锈蚀颗粒→加速磨损、堵塞滤芯/阀口
    - 防锈剂在金属表面形成保护膜 → 隔离水/氧

  8.**与密封材料的相容性**
    - 矿物油会使某些橡胶膨胀/收缩/硬化（如丁腈橡胶 NBR 对矿物油耐性好、但聚氨酯不耐）
    - 选油时须与系统所有密封件相容——否密封失效

## 三·液压油的选择原则

  1.**按工作压力选择类型**：

  | 压力等级 | 压力范围 | 推荐油品 |
  |---------|---------|---------|
  | 低压 | < **7 MPa** | HL、HM |
  | 中压 | **7~16 MPa** | HM（抗磨液压油） |
  | 高压 | **16~32 MPa** | HM、HV（高 VI） |
  | 超高压 | > **32 MPa** | HV、抗燃液 |

  → HM（抗磨液压油）是**最通用的选择**——适用于绝大多数工业液压设备。

  2.**按工作温度选择黏度等级**：
    - 环境温度 **-10~40°C** → ISO VG **32/46**（室内通用）
    - 环境温度 **20~70°C** → ISO VG **46/68**
    - 环境温度 **> 70°C** → VG 100 或更高
    - 低温环境 < -10°C → 低温液压油（HV 或 HS）
    - 黏度等级（ISO VG 32 意为在 **40°C** 下中心黏度 **32 mm²/s**）

  3.**按泵类型选择黏度**（不同泵类型对入口黏度要求不同）：

  | 泵类型 | 推荐工作黏度 mm²/s | 最高允许入口黏度（冷启动）|
  |-------|------------------|----------------------|
  | 齿轮泵 | **30~100** | ~ **1000** mm²/s |
  | 叶片泵 | **25~80** | ~ **800** mm²/s |
  | 柱塞泵 | **20~60** | ~ **500** mm²/s |

  > [!tip] 补充
  > 简单记忆：柱塞泵间隙最小 → 黏度要求最高、对高黏度最敏感（冷启动危险）；齿轮泵间隙最大 → 高黏度容忍度最高。此外，环境温度高或系统设计紧凑散热差 → 选高一号黏度；系统大型且长管路 → 选低黏度减小流动阻力。

## 速记卡

- 矿物油（占 85~90%，HM 抗磨油最主流）、乳化型（抗燃）、合成型（特殊/航空）
- 液压油的 8 大性能：黏度+VI、润滑抗磨、抗氧化、抗剪切、抗乳化、抗泡沫、防锈防腐、相容性
- 三原则选油：压力选类型（HM 通用）、温度选黏度等级（VG 32/46/68）、泵选具体黏度段
- ==抗磨液压油 HM 是工业液压的"万金油"==；抗燃液必须用在明火环境
- 柱塞泵冷启动最危险（间隙小、黏度敏感）——不要忽视冷启动黏度上限
