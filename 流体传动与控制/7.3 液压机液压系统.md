## 一·系统概述

一·**设备**：**双动薄板冲压液压机**（≈ **450 kN** / ~45 吨），用于薄板拉伸、冷冲压、冷挤压、压装、调直等工艺。

二·**三缸结构"双动"含义**：
  - **主缸**（拉伸滑块）：完成拉深行程（出大力、慢速、保压）
  - **压边缸**（压边滑块）：在拉伸过程中压住板料边缘、防止起皱（压力小、行程短）
  - **顶出缸**：下腔顶出工件（行程末段——完成加工后把工件从下模顶出）
  - 三缸按工艺顺序协同工作，被称为"双动"实际是双滑块（拉伸+压边）双动形成复合动作。

三·**主要液压元件**：
  - 变量泵（节能：大流量快进→小流量工进保压）
  - 电磁溢流阀（安全限压 + 卸荷）
  - 远程调压阀（设定不同压力等级）
  - 电液动换向阀×2（分别控主缸/压边缸和顶出缸）
  - 节流阀（减速—回油节流）
  - 压力继电器（检测到顶出缸顶出到位）
  - 高位油箱（充液——在拉伸滑块快下时从高挂油箱补油）
  - 先导溢流阀 + 手动换向阀等

## 二·工作循环与动作原理

一·**第 1 步——泵空载启动**
  - 所有电磁铁断电。电磁溢流阀（远控卸荷）+ 电液换向阀均处中位
  - 变量泵出油经电磁溢流阀直接回油箱（卸荷）——电机轻载启动

二·**第 2 步——拉伸滑块 + 压边滑块快速下行**
  - 1Y、4Y、5Y 得电
  - **拉伸滑块（主缸）**：
    - 变量泵 → 电液换向阀左位 → 主缸上腔（无杆腔）
    - 主缸上腔快下时产生局部真空→从高位油箱经充液阀吸入（补油满足大流量需求）——高位油箱高于主缸最高液位，靠静压和抽吸双重补油
  - **压边滑块（压边缸）**：
    - 变量泵（另一部分油）→ 电液换向阀→ 压边缸上腔
    - 压边缸也有自己的高位油箱补油
  - 这步中滑块靠自重加速下行补油——速度快省泵流量

三·**第 3 步——减速（转慢速接近工件）**
  - 滑块接近工件前——行程开关或预选位置发信号→ 部分电磁铁切换→ 主缸回油路串联节流阀产生背压→ 滑块减速

四·**第 4 步——拉伸、压紧**
  - 压边缸先压住板料边缘（设定压紧力——不能太大导致板料被拉断，也不能太小导致起皱）
  - 主缸继续下行——进行拉伸（工件成形）
  - 系统压力升高 → 变量泵自调至小排量工进状态（节能）
  - 关键：==拉深过程压力同步控制——压边力恒定（溢流阀限压）、拉伸力自动随工艺变化（慢速稳速）==

五·**第 5 步——保压**
  - 主缸拉伸到底后——进入保压阶段（停留一段时间，让工件定形）
  - 泵可卸荷（电磁溢流阀2Y）或由蓄能器/泵补充泄漏维持压力
  - 保压时间由压力继电器检测+延时继电器控制

六·**第 6 步——快速回程**
  - 保压延时到期 → 电磁铁切换
  - 主缸换向阀切换右位 → 油进入主缸下腔（有杆腔）→ 主缸上行→滑块抬起
  - 主缸上腔回油通过充液阀冲回高位油箱（充液阀大开）
  - 压边缸也同时回程

七·**第 7 步——原位停止**
  - 滑块回到原位 → 行程开关 → 全部电磁铁断电
  - 换向阀回中位 → 缸两腔油液通油箱 → 滑块停止

八·**第 8 步——顶出缸上升（顶出工件）**
  - 6Y 得电：油进入顶出缸下腔 → 顶出活塞上升 → 把已拉伸成形的工件从下模顶出
  - 工件取出后 → 6Y 断电 → 顶出缸靠重力下落（或弹簧复位）
  - 完成一个完整工作循环 → 回到第 1 步等待下一个

## 三·电磁铁动作顺序表

| 工况 / 步骤 | 1Y (主缸换向) | 2Y (电磁溢流卸荷) | 3Y (主缸减速节流) | 4Y (压边缸下行) | 5Y (压边缸保压) | 6Y (顶出缸) |
|------------|:---:|:---:|:---:|:---:|:---:|:---:|
| 1 空载启动 | - | +（卸荷） | - | - | - | - |
| 2 快下 | + | - | - | + | + | - |
| 3 减速 | + | - | + | + | + | - |
| 4 拉伸压紧 | + | - | + | - | + | - |
| 5 保压 | - | -（或交替） | - | - | - | - |
| 6 回程 | -（换向阀右位） | - | - | - | - | - |
| 7 停止 | -（中位） | +（卸荷） | - | - | - | - |
| 8 顶出 | - | +（卸荷） | - | - | - | + |

（+ 表示电磁铁通电；- 表示断电）

> [!tip] 补充
> 注意：表中 2Y（电磁溢流阀动作）的模式——在拉伸/保压时 2Y 断电——电磁溢流阀不卸荷（恢复限压功能），系统保持工作压力；快下/快回时 2Y 通电使溢流阀卸荷允许系统低压大流量回油。
> 实际控制中，2Y 可能并非单纯的"通/断"而是配合压力要求进行 PWM 调节——课件中为简化用开关描述。

## 速记卡

- 双动薄板压机 = 主缸（拉伸）+ 压边缸（压边防皱）+ 顶出缸（顶出工件）= 三缸协同
- ==工作循环 8 步：泵启动→快下→减速→拉伸压紧→保压→回程→停止→顶出==
- 高位油箱充液——快速下行时给主缸/压边缸上腔补油（省泵流量）
- 压边力需精确控制（太小起皱、太大拉裂）——通常使用独立溢流阀限压恒定
- 变量泵工进时自动降排量匹配→节能
- ==液压机 = 力巨大 + 速度可控 + 保压时长短可调——是所有冲压成形工艺的万能底座==
