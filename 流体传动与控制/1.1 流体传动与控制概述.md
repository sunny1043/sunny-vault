## 一·流体传动定义与分类

一·**定义**：流体传动与控制是利用**液体**（液压）或**气体**（气压）作为工作介质进行能量传递、转换和控制的传动方式。

二·**两大分支**：
  1.**液压传动**：
    - 工作介质：液压油（矿物油/乳化液）
    - 工作压力范围：通常 **0.1 ~ 35 MPa**（最高可达 70~100 MPa，超高压）
    - 特点：功率密度大、力大、低速力大
    - 应用：工程机械、机床、航空
  2.**气压传动**：
    - 工作介质：压缩空气
    - 工作压力范围：通常 **0.1 ~ 1 MPa**
    - 特点：清洁安全、低成本、动作快
    - 应用：轻工自动化、食品医药、机器人

  > [!note] 核心理解
  > 液压 vs 气动的本质区别在于介质可压缩性：油近乎不可压缩 → 刚性好、力大、定位准；空气可压缩 → 柔顺、安全、不防水但精度差。

## 二·液压传动的优缺点

| ==优点== | ==缺点== |
|---------|---------|
| 功率密度大，同等功率下体积小、重量轻 | 效率相对较低（泄漏 + 摩擦损失，总效率一般 70~85%） |
| 易于实现**无级调速**（调速阀/变量泵） | 对油温变化敏感（黏度随温变 → 系统性能漂移） |
| 传动平稳、冲击小（油液缓冲吸振） | 对油液清洁度要求高（阀芯间隙 ~5~30 μm，颗粒易卡阀） |
| 易于实现过载保护（溢流阀限压） | 外泄漏污染环境（油液渗漏 == 安全隐患） |
| 与电控结合可实现复杂自动化（比例/伺服控制） | 制造精度要求高 → 初始成本高、维护要求高 |
| 可在极低速下输出大推力（低速大转矩马达） | 管路敷设受限制（硬管 + 软管 + 管接头 == 潜在脆弱点） |

## 三·主要应用领域

  1.**工程机械**（最大市场）：挖掘机（~5~8泵多阀系统）、推土机、装载机、压路机、起重机。
  2.**航空航天**：飞行控制系统（舵机液压伺服）、起落架收放、刹车系统。
  3.**制造业**：注塑机、压力机、冲压机床、组合机床（如动力滑台）。
  4.**船舶与海洋工程**：舵机、锚机、起重机、平台升降系统。
  5.**农业机械**：拖拉机液压悬挂、收割机、联合装载。
  6.**军事装备**：坦克炮塔伺服、导弹发射架、雷达起竖。

## 四·液压系统的基本组成

一个完整的液压系统由**五大部分**组成（==五大组成==）：

```
         ┌──────────────────────────────────────────────┐
         │             原动机（电动机/内燃机）             │
         │                    │                          │
         ▼                    │                          │
    ┌────────┐               │                          │
    │ 液压泵 │ ◄── 机械能 ──┘                          │
    │(动力)  │——→ 输出液压能（压力油 p,Q）              │
    └───┬────┘                                          │
        │                                    ┌────┐     │
        │   ┌────────────────────────── ─ ─ → │阀组│     │
        │   │  (控制元件:方向/压力/流量阀)       │     │
        │   ▼                            └──┬─┘     │
    ┌────┴────────┐                  ┌──────┘        │
    │ 执行元件     │                  │               │
    │(液压缸/马达) │◄────── 控制 ────┘               │
    │ → 机械能    │                                  │
    └──────┬──────┘                                  │
           │ 回油                                     │
           ▼                                          │
    ┌────────────┐                                    │
    │ 油箱+滤油器 │◄———— 散热、沉淀、过滤                  │
    │ (辅助)    │           + 蓄能器+密封+管件          │
    └────────────┘          + 工作介质（液压油）         │
    └──────────────────────────────────────────────┘
```

  1.**动力元件（液压泵 == 心脏）** ：将原动机机械能转换为液压能。包括齿轮泵、叶片泵、柱塞泵。
  2.**执行元件（液压缸/马达 == 手脚）** ：将液压能转回机械能。缸 → 直线运动；马达 → 旋转运动。
  3.**控制元件（各种阀 == 大脑/神经）** ：控制油液的压力、流量和方向。含溢流阀（压力限定）、节流阀（调速）、换向阀（换向）三大类。
  4.**辅助元件（油箱/滤油器/蓄能器/管件/密封 == 装备保障）** ：储油、散热、过滤、储能、连接、密封。
  5.**工作介质（液压油 == 血液）** ：传递能量的载体。润滑、冷却、防锈、传递力。

  > [!question] 气压系统的组成是否相同？气压的"辅助"中冷却需求小得多（空气可压缩放热被扩散），但后处理（干燥机/油水分离器）更复杂。

## 速记卡

- 液压传动：液体为介质，压力 0.1~35 MPa，功率密度大、刚性高
- 气压传动：空气为介质，压力 0.1~1 MPa，清洁安全、精度低
- 液压五大组成：泵（动力）、缸/马达（执行）、阀（控制）、辅助（油箱/滤/蓄/密封）、液压油（介质）
- 液压的"穷"与"富"：功率密度大（富），效率低 + 清洁度敏感（穷）
