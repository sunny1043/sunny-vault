## 一·电液比例阀

一·**定义**：电液比例阀的输出量（压力或流量）与输入电信号（电流 ~ 比例电磁铁）**连续成比例**的液压阀。介于普通开关阀（通/断）和电液伺服阀（高精度）之间的中间产品。

二·**工作原理**：
  1.比例电磁铁：输入电流（通常 **0~800 mA**）→ 电磁力与电流近似成比例。
  2.电磁力直接推动阀芯（比例节流阀/溢流阀）或先导阀芯。
  3.平衡时：电磁力 = 液压反力 + 弹簧力 → 阀芯开度与电流成比例 → 输出压力/流量与电流信号对应。
  $$ i \uparrow \Rightarrow F_{mag} \uparrow \Rightarrow x_{valve} \uparrow \Rightarrow A_{th} \uparrow \Rightarrow Q \uparrow (或 p \uparrow) $$

三·**特点**：
  1.**优点**：
    - 可连续控制（而非开关量）——平滑调速、分级调压
    - 比伺服阀便宜（精度稍低但成本低 ~1/3~1/5）
    - 抗污染能力比伺服阀好（阀芯间隙 ~**10~30 μm**，伺服阀 ~1~5 μm）
  2.**局限**：
    - 精度（滞环、死区）不如伺服阀——滞环约 **3~7%**（伺服阀 <1%）
    - 响应速度比伺服阀慢（约 **50~200 ms**，伺服阀可达 10 ms 以下）
  3.**应用**：工业液压中大量使用——注塑机、锻压机、工程机械，只要"有一定精度"但不需要"极高精度"的场景。

## 二·电液伺服阀

一·**定义**：以极高精度和响应速度控制液压执行元件的位置、速度或力的闭环控制阀。输出量（流量/压力）与输入电信号成正比，且死区小、滞环极低。液压系统的"零阶保持器"。

二·**工作原理**：
  1.典型结构：力矩马达 + 射流管/喷嘴挡板（先导级）+ 滑阀（功率级）。
  2.小电流 → 力矩马达偏转 → 挡板偏位 → 两侧喷嘴背压变化 → 推动主阀芯移动 → 主阀口开度对应电流大小与方向。

三·**特点**：
  1.**优点**：
    - 精度极高：滞环 <**1%**、死区可忽略
    - 响应极快：频宽可达 **50~300 Hz**（比例阀一般 5~30 Hz）
    - 适合闭环控制：位置、力、压力伺服系统
  2.**缺限**：
    - **非常昂贵**（单阀几百到几千美元）
    - 对油液清洁度要求极高（**NAS 4~6 级**；比例阀 ~ NAS 7~9）
    - 故障率高（微小颗粒卡住阀芯就会失效）
  3.**应用**：航空航天作动器、锻压机压制力伺服、材料试验机、轧机、机器人。

## 三·插装阀（逻辑阀）

一·**定义**：一种标准化、高集成度的二通逻辑阀——由阀芯（插装件）+ 盖板 + 先导阀组成。通过盖板上的控制油口（X/Y/Z）驱动主阀芯开关，可组合实现方向/压力/流量控制。本质上是"乐高"式的液压阀积木。

二·**结构**：
  ```
                    盖板（带先导控制系统）
                    │
          控制油口X │
                    │
        ┌───────────▼──────────┐
        │  弹簧 + 插装阀芯      │
        │                      │
   A ───┤   ┌──────────┐      ├─── B
        │   │ 阀口座面  │      │
        └───┤──────────┤──────┘
            └──────────┘
  ```
  插装阀芯的开启由 A 腔、B 腔和控制腔 X 之间的压差控制。

三·**工作原理**：
  1.当 X 腔通过先导阀接回油箱（X 卸荷），插装阀开启：A↔B 通
  2.当 X 腔引入高压油（X 压力≥A 腔压力+弹簧力），插装阀关闭：A↔B 断
  3.通过电磁换向阀/比例阀切换 X 的通/断/调压 → 实现不同功能

四·**特点**：
  1.**优点**：
    - 集成度极高、功率密度大（一台阀块可集成几十个阀功能）
    - 无管路连接 → 泄漏点少、故障率低、结构紧凑
    - 切换速度快（阀芯质量小 + 行程短）
    - 适合高压大流量（可达 **> 400 L/min** 单阀芯通过量）
  2.**缺限**：
    - 设计和装配需专业知识（阀块流道设计复杂）
    - 修理困难（故障定位依赖流道图）
    - 成本对批量敏感（单个或少量的阀块成本很高）
  3.**应用**：大型压机、注塑机、连铸机、船舶、矿山——总之高压大流量集成系统。

---

## 四·三类阀对比总结

| 项目 | 电液比例阀 | 电液伺服阀 | 插装阀 |
|------|----------|----------|-------|
| 控制精度 | 中（滞环 3~7%） | 高（滞环 <1%） | 取决于先导阀 |
| 响应频宽 | **5~30 Hz** | **50~300 Hz** | 较高（取决于先导阀） |
| 抗污染能力 | 中（10~30 μm） | 差（1~5 μm，NAS 4~6） | 较强（取决于先导） |
| 成本 | 中 | 极高 | 中高（开发费高） |
| 功能灵活性 | 单一功能 | 单一功能 | 可组合多功能 |
| 应用趋势 | 应用最广（性价比最优） | 精密伺服专属 | 高压大流量 + 集成化 |

---

## 速记卡

- 比例阀：电磁力输入∝输出；精度中、成本中、抗污染中——性价比之选
- 伺服阀：极高精度+极高响应，但极昂贵+极娇气（对清洁度敏感）
- 插装阀：标准化插件+盖板+先导→可组合";"乐高"式集成；高压大流量系统最优方案
- ==插装阀集成的优点：减少管路泄漏点 ≈ 降低故障率 ≈ 提升系统可靠性==
