StudyVaults 是基于 Obsidian + Quartz 5 构建的个人知识管理系统，覆盖航天推进（涡扇发动机、轴流压气机、涡轮叶片强度/振动、转子平衡）、工程热力学（布雷顿循环、火用分析）、流体传动及数学等理工科领域。写作上采用提纲式短句 + 高信息密度风格，每个核心概念配"物理直觉"与"数学本质"双维度解读，图示支持 TikZJax（精确数学图）、Excalidraw（手绘示意）和 Mermaid（流程图），公式用 LaTeX 渲染，内置五件套笔记模板与速记卡体系，支持双向链接与图谱可视化，通过 GitHub Actions 自动构建并部署至 GitHub Pages（vault.sunny1043.top）。
