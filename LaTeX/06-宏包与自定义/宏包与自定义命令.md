# 宏包管理与自定义命令

#latex #package #macro

## 自定义命令 — \newcommand

```latex
\newcommand{\R}{\mathbb{R}}         % 实数域
\newcommand{\vect}[1]{\mathbf{#1}}  % 向量
\newcommand{\pd}[2]{\frac{\partial #1}{\partial #2}}  % 偏导
```

使用:
```latex
$\R$ 上的向量 $\vect{x}$，偏导数 $\pd{f}{x}$
```

> [!warning] `\newcommand` 要求命令名未被占用。覆盖已有命令用 `\renewcommand`。

## 带可选参数的命令

```latex
\newcommand{\abs}[2][]{%
  \if\relax\detokenize{#1}\relax
    |#2|%         无默认值
  \else
    |#2|_{#1}%    有默认值
  \fi
}
% 用法: \abs{x} → |x|, \abs[p]{x} → |x|_p
```

## 自定义环境 — \newenvironment

```latex
\newenvironment{proof}[1][证明]{%
  \par\noindent\textbf{#1}\quad
}{%
  $\square$\par
}

% 用法:
\begin{proof}
  因为 A=B，所以结论成立。
\end{proof}
```

## 常用实用命令

```latex
% 导数
\newcommand{\dd}{\mathrm{d}}
\newcommand{\dv}[2]{\frac{\dd #1}{\dd #2}}

% 偏导
\newcommand{\pdv}[2]{\frac{\partial #1}{\partial #2}}

% 期望
\newcommand{\E}{\mathbb{E}}

% 注释
\newcommand{\note}[1]{\textcolor{red}{[#1]}}
```
