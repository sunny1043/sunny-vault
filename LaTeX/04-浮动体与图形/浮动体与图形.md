# 浮动体与图形

#latex #figure #float

## 插入图片 — graphicx

```latex
\usepackage{graphicx}
\begin{figure}[htbp]
  \centering
  \includegraphics[width=0.8\textwidth]{figure.pdf}
  \caption{图片标题}
  \label{fig:myfigure}
\end{figure}
```

> 推荐使用 PDF/PNG 格式的图片。`[htbp]` 控制浮动位置。

## 浮动位置参数

| 参数 | 含义 |
|:----:|------|
| h | here — 尽量在此处 |
| t | top — 页面顶部 |
| b | bottom — 页面底部 |
| p | page — 单独一页 |
| ! | 忽略美学约束 |

## 常用选项

| 选项 | 示例 |
|------|------|
| `width=0.8\textwidth` | 宽度为文本宽度的80% |
| `height=6cm` | 固定高度 |
| `scale=0.8` | 缩放80% |
| `angle=90` | 旋转90度 |
| `trim=1cm 2cm 3cm 4cm, clip` | 裁剪(左 下 右 上) |

## 子图 — subcaption 宏包

```latex
\usepackage{subcaption}
\begin{figure}
  \centering
  \begin{subfigure}{0.45\textwidth}
    \includegraphics{fig1.pdf}
    \caption{子图一}
  \end{subfigure}
  \hfill
  \begin{subfigure}{0.45\textwidth}
    \includegraphics{fig2.pdf}
    \caption{子图二}
  \end{subfigure}
  \caption{总标题}
\end{figure}
```

## TikZ — 程序化绘图

```latex
\usepackage{tikz}
\begin{tikzpicture}
  \draw[->] (0,0) -- (4,0) node[right] {$x$};
  \draw[->] (0,0) -- (0,4) node[above] {$y$};
  \draw[blue, thick] (0,0) parabola (3,4);
  \node at (2,2) {$y=x^2$};
\end{tikzpicture}
```
