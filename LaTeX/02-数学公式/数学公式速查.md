# 数学公式速查

#latex #math

## 行内与行间公式

| 方式 | 语法 | 示例 |
|------|------|------|
| 行内 | `$...$` | $E=mc^2$ |
| 行间(无编号) | `\[...\]` 或 `$$...$$` | 独占一行居中 |
| 行间(有编号) | `\begin{equation}...\end{equation}` | 自动编号 |
| 行间(无编号变体) | `\begin{equation*}...` | amsmath 提供 |

## 多行公式对齐 — align 环境

```latex
\begin{align}
  a &= b + c \\
    &= d + e + f \\
    &= g
\end{align}
```

`&` 标记对齐点，`\\` 换行。每行独立编号。



## Obsidian 中使用公式

Obsidian 内置 **MathJax** 渲染引擎，支持所有标准 LaTeX 数学命令。

### 快速输入

| 操作 | 快捷键/语法 |
|------|------------|
| 创建公式块 | 输入 `$$` 后回车，自动生成 `$$ ... $$` 块 |
| 行内公式 | `$...$` |
| 预览切换 | `Cmd+E` (macOS) / `Ctrl+E` (Windows) |
| 实时预览 | 设置 → 编辑器 → 开启「实时预览」 |

### 工程热力学常用公式模板

多行推导 — `aligned` 环境：
```latex
$$
\begin{aligned}
c_p - c_v &= \left(\frac{\partial h}{\partial T}\right)_p 
            - \left(\frac{\partial u}{\partial T}\right)_v \\
          &= R_g \qquad \text{(迈耶公式)}
\end{aligned}
$$
```

分段函数 — `cases`：
```latex
$$
r(A^*) = \begin{cases}
n,   & r(A)=n \\
1,   & r(A)=n-1 \\
0,   & r(A)<n-1
\end{cases}
$$
```

矩阵 — `pmatrix`：
```latex
$$
\begin{pmatrix}
a & b \\ c & d
\end{pmatrix}
$$
```

### Obsidian 独有技巧

- **Callout 内嵌公式**: `> [!important] $$公式$$` — 公式在折叠块中正常渲染
- **表格内公式**: Obsidian 表格支持内嵌 `$...$`，但不支持 `$$...$$`
- **跨笔记引用**: `[[笔记名]]` + `$$\boxed{公式}$$` 比纯 LaTeX 更适合学习笔记
- **预览模式**: 写完公式后 `Cmd+E` 切到预览模式确认渲染正确

## 常用数学符号

| 类型   | 命令                                             |    输出     |
| ---- | ---------------------------------------------- | :-------: |
| 希腊字母 | `\alpha \beta \gamma \pi \theta`               | α β γ π θ |
| 希腊大写 | `\Gamma \Delta \Lambda \Omega`                 |  Γ Δ Λ Ω  |
| 上下标  | `x^2 \quad a_{ij}`                             |  x² a_ij  |
| 分数   | `\frac{a}{b}`                                  |    a/b    |
| 根号   | `\sqrt{x} \sqrt[n]{x}`                         |  √x ⁿ√x   |
| 求和   | `\sum_{i=1}^n`                                 |   Σⁿᵢ₌₁   |
| 积分   | `\int_0^\infty`                                |   ∫₀^∞    |
| 求导   | `\frac{\partial f}{\partial x}`                |   ∂f/∂x   |
| 矩阵   | `\begin{pmatrix} a & b \\ c & d \end{pmatrix}` |   (行列式)   |

## 常用字体

| 命令 | 效果 | 用途 |
|------|:----:|------|
| `\mathrm{R}` | R | 正体（单位、算子） |
| `\mathbf{R}` | **R** | 粗体（向量、矩阵） |
| `\mathcal{L}` | ℒ | 花体（集合、空间） |
| `\mathbb{R}` | ℝ | 黑板粗体（实数域） |
| `\mathscr{A}` | 𝒜 | 手写体 |

## cases 环境 — 分段函数

```latex
\[
f(x) = \begin{cases}
  0, & x < 0 \\
  x^2, & 0 \leq x < 1 \\
  1, & x \geq 1
\end{cases}
\]

## 示例：带推导的公式组合

```latex
\begin{align}
  c_p - c_v &= \left(\frac{\partial h}{\partial T}\right)_p 
              - \left(\frac{\partial u}{\partial T}\right)_v \\
            &= \frac{\partial (u+pv)}{\partial T} - c_v \\
            &= R_g \qquad \text{(迈耶公式)}
\end{align}
```
