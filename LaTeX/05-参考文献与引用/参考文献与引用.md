# 参考文献与交叉引用

#latex #reference #bibtex

## 交叉引用 — \label 和 \ref

```latex
\section{方法论}
\label{sec:method}

如图 \ref{fig:data} 所示...
式 (\ref{eq:einstein}) 表明...

\begin{figure}
  \caption{实验数据}
  \label{fig:data}
\end{figure}
```

引用规则：章节用 `sec:`、图用 `fig:`、表用 `tab:`、公式用 `eq:` 前缀。

## BibTeX — 传统方式

`.bib` 文件存储参考文献：

```bibtex
@article{einstein1905,
  author  = {Albert Einstein},
  title   = {On the Electrodynamics of Moving Bodies},
  journal = {Annalen der Physik},
  year    = {1905},
  volume  = {322},
  pages   = {891--921},
}
```

在正文中引用并生成文献列表：

```latex
\cite{einstein1905}
\bibliographystyle{plain}
\bibliography{references}
```

编译流程：
```bash
pdflatex myfile
bibtex myfile
pdflatex myfile
pdflatex myfile   # 需要两次以解析引用
```

## biblatex — 现代方式（推荐）

```latex
\usepackage[backend=biber, style=numeric]{biblatex}
\addbibresource{references.bib}
...
\cite{einstein1905}
\printbibliography
```

编译：`xelatex → biber → xelatex`

## 常用文献格式

| 条目类型 | 用途 |
|----------|------|
| `@article` | 期刊论文 |
| `@book` | 书籍 |
| `@inproceedings` | 会议论文 |
| `@phdthesis` | 博士论文 |
| `@online` | 网页 |
