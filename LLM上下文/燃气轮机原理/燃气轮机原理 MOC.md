## 学习路径

```
Ch1 绪论（循环 / 发动机类型 / 推重比）
        ↓
Ch2 轴流压气机（基元级 → 速度三角形 → 叶栅气动力 → 多级设计）
        ↓
   ┌─────────────┬──────────────┐
Ch3 叶片强度    Ch4 轮盘强度      ↓（力学）
   离心/弯曲应力  离心+热应力
   └─────┬───────┘
        ↓
Ch5 叶片振动 → Ch6 轮盘振动（频率 / 共振 / 颤振）
        ↓
Ch7 转子振动与临界转速（涡动 / 陀螺 / 刚性 vs 柔性转子）
```
