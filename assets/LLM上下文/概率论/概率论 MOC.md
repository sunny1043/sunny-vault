## 学习路径

```
Ch1 随机事件与概率(公理 + 古典/几何 + 贝叶斯)
      ↓
Ch2 一维 r.v.(分布函数 + 离散/连续常见分布 + 函数的分布)
      ↓
Ch3 多维 r.v.(联合/边缘/条件 + 独立性 + 二维正态 + 卷积)
      ↓
Ch4 数字特征(期望/方差 + 协方差/相关系数 + 切比雪夫不等式)
      ↓
Ch5 极限定理(切比雪夫/辛钦/伯努利 大数 + 林德伯格-列维 CLT)
```
复习策略: Ch1→Ch2 是"概率语言"建立,Ch3→Ch4 是"代数工具"建立,Ch5 是"极限行为",渐进式向前依赖。

## 三主线(一条逻辑链)

```
概率语言建立 → 变量工具 → 极限思想
   Ch1           Ch2-4        Ch5

公理化概率   分布/数字特征  大数定律+CLT
   ↓             ↓             ↓
样本空间-事件  期望/方差/相关   频率稳定性
公理推出常用   F/f/卷积         依概率收敛
条件概率贝叶斯 二维正态独立性    标准化→N(0,1)
```
- 主线 1 — **随机性**: 从事件语言到变量语言,Ch1→Ch2,核心是"分布函数"统一离散/连续
- 主线 2 — **数字刻画**: Ch4 用期望方差把分布"压缩成几个数",便于比较与极限处理
- 主线 3 — **极限行为**: Ch5 把 Ch1 的频率稳定性大 вызов→概率,Ch2 的正态分布→CLT 的极限归宿

## 关键公式速记(简表)

| 主题 | 公式 |
|------|------|
| 加法公式 | $P(A\cup B)=P(A)+P(B)-P(AB)$ |
| 全概率 | $P(A)=\sum P(B_i)P(A\mid B_i)$ |
| 贝叶斯 | $P(B_k\mid A)=\dfrac{P(B_k)P(A\mid B_k)}{\sum P(B_i)P(A\mid B_i)}$ |
| 二项 | $P\{X=k\}=C_n^k p^k q^{n-k}$ |
| 泊松 | $P\{X=k\}=\dfrac{\lambda^k e^{-\lambda}}{k!}$ |
| 正态标准化 | $Z=\dfrac{X-\mu}{\sigma}\sim N(0,1)$ |
| 方差展开式 | $DX=EX^2-(EX)^2$ |
| 切比雪夫 | $P\{|X-EX|\ge \varepsilon\}\le DX/\varepsilon^2$ |
| 林德伯格-列维 CLT | $\dfrac{\sum X_k - n\mu}{\sigma\sqrt n}\to N(0,1)$ |
