## 基本原理 → [[笔记/工程学/流体传动与控制/1.2 液压传动基本原理与液压油特性|1.2 液压传动基本原理与液压油特性]]
| 公式 | 说明 |
|------|------|
| $F_1/A_1 = F_2/A_2 = p$ | 帕斯卡定律：压力等值传递 |
| $F_2/F_1 = A_2/A_1$ | 力放大比 = 面积比 |
| $v_1 A_1 = v_2 A_2 = q$ | 连续性方程：流量守恒 |
| $P = p\cdot q$ | 液压功率 |
| $P=p\cdot Q,\quad Q=V\cdot n$ | 动力元件流量 |

## 动力元件 → [[笔记/工程学/流体传动与控制/2.2液压泵的工作原理，分类与主要性能参数|2.2液压泵的工作原理，分类与主要性能参数]]
| 公式 | 说明 |
|------|------|
| $q = V\cdot n$ | 理论流量 = 排量×转速 |
| $\eta_V = q_{实际}/q_{理论}$ | 容积效率 |
| $\eta_m = T_{理论}/T_{实际}$ | 机械效率 |
| $\eta = \eta_V\cdot\eta_m$ | 总效率 |
| $V = 2z\,A_{槽}\,b$ | 齿轮泵排量（工程近似） |
| $p\cdot q_{理论}=T_{理论}\cdot\omega$ | 能量守恒（泵） |

## 控制元件 → [[笔记/工程学/流体传动与控制/3.1 液压阀概述|3.1 液压阀概述]]
| 类别 | 阀种 | 功能 |
|------|------|------|

## 溢流阀 vs 减压阀 vs 顺序阀 → [[笔记/工程学/流体传动与控制/3.3 压力控制阀|3.3 压力控制阀]]
| 阀 | 控制信号 | 出口压力 | 常态 |
|----|----------|----------|------|
| 溢流阀 | 进口压力 | 回油箱（出口≈0） | 常闭 |
| 减压阀 | 出口压力 | 出口恒定低于进口 | 常开 |
| 顺序阀 | 进口压力 | 出口接后续回路 | 常闭 |

## 调速阀 → [[笔记/工程学/流体传动与控制/3.4 流量控制阀|3.4 流量控制阀]]
| 公式 | 说明 |
|------|------|
| $q = C_d\,A_T\sqrt{2\Delta p/\rho}$ | 节流阀流量公式 |
| 调速阀=减压阀+节流阀 | 保持节流阀前置 Δp 恒定 → q 不随负载变化 |

## 执行元件 → [[笔记/工程学/流体传动与控制/4.1 液压缸类型与工作参数|4.1 液压缸类型与工作参数]]
| 公式 | 说明 |
|------|------|
| $F = p\cdot A$ | 推力 = 压力×有效面积 |
| $v = q/A$ | 速度 = 流量/面积 |
| 差动连接 $F=(A_1-A_2)p,\ v=q_总/(A_1-A_2)$ | 有杆腔回油接进油 → 快进 |
| 液压马达 $T=\frac{\Delta p\cdot V}{2\pi},\ n=\frac{q}{V}$ | 扭矩、转速 |
