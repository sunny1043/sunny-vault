# _templates — 笔记模板（改造自 groepl/Obsidian-Templates）

适配 Sunny 精读风格，去掉了原版的 Dataview 依赖。三个模板：

| 模板 | 用途 |
|---|---|
| `精读笔记模板.md` | 新章节/知识点精读笔记（定义/性质/公式推导/图/核心理解 五件套） |
| `考点问答模板.md` | 真题/考点问答（含解题思路、易错点、同类变形） |
| `术语速查卡模板.md` | 术语/公式速查卡 |

## 启用（用 Obsidian 核心 Templates 插件，无需装新插件）
1. 设置 → 核心插件 → 打开 **模板 (Templates)**
2. 设置 → 模板 → 模板文件夹位置 → 填 `_templates`
3. 新建笔记后，命令面板搜「插入模板」选对应模板即可。
   - `{{title}}` 自动填笔记名，`{{date}}` 自动填日期。
   - 文中 `+++ ... +++` 是占位提示，填完删掉。

> 若以后想要"缺定义/缺配图章节自动巡检"的看板，需要再装 Dataview 插件——本次按你的选择只做了模板。
