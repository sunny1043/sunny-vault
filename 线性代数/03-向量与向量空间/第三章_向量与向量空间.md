# 第三章 向量与向量空间 ★★★

#linear-algebra #vector-space

## 核心问题

向量组的**线性相关性**是刻画方程组解结构和空间维数的关键。

## 3.1 线性相关与无关

**定义**: 若 $k_1\alpha_1+\cdots+k_s\alpha_s=0$ 只有全零解 → **线性无关**；存在不全为零的解 → **线性相关**。

**几何直觉**: 2向量相关↔共线；3向量相关↔共面；4+向量在三维空间必相关。

**代数判定**: 以向量为列构造矩阵 $A$：
- 线性无关 $\iff r(A)=s$
- 线性相关 $\iff r(A)<s$

## 3.2 核心定理

| 定理 | 内容 |
|------|------|
| 部分与整体 | 部分相关→整体相关；整体无关→部分无关 |
| 延长与缩短 | 无关延长→仍无关；相关缩短→仍相关 |
| 维数定理 | $n$ 维空间 $s>n$ 个向量必相关 |
| 等价组性质 | 线性无关组可被另一组表示→前者个数 ≤ 后者个数 |

## 3.3 向量空间（数学一专属）

**基与维数**: $n$ 个线性无关向量张成整个空间 → $\dim(V)=n$

**过渡矩阵**: $(\beta_1,\dots,\beta_n)=(\alpha_1,\dots,\alpha_n)P$，$P$ 可逆。

**坐标变换**: $x=Py$ 或 $y=P^{-1}x$，其中 $x$ 是 $\alpha$ 基下的坐标，$y$ 是 $\beta$ 基下的坐标。
