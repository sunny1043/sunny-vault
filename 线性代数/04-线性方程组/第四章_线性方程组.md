## §4.0 一张图统领全章

一·**全章总纲**：本章把"方程组有没有解、有几个解、解长什么样"全部翻译成 ==秩== 的比较。$A$ 为 $m\times n$（$n$ 个未知数），增广矩阵 $\tilde A=(A\,|\,b)$。

```
                 比较 r(A) 与 r(A|b)
                          │
        ┌─────────────────┼──────────────────┐
   r(A)<r(A|b)        r(A)=r(A|b)=n      r(A)=r(A|b)=r<n
        │                  │                   │
      无解              唯一解             无穷多解
   (出现 0=非零)                    (自由未知量 n-r 个 → 基础解系 n-r 个)
```

二·**记死一句话**：==解的有无看 $r(A)$ 与 $r(\tilde A)$ 是否相等；解的个数看 $r$ 与未知数 $n$ 是否相等==。

> [!note] 核心理解
> $Ax=b$ 即"把 $b$ 表示成 $A$ 各列的线性组合"。有解 $\iff b$ 落在列空间里 $\iff$ 添 $b$ 这列不涨秩（$r(A)=r(\tilde A)$）；解唯一 $\iff$ 列无关、没有可自由调节的方向（$r=n$）。整章是 [[第三章_向量与向量空间]] "能否表示 / 表示是否唯一"的方程组化身。

## §4.1 解的存在性判定（秩的语言）

一·**定理（Kronecker–Capelli / 增广矩阵秩判别）**：对 $Ax=b$，
$$\begin{cases}r(A)<r(\tilde A)&\Rightarrow\ \text{无解}\\ r(A)=r(\tilde A)=n&\Rightarrow\ \text{唯一解}\\ r(A)=r(\tilde A)=r<n&\Rightarrow\ \text{无穷多解（含 }n-r\text{ 个自由未知量）}\end{cases}$$

二·**公式与推导**（为何这样判）：
  1. 化 $\tilde A$ 为行阶梯形。无解 $\iff$ 出现矛盾行 $[\,0\ 0\ \cdots\ 0\ |\ d\,],\ d\neq0$（即 $0=d$）。　← 矛盾行使 $r(\tilde A)>r(A)$
  2. 这恰好等价于 $r(A)<r(\tilde A)$——$b$ 列给阶梯形多撑出一个非零行。
  3. 有解时，主元个数 = $r$；剩下 $n-r$ 个未知量没有主元约束 → **自由未知量**，可任意取值 → $r=n$ 时无自由量（唯一），$r<n$ 时有 $n-r$ 个（无穷多）。

三·**特例——齐次组 $Ax=0$**：$b=0$ 永不产生矛盾行，故 ==齐次组必有解==（至少零解）。
  1. 只有零解 $\iff r(A)=n$；有非零解 $\iff r(A)<n$。
  2. 方阵特例：$Ax=0$ 有非零解 $\iff|A|=0$。

> [!note] 核心理解
> "$0=d\neq0$"这行矛盾，几何上就是 $b$ 戳出了 $A$ 列空间之外，怎么组合都够不着 → 无解。自由未知量 = 解里能自由拨动的旋钮，旋钮个数 $n-r$ 决定了解集的维数。

> [!warning] 易错 / 边界
> 判存在性必须看**增广矩阵**的秩，单看 $r(A)$ 不够；化阶梯形时一旦出现矛盾行立即判无解。

## §4.2 齐次方程组 Ax=0 的解结构

一·**定义**：$Ax=0$ 的全部解构成**解空间（零空间）** $N(A)=\{x\mid Ax=0\}$，它是 $\mathbb R^n$ 的子空间。
  1. ==维数定理==：$\dim N(A)=n-r(A)$（自由未知量个数）。

二·**基础解系**：解空间的一组**基**，即 $n-r(A)$ 个线性无关的解 $\xi_1,\dots,\xi_{n-r}$。

三·**通解**：
$$x=c_1\xi_1+c_2\xi_2+\cdots+c_{n-r}\xi_{n-r}\qquad(c_i\text{ 任意})$$

四·**公式与推导**（基础解系个数 = $n-r$）：
  1. 化 $A$ 为行最简形，设主元列 $r$ 个、自由列 $n-r$ 个。　← 秩 = 主元数
  2. 每个自由未知量轮流取 $1$、其余自由量取 $0$，回代解出一个解向量 → 共得 $n-r$ 个解 $\xi_i$。　← 构造基础解系
  3. 这 $n-r$ 个解线性无关（自由量位置构成单位向量，天然无关），且任一解都是它们的组合 → 它们是解空间的基。∎

> [!note] 核心理解
> 解空间是"被 $A$ 压成零的所有方向"。$A$ 把 $n$ 维输入压成 $r$ 维像，必然"碾平"了 $n-r$ 个维度——这些被碾平的方向就是零空间，维数 $n-r$。这正是 [[第二章_矩阵]] 里 $AB=O\Rightarrow r(A)+r(B)\le n$ 的同一回事（$B$ 的列都在 $N(A)$ 中）。

> [!warning] 易错 / 边界
> 基础解系**不唯一**（基可换），但个数 $n-r$ 唯一；$r=n$ 时基础解系为空，只有零解。

## §4.3 非齐次方程组 Ax=b 的解结构

一·**结构定理**：$Ax=b$（有解）的通解 = **一个特解 $\eta_0$** + **对应齐次组 $Ax=0$ 的通解**：
$$x=\eta_0+c_1\xi_1+\cdots+c_{n-r}\xi_{n-r}$$

二·**性质 / 推导**（两条核心，必会证）：
  1. 两个非齐次解之差是齐次解：$A(\eta_1-\eta_2)=A\eta_1-A\eta_2=b-b=0$ → $\eta_1-\eta_2\in N(A)$。
  2. 非齐次解 + 齐次解仍是非齐次解：$A(\eta_0+\xi)=A\eta_0+A\xi=b+0=b$。
  → 合起来：解集 = 把齐次解空间整体**平移** $\eta_0$ 得到的"仿射集"。
  3. 所以非齐次解集**不是子空间**（一般不含零向量），但形状（维数 $n-r$）和齐次解空间一样。
  4. 求通解 = 求**一个特解** + 求**齐次基础解系**，两件事分头做再拼起来。

```
齐次解空间 N(A)   ——平移 η0——→   非齐次解集 {η0 + N(A)}
  (过原点的子空间)              (不过原点的平行"平面")
```

> [!note] 核心理解
> 物理直觉：非齐次的"特解"给出一个落点，齐次通解给出"在不破坏 $Ax=b$ 的前提下还能怎么挪"。挪动方向正好是零空间——因为沿零空间走 $A$ 的输出不变。这是"通解 = 特解 + 齐次通解"最本质的解释。

> [!warning] 易错 / 边界
> 特解只要**任取一个**即可；$\xi_i$ 必须来自**齐次组**而非非齐次；写通解别忘了任意常数 $c_i$。

## §4.4 求解通法（初等行变换）

一·**统一流程**（齐次/非齐次通用）：
  1. 写出增广矩阵 $\tilde A=(A|b)$（齐次则 $b=0$，可只对 $A$ 操作）。
  2. ==只用初等行变换==化为**行最简形**（主元为 1、主元所在列其余为 0）。
    （1）⚠️解方程组**绝不能用列变换**——列变换会打乱未知量、改变解集（见 [[第二章_矩阵]]）。
  3. 由阶梯判存在性（§4.1）：有矛盾行 → 无解，停。
  4. 选定自由未知量（非主元列对应），写齐次基础解系 $\xi_i$。
  5. 非齐次再令自由量全 $=0$ 回代得一个特解 $\eta_0$。
  6. 组装通解 $x=\eta_0+\sum c_i\xi_i$。
二·**含参数方程组**：先按 $r(A)$ 与 $r(\tilde A)$ 随参数变化**分类讨论**（常令某行列式 / 主元 = 0 找临界参数值），再各自求解。

> [!note] 核心理解
> 行最简形把方程组"摊到最透"：每个主元未知量被自由未知量显式表示出来，通解直接读出。行变换 = 对方程作"等价变形（倍加、互换、缩放）"，不改变解集，这是整套方法合法的根据。

## §4.5 公共解与同解

一·**公共解**：同时满足 $Ax=0$ 与 $Bx=0$ 的解。
  1. 求法：联立成 $\begin{pmatrix}A\\B\end{pmatrix}x=0$，对堆叠矩阵求基础解系即得公共解。
  2. 也可由各自通解令其相等，解出公共部分。
二·**同解**：$Ax=0$ 与 $Bx=0$ 解集完全相同。
  1. 充要条件：$r(A)=r(B)=r\!\begin{pmatrix}A\\B\end{pmatrix}$。
  2. 直觉：堆起来秩不涨，说明 $B$ 的每个方程都已被 $A$ 蕴含（反之亦然）→ 约束等价 → 解集相同。

> [!note] 核心理解
> 公共解 = 两个零空间的**交集** $N(A)\cap N(B)$，自然由"上下堆叠"的方程组刻画。同解 = 两零空间相等，等价于"行空间相等"，故用堆叠后秩不增来判别。

> [!warning] 易错 / 边界
> 公共解要的是"同时满足"，对应**堆叠**（行拼接）不是相加；同解判别三个秩要全相等，缺一不可。

## 速记卡 / 一句话总结

- **存在性**：$r(A)<r(\tilde A)$ 无解；$=n$ 唯一解；$=r<n$ 无穷多解（自由量 $n-r$ 个）。齐次组恒有解。
- **齐次**：解空间维数 $=n-r(A)$；基础解系 $n-r$ 个无关解；通解 $x=\sum c_i\xi_i$。方阵非零解 $\iff|A|=0$。
- **非齐次**：通解 = 特解 $\eta_0$ + 齐次通解；$\eta_1-\eta_2$ 是齐次解；解集 = 零空间平移 $\eta_0$（不过原点）。
- **解法**：增广矩阵 → ==只行变换==化行最简 → 判存在 → 写基础解系 + 特解 → 组装。含参数先按秩分类讨论。
- **公共解**：堆叠 $\binom{A}{B}x=0$ 求解（= 两零空间交集）。**同解** $\iff r(A)=r(B)=r\binom{A}{B}$。
