# 第四章 线性方程组 ★★★

#linear-algebra #linear-system

## 4.1 Kronecker-Capelli 定理

对于 $Ax=b$（增广矩阵 $\tilde{A}=(A|b)$）：

| 条件 | 结论 |
|------|------|
| $r(A)<r(\tilde{A})$ | **无解**（出现 $[0\cdots0|d], d\neq 0$） |
| $r(A)=r(\tilde{A})=n$ | **唯一解** |
| $r(A)=r(\tilde{A})<n$ | **无穷多解** |

## 4.2 解的结构

### 齐次方程组 $Ax=0$

- 解集构成**解空间**（零空间），维数 = $n-r(A)$
- **基础解系**: 解空间的一组基，共 $n-r(A)$ 个线性无关解
- **通解**: $x=c_1\xi_1+\cdots+c_{n-r}\xi_{n-r}$

### 非齐次方程组 $Ax=b$

- $\eta_1-\eta_2$ 是对应齐次解
- $\eta+\xi$ 仍是 $Ax=b$ 的解（$\xi$ 为齐次解）
- **通解**: $x=\eta_0+c_1\xi_1+\cdots+c_{n-r}\xi_{n-r}$

## 4.3 公共解与同解

| 概念 | 条件 |
|------|------|
| 公共解 | 联立 $\begin{pmatrix}A\\B\end{pmatrix}x=0$ |
| 同解 | $r(A)=r(B)=r\begin{pmatrix}A\\B\end{pmatrix}$ |
