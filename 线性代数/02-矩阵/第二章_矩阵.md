# 第二章 矩阵 ★★★

#linear-algebra #matrix

## 矩阵的本质

矩阵不仅是数据的排列，更代表了**线性空间之间的线性映射**。

## 2.1 基本运算

| 运算 | 关键性质 |
|------|----------|
| 乘法 | $c_{ij}=\sum a_{ik}b_{kj}$，**不可交换** ($AB\neq BA$)，**不可消去** ($AB=O\not\Rightarrow A=O$) |
| 转置 | $(AB)^T = B^T A^T$ |
| 幂 | $A^k = P\Lambda^k P^{-1}$（对角化后），$r(A)=1$ 时 $A^k=[\text{tr}(A)]^{k-1}A$ |

## 2.2 伴随矩阵 $A^*$（十大核心公式）

$$AA^* = A^*A = |A|E$$

| # | 公式 |
|---|------|
| 1 | $AA^* = |A|E$ |
| 2 | $A^{-1} = A^*/|A|$（$A$可逆） |
| 3 | $|A^*| = |A|^{n-1}$ |
| 6 | $(kA)^* = k^{n-1}A^*$ |
| 7 | $(AB)^* = B^*A^*$ |
| 8 | $(A^*)^* = |A|^{n-2}A$ |

### 秩的断层定理（最重要）

$$\boxed{r(A^*) = \begin{cases} n, & r(A)=n \\ 1, & r(A)=n-1 \\ 0, & r(A)<n-1 \end{cases}}$$

## 2.3 秩的核心不等式

| 不等式 | 公式 |
|--------|------|
| 加法 | $r(A+B) \le r(A)+r(B)$ |
| 乘法 | $r(AB) \le \min(r(A),r(B))$ |
| $AB=O$ | $r(A)+r(B) \le n$ |
| **Sylvester** | $r(AB) \ge r(A)+r(B)-n$ |
| Frobenius | $r(ABC)+r(B) \ge r(AB)+r(BC)$ |

## 2.4 初等变换与等价

- 行变换 = **左乘**初等矩阵；列变换 = **右乘**初等矩阵
- $A \simeq B \iff r(A)=r(B) \iff PAQ=B$（$P,Q$可逆）

## 例题

### [例] Sylvester不等式证明思路

构造 $M=\begin{pmatrix}A&O\\E_n&B\end{pmatrix}$，初等变换得 $r(M)=n+r(AB)$，又 $r(M)\ge r(A)+r(B)$ → $r(AB)\ge r(A)+r(B)-n$
