## §2.0 矩阵的本质

一·**两副面孔**：矩阵既是 **$m\times n$ 个数的矩形排列**（数据表），更是 **线性空间之间的线性映射** $T:\mathbb{R}^n\to\mathbb{R}^m,\ x\mapsto Ax$。
  1. ==整章一切运算的意义都要回到"映射"这副面孔上理解==：乘法 = 映射复合，逆 = 映射反演，秩 = 映射的"有效维数"。
二·**与行列式的分工**：行列式是"一个数"（只对方阵），矩阵是"一张表 / 一个变换"（任意形状）。$|A|$ 只是矩阵的一个标量指标。

> [!note] 核心理解
> 把矩阵 $A$ 看成"作用"：$Ax$ 是把向量 $x$ 送进变换得到的像。列向量 $A$ 的第 $j$ 列 = 基向量 $e_j$ 的像。理解了"列 = 基的去向"，矩阵乘法、秩、可逆全都顺理成章。

## §2.1 基本运算

### 矩阵乘法

一·**定义**：$A_{m\times s}B_{s\times n}=C_{m\times n}$，其中
$$c_{ij}=\sum_{k=1}^{s}a_{ik}b_{kj}$$
即 $C$ 的 $(i,j)$ 元 = $A$ 第 $i$ 行与 $B$ 第 $j$ 列的内积。**前列数 = 后行数**才能乘。

二·**性质 / 条件**（全是易错点）：
  1. **不可交换**：一般 $AB\neq BA$（连形状都可能不同）。可交换是特例。
  2. **不可消去**：$AB=AC\not\Rightarrow B=C$（除非 $A$ 可逆）。
  3. **有零因子**：$AB=O\not\Rightarrow A=O$ 或 $B=O$。
    （1）==但有秩约束==：$AB=O\Rightarrow r(A)+r(B)\le n$（$n$ 为 $A$ 的列数），见 §2.4。
  4. 成立的：结合律 $(AB)C=A(BC)$、分配律、$(AB)^T=B^TA^T$（转置反序）。

### 矩阵的幂

一·**相似对角化降幂**（一般首选）：若 $A=P\Lambda P^{-1}$，则
$$A^k=P\Lambda^k P^{-1}$$
对角阵的幂只需把对角元各自 $k$ 次方，见 [[第五章_特征值与特征向量]]。

二·**秩 1 矩阵的幂**（高频技巧）：若 $r(A)=1$，可写 $A=\boldsymbol{\alpha}\boldsymbol{\beta}^T$，则
$$A^k=(\boldsymbol{\beta}^T\boldsymbol{\alpha})^{k-1}A=[\operatorname{tr}(A)]^{k-1}A$$
  1. 推导：$A^2=\boldsymbol{\alpha}(\boldsymbol{\beta}^T\boldsymbol{\alpha})\boldsymbol{\beta}^T=(\boldsymbol{\beta}^T\boldsymbol{\alpha})\,\boldsymbol{\alpha}\boldsymbol{\beta}^T=(\boldsymbol{\beta}^T\boldsymbol{\alpha})A$。　← 中间 $\boldsymbol{\beta}^T\boldsymbol{\alpha}$ 是标量可外提
  2. 又 $\boldsymbol{\beta}^T\boldsymbol{\alpha}=\sum a_{ii}=\operatorname{tr}(A)$（秩 1 矩阵的迹 = 这个标量）→ 归纳得 $A^k=[\operatorname{tr}(A)]^{k-1}A$。

> [!note] 核心理解
> 乘法"不可交换"的根：映射复合本来就有先后（先转再缩 ≠ 先缩再转）。"有零因子"的根：两个非零变换的像/核可以互相抵消（$A$ 把所有东西压进 $B$ 的核里），所以乘积为零不要求某个为零——这正是秩不等式 $r(A)+r(B)\le n$ 的来源。

## §2.2 逆矩阵

一·**定义**：方阵 $A$ 若存在同阶方阵 $B$ 使 $AB=BA=E$，则 $A$ **可逆**，$B=A^{-1}$ 唯一。
  1. 注：只要 $AB=E$（一边）对方阵就够，自动有 $BA=E$。

二·**性质 / 等价判定**（可逆的等价命题链，==全科最重要的"七连环"==）：
$$A\text{ 可逆}\iff|A|\neq0\iff r(A)=n\iff A\text{ 列（行）满秩}\iff Ax=0\text{ 只有零解}\iff A\text{ 的列向量线性无关}\iff 0\text{ 不是 }A\text{ 的特征值}$$

三·**公式与求法**：
  1. **伴随法**：$A^{-1}=\dfrac{1}{|A|}A^{*}$（理论 + 二阶速算用，高阶太累）。
    （1）二阶速记：$\begin{pmatrix}a&b\\c&d\end{pmatrix}^{-1}=\dfrac{1}{ad-bc}\begin{pmatrix}d&-b\\-c&a\end{pmatrix}$（==主对调、副变号==）。
  2. **初等行变换法**（实算首选）：$(A\,|\,E)\xrightarrow{\text{行变换}}(E\,|\,A^{-1})$。
    （1）原理：行变换 = 左乘初等矩阵 $P_k$，做到 $P_k\cdots P_1 A=E$ 即 $P_k\cdots P_1=A^{-1}$，同一串变换作用在 $E$ 上就直接得到 $A^{-1}$。
  3. 运算律：$(AB)^{-1}=B^{-1}A^{-1}$（反序）、$(A^T)^{-1}=(A^{-1})^T$、$(kA)^{-1}=\frac1k A^{-1}$、$|A^{-1}|=|A|^{-1}$。

> [!note] 核心理解
> 可逆 = 映射**一一对应、可反演**，没有信息塌缩（$|A|\neq0$ 即体积不被压扁）。"七连环"是把"满秩 / 行列式非零 / 解唯一 / 无关 / 无零特征值"这几张面孔焊在一起——考场上常给你其中一个、让你推另一个。

> [!warning] 易错 / 边界
> 只有**方阵**才谈可逆；$|A|=0$ 一票否决可逆性。

## §2.3 伴随矩阵 A*

### 定义与核心恒等式

一·**定义**：$A^*$ 的 $(i,j)$ 元 = $A$ 的 $(j,i)$ 元的**代数余子式** $A_{ji}$（==注意转置：行列下标互换==）：
$$A^*=(A_{ji})=\begin{pmatrix}A_{11}&A_{21}&\cdots&A_{n1}\\ A_{12}&A_{22}&\cdots&A_{n2}\\ \vdots&\vdots&&\vdots\\ A_{1n}&A_{2n}&\cdots&A_{nn}\end{pmatrix}$$

二·**核心恒等式（一切公式之母）**：
$$\boxed{AA^*=A^*A=|A|E}$$

三·**公式与推导**：
  1. 算 $AA^*$ 的 $(i,j)$ 元 $=\sum_k a_{ik}(A^*)_{kj}=\sum_k a_{ik}A_{jk}$。　← 按定义 $(A^*)_{kj}=A_{jk}$
  2. 由 [[第一章_行列式]] 的展开 / 异乘变零定理：$\sum_k a_{ik}A_{jk}=|A|\,\delta_{ij}$（$i=j$ 给 $|A|$，否则 0）。　← 同乘得 $|A|$、异乘得 0
  3. 于是 $AA^*=|A|E$。同理 $A^*A=|A|E$。

四·**十大常用公式**（由恒等式 + 行列式性质导出）：
  1. $AA^*=A^*A=|A|E$。
  2. $A$ 可逆时 $A^{-1}=\dfrac{A^*}{|A|}$，且 $A^*=|A|A^{-1}$。
  3. $|A^*|=|A|^{\,n-1}$。　← 对 $AA^*=|A|E$ 两边取行列式：$|A|\,|A^*|=|A|^n$
  4. $(A^*)^{-1}=(A^{-1})^*=\dfrac{A}{|A|}$（$A$ 可逆）。
  5. $(A^T)^*=(A^*)^T$。
  6. $(kA)^*=k^{\,n-1}A^*$。　← $kA$ 的每个代数余子式带 $k^{n-1}$
  7. $(AB)^*=B^*A^*$（反序，与逆同构）。
  8. $(A^*)^*=|A|^{\,n-2}A$（$n\ge2$）。
  9. $r(A^*)$ 见下断层定理。
  10. 特征值：若 $A$ 可逆、$\lambda$ 是 $A$ 的特征值，则 $A^*$ 有特征值 $|A|/\lambda$。

### 秩的断层定理（最重要）

一·**定理**：
$$\boxed{r(A^*)=\begin{cases}n,&r(A)=n\\[2pt]1,&r(A)=n-1\\[2pt]0,&r(A)\le n-2\end{cases}}$$

二·**推导（三段）**：
  1. $r(A)=n$：$A$ 可逆 → $A^*=|A|A^{-1}$ 也可逆 → $r(A^*)=n$。
  2. $r(A)\le n-2$：所有 $n-1$ 阶子式全为 0 → 所有代数余子式 $A_{ij}=0$ → $A^*=O$ → $r(A^*)=0$。　← 余子式是 $n-1$ 阶子式
  3. $r(A)=n-1$：$|A|=0\Rightarrow AA^*=O\Rightarrow r(A)+r(A^*)\le n\Rightarrow r(A^*)\le1$；又 $r(A)=n-1$ 保证存在非零 $n-1$ 阶子式 → $A^*\neq O\Rightarrow r(A^*)\ge1$。两边夹 → $r(A^*)=1$。

> [!note] 核心理解
> $A^*$ 的设计目的就是"凑出 $|A|E$"，让求逆有闭式公式。断层定理的跳变（$n\to1\to0$）来自：代数余子式是 $n-1$ 阶子式，$A$ 的秩一旦掉到 $n-2$ 以下，所有 $n-1$ 阶子式集体归零，故 $A^*$ 直接塌成零矩阵——没有"$r(A^*)=2,3,\dots$"的中间地带。

> [!warning] 易错 / 边界
> $A^*$ 定义自带转置，下标顺序 $A_{ji}$ 千万别写成 $A_{ij}$；$|A^*|=|A|^{n-1}$ 在 $|A|=0$ 时两边都为 0 仍成立。

## §2.4 矩阵的秩

一·**定义**：$r(A)$ = $A$ 中**非零子式的最高阶数**；等价地 = 行阶梯形中**非零行的行数** = 行（列）向量组的极大无关组所含向量个数。
  1. 三个面孔等价：最高阶非零子式 / 阶梯非零行数 / 向量组的"有效维数"。

二·**核心不等式**（证明题弹药库）：
  1. $0\le r(A_{m\times n})\le\min(m,n)$。
  2. $r(A\pm B)\le r(A)+r(B)$。　← 列空间之和的维数次可加
  3. $r(AB)\le\min\{r(A),r(B)\}$。　← 乘法只会"压缩"信息，秩不增
  4. $AB=O\Rightarrow r(A)+r(B)\le n$（$A$ 列数 = $B$ 行数 = $n$）。
    （1）直觉：$AB=O$ 说明 $B$ 的每一列都落在 $A$ 的**零空间**里 → $r(B)\le\dim N(A)=n-r(A)$。
  5. **Sylvester 不等式**：$r(AB)\ge r(A)+r(B)-n$。
  6. **Frobenius 不等式**：$r(ABC)+r(B)\ge r(AB)+r(BC)$。
  7. 可逆矩阵不改变秩：$P,Q$ 可逆 $\Rightarrow r(PAQ)=r(A)$。

### [例] Sylvester 不等式的分块证明

一·**思路**：构造分块矩阵，用"初等变换不改秩 + 分块下界"两头夹。
  1. 令 $M=\begin{pmatrix}A&O\\E_n&B\end{pmatrix}$（$E_n$ 为 $n$ 阶单位阵）。
  2. 对 $M$ 作分块初等变换（用 $E_n$ 这块去消 $A$、消 $B$）：可化到 $\begin{pmatrix}O&-AB\\E_n&O\end{pmatrix}$ → $r(M)=n+r(AB)$。　← 初等变换不改秩
  3. 另一方面，$M$ 含有 $A$ 行块与 $B$ 列块，恒有 $r(M)\ge r(A)+r(B)$。　← 子块秩下界
  → 联立：$n+r(AB)\ge r(A)+r(B)$ 即 $r(AB)\ge r(A)+r(B)-n$。∎

> [!note] 核心理解
> 秩 = 矩阵作为映射的"**像空间维数**" = 输出里真正独立的维数。所有秩不等式都在说一句话："映射只会丢信息、不会凭空造信息"。乘法 $r(AB)\le\min$ 是"两次压缩取更狠的那次"；$AB=O$ 的秩约束是"$B$ 的像被塞进 $A$ 的核"。

## §2.5 初等变换与初等矩阵

一·**三类初等行（列）变换**：①两行互换；②某行乘非零常数 $k$；③某行的 $k$ 倍加到另一行。
二·**初等矩阵**：对 $E$ 做一次初等变换得到的矩阵，记 $P$。每个都可逆，逆仍是初等矩阵。
三·**左行右列法则**：==对 $A$ 作一次初等行变换 = 左乘对应初等矩阵；作一次初等列变换 = 右乘对应初等矩阵==。

```
行变换 ←─ 左乘 P ─→  P·A        （操作"行"，乘在左边）
列变换 ←─ 右乘 Q ─→  A·Q        （操作"列"，乘在右边）
A ──行列变换──→ 标准形 (E_r O; O O)，  PAQ = (E_r O; O O)
```

四·**等价（相抵）**：$A\simeq B$ 指 $A$ 经有限次初等行/列变换得 $B$。
  1. 充要条件：$A\simeq B\iff r(A)=r(B)\iff\exists$ 可逆 $P,Q$ 使 $PAQ=B$。
  2. 等价标准形：任一 $r(A)=r$ 的矩阵都 $\simeq\begin{pmatrix}E_r&O\\O&O\end{pmatrix}$。==秩是等价关系的唯一不变量==。
五·**应用**：求逆 $(A|E)\to(E|A^{-1})$、解 $AX=B$、求秩，全靠初等变换。

> [!note] 核心理解
> 初等矩阵把"手工换行换列"翻译成"矩阵乘法"，于是"变换序列"可写成可逆矩阵 $P,Q$。这就是为什么"$A$ 可逆 $\iff A$ 是若干初等矩阵之积"——可逆矩阵就是一串可还原的基本操作。

> [!warning] 易错 / 边界
> 左行右列别记反；初等"列"变换会改变解空间，**解方程组只能用行变换**（增广矩阵作列变换会破坏方程含义）。

## 速记卡 / 一句话总结

- **本质**：矩阵 = 线性映射；列 = 基的去向。乘法 = 复合（不可交换、不可消去、有零因子）。
- **秩 1 幂**：$A=\boldsymbol\alpha\boldsymbol\beta^T\Rightarrow A^k=[\operatorname{tr}(A)]^{k-1}A$。
- **可逆七连环**：可逆 $\iff|A|\neq0\iff r=n\iff$ 满秩 $\iff Ax=0$ 仅零解 $\iff$ 列无关 $\iff 0$ 非特征值。
- **求逆**：$(A|E)\to(E|A^{-1})$；二阶"主对调副变号 / 除以 $ad-bc$"。
- **伴随**：$AA^*=|A|E$；$A^{-1}=A^*/|A|$；$|A^*|=|A|^{n-1}$；$(kA)^*=k^{n-1}A^*$；$(AB)^*=B^*A^*$；$(A^*)^*=|A|^{n-2}A$。
- **断层定理**：$r(A^*)=n / 1 / 0$ 对应 $r(A)=n / n{-}1 / \le n{-}2$。
- **秩不等式**：$r(AB)\le\min$；$AB=O\Rightarrow r(A)+r(B)\le n$；Sylvester $r(AB)\ge r(A)+r(B)-n$。
- **初等变换**：左行右列；$A\simeq B\iff r(A)=r(B)\iff PAQ=B$；标准形 $\begin{pmatrix}E_r&O\\O&O\end{pmatrix}$。
