## §5.1 定义与特征多项式

### 特征值与特征向量

一·**定义**：方阵 $A$（$n$ 阶），若存在数 $\lambda$ 和**非零**向量 $x$ 使
$$\boxed{Ax=\lambda x\quad(x\neq\boldsymbol0)}$$
则 $\lambda$ 称 $A$ 的**特征值**，$x$ 称对应于 $\lambda$ 的**特征向量**。
  1. ==特征向量必须非零==（$x=0$ 恒成立，无意义）；特征值可以为 0。

二·**求法（特征方程）**：把定义移项 $(\lambda E-A)x=0$，要它有非零解：
$$\det(\lambda E-A)=0\quad\text{——特征方程}$$

三·**公式与推导**：
  1. $Ax=\lambda x\iff(\lambda E-A)x=0$，这是齐次方程组。　← 移项、提出 $x$（注意补 $E$）
  2. 有非零解 $\iff$ 系数阵不可逆 $\iff\det(\lambda E-A)=0$。　← 齐次非零解条件，见 [[第四章_线性方程组]]
  3. $f(\lambda)=\det(\lambda E-A)$ 是 $\lambda$ 的 $n$ 次多项式（**特征多项式**），其 $n$ 个根即全部特征值（含重根、复根）。
  4. 对每个 $\lambda_i$，解齐次组 $(\lambda_i E-A)x=0$ 的**基础解系**即特征向量（的极大无关组）。

> [!note] 核心理解
> 特征向量 = 被变换 $A$ 作用后**方向不变、只被拉伸 $\lambda$ 倍**的特殊方向；特征值 = 拉伸倍数。几何上 $A$ 把空间"沿各特征方向独立伸缩"。数学本质：求特征值 = 找让 $(\lambda E-A)$ 退化（不可逆）的 $\lambda$，即把矩阵问题化成"$\det=0$"的代数方程。

> [!warning] 易错 / 边界
> $\det(\lambda E-A)$ 与 $\det(A-\lambda E)$ 差 $(-1)^n$，但根相同；特征向量是齐次组的非零解，写答案要给基础解系并说明"$k\neq0$"。

## §5.2 特征值/向量的核心性质

一·**迹与行列式**（最常用的两条数值关系）：
$$\sum_{i=1}^{n}\lambda_i=\operatorname{tr}(A)=\sum_i a_{ii},\qquad\prod_{i=1}^{n}\lambda_i=|A|$$
  1. 推导：特征多项式 $f(\lambda)=\prod(\lambda-\lambda_i)$，比较 $\lambda^{n-1}$ 系数得 $\sum\lambda_i=\operatorname{tr}A$，比较常数项（$\lambda=0$）得 $\prod\lambda_i=|A|$。
  2. ==推论：$A$ 可逆 $\iff$ 所有 $\lambda_i\neq0$（$|A|=\prod\lambda_i\neq0$）==。
二·**矩阵函数的特征值平移**（设 $Ax=\lambda x$）：
  1. $A^k$ 有特征值 $\lambda^k$；$aA+bE$ 有 $a\lambda+b$；多项式 $g(A)$ 有 $g(\lambda)$；特征向量 $x$ 不变。
  2. $A^{-1}$（可逆）有特征值 $1/\lambda$；$A^*$ 有 $|A|/\lambda$。
  3. 推导（以 $A^2$ 为例）：$A^2x=A(\lambda x)=\lambda Ax=\lambda^2 x$。　← 反复用 $Ax=\lambda x$
三·**不同特征值的特征向量线性无关**：$\lambda_1\neq\cdots\neq\lambda_m$ 对应的特征向量 $x_1,\dots,x_m$ 必无关。
  1. 这是对角化"凑齐 $n$ 个无关向量"的基础。
四·**几何重数 $\le$ 代数重数**：$1\le n-r(\lambda_i E-A)\le k_i$（$k_i$ 为 $\lambda_i$ 作为根的重数）。

> [!note] 核心理解
> 性质二是"做题神器"：碰到 $A^3-2A+E$ 之类，直接对每个 $\lambda$ 算 $g(\lambda)$ 即可，不用真去乘矩阵。本质是特征向量张成的方向上，矩阵作用 = 标量乘，于是任何多项式 / 逆都只作用在标量 $\lambda$ 上。

> [!warning] 易错 / 边界
> 性质三只保证"不同特征值之间"无关；同一特征值的多个特征向量是否够数，要看几何重数（§5.4）。

## §5.3 相似矩阵

一·**定义**：若存在**可逆** $P$ 使 $P^{-1}AP=B$，则 $A$ **相似**于 $B$，记 $A\sim B$。

二·**性质（相似不变量）**：$A\sim B\Rightarrow$ 二者
  1. 特征多项式相同 → 特征值（含重数）全同；
  2. 行列式相同 $|A|=|B|$、迹相同 $\operatorname{tr}A=\operatorname{tr}B$、秩相同 $r(A)=r(B)$；
  3. $A^k\sim B^k$，$g(A)\sim g(B)$。
  → ==以上都是**必要**非充分==：特征值 / 迹 / 行列式全同也未必相似（差在能否对角化、Jordan 结构）。

三·**公式与推导**（特征多项式相似不变）：
  1. $\det(\lambda E-B)=\det(\lambda E-P^{-1}AP)=\det(P^{-1}(\lambda E-A)P)$。　← $\lambda E=P^{-1}(\lambda E)P$
  2. $=\det(P^{-1})\det(\lambda E-A)\det(P)=\det(\lambda E-A)$。　← 行列式可乘、$\det P^{-1}\det P=1$
  → 特征多项式不变，故特征值、迹、行列式全是相似不变量。∎

> [!note] 核心理解
> 相似 = **同一个线性变换在不同基下的两个矩阵表示**，$P$ 是换基（过渡）矩阵。变换本身没变，所以"内禀量"（特征值、迹、行列式、秩）都不变——它们刻画的是变换，不是某一组坐标。

## §5.4 相似对角化

一·**定义**：若 $A$ 相似于对角阵 $\Lambda$（$P^{-1}AP=\Lambda$），称 $A$ **可对角化**，$\Lambda$ 的对角元就是特征值，$P$ 的列就是对应特征向量。

二·**充要条件**：
$$A\text{ 可对角化}\iff A\text{ 有 }n\text{ 个线性无关的特征向量}$$
  1. 等价的可操作判据（==考试主力==）：
$$\boxed{\text{每个特征值 }\lambda_i:\quad n-r(\lambda_i E-A)=k_i\ (\text{几何重数 = 代数重数})}$$

三·**公式与推导**（为何"$n$ 个无关特征向量 $\iff$ 可对角化"）：
  1. 设 $P=(x_1,\dots,x_n)$ 各列为特征向量，则 $AP=(\lambda_1x_1,\dots,\lambda_nx_n)=P\Lambda$。　← 逐列用 $Ax_i=\lambda_ix_i$
  2. $P$ 可逆 $\iff$ 这 $n$ 个特征向量线性无关。　← 方阵可逆 = 列无关
  3. 此时 $P^{-1}AP=\Lambda$。反之若 $P^{-1}AP=\Lambda$，倒推各列即特征向量且无关。∎
  → 凑无关向量的关键：每个 $\lambda_i$ 能贡献的无关特征向量数 = 几何重数 $n-r(\lambda_iE-A)$；只有它**等于**代数重数 $k_i$，总数才凑够 $n$。

四·**充分条件（快速判定）**：
  1. $A$ 有 $n$ 个**互不相同**的特征值 → 必可对角化（不同特征值向量天然无关，见 §5.2）。
  2. $A$ 为**实对称矩阵** → 必可（正交）对角化（见 §5.5）。

五·**对角化的用途**：
  1. ==求高次幂== $A^k=P\Lambda^kP^{-1}$（对角阵幂 = 对角元各自 $k$ 次方）。
  2. 求矩阵多项式、解线性微分方程组等。

> [!note] 核心理解
> 可对角化 = "能找到一组特征向量基，使变换在该基下就是各方向独立伸缩（对角阵）"。失败的根源只有一个：某个重特征值给不出足够多的无关方向（几何重数 < 代数重数），方向"亏了"，凑不满 $n$ 个 → 只能化到 Jordan 形而非对角形。

> [!warning] 易错 / 边界
> 判定要**逐个**特征值核对几何重数 = 代数重数，单重根自动满足，只需查重根；$n$ 个不同特征值是充分非必要（重根也可能可对角化）。

## §5.5 实对称矩阵（三大定理 + 正交对角化）

实对称矩阵 $A^T=A$（实元素）性质极好，是 [[第六章_二次型]] 的核心载体。

一·**三大定理**：
  1. 特征值**全为实数**。
  2. **不同**特征值对应的特征向量**相互正交**。
    （1）推导：设 $Ax_1=\lambda_1x_1,\ Ax_2=\lambda_2x_2,\ \lambda_1\neq\lambda_2$。
      - $\lambda_1(x_1,x_2)=(Ax_1)^Tx_2=x_1^TA^Tx_2=x_1^TAx_2=\lambda_2(x_1,x_2)$。　← 用 $A^T=A$
      - $(\lambda_1-\lambda_2)(x_1,x_2)=0$，而 $\lambda_1\neq\lambda_2$ → $(x_1,x_2)=0$，正交。∎
  3. **必可正交对角化**：存在**正交矩阵** $Q$（$Q^TQ=E$）使
$$Q^{-1}AQ=Q^TAQ=\Lambda=\operatorname{diag}(\lambda_1,\dots,\lambda_n)$$
    （1）实对称必可对角化，且每个特征值几何重数恒 = 代数重数（不会"亏方向"）。

二·**正交对角化五步法（求 $Q$）**：
  1. 解 $\det(\lambda E-A)=0$ 求全部特征值 $\lambda_i$ 及重数。
  2. 对每个 $\lambda_i$ 解 $(\lambda_iE-A)x=0$ 求特征向量。
  3. ==重根==对应的特征向量先做**施密特正交化**（不同特征值的已自动正交，无需处理）。见 [[第三章_向量与向量空间]] §3.6。
  4. 全部特征向量**单位化** $\eta_i=\beta_i/\|\beta_i\|$。
  5. 以 $\eta_i$ 为列拼成 $Q=(\eta_1,\dots,\eta_n)$，则 $Q^TAQ=\Lambda$。

> [!note] 核心理解
> 实对称矩阵代表"无旋转的纯伸缩变换"，所以总能找到一组**互相垂直**的特征方向（主轴），用正交矩阵（只转不缩、保长保角）就能把它摆正成对角阵。这正是二次型"正交变换化标准形 = 找椭球主轴"的几何根据。

> [!warning] 易错 / 边界
> 只有**重根**才需施密特正交化，不同特征值的特征向量本就正交别白做；$Q$ 必须单位化后才是正交矩阵；正交对角化用 $Q^T=Q^{-1}$，比一般对角化的 $P^{-1}$ 更省事。

## 速记卡 / 一句话总结

- **定义**：$Ax=\lambda x\ (x\neq0)$；特征值由 $\det(\lambda E-A)=0$ 求，特征向量解 $(\lambda E-A)x=0$。
- **数值**：$\sum\lambda_i=\operatorname{tr}A$，$\prod\lambda_i=|A|$；可逆 $\iff$ 无零特征值。
- **平移**：$Ax=\lambda x\Rightarrow A^k\!\to\lambda^k,\ aA+bE\!\to a\lambda+b,\ A^{-1}\!\to1/\lambda,\ A^*\!\to|A|/\lambda$（向量不变）。
- **相似**：$P^{-1}AP=B$；特征多项式 / 特征值 / 迹 / 行列式 / 秩全为相似不变量（必要非充分）。
- **对角化充要**：$n$ 个无关特征向量 $\iff$ 每个 $\lambda_i$ 几何重数 $n-r(\lambda_iE-A)=$ 代数重数 $k_i$。充分：$n$ 个相异特征值 / 实对称。用途：$A^k=P\Lambda^kP^{-1}$。
- **实对称**：特征值全实、异值向量正交、必正交对角化 $Q^TAQ=\Lambda$；五步法（求值→求向量→重根施密特→单位化→拼 $Q$）。
