## 一·随机变量与分布函数

  一·**随机变量定义**
    - 设 E 的样本空间为 $\Omega$,若对每个 $\omega \in \Omega$ 都有一个实数 $X(\omega)$ 与之对应,且 $X$ 是单值确定的,则称 $X$ 为**随机变量**。
    - 作用:把"事件语言"翻译成"实数语言",使概率问题可借助微积分处理。

  二·**分布函数定义**
    - 对任意实数 $x$,定义：
      $$F(x) = P\{X \le x\}$$
    - $F(x)$ 是定义在 $\mathbb R$ 上的实值函数。离散型与连续型都可用它统一刻画。

  三·**分布函数的性质**
    （1）**单调非减**：$x_1 < x_2 \Rightarrow F(x_1) \le F(x_2)$
    （2）**右连续**：$F(x+0) = F(x)$
    （3）==边界==：$F(-\infty) = 0$，$F(+\infty) = 1$
    （4）**由 F 求概率**：
      $$P\{a < X \le b\} = F(b) - F(a)$$
      $P\{X = a\} = F(a) - F(a-0)$（左极限,刻画单点概率）

  > [!note] 核心理解
  > 分布函数把任何类型的随机变量(离散/连续/混合)统一描述,关键是**"左开右闭"区间** $P\{a < X \le b\}$=端点差,这个约定让离散处用阶梯函数也自洽。

## 二·离散型随机变量

  一·**分布律**
    - 离散型 r.v. $X$ 取值 $x_k$ 的概率 $p_k = P\{X = x_k\}$ 必须满足：
      $$p_k \ge 0,\quad \sum_k p_k = 1$$
    - 表格形式:$(x_k, p_k)$ 全表。

  二·**常见离散分布**(考研数一必背四种)
    （1）**0-1 分布**(两点分布)：$P\{X=1\}=p,\ P\{X=0\}=q=1-p$
    （2）==二项分布== $X \sim B(n,p)$：
      $$P\{X=k\} = C_n^k\, p^k\, q^{n-k},\quad k=0,1,\dots,n$$
      - n 重伯努利试验成功次数的分布。
    （3）**泊松分布** $X \sim P(\lambda)$：
      $$P\{X=k\} = \frac{\lambda^k\, e^{-\lambda}}{k!},\quad k=0,1,2,\dots$$
      - $EX=DX=\lambda$。
    （4）**几何分布**(等待首次成功)：
      $$P\{X=k\} = q^{k-1}\, p,\quad k=1,2,\dots$$
      - **无记忆性**: $P\{X>s+t\mid X>s\} = P\{X>t\}$。
    （5）**超几何分布**(无放回抽样)：$N$ 件含 $M$ 件次品,取 $n$ 件中次品数 $X$：
      $$P\{X=k\} = \frac{C_M^k\, C_{N-M}^{n-k}}{C_N^n}$$

  三·**泊松定理**(二项→泊松近似)
    - 设 $n \to \infty$, $p_n \to 0$ 且 $np_n \to \lambda > 0$,则 $B(n, p_n) \xrightarrow{d} P(\lambda)$。
    - 实务判定:$n \ge 20,\ p \le 0.05$ 或 $np \le 5$ 时常用泊松近似二项,简化计算。
    - source:: 待核(系数阈值,教材略不同)

  > [!note] 核心理解
  > 泊松分布描述"单位时间/空间内稀有事件发生次数",是二项分布在大 n 小 p 下的极限,也是许多重要模型(排队、计数、风险)的基础。

## 三·连续型随机变量

  一·**概率密度函数**
    - $X$ 为连续型 ⟺ 存在 $f(x) \ge 0$ 使得 $F(x) = \int_{-\infty}^{x} f(t)\, dt$。
    - 性质:
      （1）$f(x) \ge 0$；但 $f(x)$ 可大于 1(不是概率!)
      （2）$\int_{-\infty}^{+\infty} f(x)\, dx = 1$
      （3）$P\{a < X < b\} = P\{a \le X \le b\} = \int_a^b f(x)\, dx$,单点概率为 0
      （4）若 $f$ 在 $x$ 连续,$F'(x) = f(x)$。

  二·**常见连续分布**
    （1）**均匀分布** $X \sim U(a,b)$：
      $$f(x) = \frac{1}{b-a},\quad a \le x \le b$$
      $EX=\frac{a+b}{2},\quad DX=\frac{(b-a)^2}{12}$
    （2）**指数分布** $X \sim Exp(\lambda)$($\lambda > 0$)：
      $$f(x) = \lambda\, e^{-\lambda x},\quad x \ge 0$$
      $EX=1/\lambda,\quad DX=1/\lambda^2$,==无记忆性==: $P\{X>s+t\mid X>s\}=P\{X>t\}$。
    （3）**正态分布** $X \sim N(\mu,\sigma^2)$：
      $$f(x) = \frac{1}{\sigma\sqrt{2\pi}}\, \exp\!\left(-\frac{(x-\mu)^2}{2\sigma^2}\right)$$
      - $EX=\mu,\quad DX=\sigma^2$；曲线关于 $x=\mu$ 对称；$\sigma$ 越大曲线越扁。
      - **标准化**：$Z = \dfrac{X-\mu}{\sigma} \sim N(0,1)$；查表用 $\Phi(z) = P\{Z \le z\}$。
      - **3$\sigma$ 性质**：$P\{|X-\mu|<3\sigma\}\approx 0.9973$。

  > [!note] 核心理解
  > 正态分布是中心极限定理的极限分布,几乎所有"独立因素叠加"的量都近似正态。无记忆性是指数的标志特性,等价于"剩余寿命与已活多久无关"。

## 四·随机变量函数的分布

  一·**离散情形** —— 列表法
    - 直接把 $g(x_k)$ 与对应 $p_k$ 合并,函数值相同的概率相加即可。

  二·==连续情形 ── 分布函数法==
    （1） 先求 $Y = g(X)$ 的分布函数：
      $$F_Y(y) = P\{Y \le y\} = P\{g(X) \le y\}$$
    （2） 再求导得密度：$f_Y(y) = F_Y'(y)$。
    - 一般情形(不要求 g 单调)首选此法,稳健不易错。

  三·**单调情形 ── 公式法**
    - 若 $y = g(x)$ 严格单调可导,反函数 $x = h(y)$,则：
      $$f_Y(y) = f_X(h(y))\, \left| h'(y) \right|$$
    - 注意:$g$ 必须严格单调,否则要分段累加。

  四·**常用特例**
    - $Y = aX + b$($a \ne 0$):
      $$f_Y(y) = \frac{1}{|a|}\, f_X\!\left(\frac{y-b}{a}\right)$$
    - $X \sim N(\mu,\sigma^2)$ ⟹ $Y=aX+b \sim N(a\mu+b, a^2\sigma^2)$。

## 速记卡

- 分布函数三性质: 单调非减、右连续、两端 0/1；用 F 求 $P\{a<X\le b\}=F(b)-F(a)$。
- 离散四件套: 0-1、二项 $C_n^k p^k q^{n-k}$、泊松 $\frac{\lambda^k e^{-\lambda}}{k!}$、几何 $q^{k-1}p$。
- 泊松定理: $n\to\infty, np\to\lambda$ ⟹ $B(n,p_n)\to P(\lambda)$。
- 连续三件套: 均匀、指数(无记忆性)、正态(标准化 $Z=(X-\mu)/\sigma$)。
- $f(x)$ 是密度,可 > 1；$\int f=1$；单点概率为 0。
- 求函数分布万能法: 先 $F_Y$ 再 $f_Y=F_Y'$；单调才用 $f_Y=f_X(h(y))|h'(y)|$。
- 正态线性变换仍是正态: $aX+b \sim N(a\mu+b,\, a^2\sigma^2)$。
