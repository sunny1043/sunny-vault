## 一·二维随机变量与联合分布

  一·**二维随机变量定义**
    - 设 $E$ 的样本空间为 $\Omega$,在 $\Omega$ 上定义两个 r.v. $X, Y$ → $(X, Y)$ 称为二维随机变量。
    - 几何直观: 把每次试验结果对应为平面上一随机点。

  二·**联合分布函数**
    - 对任意 $x, y \in \mathbb R$：
      $$F(x, y) = P\{X \le x,\ Y \le y\}$$
    - 性质:
      （1）关于 $x$、$y$ 分别单调非减
      （2）对每个变量右连续
      （3）$F(-\infty, y) = F(x, -\infty) = 0$，$F(+\infty, +\infty) = 1$
      （4）对任意 $x_1<x_2,\ y_1<y_2$:
        $$P\{x_1 < X \le x_2,\, y_1 < Y \le y_2\} = F(x_2,y_2)-F(x_1,y_2)-F(x_2,y_1)+F(x_1,y_1) \ge 0$$

  三·**离散型联合分布律**
    - 联合分布律 $p_{ij} = P\{X=x_i,\ Y=y_j\}$,满足：
      $$p_{ij} \ge 0,\quad \sum_i\sum_j p_{ij} = 1$$
    - 表格形式:行描述 X 取值、列描述 Y 取值。

  四·**连续型联合密度**
    - 若存在 $f(x,y) \ge 0$,使得：
      $$F(x,y) = \int_{-\infty}^{x}\int_{-\infty}^{y} f(u,v)\, dv\, du$$
      则 $(X,Y)$ 为连续型,$f(x,y)$ 为联合密度,满足 $\int_{-\infty}^{+\infty}\int_{-\infty}^{+\infty} f = 1$。
    - $P\{(X,Y) \in G\} = \iint_G f(x,y)\, dx\, dy$（在区域 G 上积分）。

## 二·边缘分布

  一·**边缘分布函数**
    - 由联合直接"压"掉另一变量:
      $$F_X(x) = F(x, +\infty),\quad F_Y(y) = F(+\infty, y)$$

  二·**离散型 — 边缘分布律**
    - 对另一变量求和：
      $$p_{X\cdot i} = \sum_j p_{ij},\qquad p_{\cdot jY} = \sum_i p_{ij}$$

  三·**连续型 — 边缘密度**
    - 对另一变量积分:
      $$f_X(x) = \int_{-\infty}^{+\infty} f(x, y)\, dy,\qquad f_Y(y) = \int_{-\infty}^{+\infty} f(x, y)\, dx$$

  > [!note] 核心理解
  > 边缘分布"压扁"了高维信息:**不同的联合分布可以有相同的边缘分布**。所以单看边缘无法反推联合。

## 三·条件分布

  一·**离散型条件分布律**
    - 在 $Y = y_j$ 条件下 X 的条件分布律：
      $$P\{X = x_i \mid Y = y_j\} = \frac{p_{ij}}{p_{\cdot jY}},\qquad p_{\cdot jY} > 0$$
    - 满足分布律非负、和为 1。

  二·**连续型条件密度**
    - 在 $Y = y$ 条件下($f_Y(y) > 0$):
      $$f_{X\mid Y}(x \mid y) = \frac{f(x, y)}{f_Y(y)}$$
    - **全概率密度公式**: $f_X(x) = \int_{-\infty}^{+\infty} f_{X\mid Y}(x\mid y)\, f_Y(y)\, dy$。
    - **贝叶斯密度形式**: $f_{Y\mid X}(y\mid x) = \dfrac{f_{X\mid Y}(x\mid y)\, f_Y(y)}{f_X(x)}$。

  三·**独立时的简化**
    - $X, Y$ 独立时:
      $$f_{X\mid Y}(x\mid y) = f_X(x),\quad P\{X=x_i\mid Y=y_j\} = p_{X\cdot i}$$
    - 即**条件分布 = 边缘分布**。

## 四·独立性

  一·**独立的等价定义**
    - 对任意 $x, y$：
      $$F(x, y) = F_X(x)\, F_Y(y)$$
    - 离散充要：$p_{ij} = p_{X\cdot i}\, p_{\cdot jY}$ 对一切 $i, j$ 成立。
    - 连续充要：$f(x, y) = f_X(x)\, f_Y(y)$ 几乎处处成立。

  二·**判别方法**
    - 看联合密度能否"因式分解"成各自边缘之积=独立。
    - 如联合密度含交叉项 $xy$ 之外的耦合因子,通常不独立。

  三·**二维正态分布**(重点)
    - $(X, Y) \sim N(\mu_1, \mu_2, \sigma_1^2, \sigma_2^2, \rho)$，联合密度形式(节选,含相关系数 $\rho$):
      $$f(x,y) = \frac{1}{2\pi\sigma_1\sigma_2\sqrt{1-\rho^2}}\exp\!\left[-\frac{1}{2(1-\rho^2)}\left(\frac{(x-\mu_1)^2}{\sigma_1^2} - 2\rho\frac{(x-\mu_1)(y-\mu_2)}{\sigma_1\sigma_2} + \frac{(y-\mu_2)^2}{\sigma_2^2}\right)\right]$$
    - **关键性质**:
      （1） $X \sim N(\mu_1, \sigma_1^2)$, $Y \sim N(\mu_2, \sigma_2^2)$（边缘正态）
      （2） 条件密度 $f_{X\mid Y}(x\mid y)$ 也是正态
      （3） ==独立的充要条件是 $\rho = 0$==；对一般情形"独立 ⟹ 不相关,反之未必",但**二维正态使两者等价**。
      （4） $X, Y$ 线性组合 $aX + bY$ 仍服从正态。

  > [!note] 核心理解
  > 二维正态是考研"独立 vs 不相关"问题的标准试金石:利用 $\rho=0 \Leftrightarrow$ 独立,可一招判别;一般情形只能得出"独立 ⟹ 不相关",不相关未必独立,反例常见于带参数的非线性联合密度。

## 五·二维随机变量函数的分布

  一·**$Z = X + Y$ 和的分布**
    - 离散型: $P\{Z=k\} = \sum_{i+j=k} p_{ij}$（按公式卷和）
    - ==连续型 — 卷积公式==：
      $$f_Z(z) = \int_{-\infty}^{+\infty} f(x, z - x)\, dx = \int_{-\infty}^{+\infty} f(z - y, y)\, dy$$
    - 若 X、Y 独立：
      $$f_Z(z) = \int_{-\infty}^{+\infty} f_X(x)\, f_Y(z - x)\, dx$$

  二·**$Z = \max(X, Y)$ 与 $\min(X, Y)$**
    - $X, Y$ 独立,分布函数分别为 $F_X, F_Y$：
      $$F_{\max}(z) = F_X(z)\, F_Y(z)$$
      $$F_{\min}(z) = 1 - \big[1-F_X(z)\big]\big[1-F_Y(z)\big]$$
    - 推广到 n 个独立变量:max 是各 F 乘积;min 是各"余"乘积再补 1。

  三·**二维正态的线性组合**
    - $(X,Y)$ 二维正态 ⟹ $Z = aX + bY + c \sim N(a\mu_1+b\mu_2+c,\ a^2\sigma_1^2 + b^2\sigma_2^2 + 2ab\,\mathrm{cov}(X,Y))$

  四·**$Z = X/Y$、$Z = X^2 + Y^2$ 等**
    - 通用思路:用分布函数法 $F_Z(z) = P\{Z \le z\}$,化为 $(X,Y)$ 落在某个区域的概率,再积分。

## 速记卡

- 联合三性质: 单调非减、右连续、两端 0/1；矩形概率 4 项组合式。
- 边缘: 离散求和、连续积分;**边缘不能反推联合**。
- 条件分布: $f_{X|Y}=f/f_Y$；独立时条件就是边缘。
- 独立等价三式: $F=F_X F_Y$ / $p_{ij}=p_{X\cdot i}\,p_{\cdot jY}$ / $f=f_X f_Y$。
- 二维正态: $\rho=0 \Leftrightarrow$ 独立 $\Leftrightarrow$ 不相关;线性组合还是正态。
- 和 Z=X+Y: 卷积 $\int f_X(x)f_Y(z-x)dx$(独立时)。
- max = 各 F 相乘;min = 各 "1−F" 相乘再 1−。
