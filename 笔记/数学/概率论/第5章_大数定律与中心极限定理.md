## 一·直观含义与依概率收敛

  一·**大数定律的物理含义**
    - **频率稳定性**: 当重复试验次数 n 增大时,事件 A 的频率 $f_n(A) = n_A/n$ 在某种意义下接近其概率 $P(A)$。
    - **样本均值收敛**: 期望相同的 r.v. 序列的平均 $\overline{X}_n = \frac{1}{n}\sum X_k$ 接近总体均值 $\mu$。
    - 大数定律是"用频率估计概率"和"用样本均值估计期望"的理论依据。

  二·**依概率收敛的定义**
    - 设 $\{X_n\}$ 为 r.v. 序列,若对任意 $\varepsilon > 0$:
      $$\lim_{n\to\infty} P\{|X_n - a| < \varepsilon\} = 1$$
      则称 $X_n$ **依概率收敛于常数 a**,记 $X_n \xrightarrow{P} a$。
    - 强度介于"几乎必然收敛"(强)与"分布收敛"(弱)之间。

  > [!note] 核心理解
  > 依概率收敛比通常的"收敛"更弱——允许"小概率偏离",但偏离幅度可任意小且其概率趋于 0。这是切比雪夫等大数定律论证的工具,也是后续统计估计的理论根基。

## 二·切比雪夫大数定律

  一·**条件**
    - $X_1, X_2, \dots$ 相互独立;
    - 方差一致有上界: 存在常数 C 使 $DX_k \le C$\ ($k = 1, 2, \dots$);
    - 注意**不要求同分布**。

  二·**结论**
    - 设 $\overline{X}_n = \frac{1}{n}\sum_{k=1}^{n} X_k$，$\overline{EX}_n = \frac{1}{n}\sum_{k=1}^{n} EX_k$，则：
      $$\overline{X}_n - \overline{EX}_n \xrightarrow{P} 0$$
    - 即样本均值(中心化后)依概率收敛于期望均值。

  三·**证明思路**(切比雪夫不等式)
    - 计算 $D(\overline{X}_n - \overline{EX}_n) = \frac{1}{n^2} \sum D X_k \le \frac{C}{n} \to 0$。
    - 用切比雪夫不等式得 $P\{|\overline{X}_n - \overline{EX}_n| \ge \varepsilon\} \le \dfrac{C}{n\varepsilon^2} \to 0$。

## 三·辛钦大数定律

  一·**条件**
    - $X_1, X_2, \dots$ ==独立同分布== (i.i.d.);
    - 期望存在: $EX_k = \mu$ 有限；
    - ==不要求方差存在== (相比切比雪夫弱化)。

  二·**结论**
    $$\overline{X}_n = \frac{1}{n}\sum_{k=1}^{n} X_k \xrightarrow{P} \mu$$
    - 样本均值依概率收敛于总体均值 $\mu$。
    - 这是**点估计理论根基**:用样本平均估计期望在大样本下是合法的。

  > [!note] 核心理解
  > 辛钦比切比雪夫"成立条件更宽松"——不要求方差有界,只要期望存在就行,但要求"独立同分布"。它对应工程实务最常见的情况:i.i.d. 抽样估计参数。

## 四·伯努利大数定律

  一·**条件**
    - n 重伯努利试验独立重复,每次试验 A 发生概率为 p。
    - 记 $n_A$ = n 次试验中 A 发生次数 → 频率 $f_n = n_A / n$。
    - 等价模型: $X_k \sim B(1, p)$ 独立同分布, $EX_k = p$。

  二·**结论**
    $$\frac{n_A}{n} \xrightarrow{P} p$$
    - 频率依概率收敛于概率,**"频率学派"概率定义的合法性来源**。

  三·**与切比雪夫/辛钦的关系**
    - 伯努利大数定律是切比雪夫大数定律的特殊情形(i.i.d. + 有界方差 $DX_k = p q$)；
    - 也是辛钦的特殊情形(i.i.d. + $EX_k = p$ 存在)。

## 五·林德伯格-列维中心极限定理

  一·**条件**
    - $X_1, X_2, \dots$ ==独立同分布==;
    - $EX_k = \mu$, $DX_k = \sigma^2$,且 $0 < \sigma^2 < \infty$。

  二·**结论**
    - 设 $Y_n = \sum_{k=1}^{n} X_k$,则其标准化变量:
      $$Z_n = \frac{Y_n - n\mu}{\sigma\sqrt{n}} \xrightarrow{d} N(0, 1),\quad n \to \infty$$
    - 实务近似: 当 n 较大时:
      $$\sum_{k=1}^{n} X_k \dot\sim N(n\mu,\ n\sigma^2)$$

  三·**意义**
    - 大样本下 i.i.d. 和的分布近似正态,无论原始分布是什么,**只要方差有限**。
    - 实际计算:对 i.i.d. 总体的样本和,可用正态近似算 $P\{\sum X_k \le a\}$。

## 六·棣莫弗-拉普拉斯 CLT

  一·**条件**
    - 设 $Y_n \sim B(n, p)$ (即 $Y_n = \sum_{k=1}^{n} I_k$, $I_k$ 为 i.i.d. 0-1 变量)。

  二·**结论**
    - 标准化后:
      $$\frac{Y_n - np}{\sqrt{npq}} \xrightarrow{d} N(0, 1),\quad n \to \infty$$
    - 用于"$Y_n \sim B(n,p)$, n 大"时的近似计算,把二项分布近似为正态。

  三·**实务判定**
    - 常用判定条件: $np \ge 5$ 且 $nq \ge 5$ 时二项可近似正态(source:: 待核,各教材略异)。
    - 与泊松近似的对比: $n$ 大 $p$ 小用泊松,$n$ 大 $p$ 不太小用正态。

## 七·三大大数定律条件对比

| 定律 | 关键条件 | 结论(依概率收敛) | 适用面 |
|---|---|---|---|
| ==切比雪夫== | 独立(不必同分布)，方差有界 $DX_k \le C$ | $\overline{X}_n - \overline{EX}_n \xrightarrow{P} 0$ | 最宽松(允许异分布) |
| ==辛钦== | i.i.d.，期望 $\mu$ 存在(**不要求方差**) | $\overline{X}_n \xrightarrow{P} \mu$ | 实务最常见(同分布抽样) |
| ==伯努利== | i.i.d. $B(1, p)$，n 重伯努利试验 | $n_A/n \xrightarrow{P} p$ | 频率→概率,伯努利特例 |

| 定理 | 条件 | 极限分布 | 用途 |
|---|---|---|---|
| 林德伯格-列维 CLT | i.i.d.，$EX,DX\in(0,\infty)$ | $\dfrac{\sum X_k - n\mu}{\sigma\sqrt{n}} \to N(0,1)$ | i.i.d. 和的近似计算 |
| 棣莫弗-拉普拉斯 CLT | $Y_n \sim B(n,p)$ | $\dfrac{Y_n-np}{\sqrt{npq}} \to N(0,1)$ | 二项近似正态 |

## 速记卡

- 依概率收敛: $\forall\varepsilon>0,\ P\{|X_n - a|<\varepsilon\}\to 1$;比 a.s. 弱比依分布强。
- 切比雪夫: 独立方差有界 ⟹ 样本均值中心化后 →0；证明靠切比雪夫不等式。
- 辛钦: i.i.d.+期望有限 ⟹ 样本均值 → μ;**不要求方差**。
- 伯努利: 频率 → 概率,**频率学派合法性根基**,是辛钦/切比雪夫的特例。
- 林德伯格-列维 CLT: i.i.d.+方差有限 ⟹ $(\sum X - n\mu)/\sigma\sqrt n \to N(0,1)$。
- 棣莫弗-拉普拉斯: $B(n,p)$ 标准化后 → $N(0,1)$;常用条件 np, nq ≥ 5。
- 大数定律论证依赖**切比雪夫不等式**;CLT 的"近似正态"是计数器,大数定律只给"≥ 几近"。
- 三者关系: **辛钦 ⊂ 切比雪夫特殊化;伯努利 ⊂ 辛钦**(伯努利是 i.i.d. 0-1 特例)。
