## 一·样本空间与随机事件

  一·**基本定义**
    - **样本空间 Ω**：随机试验 E 的所有可能结果组成的集合。
    - **样本点 ω**：样本空间中的元素,即一个不可分的试验结果。
    - **随机事件**：Ω 的子集,简称事件。事件 A 发生 ⟺ A 中某一样本点出现。
    - **必然事件 Ω**、**不可能事件 ∅**：分别恒发生、恒不发生。
    - **基本事件**：由单个样本点构成的事件。

  二·**事件的关系**
    1.**包含** $A \subset B$：A 发生必导致 B 发生。
    2.**相等** $A = B$：$A \subset B$ 且 $B \subset A$。
    3.**和(并)** $A \cup B$：A、B 至少一个发生。
    4.**积(交)** $A \cap B$（记 $AB$）：A、B 同时发生。
    5.**互斥** $AB = \emptyset$：A、B 不能同时发生。
    6.**对立** $A \cup B = \Omega$ 且 $AB = \emptyset$：B 记作 $\bar A$。
    7.**差** $A - B = A \bar B$：A 发生而 B 不发生。

  三·**运算律**
    （1）交换律：$A \cup B = B \cup A$，$AB = BA$
    （2）结合律：$(A \cup B) \cup C = A \cup (B \cup C)$
    （3）分配律：$A(B \cup C) = AB \cup AC$
    （4）==德摩根律==：
      $$\overline{A \cup B} = \bar A \cap \bar B, \qquad \overline{A \cap B} = \bar A \cup \bar B$$
      推广:$\overline{\bigcup_i A_i} = \bigcap_i \bar A_i$，$\overline{\bigcap_i A_i} = \bigcup_i \bar A_i$。

  > [!note] 核心理解
  > 德摩根律的本质：**"或"取反变成"且",且每个都取反**。集合运算与逻辑"或/且/非"一一对应,德摩根就是逻辑反演的对偶。

## 二·概率定义与性质

  一·**古典概型**(等可能·有限)
    1.**条件**：Ω 有限;每个基本事件等可能。
    2.**公式**：
      $$P(A) = \frac{k}{n} = \frac{\text{A 中基本事件数}}{\text{Ω 中基本事件总数}}$$
    3.**计数工具**：排列 $A_n^k$、组合 $C_n^k$、可重排列 $n^k$、分组数。

  二·**几何概型**
    1.**条件**：Ω 是可测区域(线/面/体);随机点落在子区域概率与测度成正比。
    2.**公式**：
      $$P(A) = \frac{m(A)}{m(\Omega)}$$
    - 注意：与区域形状和位置无关,只与测度有关。

  三·**频率与统计概率**
    - 频率 $f_n(A) = n_A / n$；当 $n \to \infty$ 时 $f_n(A) \to P(A)$（频率稳定性）。

  四·**概率公理化**(柯尔莫哥洛夫三公理)
    （1）**非负性**：$P(A) \ge 0$
    （2）**规范性**：$P(\Omega) = 1$
    （3）**可列可加性**：若 $A_i A_j = \emptyset\ (i \ne j)$,则 $P\!\left(\bigcup_i A_i\right) = \sum_i P(A_i)$

  五·**由公理推出的重要性质**
    1.**有限可加性**：互斥完备 $A_1,\dots,A_n$ ⟹ $P(\cup A_i) = \sum P(A_i)$
    2.**对立公式**：
      $$P(\bar A) = 1 - P(A)$$
    3.==加法公式==（一般情形,不要求互斥）：
      $$P(A \cup B) = P(A) + P(B) - P(AB)$$
      三事件版：$P(A\cup B\cup C) = P(A)+P(B)+P(C) - P(AB)-P(AC)-P(BC) + P(ABC)$
    4.**单调性**：$A \subset B \Rightarrow P(A) \le P(B)$
    5.**减法公式**：$P(A-B) = P(A) - P(AB)$

  > [!note] 核心理解
  > 公理化不解决"概率怎么算",只规定任何合法 P 必须满足的三条约束。古典/几何/频率定义各自是公理在不同场景下的具体实现。

## 三·条件概率与乘法公式

  一·**条件概率定义**
    - 在 B 已发生的条件下,A 发生的概率：
      $$P(A \mid B) = \frac{P(AB)}{P(B)}, \qquad P(B) > 0$$
    - 几何直觉：B 把样本空间缩到 B,在 B 内看 A 占的比例。

  二·**乘法公式**
    1.两事件：$P(AB) = P(B)\,P(A \mid B) = P(A)\,P(B \mid A)$
    2.==n 个事件的链式法则==：
      $$P(A_1 A_2 \cdots A_n) = P(A_1)\,P(A_2 \mid A_1)\,P(A_3 \mid A_1 A_2)\,\cdots\,P(A_n \mid A_1 \cdots A_{n-1})$$

  三·**全概率公式**
    - 若 $B_1,\dots,B_n$ 互斥完备($B_i B_j = \emptyset,\ \cup B_i = \Omega,\ P(B_i) > 0$),则对任意 A：
      $$P(A) = \sum_{i=1}^{n} P(B_i)\,P(A \mid B_i)$$
    - 直觉：把"求 A"拆成"在每种原因 $B_i$ 下 A 的概率"的加权平均,权重为先验 $P(B_i)$。

## 四·贝叶斯公式

  一·**后验公式**
    - 在全概率公式条件下,已知 A 发生求原因 $B_k$ 的概率：
      $$P(B_k \mid A) = \frac{P(B_k)\,P(A \mid B_k)}{\sum_{i=1}^{n} P(B_i)\,P(A \mid B_i)}$$
    - 分子是 $B_k$ 引起 A 的"联合强度",分母是 A 发生的总概率(归一化)。

  二·**三要素**
    （1）**先验** $P(B_i)$：未观测 A 之前对原因的信念。
    （2）**似然** $P(A \mid B_i)$：在原因 $B_i$ 下观测到 A 的概率。
    （3）**后验** $P(B_i \mid A)$：观测到 A 之后对原因的更新信念。

  > [!note] 核心理解
  > 贝叶斯的方向是**由果溯因**。它把"看到结果反推原因"的问题,严格分解成"原因→结果"(似然)与"原因的先验"的乘积,再归一化。

  三·**经典例：疾病检测**(source:: 待核,常见教材例)
    - 患病率 $P(D) = 0.001$；灵敏度 $P(+\mid D) = 0.99$；假阳率 $P(+\mid \bar D) = 0.01$。
    - 求 $P(D \mid +)$：
      $$P(D \mid +) = \frac{0.001 \times 0.99}{0.001 \times 0.99 + 0.999 \times 0.01} \approx \frac{0.00099}{0.010989} \approx 0.090$$
    - 即阳性者真正患病的概率约 9%,远低于直觉 99%,因基数极低。

## 五·事件独立性

  一·**两事件独立**
    - 定义：
      $$P(AB) = P(A)\,P(B)$$
    - 与互斥的关系：若 $P(A), P(B) > 0$,**独立 ⟹ 不互斥,互斥 ⟹ 不独立**（因互斥 $P(AB)=0 \ne P(A)P(B)$）。

  二·**n 个事件相互独立**
    - 任取 $1 \le i_1 < \cdots < i_k \le n$，都有 $P(A_{i_1}\cdots A_{i_k}) = P(A_{i_1})\cdots P(A_{i_k})$。
    - ==相互独立 ⟹ 两两独立,反之未必==。

  三·**伯努利概型**
    1. n 重伯努利试验：n 次独立重复,每次只有 $A/\bar A$ 两种结果,$P(A) = p$。
    2. **二项概率公式**：A 发生 k 次的概率：
       $$P(X = k) = C_n^k\, p^k\, (1-p)^{n-k}, \quad k = 0,1,\dots,n$$
    3. 典型应用：产品抽样合格率、命中率累加。

## 速记卡

- 德摩根律：**并取反=各自反的交**,交取反=各自反的并。
- 加法公式：$P(A\cup B) = P(A)+P(B)-P(AB)$（==互斥时才退化成 $P(A)+P(B)$==,因为 $P(AB)=0$）。
- 条件概率三件套:**条件** $P(A|B)=P(AB)/P(B)$、**乘法** $P(AB)=P(B)P(A|B)$、**全概率** $P(A)=\sum P(B_i)P(A|B_i)$。
- 贝叶斯方向:$B_i \xrightarrow{\text{引起}} A \xrightarrow{\text{反推}} P(B_i|A)$,分子=联合,分母=归一化。
- 独立 vs 互斥：**两者互斥互推不到**(概率 >0 时互斥必不独立,独立必不互斥)。
- 伯努利：n 次独立 0-1 试验,成功 k 次的概率 $C_n^k p^k q^{n-k}$。
- 公理三条：非负、规范、可列可加。
