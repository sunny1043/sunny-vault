## §3.0 核心问题

一·**主线**：向量组的**线性相关性**是本章主线，它一手连两头：
  1. 往下接 [[笔记/数学/线性代数/04-线性方程组/第四章_线性方程组|第四章_线性方程组]]：解的存在唯一、基础解系个数全由相关性决定。
  2. 往上接 [[笔记/数学/线性代数/02-矩阵/第二章_矩阵|第二章_矩阵]]：相关性 $\iff$ 矩阵的秩 $\iff$ 行列式是否为零。
二·**一句话**：==相关性 = 向量组里"有没有多余的方向"==。无关 = 个个都贡献新方向；相关 = 至少一个能被别人表示出来（冗余）。

> [!note] 核心理解
> 把每个向量看成空间里的一个"方向"。线性相关 = 这些方向挤在一个更低维的子空间里（共线、共面…）；线性无关 = 它们撑开了满维数的空间。整章就是在量化"撑开了几维"——这个数就是秩。

---

## §3.1 线性组合与线性表示

一·**定义**：
  1. **线性组合**：$k_1\boldsymbol\alpha_1+k_2\boldsymbol\alpha_2+\cdots+k_s\boldsymbol\alpha_s$（$k_i$ 为数）。
  2. **$\boldsymbol\beta$ 可由 $\boldsymbol\alpha_1,\dots,\boldsymbol\alpha_s$ 线性表示**：存在数 $k_i$ 使 $\boldsymbol\beta=\sum k_i\boldsymbol\alpha_i$。

二·**判定（化为方程组）**：把 $\boldsymbol\alpha_i$ 作列拼成 $A=(\boldsymbol\alpha_1,\dots,\boldsymbol\alpha_s)$，则
$$\boldsymbol\beta\text{ 可被表示}\iff Ax=\boldsymbol\beta\text{ 有解}\iff r(A)=r(A\,|\,\boldsymbol\beta)$$
  1. 表示**唯一** $\iff r(A)=r(A|\boldsymbol\beta)=s$（列满秩）；表示**无穷多** $\iff r(A)=r(A|\boldsymbol\beta)<s$。

> [!note] 核心理解
> "能否表示"= "$\boldsymbol\beta$ 在不在 $\boldsymbol\alpha$ 们张成的空间里"，翻译成代数就是"$Ax=\boldsymbol\beta$ 有没有解"。秩相等 = 添了 $\boldsymbol\beta$ 这一列没扩大列空间 = $\boldsymbol\beta$ 早就在里面。这把几何问题彻底化成了 §4 的方程组问题。

---

## §3.2 线性相关与线性无关

一·**定义**：对向量组 $\boldsymbol\alpha_1,\dots,\boldsymbol\alpha_s$，看齐次方程
$$k_1\boldsymbol\alpha_1+\cdots+k_s\boldsymbol\alpha_s=\boldsymbol0$$
  1. **只有全零解**（$k_1=\cdots=k_s=0$）→ **线性无关**。
  2. **存在不全为零的解** → **线性相关**（即至少一个向量是其余的线性组合）。

二·**几何直觉**：
  1. 二向量相关 $\iff$ 共线；三向量相关 $\iff$ 共面。
  2. ==$n$ 维空间里 $>n$ 个向量必相关==（鸽巢式结论，见 §3.3 维数定理）。

三·**代数判定（核心）**：把向量作列拼成 $A=(\boldsymbol\alpha_1,\dots,\boldsymbol\alpha_s)_{n\times s}$：
$$\text{无关}\iff r(A)=s\ (\text{满列秩})\qquad\text{相关}\iff r(A)<s$$
  1. 特例：$s=n$（方阵）时，无关 $\iff|A|\neq0$；相关 $\iff|A|=0$。
  2. 特例：$s>n$（向量个数 > 维数）时 $r(A)\le n<s$ → **必相关**。

四·**公式与推导**（为何"相关 $\iff r(A)<s$"）：
  1. $\sum k_i\boldsymbol\alpha_i=\boldsymbol0$ 写成矩阵形式就是齐次方程组 $Ax=\boldsymbol0$，$x=(k_1,\dots,k_s)^T$。　← 线性组合 = 矩阵乘列
  2. $Ax=\boldsymbol0$ 有非零解 $\iff r(A)<s$（未知数个数 $s$ > 秩）。　← 齐次方程非零解条件，见 [[笔记/数学/线性代数/04-线性方程组/第四章_线性方程组|第四章_线性方程组]]
  3. 有非零解 = 存在不全零的 $k_i$ = 线性相关。∎

> [!note] 核心理解
> 相关/无关本质是"齐次方程组有无非零解"的几何说法。秩 $r(A)$ 就是这组向量真正撑开的维数：$r=s$ 说明 $s$ 个方向个个独立（无关），$r<s$ 说明有 $s-r$ 个方向是多余的（相关）。

> [!warning] 易错 / 边界
> 含**零向量**的组必相关；只有**一个**非零向量必无关；判定务必"向量作列、看秩"，别凭感觉。

---

## §3.3 核心定理

四条定理常以"证明 / 判断"形式直接考。

一·**部分与整体**：
  1. 部分组相关 $\Rightarrow$ 整体组相关（"一部分挤在低维，整体也逃不掉"）。
  2. 整体组无关 $\Rightarrow$ 任一部分组无关（逆否命题）。
二·**延长与缩短**（对"每个向量加/减分量"而言）：
  1. 无关组**接长维数**（每个向量末尾添分量）→ 仍无关（添的信息只会更难相关）。
  2. 相关组**截短维数**（每个向量删同样位置分量）→ 仍相关。
三·**维数定理**：$n$ 维空间中，任意 $s>n$ 个向量必线性相关。
  1. 证：作 $A_{n\times s}$，$r(A)\le n<s$ → 由 §3.2 判定即得。
四·**表示与个数（替换定理推论）**：若无关组 $\boldsymbol\alpha_1,\dots,\boldsymbol\alpha_p$ 能被 $\boldsymbol\beta_1,\dots,\boldsymbol\beta_q$ 线性表示，则 ==$p\le q$==。
  1. 推论：两个**等价**的线性无关组所含向量个数**相等**（这是"维数 / 秩定义良好"的根据）。
  2. 直觉：能被 $q$ 个向量表示的，最多只能有 $q$ 个互相独立——"被表示者不可能比表示者更丰富"。

> [!note] 核心理解
> 这组定理全是"维数 / 秩不会被夸大"的不同侧面。定理四（$p\le q$）是全章的逻辑基石：正因为它，"极大无关组的个数"才唯一、"维数"才是良定义的概念。

## §3.4 极大无关组与秩

一·**定义**：向量组的**极大线性无关组** = 选出尽量多的向量使其无关，且再添任何一个原组向量都变相关。
  1. 它含的向量个数 = 向量组的**秩** $r$；原组任一向量都能由它**唯一**线性表示。
  2. 同一向量组的极大无关组**不唯一**，但**个数（秩）唯一**（由 §3.3 定理四保证）。

二·**求法（初等行变换）**：
  1. 向量作列拼成 $A$，化**行阶梯形**。　← 行变换不改变列之间的线性关系
  2. 阶梯形中**主元（首非零元）所在的列**对应的原向量 → 构成一个极大无关组。
  3. 非主元列的向量 = 主元列向量的组合，组合系数直接从阶梯形读出（化最简行阶梯更直观）。

三·**等价向量组**：两组能互相线性表示称等价。
  1. 等价 $\Rightarrow$ 秩相等（反之不一定，秩相等未必等价）。

> [!note] 核心理解
> 极大无关组 = 向量组的"骨架基底"，秩 = 骨架的根数。行变换不改变"列与列之间谁能表示谁"的关系，所以能在阶梯形上一眼挑出独立列——这是求极大无关组只用行变换、且要看原向量的原因。

## §3.5 向量空间·基·维数·过渡矩阵（数一专属）

### 基与维数

一·**定义**：
  1. **向量空间 $V$**：对加法和数乘**封闭**的向量集合（含零向量）。
  2. **基**：$V$ 中一组**线性无关**且能**张成整个 $V$** 的向量 $\boldsymbol\alpha_1,\dots,\boldsymbol\alpha_n$。
  3. **维数** $\dim V=n$：基所含向量个数（与基的选取无关）。
  4. **坐标**：$V$ 中任一 $\boldsymbol\xi$ 在基 $\{\boldsymbol\alpha_i\}$ 下唯一写成 $\boldsymbol\xi=\sum x_i\boldsymbol\alpha_i$，$(x_1,\dots,x_n)^T$ 即坐标。

### 过渡矩阵与坐标变换

一·**定义**：从基 $\{\boldsymbol\alpha_i\}$ 到基 $\{\boldsymbol\beta_i\}$ 的**过渡矩阵** $P$ 满足
$$(\boldsymbol\beta_1,\dots,\boldsymbol\beta_n)=(\boldsymbol\alpha_1,\dots,\boldsymbol\alpha_n)P$$
$P$ 必可逆（两基都无关）。

二·**坐标变换公式**：同一向量在两基下的坐标 $x$（$\alpha$ 基）、$y$（$\beta$ 基）满足
$$x=Py\qquad\Longleftrightarrow\qquad y=P^{-1}x$$

三·**公式与推导**：
  1. 同一向量 $\boldsymbol\xi=(\boldsymbol\alpha)x=(\boldsymbol\beta)y$（行向量基 × 列坐标）。　← 坐标的定义
  2. 代入 $(\boldsymbol\beta)=(\boldsymbol\alpha)P$：$\boldsymbol\xi=(\boldsymbol\alpha)Py$。
  3. 与 $\boldsymbol\xi=(\boldsymbol\alpha)x$ 比较，且 $\alpha$ 基无关（表示唯一）→ $x=Py$。∎
  → ⚠️==注意"基变换"与"坐标变换"方向相反==：基 $\beta=\alpha P$，而坐标 $y=P^{-1}x$。这是最常错的点。

四·**求 $P$（具体基）**：$(\boldsymbol\alpha\,|\,\boldsymbol\beta)\xrightarrow{\text{行变换}}(E\,|\,P)$，即 $P=\alpha^{-1}\beta$（把基都看成列拼的方阵）。

```
基:    {α} ──(右乘 P)──→ {β}        β = αP
坐标:   x ──(乘 P⁻¹)──→ y           y = P⁻¹x      （方向相反！）
含义: 基向量"变多/变长"，同一点的坐标数值就"变少/变短"
```

> [!note] 核心理解
> 基是"量空间的尺子"，坐标是"用这把尺子量某向量得到的读数"。换一把尺子（换基），向量本身没动，但读数（坐标）按 $P^{-1}$ 变——尺子和读数天生反向，正如"尺子刻度变密，读数变大"。

> [!warning] 易错 / 边界
> 基必须既"无关"又"张成"，缺一不可；$\dim$ 与基的选择无关；过渡矩阵一定可逆。

## §3.6 内积·正交·施密特正交化

一·**定义**：
  1. **内积** $(\boldsymbol\alpha,\boldsymbol\beta)=\boldsymbol\alpha^T\boldsymbol\beta=\sum a_ib_i$；**长度（范数）** $\|\boldsymbol\alpha\|=\sqrt{(\boldsymbol\alpha,\boldsymbol\alpha)}$。
  2. **正交**：$(\boldsymbol\alpha,\boldsymbol\beta)=0$。**标准正交基**：两两正交且都为单位长度的基。
  3. **正交矩阵** $Q$：$Q^TQ=E$（即 $Q^{-1}=Q^T$，列向量构成标准正交组）。

二·**施密特（Schmidt）正交化**：把无关组 $\boldsymbol\alpha_1,\dots,\boldsymbol\alpha_s$ 化成正交组 $\boldsymbol\beta_1,\dots,\boldsymbol\beta_s$。
$$\boldsymbol\beta_1=\boldsymbol\alpha_1,\qquad \boldsymbol\beta_k=\boldsymbol\alpha_k-\sum_{j=1}^{k-1}\frac{(\boldsymbol\alpha_k,\boldsymbol\beta_j)}{(\boldsymbol\beta_j,\boldsymbol\beta_j)}\boldsymbol\beta_j$$

三·**公式与推导**（第 $k$ 步的几何）：
  1. 想让新 $\boldsymbol\beta_k$ 与已得的 $\boldsymbol\beta_1,\dots,\boldsymbol\beta_{k-1}$ 都正交。　← 目标
  2. 做法：从 $\boldsymbol\alpha_k$ 中**减去它在每个 $\boldsymbol\beta_j$ 上的投影** $\dfrac{(\boldsymbol\alpha_k,\boldsymbol\beta_j)}{(\boldsymbol\beta_j,\boldsymbol\beta_j)}\boldsymbol\beta_j$。　← 投影 = 平行分量
  3. 减掉所有平行分量，剩下的就是与前面全部正交的"垂直分量"。
  4. 最后**单位化** $\boldsymbol\eta_k=\boldsymbol\beta_k/\|\boldsymbol\beta_k\|$ 得标准正交组。

```
α_k = (在 β_j 方向的投影分量之和)  +  (垂直分量)
                ↓ 减掉投影                 ↓ 保留
              平行部分丢弃            β_k ⟂ 所有 β_j
```

> [!note] 核心理解
> 正交化 = "逐个把后来的向量里、与前面方向重复的部分扣掉"，留下纯粹的新方向。几何就是反复做"减投影"。它是 [[笔记/数学/线性代数/06-二次型/第六章_二次型|第六章_二次型]] 正交变换、实对称矩阵正交对角化的前置工序——重根特征向量必须先正交化才能拼出正交矩阵 $Q$。

> [!warning] 易错 / 边界
> 施密特要求输入是**无关组**（相关组中途会出 $\boldsymbol\beta_k=\boldsymbol0$）；正交化后顺序不能乱（$\boldsymbol\beta_k$ 依赖前面所有 $\boldsymbol\beta_j$）。

## 速记卡 / 一句话总结

- **主线**：相关性 = 向量组里有没有多余方向；秩 = 真正撑开的维数。
- **判定**：向量作列拼 $A_{n\times s}$ → 无关 $\iff r(A)=s$；相关 $\iff r(A)<s$。$s>n$ 必相关；含零向量必相关。
- **表示**：$\boldsymbol\beta$ 可被表示 $\iff r(A)=r(A|\boldsymbol\beta)$；唯一 $\iff$ 还等于 $s$。
- **四定理**：部分相关→整体相关；无关延长仍无关、相关缩短仍相关；$n$ 维 $>n$ 个必相关；无关组被另一组表示 → $p\le q$，等价无关组个数相等。
- **极大无关组**：行变换化阶梯，主元列对应的原向量即是；个数 = 秩（唯一）。
- **基/维数/过渡阵**：$\beta=\alpha P$（$P$ 可逆），坐标 $x=Py,\ y=P^{-1}x$（==基与坐标反向==）；求 $P$：$(\alpha|\beta)\to(E|P)$。
- **施密特**：$\boldsymbol\beta_k=\boldsymbol\alpha_k-\sum\frac{(\alpha_k,\beta_j)}{(\beta_j,\beta_j)}\beta_j$（减投影），再单位化；正交矩阵 $Q^TQ=E$。
