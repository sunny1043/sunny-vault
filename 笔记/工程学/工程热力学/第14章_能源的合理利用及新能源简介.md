## 一· 能量与能源的分类

一· 六种能量形式

1. **机械能**：与宏观机械运动或空间状态相关
   - 动能：因机械运动具有的做功能力
   - 势能：重力势能（高度差）+ 弹性势能（形变）
2. **热能**：分子动能 + 分子势能的总和 → ==所有其他形式的能量都可以完全转换为热能==
3. **电能**：与电子流动和积累相关，由化学能/机械能转换而来
4. **辐射能**：以电磁波形式发射，含 γ射线/X射线/热辐射/微波/无线电波。热辐射 = 紫外线+可见光+红外线
5. **化学能**：原子核外化学反应释放的物质结构能，通常用燃料发热量表示
6. **核能**：原子核结构变化释放的能量，分核衰变/核裂变/核聚变

> [!note] 核心理解
> 物理上：六种能量形式的本质区别在于**能量载体层级**——机械能（宏观物体运动）→ 热能（分子运动）→ 化学能（核外电子重排）→ 核能（核内核子重排），越往下能量密度越高。数学上：质能方程 E = mc² 说明了核能的巨大来源——质量亏损转换为能量。

二· 能源的五大分类维度

1. 按来源
   - （1）地球自身：核能、地热能
   - （2）地球外天体：太阳能及其衍生（煤/石油/天然气/生物质/水能/风能/波浪能）
   - （3）天体相互作用：潮汐能（月球引力）

2. 按获得方法
   - （1）**一次能源**：自然界存在、未经加工（煤、石油、天然气、水能、风能）
   - （2）**二次能源**：由一次能源加工转换（蒸汽、煤气、电、氢、酒精、焦炭、激光）

3. 按可否再生
   - （1）可再生能源：水能、风能、潮汐能、太阳能
   - （2）非再生能源：煤、石油、天然气、核燃料

4. 按污染
   - （1）清洁能源：太阳能、水能、海洋能
   - （2）非清洁能源：煤、石油

5. 按技术水平
   - （1）**常规能源**：技术成熟、大量生产（煤、石油、天然气、水能）
   - （2）**新能源**：正在研发（太阳能、地热能、生物质能、潮汐能；核裂变有争议——部分学者划入常规）

## 二· 能源的合理利用

一· ==能量的品位==

1. 三级品位划分
   - **高级能**（λ=1，完全有序）：机械能、电能、原子能、化学能 → 适合直接做功
   - **中级能**（0<λ<1，部分有序）：热能、热辐射能 → 品位随温度升高
   - **低级能**：与环境平衡、做功能力为零的废能

2. 能级 λ（有效度）量化示例
   - 机械能/电能/化学能（理论值）：λ = **1**
   - 重油（化学能，41860 kJ/kg）：λ = **0.706**
   - 焦炉煤气（16744 kJ/m³）：λ = **0.701**
   - 蒸汽（热能，500°C）：λ = **0.614**
   - 热水（热能，100°C）：λ = **0.201**

二· 能量的梯级利用

1. ==核心原则："分配得当、各得其所、温度对口、梯级利用"==

2. 热能梯级利用路径
   - **高温段（≥1000°C）**：燃煤/燃气→高品位热能，适合做功发电（效率 **30%~40%**）
   - **中温段（400~600°C）**：燃气轮机排气→余热锅炉产蒸汽→蒸汽轮机发电（联合循环，总效率可超 **55%**）
   - **中低温段（100~300°C）**：有机朗肯循环（ORC）发电，或热泵/热管用于干燥、供热、制冷
   - **低温段（<100°C）**：供暖、生活热水

> [!note] 核心理解
> 物理上：梯级利用的本质是**"高能高用、低能低用"**——不让高温热能直接供暖（巨大㶲损失），而是先做功发电，再用余热供热。数学上：卡诺效率 η = 1 − T_L/T_H → 温差越大做功潜力越大，直接供暖等于浪费温差。

三· 能源的综合利用——四种典型方案

1. **燃气-蒸汽联合循环**
   - 燃气轮机初温 **>1300°C**，排气 **400~600°C**
   - 排气→余热锅炉产蒸汽→蒸汽轮机做功
   - 发电效率超 **55%**（单独燃机/蒸汽均不超 **45%**）
   - IGCC（整体煤气化联合循环）：煤气化 + 联合循环

2. **功热并供（热电联产）**
   - 热机同时输出功（电）+ 工艺/生活用热
   - 燃机优势：排气流量大、温度适中（**400~600°C**），可用补燃调节功热比

3. **动力-化工多联产**
   - IGCC等动力系统 + 煤基化工生产
   - 发电供热的同时产出甲醇、二甲醚、氢气
   - 兼顾动力、化工、环保

4. **多能源互补**
   - 可再生能源（太阳能、风能）不连续不稳定
   - → 化石能源-太阳能互补、燃料电池-太阳能联合、燃机-风电联合

四· 能量转换新技术

1. **磁流体发电（MHD）**
   - 原理：高温导电流体高速通过磁场→切割磁力线→感应电动势→热能→电能直接转换
   - 优点：无高速转动部件、噪声小、体积小、启动快
   - 工质温度可达 **2500°C** 以上，加"种子"（钾盐）提高电导率
   - 磁流体-蒸汽联合循环：总发电效率 **50%~60%**
   - 缺点：闭式循环需惰性气体/液态金属，技术复杂

2. **燃料电池**
   - 原理：电化学反应将化学能直接转为电能（燃料电池 = 连续供燃料的化学电池）
   - ==效率不受卡诺循环限制==，理论可达 **85%~90%**
   - 结构：阳极（燃料氧化）| 电解质（离子运输）| 阴极（氧化剂还原）
   - 类型：磷酸盐型、熔融碳酸盐型、高温固态氧化物型、聚合物电解质型
   - 优点：高效、低污染无噪声、燃料适应性强（H₂/CH₃OH/天然气/煤）、体积小、用途广
   - 可与燃机-蒸汽联合循环结合：整体煤气化燃料电池联合循环

## 三· 新能源

一· 核能

1. 基本原理
   - 来源：原子核内核力（远大于化学键力）→ 核反应释放能量为化学能的几百万倍
   - 质能方程：E = mc²

2. 核裂变
   - 重原子核（²³⁵U、²³⁹Pu）分裂→释放 **2~3个中子** + 核能
   - 中子→继续轰击其他核→链式反应
   - 能效：**1kg 铀 ≈ 2500t 标准煤**
   - 已实用化：轻水堆、重水堆、气冷堆、快中子增殖堆
   - （补：目前核电占全球发电量的 ~10%；中国以压水堆为主）

3. 核聚变
   - 轻原子核在超高温下聚合成重核：²H + ³H → ⁴He + n + 能量
   - 能效：**1kg 氘氚 ≈ 5倍铀裂变**
   - 燃料：氘可从海水提取（每升 **0.034g**），储量极为丰富
   - ==真正的清洁能源==：产物只有氦和中子，无长寿命放射性废物
   - 技术难点：需几千万度~上亿度超高温；等离子体约束（磁约束/惯性约束）
   - 当前状态：尚不可控

二· 太阳能

1. 基本数据
   - 太阳表面温度约 **6000K**，以 **3.8×10²³ kW** 辐射能量
   - 地球接收：**1.73×10¹⁴ kW**（仅22亿分之一），到达地面约 **8.5×10¹⁰ kW**（几十万倍全球发电量）
   - 特点：分布广泛、可再生、清洁；但能量密度低、间断不稳定

2. 四种利用方式
   - （1）**光热利用**：集热器（非聚光式≤200°C；聚光式可达 **3000°C** 以上）→ 热水器/温室/空调/热发电
   - （2）**光电利用**：光伏效应（太阳能电池）→ PN结内建电场分离光生电子-空穴对
   - （3）**光化学利用**：太阳能分解水制氢（需光催化剂）
   - （4）**光生物利用**：光合作用→速生植物/油料作物/海藻

3. 储能是关键瓶颈
   - 储热（水/卵石/熔盐）+ 储电（蓄电池）
   - 互补系统：太阳能-水电、太阳能-氢能

三· 风能

1. 来源：太阳能→大气温差→压差→空气流动（太阳能中 **1%~3%** 转为风能）
2. 利用形式
   - 风帆助航（电脑控制，节油率可达 **15%**）
   - 风力泵水（低扬程大流量/中扬程/高扬程小流量三档）
   - ==风力发电（最主要形式）==：独立运行/互补运行/并网运行
   - 风力制热（直接制热效率最高，发电制热最低）
3. 局限：能量密度低、不稳定、地区差异大、装置体积大投资高

四· 生物质能

1. 来源：光合作用→生物质，地球年产生物质能 **3×10¹⁵ kJ**（≈全球总能耗10倍）
2. 三大转化技术
   - （1）**物理转化**：固化成型（提高体积/能量密度）；高含油率生物质直接压榨取油
   - （2）**化学转化**：直接燃烧 + 热解（>400°C，惰性氛围→生物油+合成气+焦炭）+ 气化（>700°C，与气化介质反应→合成气）+ 液化（高压/催化剂→液体烃类燃料）
   - （3）**生物转化**：厌氧发酵→沼气（CH₄ **60%~70%** + CO₂ **30%~40%**）；发酵→燃料乙醇
3. 趋势：从废弃物利用→主动种植能源植物

五· 地热能

1. 储量：地壳表层10km内 **1.26×10²⁷ kJ**（≈4.6×10¹⁶t标准煤）
2. 四种类型
   - 水热型（高温≥150°C/中温90~150°C/低温<90°C）——目前主要开发
   - 地压型（高压热水150~260°C，含甲烷）
   - 干热岩型（150~650°C，需人造地热系统注水取热）
   - 岩浆型（700~1200°C，技术难度极大）
3. 发电方式：地热蒸汽直接发电 / 扩容法（闪蒸）/ 双循环（低沸点工质）
4. 原则：一水多用、逐级开发、综合利用 + 回灌防沉降

六· 海洋能

1. **潮汐能**：月球/太阳引力→海水涨落→筑坝蓄水→水位差发电。已实用化，需防腐防污防淤
2. **波浪能**：风+重力→海水起伏（动能+势能）。气动式最多。小型装置（航标灯）已推广
3. **海洋温差能**：表层与深层温差（热带可达 **20°C**）→朗肯循环发电
4. **盐差能**：不同浓度海水间化学能（初期探索）
5. **海流能**：持续海水环流，稳定但利用难度大

七· ==氢能==

1. 基本特性
   - 发热量为汽油的 **3倍**，燃烧温度可达 **2000°C**
   - ==终极清洁燃料==：燃烧产物只有水，水又是制氢原料
   - 现状：地球上氢单质极少，需人工制取→氢是二次能源

2. 制氢方法
   - （1）化石燃料制氢：当前工业主流，但效率低、污染大、消耗化石资源
   - （2）电解水制氢：消耗高品位电能、总效率低，但可作储能手段
   - （3）研究前沿：分解水法（>3000K）、光化制氢（光催化剂）、热化学制氢（生物质气化/裂解）、生物制氢（光合微生物/厌氧发酵）

3. 利用方式
   - 作为动力能源（汽车/飞机/舰艇燃料）
   - 氢-氧燃料电池发电（电解水逆反应，最高效简单）
   - ==作为能量载体==：解决太阳能/风能不连续问题——多余电力→制氢→储存→需要时发电

4. 储氢瓶颈
   - 高压钢瓶储气：量小、搬运不便
   - 低温液化（杜瓦瓶）：耗能巨大、昂贵
   - → 研究热点：**金属氢化物储氢**（FeTi合金、Mg合金等，可逆吸放氢）

## 速记卡 / 一句话总结

- ==六大能量形式：机械能→热能→电能→辐射能→化学能→核能，能量密度逐级升高==
- 能源五维分类：来源/获得方法/可否再生/污染/技术水平
- ==能量品位三级：高级（λ=1，直接做功）→ 中级（0<λ<1，温度越高品位越高）→ 低级（废能）==
- 梯级利用铁律："温度对口，梯级利用"，高温先做功再供热
- 联合循环发电效率：燃机单独≤45%，联合循环超 **55%**
- 燃料电池不受卡诺效率限制，理论效率 **85%~90%**
- 核裂变 **1kg 铀 = 2500t 标准煤**；核聚变更高5倍且真正清洁（产物只有氦）
- 太阳能：地球接收 **8.5×10¹⁰ kW** = 全球发电量几十万倍
- 氢能：热值=汽油的3倍、燃烧温度2000°C、产物只有水——终极清洁载体
- 氢能核心瓶颈：廉价制氢 + 安全高效储运（金属氢化物最有前景）
