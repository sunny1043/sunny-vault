## 2-1 热力学第一定律的实质及表达式

### 一·热力学第一定律的实质

**定义**：热力学第一定律是能量守恒与转换定律在热力学中的具体体现——在伴有热效应的各种过程中，**各种不同形式的能量在转移和转换时，其总量保持不变**。在工程热力学中，它主要说明**热能和机械能在转移和转换时，能量的总量必定守恒**。

> [!note] 核心理解
> 物理上：能量不会凭空产生或消失——输入能量 − 输出能量 = 系统储能增量，就是「能量账本」的平衡方程。数学上：这是热力系统的一个守恒律标量方程，将过程量（Q、W）和状态量（E）通过等式关联起来。

---

### 二·一般热力系的能量方程（基本表达式）

**定义**：取任意热力系（含闭口系和开口系的普遍情况），在微小时间段 dτ 内，外界传入热量、流入流出工质带入/带出的能量、对外做功，与系统总能量增量之间满足的平衡关系。

**热力系总能量 E 的组成**：
- E = **U** + **Eₖ** + **Eₚ**
  - U：热力学能（内能）
  - Eₖ：宏观动能
  - Eₚ：重力位能

每千克工质的总能量：**e** = **u** + c²/2 + gz

**公式与推导**——一般热力系微元过程的能量平衡：

（1）取极短时间 dτ，热力系从外界吸收微小热量 **δQ**，流入质量 **δm₁**（每千克总能 e₁），流出质量 **δm₂**（每千克总能 e₂），对外做微小总功 **δW_tot**。
← 用 δ 表示过程量的微小量（非全微分），d 表示状态量的微小增量

（2）根据质量守恒：热力系质量变化 dm = **δm₁ − δm₂** …(2-2)
← dm 是状态量的变化（全微分），δm₁、δm₂ 是过程量

（3）根据热力学第一定律——进入热力系的能量总和减去离开热力系的能量总和等于热力系总能量的增量：
```
(δQ + e₁δm₁) − (δW_tot + e₂δm₂) = (E + dE) − E
```
← 进入项：热量 δQ + 流入工质带入能 e₁δm₁；离开项：总功 δW_tot + 流出工质带出能 e₂δm₂

（4）化简得**微分式**：
```
δQ = dE + (e₂δm₂ − e₁δm₁) + δW_tot  …(2-3)
```
← 这是热力学第一定律的最基本表达式

（5）对有限长时间 τ 积分得**积分式**：
```
Q = ΔE + ∫(e₂δm₂ − e₁δm₁) + W_tot  …(2-4)
```

> [!tip] 适用条件
> 式 (2-3) 和 (2-4) 适用于**任何热力系**（闭口/开口）、**任何工质**、**任何过程**（无摩擦/有摩擦）。

**易错 / 边界**：
- δ 和 d 的区别是关键——Q 和 W 是过程量，没有「在某状态下的值」；E 是状态量，有全微分 dE
- 总功 W_tot 包含所有形式的功（膨胀功、轴功、推动功等），后续才针对不同系统分解

---

### 三·闭口系的能量方程

**定义**：闭口系指与外界**无物质交换**（δm₁ = δm₂ = 0）且**宏观动能和重力位能不变**（ΔEₖ = ΔEₚ = 0）的热力系。此时系统总能量变化 ΔE = ΔU。

**公式与推导**——从基本表达式 (2-4) 化简：

（1）列出 (2-4) 各项在闭口系中的简化：
- Q = Q（热量保持不变）
- ΔE = ΔU + ΔEₖ + ΔEₚ = **ΔU** ← 动能和位能无变化
- 物质交换项：∫(e₂δm₂ − e₁δm₁) = **0** ← 无物质交换
- 总功中只有**膨胀功** W（体积变化对外做功），即 W_tot = W

（2）代入得**积分式**：
```
Q = ΔU + W = U₂ − U₁ + W  …(2-5)
```
← 闭口系吸热量 = 热力学能增加 + 对外膨胀功

（3）每千克工质：
```
q = Δu + w = u₂ − u₁ + w  …(2-6)
```

（4）微元过程微分：
```
δq = du + δw  …(2-7)
```
← 这是后续推导无摩擦平衡过程 δq = du + pdv 的起点

> [!note] 核心理解
> 物理上：闭口系的能量只有两条路——要么变成热力学能存起来（升温），要么变成膨胀功对外输出。数学上：这是热力学第一定律在控制质量下的退化形式，去掉物质流项后退化为 δq = du + δw。

**适用条件**：闭口系、任何工质、任何过程（有/无摩擦均可）。

---

### 四·开口系的能量方程

**定义**：开口系与外界**既有能量交换也有物质交换**。典型如活塞式动力机械的进气→封闭膨胀→排气循环——进气时工质流入，排气时工质流出，完成一个工作周期后系统状态复原（ΔE = 0）。

**核心概念——推动功**：工质流入/流出控制体时，**外界必须克服截面压力将工质推入/推出**，这一部分功 = pV。每千克工质的推动功 = **pv**。

**核心概念——技术功 W_t**：动力机械在一个工作周期中**真正获得的可利用功**，定义为进气功 W_in + 膨胀功 W − 排气功 W_out：
```
W_t = W_in + W − W_out  …(2-9)
```

**公式与推导**——从基本表达式 (2-4) 展开：

（1）分析一个完整工作周期（进气→封闭→膨胀→排气→复位）：
- Q = Q（周期内传入的总热量）
- ΔE = **0** ← 周期结束后系统回到初始空缸状态
- 物质交换项：∫(e₂δm₂ − e₁δm₁) = **U₂ − U₁** ← 考虑简化（位能不变，宏观动能并入滞止参数）
- 总功 W_tot 的分解：W_tot = **−p₁V₁ + W_in + W − W_out + p₂V₂**
← 外界对系统做推动功 p₁V₁（进）取负；系统对外做推动功 p₂V₂（出）取正；W_in 为进气功（外界对系统，取负）、W_out 为排气功（系统对外界，取正）

（2）代入 (2-4)：
```
Q = 0 + (U₂ − U₁) + (−p₁V₁ + W_in + W − W_out + p₂V₂)
  = U₂ − U₁ + p₂V₂ − p₁V₁ + (W_in + W − W_out)
```
← 将 p₁V₁ 和 p₂V₂ 移到 U 旁边，定义**焓** H = U + pV

（3）用焓 H = U + pV 和 H₂ − H₁ = (U₂ − U₁) + (p₂V₂ − p₁V₁) 代换：
```
Q = H₂ − H₁ + W_t  …(2-10)
```

（4）每千克工质：
```
q = h₂ − h₁ + w_t  …(2-11)
```

（5）微元过程微分：
```
δq = dh + δw_t  …(2-12)
```

> [!note] 核心理解
> 物理上：开口系中，不是所有膨胀功都能对外输出——膨胀功的一部分要用来「填坑」：弥补排气推动功与进气推动功的差值 (p₂v₂ − p₁v₁)，剩下的才是技术功 w_t。**焓 h = u + pv 引入了「随物质流动携带的推动功」**，使开口系能量方程形式上退化为和闭口系相似的简洁形式。数学上：焓是一个组合状态参数，将热力学能和推动功打包，让能量方程在开口系下仍保持简单线性结构。

**适用条件**：开口系、任何工质、任何过程。

**三种功的关系链（本节核心）**：
```
膨胀功 w  ———→  技术功 w_t = w − (p₂v₂ − p₁v₁)  ———→  轴功 w_sh = w_t − ½(c₂²−c₁²) − g(z₂−z₁)
          ← 扣推动功差                        ← 扣动能差 + 位能差
```

> [!warning] 易错
> 并不能简单说「膨胀功 > 技术功 > 轴功」——因为 (p₂v₂ − p₁v₁)、½(c₂²−c₁²)、g(z₂−z₁) **三项皆可正可负**。最终三者大小关系取决于具体过程的参数变化方向。

---

### 五·稳定流动的能量方程

**定义**：**稳定流动**指流道中**任何位置上**流体的流速及所有状态参数（温度、压力、比体积、比热力学能等）都**不随时间变化**的流动。各种工业设备（汽轮机、压气机、锅炉等）正常运行时的流动工质所经历的过程都接近于稳定流动。

**公式与推导**——取流道进出口截面之间的开口系：

（1）设定条件：
- 进出口截面上参数均匀一致（否则取平均值）：
  - 进口截面：p₁, v₁, u₁, c₁, z₁, e₁
  - 出口截面：p₂, v₂, u₂, c₂, z₂, e₂
- 稳定流动 → 开口系总能量 E = 定值 → **ΔE = 0**
- 取时段 τ，恰好有 **1 kg** 流体通过流道（因稳定流动，此时段内流过任意截面的流体都是 1 kg）

（2）列出 (2-4) 各项：
- Q = **q**（每千克流体吸收的热量）
- ΔE = **0**
- 物质交换项：(e₂ − e₁) ← 1 kg 进、1 kg 出
- 总功 W_tot = w_sh − p₁v₁ + p₂v₂ ← 轴功 w_sh（通过叶轮轴对外输出）+ 两截面推动功

（3）代入 (2-4)：
```
q = 0 + (e₂ − e₁) + (w_sh − p₁v₁ + p₂v₂)
```

（4）展开 e₂ − e₁ = (u₂ + c₂²/2 + gz₂) − (u₁ + c₁²/2 + gz₁)，结合推动功项：
```
q = (u₂ + c₂²/2 + gz₂ + p₂v₂) − (u₁ + c₁²/2 + gz₁ + p₁v₁) + w_sh
```
← 将 u + pv = h 合并为焓

（5）最终得**稳定流动能量方程**：
```
q = (h₂ − h₁) + ½(c₂² − c₁²) + g(z₂ − z₁) + w_sh  …(2-13)
```

**适用条件**：稳定流动开口系、任何工质、任何过程。

> [!note] 核心理解
> 物理上：对流动工质，**焓 h 可理解为流体向下游传送的「热力学能 + 推动功」之和**。稳定流动能量方程本质是——传入的热量 q，一部分提升流体的「携带能量」（焓差 + 动能差 + 位能差），剩下的做轴功 w_sh 对外输出。数学上：这是控制体积法下的能量守恒，将流体通过控制面的能量通量以焓的形式优雅地打包。

---

### 六·三个能量方程的内在联系 ——「热变功」的本质

**关键结论**：==归根结底，反映热能和机械能转换核心关系的是闭口系方程 q = Δu + w。==

**从稳定流动方程退化为开口系方程**：

（1）将稳定流动方程 (2-13) 中动能增量 ½(c₂²−c₁²) 和位能增量 g(z₂−z₁) 看作暂存于流体而尚未对外输出的功，与轴功 w_sh 合并：
```
q = h₂ − h₁ + [½(c₂²−c₁²) + g(z₂−z₁) + w_sh] = h₂ − h₁ + w_t  …(2-14)
```
← 合并后的功项就是技术功 w_t，与开口系方程 (2-11) 完全一致

（2）从开口系方程退化为闭口系方程：
```
q = (u₂ + p₂v₂) − (u₁ + p₁v₁) + w_in + w − w_out
```
- 因 w_in = p₁v₁（进气推动功）
- 因 w_out = p₂v₂（排气推动功）
→ 代入后 p 项抵消：
```
q = u₂ − u₁ + w  …(2-6)
```

**热变功的本质**：
```
w = (u₁ − u₂) + q = −Δu + q
```
→ 在任何情况下，**膨胀功只能从热力系本身的热力学能减少或从外界供给的热量转变而来**。所不同的是：
- **闭口系**中：膨胀功 w 全部向外界输出
- **开口系**中：膨胀功的一部分用于弥补 p₂v₂ − p₁v₁，剩余为技术功 w_t 输出
- **稳定流动**中：膨胀功进一步扣除动能差和位能差后，剩余为轴功 w_sh 输出

**三种功的完整代数关系**：
```
w = (p₂v₂ − p₁v₁) + w_t  …(2-15)
w = (p₂v₂ − p₁v₁) + ½(c₂²−c₁²) + g(z₂−z₁) + w_sh  …(2-16)
```

> [!note] 核心理解
> 物理上：膨胀功是热能和机械能转换的「总枢纽」——它从 Δu 和 q 转化而来，再根据系统类型层层分配（→推动功差 → 技术功 → 动能/位能 → 轴功）。不同能量方程只是在不同边界条件下描述同一守恒律的变体。数学上：三个方程结构一致，差异仅在于「边界穿越项」的取舍——闭口系无物质流项，开口系加推动功项 = 引入焓，稳定流动再加动能/位能 = 全版控制体方程。

## 2-2 功和热量的计算及其在压容图和温熵图中的表示

### 一·有摩擦时功的计算

**热力系选取**：取**气缸内气体 + 活塞 + 气缸**一并作为热力系（摩擦变成内摩擦）。假定摩擦生成的热全部由气体吸收，活塞和气缸热力状态不变，分析时可不考虑活塞和气缸本身。

**力学平衡条件**：
```
pA = F + F_f
```
- p：气体压力，A：活塞截面积
- F：外力（外界对活塞的作用力）
- F_f：活塞与缸壁间的摩擦力

**公式与推导**——有摩擦的膨胀功：

（1）外界向气体加入热量后，气体在平衡状态下膨胀，活塞移动 dx

（2）气体对外界做的功：
```
δw = F·dx = (pA − F_f)·dx = p·Adx − F_f·dx
   = pdv − δw_l  …(2-17)
```
← p·Adx = pdv（气体膨胀功的理想部分），F_f·dx = δw_l（摩擦功损）

（3）结论：**有摩擦时 δw < pdv**
← 摩擦消耗了一部分膨胀功，真正对外输出的功小于 pdv

（4）**功损** δw_l 转化为**热产** δq_g：
```
δq_g = δw_l  …(2-18)
```
← 摩擦功全部变成热量，被系统内工质吸收

> [!warning] 重要——压缩过程同样适用
> 压缩过程中摩擦力反向，外界消耗的功 |δw| > |pdv|（多消耗的用来克服摩擦）。但由于 δw 和 pdv 均为负值，**不等式 δw < pdv 对压缩过程仍然成立**（例如 −5 < −3）。

---

### 二·无摩擦内平衡过程的功与热量计算

**条件**：δw_l = 0（无摩擦）、气体始终处于内平衡状态。

**（1）膨胀功 w 的计算式**：

无摩擦时 δw = pdv，积分：
```
w = ∫₁² pdv  …(2-20)
```
← 膨胀功 = p-v 图中过程曲线**下方**的面积

**（2）技术功 w_t 的计算式**：

由 w_t = w − (p₂v₂ − p₁v₁) 及 w = ∫pdv：
```
w_t = ∫₁² pdv − (p₂v₂ − p₁v₁)
```
← 分部积分：d(pv) = pdv + vdp → ∫pdv = p₂v₂ − p₁v₁ − ∫vdp

（1）代入消去 p₂v₂ − p₁v₁：
```
w_t = −∫₁² vdp  …(2-21)
```
← 技术功 = p-v 图中过程曲线**左边**（向 p 轴投影）的面积

**（3）热量 q 的计算式**：

（1）由熵的定义式：ds = (δq/T)_rev，无摩擦内平衡下：
```
δq = du + pdv = du + δw
```
← 又由热力学恒等式 Tds = du + pdv → 对可逆过程 δq = Tds

（2）积分：
```
q = ∫₁² Tds  …(2-22)
```
← 热量 = T-s 图中过程曲线**下方**的面积

**图**——p-v 图和 T-s 图中过程曲线面积的物理含义：

```
       p ↑                          T ↑
         |     2                       |     2
         |    /                        |    /
         |   /  膨胀功 w               |   /  热量 q
         |  /   = 曲线下方面积          |  /   = 曲线下方面积
         | /                           | /
         |/                            |/
         +——————→ v                   +——————→ s
         1                             1

      技术功 w_t = −∫vdp = 曲线左边面积（向 p 轴投影）
```

> [!note] 核心理解
> 物理上：p-v 图是「力 × 位移」的图形化——pdv 即压力在体积变化上做的微元功，积分就是过程总功；T-s 图同理，Tds 是热量的微元贡献（温度是热传递的「驱动力」，熵是热传递的「位移」）。数学上：p-v 图功是过程曲线的「纵向」积分 ∫pdv；技术功是同一曲线的「横向」积分 −∫vdp，二者由分部积分 d(pv) = pdv + vdp 衔接。T-s 图同理，可逆过程 δq_rev = Tds。

**易错 / 边界**：
- 以上面积表示**仅在无摩擦内平衡（可逆）过程成立**。有摩擦时 δq ≠ Tds、δw ≠ pdv，面积不等于实际功/热量
- w_t = −∫vdp 中的负号是因为 vdp 积分方向与 pdv 积分方向不同（曲线向 p 轴投影时，积分方向沿纵轴）

### 三·无摩擦内平衡循环的功和热量

**定义**：工质经历一系列过程后回到初始状态 → 完成一个**循环**。无摩擦内平衡循环中，所有过程都是可逆的。

**循环的膨胀功和技术功**：

对无摩擦内平衡循环：
```
w₀ = ∮pdv = ∮(−vdp) = w_t₀
```
→ ==膨胀功 = 技术功，用循环的 w₀ 表示==
→ **等于 p-v 图循环曲线包围的面积** abcda

**循环的热量**：
```
q₀ = ∮Tds
```
→ **等于 T-s 图循环曲线包围的面积** ABCDA

**循环积分净功与净热量的关系**：

对能量方程 δq = du + δw 循环积分：
```
∮δq = ∮du + ∮δw
```
← 因 u 是状态量，∮du = 0（回到原状态后 Δu = 0）

（1）可得：
```
q₀ = w₀  …(2-23)
```
→ ==循环的净热量等于循环的净功==

> [!note] 核心理解
> 物理上：工质走一圈回到原状态 → 热力学能复原 → 净吸热量必然等于净输出功，否则能量不守恒。这揭示了热机循环的根本原理：**从热源吸热 → 热力学能暂存 → 对外做功 → 回到初态**，净效果 = 热→功。数学上：这是 ∮du = 0 的直接结果——状态量的闭合回路积分为零，所以过程量的循环积分相等。

> [!tip] 关键补充
> 对于**有摩擦的内不平衡循环**，虽然 w₀ 和 q₀ **不能用** p-v 图和 T-s 图曲线包围的面积表示（因为 δw ≠ pdv、δq ≠ Tds），但由于 ∮du = 0 是状态量性质（与过程是否可逆无关），**q₀ = w₀ 的结论仍然成立**。这是能量守恒与转换定律的刚性约束，与过程是否理想无关。

### 五·典型例题关键数字整理

**【例 2-1】——水泵送水 50 m**

- 理论最小功：w_min = gΔz = **9.807 m/s² × 50 m = 0.4904 kJ/kg**
- 泵效率 η = **70%** → 实际功：w_act = 0.4904 / 0.70 = **0.7006 kJ/kg**
- 功损 → 热产：w_l = 0.7006 − 0.4904 = **0.2102 kJ/kg**（使水热力学能增加）
- 过程 1→3（扬水完成后）：焓变 Δh = 0.2102 kJ/kg（仅余功损转化部分）

**【例 2-2】——燃气轮机装置**

- 空气质量流量：q_m = **10 kg/s**
- 压气机：进口焓 h₁ = **290 kJ/kg** → 出口 h₂ = **580 kJ/kg** → 耗功 w_c = **290 kJ/kg** → 功率 P_c = **2900 kW**
- 燃烧室：h₂ = 580 → h₃ = **1250 kJ/kg** → 吸热量 q = **670 kJ/kg**
- 燃料消耗量：q_mf = (10 × 670) / 43960 = **0.1524 kg/s**
- 燃气轮机：h₃ = 1250 → h₄ = **780 kJ/kg** → 做轴功 w_t = **470 kJ/kg** → 功率 P_t = **4700 kW**
- 装置输出功率：P = 4700 − 2900 = **1800 kW**

### 六·课本思考题要点

**1. 热量和热力学能的区别与联系？**
- 热力学能 U 是**状态量**（取决于工质本身状态）
- 热量 Q 是**过程量**（只在传递过程中有意义）
- 联系：热量可以转变为热力学能（δq = du + δw），且 δq 不是全微分而 du 是

**2. δq = du + pdv 与 δq = dh − vdp 的适用范围？**
- 两式仅对**无摩擦内平衡（可逆）过程**成立
- 通用式是 δq = du + δw 和 δq = dh + δw_t（对任何过程）
- pdv 和 −vdp 仅在没有功损时等于 δw 和 δw_t

**3. δq = du + pdv 与 dh = du + d(pv) 的对比？**
- dh = du + d(pv) 是**焓的定义微分**，无论过程是否可逆都成立（h 是状态量）
- δq = du + pdv 只在可逆过程成立（因为 δq 不是状态量，必须指定路径）
- 形式上相似但本质不同：一个是状态函数关系，一个是过程关系

**4. 绝热刚性容器 → 隔板抽去 → 气体自由膨胀到真空？**
- ΔU = 0（绝热 Q = 0 + 向真空膨胀不做功 W = 0）
- 不能用 δq = du + pdv 分析——因为这不是准静态过程，p 没有统一定义，pdv 没有意义
- 这是典型的**不可逆过程**

## 速记卡 / 一句话总结

- ==热力学第一定律 = 能量守恒在热力学中的体现==：输入能量 − 输出能量 = 系统储能增量
- 总能量 E = U + Eₖ + Eₚ；基本表达式：δQ = dE + (e₂δm₂ − e₁δm₁) + δW_tot **适用于任何系统、任何工质、任何过程**
- **闭口系**（无物质交换）：q = Δu + w，δq = du + δw —— 最核心的「热→功」关系式
- **开口系**（有物质流）：q = Δh + w_t —— 引入**焓 h = u + pv**（热力学能 + 推动功的打包）
- **稳定流动**：q = Δh + ½Δ(c²) + gΔz + w_sh —— 焓差 + 动能差 + 位能差 + 轴功
- ==膨胀功 w 是总枢纽==：闭口系全输出 → 开口系扣 (p₂v₂−p₁v₁) 得技术功 w_t → 稳定流动再扣动能/位能差得轴功 w_sh。三者大小关系不确定（三项修正皆可正可负）
- **无摩擦内平衡过程**：w = ∫pdv（p-v 图曲线下方面积）、w_t = −∫vdp（曲线左边面积）、q = ∫Tds（T-s 图曲线下方面积）
- **有摩擦**：δw = pdv − δw_l < pdv，功损 δw_l = 热产 δq_g；**压缩过程也成立**
- ==循环净热量 = 循环净功==：q₀ = w₀（无论有无摩擦，因为 ∮du = 0 是状态量刚性约束）
- **功看 p-v，热看 T-s**——p-v 图面积 = 功，T-s 图面积 = 热量（仅对可逆过程严格成立）
- 焓 h = u + pv 是**状态参数**（与路径无关）；热量 q 和功 w 是**过程量**（与路径有关）——这是区分状态量和过程量的经典案例
