## §9-1 水蒸气的饱和状态

### 核心概念：汽化与饱和

一·基本定义

- 1. **汽化**：液体转变为气体的过程
  - （1）**蒸发**：液体表面在任何温度下进行的缓慢汽化过程
  - （2）**沸腾**：液体内外同时进行的剧烈汽化 → 只在饱和温度发生
- 2. **液化**（凝结）：蒸气或气体转变为液体的过程
- 3. ==饱和状态是汽化和液化达到动态平衡共存的状态==

> [!note] 核心理解
> 微观上：水面附近动能大的分子挣脱引力逸出 → 汽化；蒸气分子碰撞水面被俘获 → 液化。当单位时间逸出数 = 进入数，宏观上压力温度不再变化 → 饱和。压力越高蒸气密度越大 → 碰撞越频繁 → 液化速度越快；温度越高水分子动能越大 → 逸出越多 → 汽化速度越快。饱和压力与饱和温度是耦合的。

二·饱和状态的核心参数

- 1. **饱和水**：饱和状态下的液态水
- 2. **饱和水蒸气**（饱和蒸气）：饱和状态下的水蒸气
- 3. **饱和压力** p_s：饱和蒸气的压力
- 4. **饱和温度** t_s（或 T_s）：与饱和压力对应的温度
- 5. 一一对应关系：==一定的饱和温度总对应着一定的饱和压力，反之亦然==；饱和温度越高 → 饱和压力越高
  - 粗略经验公式（仅适用于 **100°C** 附近）：p_s = (t_s / 100)^4，p_s 单位为 **atm**（bar）
  - 精确公式（严家騄）：从三相点到临界点全部符合 **1985** 年国际骨架表允差

三·关键边界：临界点与三相点

- 1. **临界点 C**
  - 临界温度 **T_c = 647.14 K**（**373.99°C**）
  - 临界压力 **p_c = 22.064 MPa**
  - 临界比体积 **v_c = 0.003106 m³/kg**
  - 意义：t > t_c 时液相不可能存在，只可能是气相；临界温度是**最高的饱和温度**，临界压力是**最高的饱和压力**
  - 临界状态 → 汽液两相模糊不清、无法区分

- 2. **三相点 A**
  - 三相点温度 **T_A = 273.16 K**（**0.01°C**）
  - 三相点压力 **p_A = 0.000611659 MPa**
  - 意义：p < p_A 时液相不存在，只可能是气相或固相；是**汽-液-固三相共存**的状态，也是**最低的饱和状态**

```
     p ↑
       |                           【p-T 相图示意】
       |         固相        液相
       |          \         /
       |   p_c ────● C（临界点）
       |          /   \
       |         /     \        气相
       |  p_A ──● A（三相点）
       |       /        \
       +——→ T
    T_A      T_c
```

## §9-2 水蒸气的产生过程

一·定压加热三阶段

- 1. **预热阶段**：未饱和水 → 饱和水（温度从 0°C 升至 t_s）
  - 吸热量 = **液体热** q' = ∫₀^{t_s} c'_p dt ≈ c'_p · t_s
  - 也等于焓增：q' = h' − h₀（h₀ 为 0°C 水的焓，h' 为饱和水焓）
  - 饱和水焓：h' = h₀ + q'
  - 压力越高 → 对应饱和温度越高 → 液体热越大
  - 数据（课本表 9-1）：p = **0.01** MPa → q' = **191.80** kJ/kg；p = **10** MPa → q' = **1397.1** kJ/kg；p = **22.064** MPa（p_c）→ q' = **2063.8** kJ/kg

- 2. **汽化阶段**：饱和水 → 饱和水蒸气（定温定压）
  - 吸热量 = **汽化潜热 r** = h'' − h' = T_s (s'' − s')
  - 压力越高 → 汽化潜热**越小**；p = p_c 时 **r = 0**
  - 数据（表 9-2）：p = **0.01** MPa → r = **2392.0** kJ/kg；p = **1** MPa → r = **2014.8** kJ/kg；p = **10** MPa → r = **1317.2** kJ/kg；p = p_c → r = **0**
  - 饱和水蒸气焓：h'' = h' + r = h₀ + q' + r

- 3. **过热阶段**：饱和水蒸气 → 过热水蒸气（温度 t_s → t）
  - 吸热量 = **过热热量** q'' = ∫_{t_s}^t c_p dt = c̄_p|_{t_s}^t · (t − t_s) = c̄_p · D
  - D = t − t_s 称为**过热度**，表示离开饱和状态的远近
  - q'' = h − h''，过热水蒸气焓：h = h'' + q'' = h₀ + q' + r + q''

```
加热全过程能量流向（定压 p）：
  0°C 水 ──q'──→ 饱和水(t_s) ──r──→ 饱和蒸气(t_s) ──q''──→ 过热蒸气(t)
                    预热                汽化                过热
总吸热量 q = q' + r + q'' = h − h₀
```

二·湿蒸汽与干度

- 1. **湿蒸汽**：饱和水与饱和水蒸气同时并存的混合物（汽化过程未完成）
- 2. **干度** x = m_v / (m_v + m_w) —— 饱和水蒸气的质量分数
- 3. **湿度** y = m_w / (m_v + m_w) = 1 − x —— 饱和水的质量分数
  - 饱和水：x = **0**，y = **1**
  - 饱和水蒸气：x = **1**，y = **0**
  - 湿蒸汽：**0 < x < 1**，**1 > y > 0**

- 4. **湿蒸汽比状态参数计算**（线性插值）：
  - v_x = (1 − x) v' + x v'' = v' + x(v'' − v')
  - h_x = (1 − x) h' + x h'' = h' + x r
  - s_x = (1 − x) s' + x s'' = s' + x·r / T_s
  - u_x = h_x − p_s v_x

> [!tip] 补充
> 湿蒸汽的压力和温度不是相互独立的——知道压力就知道对应的饱和温度。因此确定湿蒸汽状态需要两个独立参数，常见组合是（p, x）或（t, x）。

## §9-3 水蒸气图表

一·三图总览：一点连双线，三区五态含

- 1. **下界线**（饱和液体线，x = 0 定干度线）：连接各定压线上开始汽化的点
- 2. **上界线**（饱和蒸气线，x = 1 定干度线）：连接各定压线上汽化完毕的点
- 3. **临界点 C**：上下界线交点
- 4. **三区**：未饱和水区（下界线左侧）、湿蒸气区（上下界线之间）、过热水蒸气区（上界线右侧）
- 5. **五态**：未饱和水、饱和水、湿蒸气、饱和水蒸气、过热水蒸气

二·p-v 图（压容图）特征

```
 p ↑
   │                                        ┌→ 过热区
   │   未饱和水区    湿蒸气区(两相)           │
   │   │←——→│←——————→│                       │
   │ C ●────────────────● A₂  ←上界线 x=1     │  p>p_c 超临界
 p_c├──● C（临界点）                         │
   │  │                                      │
   │ A₁●────────────────┐                    │
   │  │←下界线 x=0      │                    │
   │  │                  │                    │
   │  │  ← 定温线很陡    │ 定温线=定压线=水平  │
   │  │  （水压缩性小）  │                    │
   +——┴————————————→ v
```

- 水的压缩性很小 → 未饱和区的定温线近乎垂直
- 湿蒸气区中定温线 = 定压线 = 水平线段
- p > p_c（超临界）→ 不再有汽化过程，定压线连续弯曲

三·T-s 图（温熵图）特征

```
 T ↑
   │                       超临界 p>p_c
   │                     ╱
   │ C ●──────────────● A₂  ←上界线 x=1
   │  │╲               ╱│
T_c├──● C               │
   │  │ ╲   湿蒸气     ╱ │    过热区
   │  │  ╲  区       ╱  │
   │ A₁●──╲────────╱───┘
   │  │    ╲      ╱  ←不同压力的定压线在未饱和区
   │  │     ╲    ╱      几乎重合（水的压缩性小→
   │  │      ╲  ╱       定熵压缩后温度几乎不变）
   │  │       ╲╱
   +——┴—————————→ s
```

- 未饱和区：不同压力定压线靠得很近，几乎与下界线重合
- 湿蒸气区：定压线 = 定温线 = 水平线段
- 过热区：定压线向右上方散开
- (∂h/∂s)_p = T → 压力越高定压线斜率越大，越陡

四·h-s 图（焓熵图 / Mollier 图）

```
 h ↑
   │                                    ╱
   │                               ╱  ← 定压线在过热区
   │                          ╱
   │  A₂ ●──────────────● C       ← 上界线 x=1
   │     │╲             ╱│
   │     │ ╲           ╱ │
   │     │  ╲  湿蒸气 ╱  │      过热区
   │     │   ╲  区   ╱   │
   │     │    ╲     ╱    │
   │  A₁ ●─────╲──╱─────┘       ← 下界线 x=0
   │            ╲╱
   │  未饱和水区 │              ← 定压线在未饱和区与
   │  定压线靠得很近              下界线几乎重合
   +—————————————→ s
```

**h-s 图关键规律**：

- 1. 斜率：**(∂h/∂s)_p = T** → 定压线上各点斜率等于该点绝对温度
- 2. 湿蒸气区：定压线 = 定温线 → T = T_s = const → 斜率恒定 → **定压线是斜直线**
- 3. 压力越高 → T_s 越高 → 定压线斜率越大 → 越陡
- 4. ==h-s 图是工程计算最常用的图==：绝热过程是竖直线（定熵 = 垂线），技术功 w_t = h₁ − h₂ 直接读纵坐标差

> [!note] 核心理解
> h-s 图的优雅之处：定熵膨胀过程只需画一条竖线，技术功 = 纵坐标差。湿蒸气区定压线是斜直线——因为定压即定温，T_s 恒定 → (∂h/∂s)_p 恒定。这是水蒸气独有的（理想气体定温线在 h-s 图中是水平线，因为 h 仅是 T 的函数）。

五·水蒸气表使用要点

- 1. **饱和水与饱和水蒸气表**
  - 附表 6：按温度排列 → 查温度整齐的饱和参数
  - 附表 7：按压力排列 → 查压力整齐的饱和参数
  - 两表均列：p_s / t_s、v'、v''、h'、h''、s'、s''、r（汽化潜热）

- 2. **未饱和水与过热水蒸气表**（附表 8）
  - 按温度和压力排列（温度压力无对应关系）
  - 粗黑线以上 = 未饱和水；粗黑线以下 = 过热水蒸气

- 3. 查表步骤（以定熵膨胀为例）：
  - （1）由（p₁, t₁）查附表 8 得 h₁ 和 s₁
  - （2）沿 s₂ = s₁ 在 p₂ 下定熵 → 若 s₂ 落在 s'(p₂) 与 s''(p₂) 之间 → 终态为湿蒸汽
  - （3）由 s_x = s' + x(s'' − s') = s₁ 反解干度：x = (s₁ − s') / (s'' − s')

> [!note] 核心理解

## §9-4 水蒸气的热力过程

一·利用图表的计算步骤

- 1. 将过程画在 **h-s 图**上 → 便于直观分析
- 2. 查出始末状态参数：T₁, p₁, v₁, h₁, s₁ 以及 T₂, p₂, v₂, h₂, s₂
- 3. 计算热量（无摩擦 → 可逆过程）

二·四种基本热力过程对照

| 过程类型 | 热量 q | 膨胀功 w | 技术功 w_t |
|:---|:---|:---|:---|
| **定容** (v = const) | q_v = u₂ − u₁ = (h₂ − h₁) − v(p₂ − p₁) | w_v = 0 | w_{t,v} = v(p₁ − p₂) |
| **定压** (p = const) | q_p = h₂ − h₁ | w_p = p(v₂ − v₁) | w_{t,p} = 0 |
| **定温** (T = const) | q_T = T(s₂ − s₁) | w_T = T(s₂−s₁) − (h₂−h₁) + (p₂v₂−p₁v₁) | w_{t,T} = T(s₂−s₁) − (h₂−h₁) |
| **定熵** (s = const) | q_s = 0 | w_s = u₁ − u₂ | w_{t,s} = h₁ − h₂ |

> [!tip] 补充
> 水蒸气与理想气体的关键区别：理想气体 u 只取决于 T → 定温过程 u₂ = u₁，w = q。水蒸气定温过程 u₂ ≠ u₁（实际气体效应），必须用 h 和 pv 算。

三·定熵膨胀典型算例

已知：p₁ = **1** MPa，t₁ = **300°C**，绝热（定熵）膨胀到 p₂ = **0.1** MPa

- （1）查附图 1（h-s 图）得：h₁ = **3053** kJ/kg，s₁ = **7.122** kJ/(kg·K)
- （2）沿 s = 7.122 线与 p₂ = 0.1 MPa 相交：h₂ = **2589** kJ/kg，x = **0.961**
- （3）技术功 w_t = h₁ − h₂ = **3053** − 2589 = **464** kJ/kg
- （4）湿度 y₂ = 1 − x = 1 − 0.961 = **0.039**（**3.9%**）

## §9-5 基本朗肯循环

一·四设备 + 四过程

```
                            ┌── 1 过热蒸气 ──→ 汽轮机 ──→ 发电机
                            │                    │
  ┌─────────────────────────┘                    │
  │                                              ▼
  │  锅炉                                      2 乏汽
  │  (定压加热)                                 │
  │                                              ▼
  │                                         凝汽器 ←── 冷却水
  │  0 ←── 水泵 ←── 3                          │
  │  (绝热压缩)   (饱和水)                (定压冷凝)
  └──────────────────────────────────────────────┘
```

- 过程 **0→1**（锅炉）：定压加热 → 未饱和水预热 → 汽化 → 过热
  - 吸热量：q₁ = h₁ − h₀
- 过程 **1→2**（汽轮机）：绝热（定熵）膨胀 → 对外做功
  - 技术功：w_T = h₁ − h₂
- 过程 **2→3**（凝汽器）：定压冷凝 → 乏汽凝结放热
  - 放热量：q₂ = h₂ − h₃
- 过程 **3→0**（水泵）：绝热压缩 → 凝结水升压
  - 水泵耗功：w_P = h₀ − h₃ ≈ v₃(p₁ − p₂)

二·T-s 图上的朗肯循环

```
 T ↑
   │
   │          ┌────── 1（新汽：过热水蒸气）
   │         ╱│
   │    T_m1╱ │
   │       ╱  │
   │      ╱   │  ← 过热区定压线（锅炉加热 p₁）
   │     ╱    │
   │ T_s├─────┤────── 湿蒸气区定压线 p₁（汽化）
   │    │╲    │╲
   │    │ ╲   │ ╲
   │    │  ╲  │  ╲ 1→2 定熵膨胀
   │    │   ╲ │   ╲
   │    │    ╲│    ╲
   │ 0  ├─────3──────2
   │    │      ╲     ╲
   │    │ 3→0   ╲    ╲     p₂ 定压线（凝汽器）
   │    │ 水泵   ╲   ╲
   │    │ 绝热压缩╲  ╲
   │ T_m2────────────────── 平均放热温度
   +————┴—————————————→ s

   循环包围面积 01230 = 净功 w₀
   面积 60176 = 吸热量 q₁
   面积 63276 = 放热量 q₂
```

三·热效率推导

- （1）循环净功：w₀ = w_T − w_P = q₁ − q₂ ← 热力学第一定律
- （2）w₀ = (h₁ − h₂) − (h₀ − h₃)
- （3）吸热量 q₁ = h₁ − h₀，放热量 q₂ = h₂ − h₃
- （4）热效率：

$$\eta_t = \frac{w_0}{q_1} = 1 - \frac{q_2}{q_1} = 1 - \frac{h_2 - h_3}{h_1 - h_0}$$

- （5）忽略水泵功（v₃ ≪ v 蒸气 → w_P ≪ w_T）：**h₀ ≈ h₃**

$$\eta_t \approx 1 - \frac{h_2 - h_3}{h_1 - h_3} = \frac{h_1 - h_2}{h_1 - h_3}$$

> [!note] 核心理解
> 朗肯循环本质：两个定压过程（加热 + 放热）+ 两个绝热过程（膨胀 + 压缩）。卡诺循环要求定温加热/放热，在湿蒸气区虽可近似实现但循环净功极小（p-v 图上定温线与绝热线斜率差太小），且压缩湿蒸气困难。朗肯循环牺牲了理论最高效率，换来了工程可行性和可观的净功。

四·典型计算（例 9-2）

参数：p₁ = **3** MPa，t₁ = **450°C**，p₂ = **0.005** MPa，流量 **60** t/h

- （1）h₁ = **3345** kJ/kg，h₀ ≈ h₂' = **137.7** kJ/kg，h₂ = **2158** kJ/kg
- （2）吸热量：Q₁ = **192.3 × 10⁶** kJ/h
- （3）理论功率：P_T = **19 780** kW；水泵功率 P_P ≈ **50** kW（仅占汽轮机功率的 **0.25%**）
- （4）理论热效率：

$$\eta_t = \frac{P_T - P_P}{Q_1 / 3600} \approx 37\%$$

- （5）考虑汽轮机相对内效率 η_T = **0.82**：实际功率 P_T' = **16 220** kW，实际绝对内效率 η_i = η_t × η_T = **30.3%**

## §9-6 蒸汽参数对热效率的影响

一·初温 T₁ ↑ 的影响

```
 T ↑                        T-s 图示意：
   │      1'                1' 在 1 右上方
   │     ╱│                T_m1' > T_m1
   │  1 ╱ │                平均吸热温度 ↑
   │   ╱  │                平均放热温度不变
   │  ╱   │
   +——┴———→ s
```

- 1. 平均吸热温度 T_m1' > T_m1 → 热效率 η_t ↑
- 2. 终态 2' 的干度 x₂' > x₂ → **降低乏汽湿度** → 减少末级叶片水蚀
- → 提高初温**一举两得**：既提效率又保安全

二·初压 p₁ ↑ 的影响

- 1. 平均吸热温度 T_m1 提高 → η_t ↑
- 2. **但单独提初压使 x₂ 减小**（乏汽湿度增大）→ 危及汽轮机末级叶片安全、降低级效率
- → ==初温初压必须**同时提高**==，才能既提效率又控湿度
- 3. 现代大型机组参数：t₁ ≈ **550°C**，p₁ > **15 MPa**

三·终压 p₂ ↓ 的影响

```
 T ↑
   │      1
   │     ╱│
   │    ╱ │                   平均吸热温度几乎不变
   │   ╱  │
   │  ╱   │                   平均放热温度显著降低 T_m2' < T_m2
   │ ╱ 2  │
   │╱     2'                   由于饱和温度 t_s(p₂) ↓
   +——┴———→ s
```

- 1. 平均放热温度 **显著降低**，平均吸热温度降低很少 → η_t **显著提高**
- 2. 限度：t_s(p₂) 最低只能降到天然冷源温度（大气/海水 ≈ **20~30°C**）
- 3. 目前大型机组：p₂ ≈ **0.004 MPa**（对应 t_s ≈ **29°C**）

四·综合策略

```
初温 T₁ ↑ ：η_t ↑ + 乏汽湿度 ↓（利大于弊）
初压 p₁ ↑ ：η_t ↑ + 乏汽湿度 ↑（需与 T₁ 配合）
终压 p₂ ↓ ：η_t ↑（效果最显著但受限于冷源温度）
```

## §9-7 蒸汽再热循环

一·基本思想

- 1. 新汽在汽轮机中**不完全膨胀到终压**
- 2. 膨胀到中间压力 p_a → 引回再热器 → 再次加热到 t₁'（常 ≈ t₁）
- 3. 再到第二段汽轮机继续膨胀到终压 p₂
- ==目的：提效率 + 降湿度==

二·T-s 图

```
 T ↑
   │
   │  ┌── 1 ──── a ──── 1'（再热后）
   │  │    ╲    │    ╲
   │  │     ╲   │     ╲
   │  │      ╲  │      ╲
   │  │       ╲ │       ╲
   │  │        ╲│       2' ← 再热后乏汽干度 ↑
   │  │         2 ← 原朗肯乏汽（湿度大）
   │  0────────3────────────────── p₂
   +——┴——————————————→ s
```

三·热效率

$$\eta_{t,再热} = 1 - \frac{q_2}{q_1} = 1 - \frac{h_{2'} - h_3}{(h_1 - h_0) + (h_{1'} - h_a)}$$

- q₁ = 锅炉一次加热 + 再热器二次加热
- 只要再热参数选得合理 → 平均吸热温度 T_m1' > T_m1（原朗肯） → η_t ↑
- ==再热还可显著降低乏汽湿度==（y₂' < y₂）

四·再热压力 p_a 的选择

- 1. 最佳准则：前段汽轮机排汽状态 a 应落在**再热循环平均吸热温度线**上
  - p_a 过高 → 附加循环吸热温度高但循环功小 → 增益小
  - p_a 过低 → 附加循环开始吸热温度低于平均吸热温度 → 拉低增益
- 2. 工程实践：p_a 选得比热力学最佳值**稍高**些更有利
- 3. 大型超高压机组几乎**全部**采用再热循环

> [!note] 核心理解
> 再热 = 在原朗肯循环上叠加一个附加循环（1→a→1'→2'→2→1）。附加循环的平均吸热温度高于原循环时 → 总平均吸热温度提高 → 热效率增加。这相当于"把朗肯循环的上半截掰开、展开"，向卡诺化方向迈进。

## §9-8 抽汽回热循环

一·基本思想（源于卡诺定理）

- 1. 朗肯循环中 0→1 段**水的低温预热严重拉低平均吸热温度**
- 2. 方案：从汽轮机中间级抽出部分蒸汽 → 用来预热锅炉给水 → 使锅炉吸热从较高温度开始
- 3. ==核心：设法让吸热过程**不包括**那段低温水的预热==

二·T-s 图（以二次抽汽为例）

```
 T ↑
   │
   │             a''     a'  a    1
   │        ┌────┼──────┼───┼────── 锅炉吸热 a''→1
   │  给水  │    │  ↑   │   │
   │  加热  │    │  │   │抽 │
   │  b''   │ b' │  │   │汽 │
   │   ┌────┼────┤  │   │a  │
   │   │    │    │  │   │   │
   │   │ c  │    │  │   │   │
   │   ├────┤    │  │   │   │
   │  0    3     │  │   │   │
   │            b  │   │   │
   │               │   │   │
   │               2   │   │
   +——┴——————————————→ s
```

- a → a'：第一次抽汽在回热器中定压凝结放热
- b → b'：第二次抽汽在回热器中定压凝结放热
- 放出的热量预热给水：c → b'' → a''

三·抽汽量计算（质量 + 能量守恒）

- 设进汽轮机总流量为 **1 kg**，第一次抽汽 α₁ kg，第二次 α₂ kg（不考虑散热）：
- （1）第一级回热器：α₁ (h_a − h_a') = (1 − α₁)(h_a' − h_b'')

$$\alpha_1 = \frac{h_{a'} - h_{b''}}{h_a - h_{b''}}$$

- （2）第二级回热器：α₂ (h_b − h_b') = (1 − α₁ − α₂)(h_b' − h_c)

$$\alpha_2 = (1 - \alpha_1)\frac{h_{b'} - h_c}{h_b - h_c}$$

四·回热循环热效率

$$\eta_{t,回} = 1 - \frac{Q_2}{Q_1} = 1 - \frac{(1 - \alpha_1 - \alpha_2)(h_2 - h_3)}{h_1 - h_{a''}}$$

- 吸热过程变为 a''→1（跳过低温段）→ 平均吸热温度 ↑ → η_t ↑
- ==抽汽回热是提高循环热效率最切实可行的方法==，几乎所有火电厂均采用
- 抽汽次数：**3~4** 次少数，**5~6** 次常见，甚至 **7~8** 次

五·回热级间分配

- 方法：焓降分配法、等焓差法、**等温升法**、等温比法等
- 严家騄观点：按==等温升法==（各级回热器给水温升相同）即可获**接近最佳**效果

> [!note] 核心理解
> 抽汽回热的物理本质：抽出的部分蒸汽不在汽轮机中继续膨胀做功，而是用于加热给水，把原本在凝汽器中白白放掉的凝结潜热"回收"到循环内部。代价是流经后段汽轮机的蒸汽量减少 → 输出功减小，但吸热量减得更多 → η_t 提高。热效率 ≈ 42~47%（现代机组）。

六·再热 + 回热联合（例 9-3）

参数：p₁ = **10** MPa，t₁ = **500°C**；再热 p_a = **1.5** MPa，t₁' = **500°C**；抽汽 p_a = 1.5 MPa，p_b = **0.13** MPa；p₂ = **0.005** MPa

- α₁ = **16.82%**，α₂ = **9.75%**
- η_t = **47.06%**，比同参数朗肯循环（**42.07%**）提高 **11.86%**

## §9-9 热电联产循环

一·概念

- 1. 现代化大型机组发电效率 ≈ **50%** → 燃料中**近一半能量**由凝汽器排向大气
- 2. 另一方面，生产生活中又额外烧燃料产中低压蒸汽供热 → 蒸汽做功潜能未利用
- 3. ==热电联产==：将热和电集中由大型热电厂提供 → 提高燃料总利用率

二·实现方式

- 1. **抽汽供热**：从汽轮机中间级抽出所需参数蒸汽 → 送往热用户
  - 代价：流经后级蒸汽量 ↓ → 汽轮机出力 ↓、η_t 略降
  - 但总能量利用率 ↑

- 2. **背压式汽轮机**：蒸汽不全膨胀到环境温度，而是到较高背压 → 全部排汽供热
  - 典型背压：**0.12** MPa（采暖，t_s ≈ **105°C**）
  - 理论上蒸汽能量利用率可达 **100%**，实际约 **70%**（含锅炉损失、管道损失）
  - 缺点：热电比例不能调节 → 用热不足时发电受限

三·能量利用率

$$K = \frac{w_0 + q_p}{q_1}$$

- w₀ = 发电能量，q_p = 供热能量，q₁ = 蒸汽从锅炉获得能量

> [!note] 核心理解
> 热电联产的哲学矛盾：η_t（热效率）下降了，但**总能量利用率** K 提高了。因为热和电的品味不同——电是高品位能，热是低品位能。评价要综合考虑，不能只看单一指标。这引向热经济学。

## §9-10 实际循环的能量分析与㶲分析

一·两种方法的依据

- 1. **能量分析**（第一定律）：能量守恒 → 只看量的平衡
- 2. **㶲分析**（第二定律）：㶲不守恒 → 看可用能的损失分布
- ==两种方法对"最大损失在哪里"的结论完全不同==

二·算例关键参数

p₁ = **13.5** MPa、t₁ = **550°C**、p₂ = **0.004** MPa、η_T = **0.88**、η_B = **0.90**、H_{u,l} = **41 000** kJ/kg(f)

- 每 kg 燃料产蒸汽 m = **11.1037** kg
- 锅炉吸热 Q₁ = **36 900.1** kJ/kg(f)
- 汽轮机输出功 W_T = **14 475.0** kJ/kg(f)
- 水泵耗功 W_P = **215.08** kJ/kg(f)
- 凝汽器放热 Q_c = **22 640.2** kJ/kg(f)

三·能量分析结果

```
能流图（100% = 燃料低发热量 41000 kJ/kg）：

  燃料低发热量 100%
     │
     ├── 锅炉热损失：10%（4100 kJ/kg）
     │
     └── 蒸汽吸热 90%
           │
           ├──→ 汽轮机做功 35.3%（14475 kJ/kg）
           │         │
           │         └── 水泵耗功 0.53% → 净功 34.78%
           │
           └──→ 凝汽器排热 55.22%（22640 kJ/kg）
```

- ==能量分析：最大损失在**凝汽器**，超过 **55%**==

四·㶲分析结果

- 燃料㶲 E_{x,u} ≈ **0.975** × H_{u,h} = **42 120** kJ/kg(f)
- 各项㶲损分布：
  - 锅炉㶲损：**59.87%**（最大！） ← 燃烧 + 传热温差大（几百到上千度）的不可逆
  - 汽轮机㶲损：**4.55%**
  - 凝汽器㶲损：**1.60%**
  - 水泵㶲损：**0.14%**
  - 输出功（有效㶲）：**33.86%**
- 㶲效率 η_{ex} = **33.55%**

```
㶲流图（100% = 燃料㶲 42120 kJ/kg）：

  燃料㶲 100%
     │
     ├── 锅炉㶲损 59.87%（燃烧不可逆 + 传热温差）
     │
     ├── 汽轮机㶲损 4.55%（摩擦 + 涡流）
     │
     ├── 凝汽器㶲损 1.60%（温差传热）
     │
     ├── 水泵㶲损 0.14%
     │
     └── 输出功 33.86%
```

五·两种分析的对照

| 对比维度 | 能量分析 | 㶲分析 |
|:---|:---|:---|
| 最大损失在哪里 | 凝汽器（> 55%） | 锅炉（≈ 60%） |
| 凝汽器排热 | 看成巨大损失 | 理论上必须排、值不了多少㶲 |
| 改进方向 | 回收排热？ | 减少锅炉燃烧不可逆 + 传热温差 |
| 理论基础 | 热力学第一定律 | 热力学第二定律 |
| 指导意义 | 容易误判 | ==更合理、更有指导意义== |

> [!note] 核心理解
> 能量分析说凝汽器是"最大损失" → 容易误导人去回收排热（实际上温度已接近环境，㶲值极低，没有做功能力）。㶲分析指出锅炉才是罪魁祸首：燃烧反应严重不可逆，火焰→蒸汽传热温差达数百度。==减少锅炉㶲损是提高能量转换效率的根本出路==，而凝汽器的排热从理论上就是必须的（第二定律要求）。

## 速记卡 / 一句话总结

- 1. **饱和** = 汽化率 = 液化率 → p_s 与 t_s 一一对应；临界点 t_c = **373.99°C** / p_c = **22.064 MPa**；三相点 t_A = **0.01°C**
- 2. **定压加热三阶段**：预热（q'）→ 汽化（r，压力越高 r 越小，临界点 r = 0）→ 过热（q''）；总吸热量 q = h − h₀
- 3. **湿蒸汽补插**：h_x = h' + x r，s_x = s' + x·r / T_s；x = 0 饱和水，x = 1 饱和蒸气
- 4. **h-s 图**：工程计算核心图——定熵过程 = 垂线，技术功直接读纵坐标差；(∂h/∂s)_p = T → 湿蒸气区定压线是直线
- 5. **朗肯循环**：锅炉定压加热 → 汽轮机绝热膨胀 → 凝汽器定压放热 → 水泵绝热压缩；η_t = 1 − (h₂−h₀)/(h₁−h₀)；典型效率 **37%**（理论）/ **30%**（实际）
- 6. **提效三招**：T₁↑（提效率 + 降湿度）、p₁↑（需与 T₁ 配合，否则湿度↑）、p₂↓（最显著，但受限于冷源 t_s ≈ **29°C**，p₂ ≈ **0.004 MPa**）
- 7. **再热循环**：膨胀到中间压力 → 再加热 → 继续膨胀；η_t ↑ + 湿度 ↓；大型超高压机组标配
- 8. **抽汽回热**：抽部分蒸汽预热给水 → 跳过低温预热段 → 平均吸热温度 ↑ → η_t ↑；级间用**等温升法**分配；联合再热 + 回热 η_t 可达 **47%**
- 9. **热电联产**：η_t 降但能量利用 K ↑；背压式 K 理论 **100%**，实际 **70%**
- 10. **能量 vs 㶲分析**：能量分析最大损失在凝汽器（**55%**），㶲分析最大损失在锅炉（**60%**）→ 㶲分析才是改进的正确指南
