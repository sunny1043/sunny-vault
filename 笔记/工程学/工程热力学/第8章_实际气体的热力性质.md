## 一· 本章导读

一· 为什么要研究实际气体

1. 理想气体的适用边界：**压力相对较低 + 温度相对较高**的空气、燃气等
2. 大多数气体不是理想气体：水蒸气、各种制冷剂 → 必须按**实际气体**处理
3. 实际气体的特征：状态方程形式复杂 → 导出的 h、s 表达式也复杂 → 工程上多采用**查图表**方法
4. 需要两张图：定性上明白实际气体为什么偏离理想气体；定量上会用通用线图查物性

二· 本章的核心逻辑链

```
理想气体 pv = RgT（基准）
      │
      ▼
实际气体偏离原因：分子有体积 + 分子间有引力
      │
      ▼
修正 → 范德瓦尔方程（定性对，定量差）
      │
      ▼
维里方程 / R-K方程（更精确的经验修正）
      │
      ▼
对应态原理 → 通用压缩因子图 / 余焓图 / 余熵图（工程查图法）
      │
      ▼
计算实际气体的 h、s 修正 → 工程热力计算完整闭环
```

> [!note] 核心理解
> 物理上：理想气体 = 分子质点（零体积 + 零作用力）；实际气体 = 分子硬球 + 引力场。从理想 → 实际的修正本质上是给分子\"还\"了两个物理属性：体积和引力。数学上：p→p+a/v²（引力修正内压），v→v−b（扣除分子自身体积），这就是范德瓦尔方程的全部物理思想。

## 二· 纯物质的热力学面（8-1）

### 热力学面的定义

**定义**：纯物质的状态方程为 F(p, v, T)=0，在 p-v-T 空间坐标系中构成一个**复杂曲面**，曲面上每一点代表一个平衡状态。这个曲面称为**热力学面**。

### 热力学面的结构——\"一面六区七线一点\"

一· 一面：F(p, v, T)=0 所确定的热力学曲面

二· 六区（三个单相区 + 三个两相区）

1. **三个单相区**：固相区、液相区、气相区 —— 单独占据热力学面的不同区域
2. **三个两相区**（饱和区）：夹在两个单相区之间
   - （1）气-液两相区
   - （2）气-固两相区
   - （3）固-液两相区
   - 在两相区内，两种相**平衡共存**，具有**相同的压力和温度** → 称为饱和状态

```
       液                气
       │                 │
       ├─── 液-气 ───────┤
       │                 │
   固──┤                 │
       │                 │
       ├─ 固-液          ├─ 固-气
       │                 │
```

三· 七线（六条饱和线 + 一条三相线）

1. 熔解线：固-液两相区与固相区的交界
2. 凝固线：固-液两相区与液相区的交界
3. 饱和液体线（气化线）：气-液两相区与液相区的交界
4. 饱和蒸气线（凝结线）：气-液两相区与气相区的交界
5. 升华线：气-固两相区与固相区的交界
6. 凝华线：气-固两相区与气相区的交界
7. 三相线：气-液、固-液、气-固三个两相区的交线 → 对应**三相点 A**

四· 一点：临界点 C

1. 定义：饱和液体线与饱和蒸气线的**汇合点**
2. 临界参数：临界温度 **Tc**、临界压力 **pc**、临界比体积 **vc**
3. 水的临界参数：pc = **22.064 MPa**，Tc = **647.14 K**（≈373.99°C），vc ≈ **0.00326 m³/kg**
4. ==临界点是气液两相共存的最高压力和最高温度==
5. 超临界现象：在超临界压力下加热 → 液→气无明显气化过程，**连续渐变**完成，始终单相

五· 三相点 A

1. 三相点参数：压力 pA、温度 TA 完全确定
2. 水的三相点：pA = **611.2 Pa**，TA = **273.16 K**
3. 低于三相点压力加热固体 → **直接升华**为气体，液相不复存在
4. 高于三相点压力加热固体 → 正常固→液→气

![[图_热力学面示意.excalidraw]]  <!-- 📌 待补图 -->
> **图注**：纯物质 p-v-T 热力学面示意。标出三个单相区（固/液/气）、三个两相区、六条饱和线、临界点 C 和三相点 A。水的 p-v-T 面在固-液边界处斜率特殊（冰熔化时体积收缩）。

> [!note] 核心理解
> 物理上：热力学面是纯物质\"相的地图\"——它告诉你给定 p、v、T 下物质是固态/液态/气态还是两相共存。临界点是这张地图上最关键的地标：超过它，液和气的区别消失。数学上：F(p,v,T)=0 是一个二维流形嵌入三维空间（2-manifold in R³），面上的折痕/边界线对应相变。

## 三· 纯物质的气-液相变和克劳修斯-克拉贝隆方程（8-2）

### 气-液相变过程

一· p-v 图上的定温压缩过程

1. 在 p-v 图中，临界点 C 和三相点 A 之间的饱和曲线 AC 是气-液两相区的边界
2. 低于临界温度的气体定温压缩：
   - （1）a→b：气体压力升高、比体积减小——**单相气体区**
   - （2）b→d：气-液相变过程，==温度 Ts 和压力 ps 保持不变==；b 点 = 干饱和蒸气，d 点 = 饱和液体，中间 = 湿蒸气（不同比例气液共存）
   - （3）d→e：液体被压缩，压力迅速升高、比体积缓慢减小——液体的可压缩性很小
   - b→d 过程放出的热量 = 凝结潜热 r（kJ/kg），d→b = 气化潜热（数值相等）

```
p ↑
  │                    C（临界点）
  │              b─────d              ← 定温定压，相变平台
  │            ／        ＼
  │    a─────              ＼
  │  ／                      e
  │／
  └──────────────────────→ v
```

3. 在临界温度 Tc 下定温压缩：在 C 点附近**整体地、逐渐地**过渡到液体，无明显的凝结过程，定温线在 C 点形成拐点
   - ∂p/∂v|T = **0**，∂²p/∂v²|T = **0**
4. 超过 Tc 下定温压缩：即使压力远超 pc，**也不会液化**
5. 低于三相点温度定温压缩：气体直接**凝华**为固体

二· 关键结论

- ==纯物质的气-液相变只发生在临界点和三相点之间==
- 临界温度 → 能发生液化的最高温度
- 饱和压力是饱和温度的**单值函数** —— ps = ps(Ts)，与比体积无关

### 克劳修斯-克拉贝隆方程

**定义**：描述相变过程中饱和压力与饱和温度关系的微分方程。

**公式与推导**：

（1）出发点——麦克斯韦关系式（7-6）第三式：
$$\left(\frac{\partial p}{\partial T}\right)_v = \left(\frac{\partial s}{\partial v}\right)_T$$

（2）由于饱和压力 ps 仅是 Ts 的函数，与 v 无关：
$$\left(\frac{\partial p}{\partial T}\right)_v = \frac{dp_s}{dT_s} = \left(\frac{\partial s}{\partial v}\right)_T$$

（3）沿定温线（相变过程 Ts 不变）积分：
$$ds_T = \frac{dp_s}{dT_s} dv_T$$
$$\int_{s'}^{s''} ds_T = \frac{dp_s}{dT_s} \int_{v'}^{v''} dv_T$$
$$s'' - s' = \frac{dp_s}{dT_s}(v'' - v')$$

（4）利用气化潜热定义 r = Ts(s'' − s')，代入得：
$$\frac{dp_s}{dT_s} = \frac{s'' - s'}{v'' - v'} = \frac{r}{T_s(v'' - v')}$$

> 其中 s'、v' 为饱和液体的比熵和比体积；s''、v'' 为饱和蒸气的比熵和比体积；r 为气化潜热。

这就是**克劳修斯-克拉贝隆方程（Clausius-Clapeyron Equation）**。

**适用范围**：不仅用于气-液两相相变，也可用于**任何两相**相变（气-固、液-固、固-固）。

**低压简化形式**：在低压下 v'' ≫ v'，且蒸气可近似为理想气体 v'' ≈ RgTs/ps，则：
$$\frac{dp_s}{dT_s} \approx \frac{r p_s}{R_g T_s^2}$$

**性质和条件**：
- 饱和曲线的斜率 dps/dTs 在给定温度下为定值
- 冰的熔解线斜率为负（v'液 < v'固，冰熔化时体积缩小）→ 可用克-克方程解释
- 一般物质的气化线和升华线斜率为正

> [!note] 核心理解
> 物理上：克-克方程告诉你\"压力升高 1 Pa，饱和温度升高多少\"——本质是相变时两相的熵差和体积差之比决定斜率。冰的熔解线斜率为负的物理原因：冰的密度比水小（v固 > v液），分母 v''−v' 为负 → dps/dT 为负 → 加压反而降低熔点。数学上：这是一阶常微分方程，给定 r(T) 和 v''(T)−v'(T) 即可积分出 ps(T) 曲线。

【例8-1】水蒸气在 **100°C** 时，v' = **0.0010434 m³/kg**，v'' = **1.6736 m³/kg**，r = **2256.6 kJ/kg**，ps = **0.101325 MPa**。
$$\frac{dp_s}{dT_s} = \frac{2256.6 \times 10^3}{(100+273.15) \times (1.6736 - 0.0010434)} = 3616 \text{ Pa/K} = 0.003616 \text{ MPa/K}$$

用 99°C→101°C 的实测饱和压力变化验证：dps/dTs ≈ **(0.104994 − 0.097762)/(2 K) = 0.003616 MPa/K** ✓

## 四· 实际气体的状态方程（8-3）

### 为何需要实际气体状态方程

一· 理想气体的局限

1. 理想气体假设：① 分子不占有体积；② 分子之间没有作用力
2. 实际情况：高压 + 低温 → 分子体积不可忽略 + 分子间引力显著 → pv ≠ RgT
3. 有了状态方程 p = p(T, v) + 理想气体比热容 cp0(T) → 通过热力学一般关系式计算全部平衡性质

二· 压缩因子 Z

**定义**：$$Z = \frac{pv}{R_g T}$$

- ==Z 反映实际气体与理想气体的偏离程度==
- Z = 1：理想气体
- Z > 1：实际气体比理想气体更难压缩（v > v0）
- Z < 1：实际气体比理想气体更易压缩（v < v0）

物理意义：$$Z = \frac{v}{v_0} = \frac{v(T,p)}{R_g T/p}$$ ，即相同 T、p 下实际气体与理想气体比体积之比。

### 范德瓦尔状态方程（van der Waals, 1873）

**定义**：
$$\left(p + \frac{a}{v^2}\right)(v - b) = R_g T$$

**物理思想**：
- （1）分子有体积 → 分子自由运动空间从 v 减小为 **v − b**（b 为与分子体积相关的修正数）
- （2）分子间有引力 → 碰撞器壁的压力比理想值小 **a/v²**（a 为与分子引力相关的修正数）

**引力修正 a/v² 的物理机制**：
- 分子与器壁碰撞时受内部引力 → 产生向内的合力 → 碰撞压力减小
- 减小量∝（碰撞强度×碰撞频率）→ 两者均∝气体密度 → 总效果∝密度² = 1/v²

```
       容器壁
    ←───── ○ ← 碰撞分子
          ↑
     ←────┼────→ 内部引力（向内）
          ↓
     ○ ○ ○ ○  （内部气体分子）
```

**整理为 v 的三次式**：
$$v^3 - \left(b + \frac{R_g T}{p}\right)v^2 + \frac{a}{p}v - \frac{ab}{p} = 0$$

- 对应每个 p、T，v 有三个根：可能一个实根+两个虚根；或一个实根+一个二重根；或三个不同实根（对应 T < Tc 时的气液共存）
- ==T < Tc 时定温线呈 S 形曲折 → Maxwell 等面积法则给出饱和压力==

**修正数 a、b 与临界参数的关系**

（1）临界定温线在临界点处：$$\left(\frac{\partial p}{\partial v}\right)_{T_c} = 0, \quad \left(\frac{\partial^2 p}{\partial v^2}\right)_{T_c} = 0$$

（2）范德瓦尔方程写为 p = RgT/(v−b) − a/v²，求偏导：
$$\left(\frac{\partial p}{\partial v}\right)_{T_c} = -\frac{R_g T_c}{(v_c - b)^2} + \frac{2a}{v_c^3} = 0$$
$$\left(\frac{\partial^2 p}{\partial v^2}\right)_{T_c} = \frac{2R_g T_c}{(v_c - b)^3} - \frac{6a}{v_c^4} = 0$$

（3）联立解得：
$$v_c = 3b, \quad a = 3p_c v_c^2 = \frac{27}{64}\frac{R_g^2 T_c^2}{p_c}, \quad b = \frac{v_c}{3} = \frac{R_g T_c}{8p_c}$$

（4）临界压缩因子：
$$Z_c = \frac{p_c v_c}{R_g T_c} = \frac{3}{8} = 0.375$$

**评价**：
- 定性：能正确反映气液两相共存、临界点、S 形定温线等主要特征
- 定量：实际大多数气体 Zc = **0.23~0.30**，距 0.375 较远 → ==范德瓦尔方程定量上不精确==

**易错/边界**：
- 范德瓦尔方程是**半经验方程**——物理意义清楚，但定量精度有限
- 在低压高温极限下（v→∞），a/v²→0，v−b≈v → 退化为理想气体方程

### 维里方程（Virial Equation, Kamerlingh Onnes, 1901）

**定义**：用幂级数形式表达的状态方程

以 1/v 展开：
$$Z = \frac{pv}{R_g T} = 1 + \frac{B}{v} + \frac{C}{v^2} + \frac{D}{v^3} + \cdots$$

以 p 展开：
$$Z = 1 + B'p + C'p^2 + D'p^3 + \cdots$$

- A/B/C/D（或 B'/C'/D'）称为**维里系数**，都是**温度的函数**
- v→∞ 时 Z→1（趋近理想气体）→ 第一维里系数 A≡1

**两种展开的关系**：
$$B' = \frac{B}{R_g T}, \quad C' = \frac{C - B^2}{(R_g T)^2}, \quad D' = \frac{D - 3BC + 2B^3}{(R_g T)^3}$$

**截断原则**：截断处以后各项总和应在实验误差以内。1/v 展开收敛更快，p 展开收敛慢 → 主要用于低密度情况。

**例子：维里形式的范德瓦尔方程**
$$Z = 1 + \frac{b - a/(R_g T)}{v} + \frac{b^2}{v^2} + \frac{b^3}{v^3} + \cdots$$

> [!note] 核心理解
> 物理上：维里方程不是一个新的物理模型，而是**状态方程的\"泰勒展开\"**——把任何状态方程都写成 Z 的无限级数。B 反映两分子相互作用，C 反映三分子相互作用，以此类推。数学上：任何解析的状态方程在 v→∞ 附近都有维里展开，维里系数可以通过统计力学从分子间势能计算出来 → 连接了宏观和微观。

### 雷特里奇-邝方程（Redlich-Kwong, 1949）

**定义**：对范德瓦尔方程中 a/v² 项的温度依赖性做修正：
$$p = \frac{R_g T}{v - b} - \frac{a}{T^{0.5} v(v + b)}$$

**修正参数与临界参数的关系**（类似范德瓦尔推导）：
$$a = 0.42748 \frac{R_g^2 T_c^{2.5}}{p_c}, \quad b = 0.08664 \frac{R_g T_c}{p_c}$$

**临界压缩因子**：
$$Z_c = \frac{1}{3} = 0.333$$

**评价**：
- 精确度**高于范德瓦尔方程**，在化工等领域广泛应用
- 有实验数据时，a、b 可由实验数据拟合 → 精度更高

## 五· 实际气体的比热容、焦-汤系数、焓和熵（8-4）

### 实际气体的比热容

一· 余比热容的概念

1. 理想气体：cp0 = cp0(T)（仅温度的函数）
2. 实际气体：cp = cp(T, p) → 比 cp0 多了压力修正
3. 定义余比热容：**Δcp = cp − cp0** = cp(T, p) − cp0(T)
4. 规律：温度升高 → Δcp 减小；压力升高 → Δcp 增大

二· 余比热容的计算式

$$\Delta c_p = (c_p - c_{p0})_T = -T \int_0^p \left(\frac{\partial^2 v}{\partial T^2}\right)_p dp_T$$

推导依据：式（7-41）来自热力学一般关系式。

三· 确定 Δcp 的两条途径

1. 直接测定：可靠但费时，实验装置复杂
2. 利用状态方程计算：已知 v(T, p) → 求二阶偏导 → 积分

四· 反过来——由比热容实验数据建立状态方程

若已知 cp(T, p) 的实验数据：
$$v = \iint -\frac{f'(T,p)}{T} dT_p^2 + C_1(p)T + C_2(p)$$

### 实际气体的焦-汤系数

**定义**：绝热节流过程中温度随压力的变化率
$$\mu_J = \left(\frac{\partial T}{\partial p}\right)_h = \frac{1}{c_p}\left[T\left(\frac{\partial v}{\partial T}\right)_p - v\right]$$

**物理图像——T-p 图上的定焓线**：

```
T ↑
  │   致热区（μJ < 0，节流升温）
  │          ╱
  │        ╱  ← 转回曲线（μJ = 0）
  │      ╱
  │    ╱    致冷区（μJ > 0，节流降温）
  │  ╱
  │╱
  └──────────────────→ p
```

- 定焓线在高压区温度上升 → **致热区**（μJ < 0）
- 经过最高点后转为下降 → **致冷区**（μJ > 0）
- 所有定焓线最高点的连线 → **转回曲线**（μJ = 0）
- 转回曲线方程：$$v = T\left(\frac{\partial v}{\partial T}\right)_p$$
- ==最高转回温度 Tmax ≈ 5Tc==
- 氢气的 Tc = **33.2 K**，常温远高于 Tmax → 常温下节流氢气不降温反而升温（历史上的教训）

**转回曲线的特殊点**（对范德瓦尔气体）：
- 最高转回温度：**T = 6.75 Tc**
- 最低转回温度：**T = 0.75 Tc**
- 最高转回压力：**p = 9 pc**

### 实际气体的焓

**以 T、p 为独立变量的焓的一般关系式**（式7-32）：
$$dh = c_p dT - \left[T\left(\frac{\partial v}{\partial T}\right)_p - v\right] dp$$

**积分求焓**——选参考状态 O(p0, T0)，沿 O→m→A 路径（先在低压下定温升温，再定温升压）：
$$h = \int_{T_0}^T c_{p0} dT - \int_0^p \left(\frac{\partial v}{\partial T}\right)_p dp_T + \int_0^p v dp_T + h_0$$

**对 p = p(T, v) 形式的状态方程**，变换为更方便的形式：
$$h = \int_{T_0}^T c_{p0} dT + \int_{\infty}^v \left[T\left(\frac{\partial p}{\partial T}\right)_v\right] dv_T - \int_{\infty}^v p dv_T + pv - R_g T + h_0$$

（推导利用循环关系式转换偏导 + 分部积分。）

- 其中 ∫cp0 dT = h0（同一温度下理想气体状态的焓值）
- 后两项 = 实际气体偏离理想气体的修正项

### 实际气体的熵

**以 T、p 为独立变量的第二 ds 方程**（式7-26）：
$$ds = \frac{c_p}{T} dT - \left(\frac{\partial v}{\partial T}\right)_p dp$$

**积分求熵**——同样沿 O→m→A 路径：
$$s = \int_{T_0}^T \frac{c_{p0}}{T} dT - \int_0^p \left(\frac{\partial v}{\partial T}\right)_p dp_T + s_0$$

**对 p = p(T, v) 形式的状态方程**：
$$s = \int_{T_0}^T \frac{c_{p0}}{T} dT + \int_{\infty}^v \left(\frac{\partial p}{\partial T}\right)_v dv_T + s_0$$

> [!note] 核心理解
> 物理上：实际气体 h、s 的计算 = 理想气体部分 + 修正。修正来自\"分子间引力 + 分子体积\"导致的 pv ≠ RgT。修正积分路径选低压→高压，因为低压下实际气体≈理想气体，好算。数学上：dh 和 ds 是全微分 → 积分与路径无关 → 选最方便路径计算。这种\"选参考态 + 绕道计算\"的手法在热力学中反复出现——本质是利用状态函数的路径无关性。

## 六· 对比状态方程和对应态原理（8-5）

### 对比参数的定义

将 p、v、T 分别除以各自的临界参数，得到**无因次**对比参数：

- 对比温度：$$T_r = \frac{T}{T_c}$$
- 对比压力：$$p_r = \frac{p}{p_c}$$
- 对比比体积：$$v_r = \frac{v}{v_c}$$
- 压缩因子：$$Z = \frac{pv}{R_g T} = \frac{p_r v_r}{T_r} Z_c$$

### 范德瓦尔对比状态方程

**推导**（从有因次的范德瓦尔方程出发）：

（1）有因次范德瓦尔方程：$$\left(p + \frac{a}{v^2}\right)(v - b) = R_g T$$

（2）代入 a = 3pc vc²、b = vc/3、Rg = (8/3)(pc vc/Tc)：
$$\left(p + \frac{3p_c v_c^2}{v^2}\right)\left(v - \frac{v_c}{3}\right) = \frac{8}{3}\frac{p_c v_c}{T_c} T$$

（3）全式除以 pc vc，引入对比参数：
$$\left(p_r + \frac{3}{v_r^2}\right)\left(v_r - \frac{1}{3}\right) = \frac{8}{3} T_r$$

这就是==范德瓦尔对比状态方程——不包含任何物质特定常数，是**通用的**==。

**用 Z 替代 vr 的等价形式**：对范德瓦尔气体 Zc = 3/8，
$$\left(Z + \frac{27 p_r}{64 Z T_r^2}\right)\left(1 - \frac{p_r}{8 Z T_r}\right) = 1$$

**理想气体的对比状态方程最简单**：Z = 1（恒等于 1）。

### 对应态原理（热力学相似原理）

一· 从范德瓦尔气体的启示

1. 所有范德瓦尔气体 → 共用同一个对比状态方程 F(Tr, pr, vr) = 0
2. → 对任何范德瓦尔气体，只要 **Tr 和 pr 相同，vr（或 Z）就必定相同**
3. → 所有范德瓦尔气体是**热力学相似**的

二· 推广到一般情况

1. 任何一组热力学相似的气体 → 在 Tr-pr-vr（或 Tr-pr-Z）三维坐标系中具有**同一热力学面**
2. **对应态原理**（Corresponding State Principle）：
   ==对热力学相似的流体，在三个对比参数 Tr、pr、vr（或 Z）中，如果有两个相同，则第三个也必定相同==

三· 对应态原理的意义

1. 只需要**一种**热力学相似气体的实验数据 → 可以推算**所有**同类气体的性质
2. 实际气体中，临界压缩因子 Zc 相近的物质 ≈ 热力学相似
3. 大多数流体的 Zc ≈ **0.27** → 可以用同一套通用线图

```
热力学相似 → 同一对比状态方程 → 通用线图可查
                                   │
                    ┌──────────────┼──────────────┐
                    ▼              ▼              ▼
              压缩因子图      对比余焓图      对比余熵图
              Z = f(Tr, pr)  Δh/RTc = f(Tr,pr)  Δs/R = f(Tr,pr)
```

> [!note] 核心理解
> 物理上：对应态原理说\"只要把温度/压力都除以各自的临界值，所有气体的行为就差不多\"——临界点是每种气体的\"标尺\"，用这个标尺归一化后，差异被消除。类比：用身高除以自己的头长（≈7头身），不同身高的人归一化后比例类似。数学上：对比状态方程消去了 a/b/Rg 等物质特定常数，只留下无因次关系 → 这就是为什么同一张图对所有 Zc≈0.27 的气体都通用。

**本书作者的贡献**（严家騄）：
- 实际气体通用状态方程：进一步考虑分子缔合现象 + 温度对分子体积和引力的影响
- 饱和蒸气通用状态方程：多重复合分子模型
- 精度：误差 < **1%**，适用范围广

## 七· 通用压缩因子图、对比余焓图和对比余熵图（8-6）

### 通用压缩因子图

一· 制作原理

1. 对 Zc ≈ 0.27 的流体，Z = Z(Tr, pr) → 在 Z-pr 坐标系中画定 Tr 线簇
2. 数据来源：**多种**气体的 p-v-T 实验数据整理
3. 从图8-6的实际数据可以看出：性质差异很大的物质，==在相同 Tr、pr 下 Z 值基本相同== → 印证对应态原理

二· 图中曲线的共同特征

1. **pr → 0 时**：所有定 Tr 线收敛于 Z = 1 → 任何气体在极低压力下都趋于理想气体
2. **Tr > 1 时**（超临界温度）：Z 单调变化，无相变特征
3. **Tr < 1 时**（亚临界温度）：Z 发生跳跃（对应气-液相变）
4. **低压下 Z < 1**（分子引力主导 → 易压缩），**高压下 Z > 1**（分子体积主导 → 难压缩）

```
Z ↑
  │                    Tr = 5.0
  │  1.0 ────────────────────────  Tr = 2.0
  │     ╱                        Tr = 1.5
  │   ╱                          Tr = 1.2
  │ ╱    （跳跃）                 Tr = 1.0（临界）
  │╱                              Tr = 0.9
  │──────────────────────────→ pr
  │
  所有线 → pr=0 时 Z→1
```

三· 用途

- 已知 T、p 和临界参数 → 计算 Tr、pr → 查图得 Z → 由 v = ZRgT/p 求比体积
- 为**缺乏专用状态方程和专用图表的流体**提供计算依据

### 通用对比余焓图

**推导过程**：

（1）在定温下积分 dh 方程：
$$(h - h_0^*)_T = \int_0^p \left[v - T\left(\frac{\partial v}{\partial T}\right)_p\right] dp_T$$

（2）用 Z 替代 v：v = ZRgT/p
$$\left(\frac{\partial v}{\partial T}\right)_p = \frac{Z R_g}{p} + \frac{R_g T}{p}\left(\frac{\partial Z}{\partial T}\right)_p$$

（3）代入并做变量替换 p = pr·pc：
$$-\Delta h_r = \frac{h_0^* - h}{R_g T_c} = T_r^2 \int_0^{p_r} \left(\frac{\partial Z}{\partial T_r}\right)_{p_r} d(\ln p_r)$$

- h0*：相同温度下理想气体的比焓
- Δhr：对比余焓（residual enthalpy），**仅是 Tr 和 pr 的函数**

**使用方法**：
1. 计算对比参数 Tr、pr
2. 查图8-8得 (h0* − h)/(RgTc) 值
3. 实际气体的焓：h = h0* + Δh_correction
4. 其中 h0* = ∫ cp0 dT（用理想气体比热容计算）

### 通用对比余熵图

**推导过程**：

（1）在定温下积分 ds 方程：
$$(s - s_0^*)_{T} = -\int_0^p \left(\frac{\partial v}{\partial T}\right)_p dp_T$$

（2）同样用 Z 替代并变换：
$$-\Delta s_r = \frac{s_0^* - s}{R_g} = \int_0^{p_r} (Z - 1) d(\ln p_r)_{T_r} + T_r \int_0^{p_r} \left(\frac{\partial Z}{\partial T_r}\right)_{p_r} d(\ln p_r)$$

（3）利用与余焓的关系：
$$\frac{s_0^* - s}{R_g} = \frac{h_0^* - h}{R_g T_c} \cdot \frac{1}{T_r} + \int_0^{p_r} (Z - 1) d(\ln p_r)_{T_r}$$

- s0*：相同 T、p 下理想气体的比熵

**使用方法**：
1. 计算 Tr、pr
2. 查图8-9得 (s0* − s)/Rg 值
3. 实际气体的熵：s = s0* + Δs_correction

> [!note] 核心理解
> 物理上：通用线图的本质是\"用图代公式\"——对比状态方程 F(Z,Tr,pr)=0 的解析式太复杂，干脆把实验数据画成曲线上图里，看图说话。余焓 = 实际焓 − 理想焓，修正量来自分子间相互作用。数学上：所有通用线图的理论依据都是热力学相似（对应态原理）——流体只要 Zc 相近，其无因次偏离量 Δh/RTc、Δs/R 就只是 Tr 和 pr 的函数，不依赖具体物质。

**通用线图的局限性**：
1. 只对 Zc ≈ 0.27 的流体精度好 → Zc 偏离大的流体需专门线图
2. 近临界区精度下降 → 状态方程的直接计算更可靠
3. 通用线图是\"对比状态方程的图解替代物\"——曲线形状复杂，难以用一个数学式概括

## 八· 典型例题解析

### 【例8-4】用通用线图计算 CO₂ 的比焓和比熵

**已知条件**：CO₂ 在 T = **450 K**，p = **15 MPa**；
参考态：**273.15 K、0.101325 MPa** 时 h0 = 0，s0 = 0。
（注意：参考态选在极低压下，CO₂ 可看作理想气体。）

**步骤**：

（1）查表8-1得临界参数：Tc = **304.2 K**，pc = **7.376 MPa**
查附表得：Rg = **0.18892 kJ/(kg·K)**
查附表得 cp0 表达式：cp0 = 0.5058 + 1.3590×10⁻³·{T}K − 0.7955×10⁻⁶·{T}K² + 0.1697×10⁻⁹·{T}K³  kJ/(kg·K)

（2）计算对比参数：
Tr = 450/304.2 = **1.479**，pr = 15/7.376 = **2.034**

（3）查图8-7（通用压缩因子图）：Tr = 1.479，pr = 2.034 → **Z = 0.845**
→ v = ZRgT/p = 0.845×0.18892×10³×450/(15×10⁶) = **0.00479 m³/kg**
（理想气体计算值：0.00567 m³/kg → 偏差约 −15.5%）

（4）查图8-8（通用对比余焓图）：Tr = 1.479，pr = 2.034 → **Δh/(RgTc) = 1.05**

（5）计算 h0*（理想气体状态下的比焓）：
h0* = ∫ cp0 dT（从273.15 K到450 K）= **153.80 kJ/kg**

（6）实际气体的比焓：
h = h0* − 1.05 × 0.18892 × 304.2 = 153.80 − 60.34 = **93.46 kJ/kg**
（理想气体计算值：153.80 kJ/kg → 修正量约 −39%）

（7）查图8-9（通用对比余熵图）：Tr = 1.479，pr = 2.034 → **Δs/Rg = 0.58**

（8）计算 s0*（理想气体状态下的比熵）：
s0* = ∫(cp0/T)dT − Rg ln(p/p0) = −0.9941 kJ/(kg·K)

（9）实际气体的比熵：
s = s0* − 0.58×0.18892 = −0.9941 − 0.1096 = **−1.1037 kJ/(kg·K)**

> [!note] 核心理解
> 例8-4展示完整的\"通用线图法\"流程：临界参数 → Tr/pr → 查Z → 查Δh → 查Δs → 分别加理想气体计算值。注意：余焓/余熵的定义是 h0*−h（理想减实际），因此实际值 = 理想值 − 修正量。

### 【例8-5】水蒸气节流过程的焓和熵变化

水蒸气从 p1 = **30 MPa**、T1 = **500°C** 节流到 p2 = **0.1 MPa**，求比焓和比熵变化。

已知：Tc = **647.14 K**，pc = **22.064 MPa**，Rg = **0.4615 kJ/(kg·K)**

**关键结果**：
- 节流过程 h1 = h2（焓不变）→ 通用线图计算 Δh ≈ **−388.3 kJ/kg**（与查水蒸气表 403.2 kJ/kg 相比误差 ~3.7%）
- Δs ≈ **3.0246 kJ/(kg·K)**（与查表 3.0383 kJ/(kg·K) 相比误差 ~0.5%）

> [!tip] 补充
> 即使水蒸气的 Zc = 0.229（离0.27较远），用 Zc = 0.27 的通用线图计算结果仍**较为满意**——说明对应态原理的\"普适性\"范围比严格推导的要宽。这正是工程上通用线图实用的原因。

## 思考题精选

1. **物质的临界状态究竟是怎样一种状态？**
   → 气液两相共存消失的极限状态。p > pc 后无气化过程，液→气连续渐变；临界点处定温线有拐点。

2. **超临界压力下加热液体 → 气体的过程如何完成？**
   → 无明显的相变平台，始终单相，在 Tc 附近以连续渐变方式完成密度和性质的过渡。

3. **冰的熔解线斜率为负 → 能否用克-克方程解释？**
   → 冰的 v'液 < v'固（冰密度小于水）→ v''−v' < 0 → dps/dT < 0 → 加压降低熔点。✓

4. **实际气体在低压下 Z < 1，高压下 Z > 1 → 为什么？**
   → 低压：分子间距大，引力主导 → 气体比理想状态更\"收缩\" → Z < 1；
   高压：分子间距小，分子自身体积占主导 → 气体比理想状态更\"膨胀\" → Z > 1。

5. **焦-汤系数在致冷区和致热区的符号及物理含义？**
   → μJ > 0（致冷区）：节流后温度下降；μJ < 0（致热区）：节流后温度上升。判断依据：当前状态是否在转回曲线之内（致冷区）。

## 速记卡 / 一句话总结

- ==热力学面 = 一面六区七线一点，临界点是气液共存消失的极限状态==
- 克-克方程：dps/dTs = r/[Ts(v''−v')]，低压可简化为 dps/dTs ≈ rps/(RgTs²)
- ==范德瓦尔方程的核心思想：分子体积 → v−b；分子引力 → p+a/v²==
- 范德瓦尔气体 Zc = **3/8 = 0.375**，实际气体 Zc = **0.23~0.30**
- R-K方程在 v−b 后加了 T^0.5 修正 → Zc = **1/3 = 0.333**，精度优于范德瓦尔
- 压缩因子 Z = pv/(RgT)，Z<1 易压缩（引力主导），Z>1 难压缩（体积主导）
- ==对应态原理：两个对比参数相同 → 第三个必相同（对热力学相似流体）==
- 最高转回温度 Tmax ≈ **5 Tc**，氢气 Tc = 33.2 K → 常温下节流升温
- 通用线图 = 对比状态方程的图解替代 → 已知 Tr、pr → 查 Z、Δh、Δs
- 实际气体的 h 和 s = 理想气体值 + 修正量，修正只取决于 Tr 和 pr
- 典型流程：临界参数 → Tr/pr → 查图（Z/Δh/Δs） → 加理想气体计算值 → 得结果
