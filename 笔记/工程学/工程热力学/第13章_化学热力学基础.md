## 一· 化学反应系统的特点

一· 与简单可压缩系统的关键差异

1. 分子结构变化 → 内部**化学能**变化 → 热力学能不仅含分子动能+位能，还含化学能
2. 组分变化 → 状态参数不仅取决于 p、T，还依赖各组分质量分数 wᵢ 或摩尔分数 xᵢ
   - U = U(T, p, w₁, w₂, …, wᵢ, …)
   - 独立变量增多 → 即使 p、T 不变，化学反应仍可进行
3. 功的形式拓展
   - 简单可压缩系统：仅容积变化功 W
   - 化学反应系统：总功 W_tot = W（容积功）+ W_u（==有用功==，非容积变化功的主要形式，如电功）
4. 物量单位：采用**摩尔**（涉及分子结构变化）

二· 两类特殊过程

- 定温-定容过程、定温-定压过程：在简单系统中 p、T不变则状态不变，但在反应系统中仍有化学变化发生

> [!note] 核心理解
> 物理上：化学反应系统多了一个自由度——化学组分。就像简单系统是 (p, v) 二维状态面，反应系统是 (p, v, {wᵢ}) 多维空间。数学上：因此需要引入化学势 μᵢ 作为组分变化对应的"广义力"，dU = TdS − pdV + Σμᵢdnᵢ。

## 二· 热力学第一定律在化学反应中的应用

一· 反应热的基本公式

1. 定容反应热（闭口系，dV=0，W=0，ΔE_k=ΔE_p=0）
   - **Q_V = ΔU = U_Pr − U_Re**
   - （Pr = 生成物 Products，Re = 反应物 Reactants）

2. 定压反应热（闭口系定压 或 开口系稳定流动无技术功）
   - **Q_p = ΔH = H_Pr − H_Re**

二· 热效应

1. **定容热效应 Q_V**：定温-定容条件下反应热
   - 只要求初终态温度相等，不要求过程始终定温（因为 ΔU 是状态量）

2. **定压热效应 Q_p**：定温-定压条件下反应热
   - Q_p − Q_V 的关系
     - 一般：Q_p − Q_V = (H_Pr−H_Re) − (U_Pr−U_Re) = (pV)_Pr − (pV)_Re
     - ==理想气体：Q_p − Q_V = RT × (n_Pr − n_Re) = Δn·RT==
     - 物理含义：定压时做的容积功（反应前后摩尔数变化→体积变化→对外做功）

3. **标准定压热效应 Q_p⁰**
   - 标准状况：t = **25°C**，p = **1 atm**
   - 后续讨论以 Q_p 为准

4. 绝热过程
   - 定容绝热：U_Pr = U_Re
   - 定压绝热：H_Pr = H_Re

三· 生成焓与燃烧焓

1. **标准生成焓 ΔH_f⁰**：标准状况下，由稳定单质生成 1 mol 化合物的焓变
2. **燃烧焓（发热量）**：1 mol 燃料完全燃烧时的焓变
   - 高热值（HHV）：产物中水为液态
   - 低热值（LHV）：产物中水为气态

> [!tip] 补充
> 工程热力学中燃烧计算多用低热值 LHV，因为实际燃烧排气温度高，水以气态排出，冷凝潜热无法利用。HHV − LHV = 水的汽化潜热 × 生成水量。

## 三· 热力学第二定律在化学反应中的应用

一· 熵与化学反应的不可逆性

1. 化学反应也存在不可逆性（方向性）
2. 孤立系的熵增原理仍然适用：dS_iso ≥ 0

二· 自由能——反应方向判据

1. **亥姆霍兹自由能 F**（定温-定容过程）
   - F = U − TS
   - ==dF_T,V ≤ 0==（定温定容下，自发过程F减小，平衡时F最小）

2. **吉布斯自由能 G**（定温-定压过程）——==最常用==
   - G = H − TS
   - ==dG_T,p ≤ 0==（定温定压下，自发过程G减小，平衡时G最小）
   - ΔG = ΔH − TΔS → 焓效应与熵效应的竞争

3. 物理含义
   - ΔG < 0 → 反应可自发进行
   - ΔG = 0 → 化学平衡
   - ΔG > 0 → 逆反应自发

三· 化学势与化学平衡

1. **化学势 μᵢ**：组分 i 的偏摩尔吉布斯自由能 = (∂G/∂nᵢ)_T,p,n_j≠i
2. 化学平衡条件：Σνᵢμᵢ = 0（反应物取负，生成物取正）
3. **平衡常数 K_p**
   - 对气相反应 aA + bB ⇌ cC + dD：
   - K_p = (p_C^c × p_D^d)/(p_A^a × p_B^b)（以分压表示）
   - ΔG⁰ = −RT ln K_p
   - ==K_p 大 → 反应偏向生成物方向==

四· 最大有用功

- 定温定压反应的最大有用功 = −ΔG
- 燃料电池正是将 −ΔG 直接转为电功，不受卡诺效率限制

> [!note] 核心理解
> 物理上：ΔG = ΔH − TΔS 揭示了化学反应的驱动力来自两方面——①焓降（放热倾向，ΔH<0 有利）；②熵增（混乱度增加倾向，ΔS>0 有利）。高温下 TΔS 项权重增大。数学上：G 是定温定压下系统的"势函数"，就像力学中势能最小=稳定平衡，化学平衡 = G 最小。

## 四· 燃烧过程的热力学分析

一· 理论燃烧温度（绝热火焰温度）

1. 定义：燃料在绝热条件下完全燃烧，释放的热量全部用于加热生成物，所能达到的最高温度
2. 能量平衡（定压绝热）：H_Pr(T_ad) = H_Re(T₀)
   - 求解 T_ad → 需迭代（生成物比热容是温度的函数）

二· 影响理论燃烧温度的因素

1. 燃料热值 ↑ → T_ad ↑
2. 过量空气系数 ↑ → 生成物稀释 → T_ad ↓
3. 空气预热温度 ↑ → T_ad ↑
4. 燃烧产物离解（高温下 CO₂→CO+½O₂, H₂O→H₂+½O₂ 等吸热反应）→ T_ad 受限

三· 燃烧效率——第二定律视角

1. 燃烧过程的㶲损失是热力系统中最大的不可逆损失来源
2. 原因：化学能→热能的转换中，温度从理论上无限高（化学能品位 λ=1）骤降到燃烧温度（~2000K），品位损失巨大
3. 改进方向：燃料电池（直接化学能→电能，绕过燃烧）

> [!tip] 补充
> 燃烧是典型的"高开低走"——化学能品位 λ≈1，但燃烧后变为 2000K 的热能，λ≈0.7。这是热力系统最大的㶲损失环节。燃料电池避开了这个损失（λ 接近 0.8~0.9），所以效率极高。

## 速记卡 / 一句话总结

- 反应系统多一个自由度（组分），需引入化学势 μᵢ
- ==定容热效应 Q_V = ΔU；定压热效应 Q_p = ΔH==
- 理想气体：**Q_p − Q_V = Δn·RT**（反应前后摩尔数变化导致容积功差）
- 标准状况：**25°C, 1 atm**，标准定压热效应 Q_p⁰
- ==吉布斯自由能判据：ΔG_T,p < 0 → 自发；ΔG = 0 → 平衡==
- ΔG⁰ = −RT ln K_p，平衡常数决定反应方向
- 理论燃烧温度：绝热条件下燃料完全燃烧的生成物最高温度
- 燃烧是系统中最大㶲损失环节（化学能 λ≈1 骤降到 2000K λ≈0.7）
- 燃料电池避开燃烧不可逆损失，不受卡诺效率限制
