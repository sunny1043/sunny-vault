## 6-1 概述

一·热机的能量转换路线

- 1.化石燃料（煤、燃油、天然气）化学能 → 燃烧放热 → 工质热能
- 2.工质热力过程（状态变化）→ 热能 → 机械功
    - 内燃式热机：燃烧产物本身作为膨胀做功工质（如内燃机、燃气轮机）
    - 外燃式热机：燃烧产物将热能传给另一种工质（如水蒸气），后者膨胀做功

二·理论循环分析的意义

- 1.实际热力循环是多样的、不可逆的，但可近似为一系列简单可逆过程的封闭循环
- 2.理论循环分析给出该循环理论上能达到的**最佳效果**，为改进实际循环、减少不可逆损失提供比较标准
- 3.只要抽象合理，理论循环计算结果在理论和实用上都有价值

> [!note] 核心理解
> 本章核心方法：把实际开式、工质变化的循环 → 抽象为闭式、工质不变的**理论循环**，在 p-v 图和 T-s 图上分析。热效率是核心指标，影响效率的关键是各循环的**特性参数**（压缩比 ε、压升比 λ、预胀比 ρ、增压比 π、升温比 τ）。

## 6-2 活塞式内燃机的混合加热循环

### 实际循环 → 理论循环的简化

一·四冲程柴油机实际循环（示功图 7 个过程）

- 1.a→b：进气过程——压力略低于大气压
- 2.b→c：绝热压缩——温压同时升高
- 3.c→d：定容燃烧——喷油遇高温压缩空气，瞬间燃烧，活塞微动
- 4.d→e：定压燃烧——活塞右移，喷油持续燃烧至烧完
- 5.e→f：绝热膨胀——燃烧后气体膨胀做功
- 6.f→g：定容排气——排气阀开，压力骤降
- 7.g→a：排气冲程——废气排出，压力略高于大气压

二·理想化简化（开式 → 闭式循环）

- 1.废气与空气成分相差不大（**≈80%** 为 N₂），→ 近似假定工质成分不变
- 2.缸内燃烧 → 看作外部向工质加热；定容排气 → 看作定容冷却
- 3.进气/排气压力接近大气压 → 进气功与排气功抵消 → 认为既不进气也不排气
- 4.各实际过程理想化：
    - b→c（绝热压缩）→ 1→2 定熵压缩
    - c→d（定容燃烧）→ 2→3 定容加热
    - d→e（定压燃烧）→ 3→4 定压加热
    - e→f（绝热膨胀）→ 4→5 定熵膨胀
    - f→g（定容排气）→ 5→1 定容冷却

→ 得到闭式理想循环 **123451**，称为 ==混合加热循环==。

### 混合加热循环的特性参数

**定义**：三个无量纲参数唯一确定循环。

一·压缩比 ε （**ε = v₁/v₂**）

- 1.含义：燃烧前气体在气缸中被压缩的程度——比体积缩小的倍率
- 2.柴油机典型值：**13～20**

二·压升比 λ （**λ = p₃/p₂**）

- 1.含义：定容燃烧时气体压力升高的倍率

三·预胀比 ρ （**ρ = v₄/v₃**）

- 1.含义：定压燃烧时气体比体积增大的倍率
- 2.也叫"定压预胀比"或"cut-off ratio"

### 热效率公式推导

**设定**：工质为定比热容理想气体，比热比 γ₀ = c_p/c_v。

**步骤**：

（1）各过程吸/放热量

- 定容加热：q₁ᵥ = cᵥ(T₃ − T₂) ← 定容，cᵥ
- 定压加热：q₁ₚ = cₚ(T₄ − T₃) ← 定压，cₚ
- 定容放热：q₂ = cᵥ(T₅ − T₁) ← 定容，cᵥ

（2）热效率定义式

η_t = 1 − q₂ / q₁ = 1 − q₂ / (q₁ᵥ + q₁ₚ)

= 1 − cᵥ(T₅ − T₁) / [cᵥ(T₃ − T₂) + cₚ(T₄ − T₃)]

= 1 − (T₅ − T₁) / [(T₃ − T₂) + γ₀(T₄ − T₃)]

（3）用 ε, λ, ρ 表达各点温度

- 1→2 定熵：T₂ = T₁(v₁/v₂)^(γ₀−1) = T₁ε^(γ₀−1) ← 定熵过程 T·v^(γ−1) = const
- 2→3 定容：T₃ = T₂(p₃/p₂) = T₂λ = T₁ε^(γ₀−1)λ ← 定容 p∝T
- 3→4 定压：T₄ = T₃(v₄/v₃) = T₃ρ = T₁ε^(γ₀−1)λρ ← 定压 v∝T
- 4→5 定熵：T₅ = T₄(v₄/v₅)^(γ₀−1) = T₄(ρ/ε)^(γ₀−1) = T₁λρ^γ₀ ← v₄/v₅ = (v₄/v₃)(v₃/v₂)(v₂/v₁) = ρ·(1)·(1/ε)；代入 T₄ 表达式后化简

（4）代入化简

η_t = 1 − (T₅ − T₁) / [(T₃ − T₂) + γ₀(T₄ − T₃)]

= 1 − (T₁λρ^γ₀ − T₁) / [T₁ε^(γ₀−1)(λ−1) + γ₀·T₁ε^(γ₀−1)λ(ρ−1)]

```
              1       λρ^γ₀ − 1
η_t = 1 − ——————— · ————————————————————    （6-4）
            ε^(γ₀−1)  (λ−1) + γ₀λ(ρ−1)
```

### 参数对热效率的影响

一·压缩比 ε 的影响

- 1.λ 和 ρ 不变时，ε ↑ → η_t ↑
- 2.原因（T-s 图解释）：ε 增大 → 循环平均吸热温度 T_m₁ 升高，平均放热温度不变 → 热效率提高
- 3.==柴油机压缩比高（13～20），既为喷油自燃所需，也为高热效率==

二·压升比 λ 和预胀比 ρ 的影响

- 1.ε 不变时，提高 λ 并降低 ρ → η_t ↑
- 2.物理含义：使燃烧过程**更多在定容下进行、更少在定压下进行**
- 3.T-s 图面积法验证：λ 增大、ρ 减小 → 循环放热量占比减少 → η_t 提高

> [!note] 核心理解
> 混合加热循环可以看作奥托循环（纯定容加热）与狄塞尔循环（纯定压加热）的连续过渡。λ 越大 → 越接近奥托；ρ 越大 → 越接近狄塞尔。ε ↑ 永远提高 η_t，λ ↑ 提高 η_t，ρ ↑ 降低 η_t。本质：**平均吸热温度越高、平均放热温度越低，热效率越高**——这是卡诺定理在循环分析中的体现。

> [!tip] 补充
> 实际柴油机中，喷油开始后燃烧室内的燃烧过程是"先定容爆发、后定压持续"的混合模式——预混燃烧阶段（定容特征）+ 扩散燃烧阶段（定压特征）。因此混合加热循环比纯奥托或纯狄塞尔更贴近柴油机实际。

## 6-3 定容加热循环（奥托循环）和定压加热循环（狄塞尔循环）

### 6-3.1 定容加热循环（奥托循环 Otto Cycle）

**定义**：燃料与空气预混后在压缩终了用电火花点燃，燃烧极快 → 看作**定容加热**。预胀比 ρ = 1。

一·特例推导

- 1.混合加热循环中令 ρ = 1 → v₃ = v₄ → 状态 4 与状态 3 重合
- 2.代入式 (6-4)：ρ^γ₀ − 1 = 0，分母中 γ₀λ(ρ−1) = 0

```
η_t,v = 1 − 1 / ε^(γ₀−1)    （6-5）
```

- 3.==热效率仅取决于压缩比 ε 和比热比 γ₀==

二·p-v 图与 T-s 图

```
奥托循环 p-v 图：               奥托循环 T-s 图：

   p ↑                            T ↑
     |     3                          |      3
     |    /|                          |     /|
     |   / |                          |    / |
     |  /  |                          |   /  |
     | 2   |                          |  2   |
     |  \  |                          |   \  |
     |   \ |                          |    \ |
     |    \|                          |     \|
     |     1——4                       |      1——4
     +————————→ v                    +————————→ s

   1→2 定熵压缩                   同左
   2→3 定容加热
   3→4 定熵膨胀
   4→1 定容放热
```

三·局限性——爆燃

- 1.点燃式内燃机中压缩的是**燃料与空气混合物**
- 2.ε 过高 → 压缩终了 p、T 太高 → 引起不正常燃烧（**爆燃**）→ 降效率 + 损发动机
- 3.→ 点燃式发动机压缩比低：**5～9**（远低于柴油机 13～20）

> [!note] 核心理解
> 奥托循环是混合加热循环在 ρ → 1 的极限。热效率公式 η = 1 − 1/ε^(γ−1) 表明 **ε 是唯一可控变量**（γ 由工质决定）。因为汽油机被爆燃限制了 ε（5～9），所以奥托循环的**理论热效率天生上限**低于柴油机。以 ε=8, γ=1.4 计算：η_t ≈ 1 − 1/8^0.4 ≈ **0.565**。

### 6-3.2 定压加热循环（狄塞尔循环 Diesel Cycle）

**定义**：柴油机燃烧主要发生在活塞离开上止点后的行程中，一面燃烧一面膨胀，缸内压力基本不变 → 看作**定压加热**。压升比 λ = 1。

一·特例推导

- 1.混合加热循环中令 λ = 1 → p₃ = p₂ → 状态 3 与状态 2 重合
- 2.代入式 (6-4)：分母中 (λ−1) = 0, γ₀λ(ρ−1) = γ₀(ρ−1)

```
              1       ρ^γ₀ − 1
η_t,p = 1 − ——————— · ——————————    （6-6）
            ε^(γ₀−1)  γ₀(ρ − 1)
```

- 3.热效率取决于 ε 和 ρ 两个参数

二·p-v 图与 T-s 图

```
狄塞尔循环 p-v 图：            狄塞尔循环 T-s 图：

   p ↑                            T ↑
     |  2———3                       |   2———3
     |  |     \                     |   |     \
     |  |      \                    |   |      \
     |  |       \                   |   |       \
     |  |        4                  |   |        4
     |  |       /                   |   |       /
     |  |      /                    |   |      /
     |  |     /                     |   |     /
     |  1————                      |   1————
     +————————→ v                  +————————→ s

   1→2 定熵压缩
   2→3 定压加热
   3→4 定熵膨胀
   4→1 定容放热
```

三·参数影响

- 1.ρ 不变，ε ↑ → η_t,p ↑
- 2.ε 不变，ρ ↑（增加发动机负荷）→ η_t,p ↓
    - 原因：γ₀ > 1 时，ρ^γ₀ − 1 比 ρ − 1 增长快 → 分子增长快于分母 × ρ
- 3.当 λ = 1 时，从 η_t − ρ 曲线明显看到 η_t 随 ρ 增大而下降

> [!note] 核心理解
> 狄塞尔是混合加热循环在 λ → 1 的极限。公式中含有 ρ^γ₀ 项——由于 γ₀≈1.4 > 1，ρ 增大对效率的惩罚是"超线性"的。物理含义：定压加热意味着燃烧后期气体温度已降低但仍吸热 → 吸热平均温度被拉低 → 热效率下降。==增大负荷（增大喷油量 → ρ 增大）会降低狄塞尔循环热效率。==

> [!tip] 补充
> 现代柴油机的实际燃烧过程包含预混燃烧阶段（有短暂压力上升），并非纯定压。严格的"纯狄塞尔"只在大型低速船用柴油机中接近实现。因而混合加热循环才是柴油机最准确的理论模型。

## 6-4 活塞式内燃机各种循环的比较

比较需要**给定相同条件**——不同条件下结论可能完全相反。

### 比较条件一：进气状态、压缩比 ε、吸热量 q₁ 相同

一·T-s 图上的比较

- 1.三种循环在 T-s 图上吸热量相同（等面积）
- 2.定容加热循环（奥托）放热量最少 → ==η_t,v > η_t > η_t,p==

```
相同 ε/q₁ 下的比较（T-s 示意图）：

   T ↑
     |     3
     |    / \     奥托放热最少
     |   /   \    混合次之
     |  2     4   狄塞尔最多
     |   \    |
     |    \   |
     |     1——5
     +————————————→ s

   放热量面积：715'6'7 < 71567 < 715"6"7
```

二·结论

- 1.点燃式内燃机按定容加热循环工作最有利
- 2.压燃式内燃机按混合加热循环比纯狄塞尔有利；越接近定容加热 → 效率越高
- 3.⚠️ 但不能得出"汽油机效率高于柴油机"——因为压缩比相差悬殊（5～9 vs 13～20），不符合"相同 ε"条件

### 比较条件二：进气状态、最高温度 T_max、最高压力 p_max 相同

一·T-s 图上的比较

- 1.三种循环放热量相同（受限于相同的 T_max 和 p_max → 定容放热起点 5 固定）
- 2.定压加热循环吸热量最多 → ==η_t,p > η_t > η_t,v==

二·结论

- 1.热强度和机械强度受限时→ 采用定压加热循环可获得较高热效率
- 2.若近似认为汽油机和柴油机具有相同 T_max 和 p_max → 柴油机（压燃式）热效率更高
- 3.==实际情况确如此：柴油机压缩比高 → 热效率显著超过汽油机==

> [!note] 核心理解
> 两个比较条件揭示了"看似矛盾的结论"：条件一 η_v > η_p，条件二 η_p > η_v。核心差别在于**比较基准不同**——条件一固化了 ε 和 q₁（柴油机优势被"锁"住），条件二固化了 T_max 和 p_max（释放了柴油机高压比优势）。实际工程中条件二更接近真相——相同材料/冷却约束下，柴油机确实更高效。

## 6-5 燃气轮机装置的循环（布雷顿循环）

### 装置与循环概述

**定义**：燃气轮机装置由 ==压气机 + 燃烧室 + 燃气轮机== 三大部分组成，按定压加热循环（布雷顿循环 Brayton Cycle）工作。

一·装置流程

```
    空气 → 压气机(定熵压缩) → 燃烧室(定压加热) → 燃气轮机(定熵膨胀) → 废气排大气(定压冷却)
     1 ————————————————— 2 ———————————————— 3 ——————————————————— 4 —————————————→ 1
```

二·两个特性参数

- 1.增压比：**π = p₂/p₁**
- 2.升温比：**τ = T₃/T₁**

三·p-v 图与 T-s 图

```
布雷顿循环 p-v 图：              布雷顿循环 T-s 图：

   p ↑                              T ↑
     |   2 ——— 3                      |      3
     |  /        \                    |     / \
     | /          \                   |    /   \
     |/            \                  |   2     4
     |              \                 |  /       \
     |               4                | /         \
     |              /                 |/           \
     |             /                  |             1
     |            1                   +—————————————→ s
     +————————————————→ v

   1→2 定熵压缩       2→3 定压加热
   3→4 定熵膨胀       4→1 定压放热
```

### 热效率公式推导

**设定**：工质为定比热容理想气体。

（1）吸/放热量

- q₁ = cₚ(T₃ − T₂) ← 定压加热
- q₂ = cₚ(T₄ − T₁) ← 定压放热

（2）热效率

η_t,p = 1 − q₂/q₁ = 1 − (T₄ − T₁)/(T₃ − T₂)

（3）用 π 表达各点温度

- 1→2 定熵：T₂ = T₁(p₂/p₁)^((γ₀−1)/γ₀) = T₁π^((γ₀−1)/γ₀) ← 定熵过程 T∝p^((γ−1)/γ)
- 3→4 定熵：T₄ = T₃(p₄/p₃)^((γ₀−1)/γ₀) = T₃/π^((γ₀−1)/γ₀) ← p₄/p₃ = p₁/p₂ = 1/π

（4）代入化简

η_t,p = 1 − (T₃/π^((γ₀−1)/γ₀) − T₁) / (T₃ − T₁π^((γ₀−1)/γ₀))

= 1 − 1/π^((γ₀−1)/γ₀)

```
η_t,p = 1 − 1 / π^((γ₀−1)/γ₀)    （6-13）
```

> [!note] 核心理解
> ==布雷顿循环理论热效率仅取决于增压比 π，与升温比 τ 无关。== 这一结论与直觉相反——直觉认为 T₃ 越高效率越高。但 T₃ 增高的同时也提高了排气温度 T₄（二者同步比例），所以 τ 对 η_t 无影响。π 增大的物理本质：提高了循环平均吸热温度 T_m₁、降低了平均放热温度 T_m₂ → 逼近卡诺效率。

**缺陷**：上述是理想情况。实际中存在压气机和燃气轮机的不可逆损失 → 实际 η_t 既与 π 有关，也与 τ 有关，且存在**最佳增压比** π_opt 使 η_t 最大。

### 回热循环（Regenerative Brayton Cycle）

一·动机

- 1.燃气轮机排气温度 T₄ 通常 **高于** 压气机出口温度 T₂
- 2.→ 用回热器回收排气余热来预热压缩空气 → 节约燃料

二·理想完全回热

- 1.假定 T₂ = T_b，T_a = T₄（回热器两端温差为零）
- 2.定压加热 2→a 所需热量由定压冷却 4→b 放热量"内部"供给
- 3.→ 从外界吸热的过程变成 a→3（而非 2→3），放热变为 b→1（而非 4→1）
- 4.→ 提高了平均吸热温度、降低了平均放热温度 → η_t 提高

```
回热循环 T-s 图：

   T ↑
     |           3
     |          /|
     |   a————/ |
     |  /|   /  |
     | / |  /   |
     |2  | /    |
     | \ |/     |
     |  \|      |
     |   1———b——4
     +————————————→ s

   虚线 a—2 和 4—b 为回热器内换热过程
```

三·理想回热循环热效率

（1）η_t,r = 1 − q₂/q₁ = 1 − cₚ(T_b − T₁) / [cₚ(T₃ − T_a)]

（2）完全回热下：T_b = T₂ = T₁π^((γ₀−1)/γ₀)，T_a = T₄ = T₃/π^((γ₀−1)/γ₀)

（3）代入化简：

η_t,r = 1 − (T₁π^((γ₀−1)/γ₀) − T₁) / (T₃ − T₃/π^((γ₀−1)/γ₀))

= 1 − π^((γ₀−1)/γ₀)·(T₁ / T₃)

```
η_t,r = 1 − π^((γ₀−1)/γ₀) / τ    （6-14）
```

四·参数影响

- 1.τ ↑（提高 T₃ 或降低 T₁）→ η_t,r ↑
- 2.π ↓ → η_t,r ↑（与不回热循环相反！）
- 3.T-s 图解释：τ 不变时降 π → 平均吸热温度升、平均放热温度降 → η_t 升

> [!note] 核心理解
> 回热循环颠覆了简单布雷顿循环的直觉：==回热后，降低 π 反而提高效率==。因为 π 越低 → T₂ 越低 → 可回收的排气余热越多 → 外部需供的热量越少。但 π 过低会 → 循环净功大幅下降 → 功率密度恶化。工程上需在效率和功率间折中。

### 中间冷却 + 中间再热 + 回热

一·措施

- 1.分段压缩 + 中间冷却：降低压缩功
- 2.分段膨胀 + 中间再热：增加膨胀功
- 3.同时采用回热

二·理想效果

- 1.在相同总压比 π_total 和完全回热下，等效于多个相同压比的理想回热循环
- 2.热效率：η_t,complex = 1 − π_total^((γ₀−1)/γ₀) / τ  （6-15）
- 3.==在相同 π 和 τ 下，复杂回热循环 η_t > 简单回热循环 η_t==

三·实际考量

- 1.增加设备 → 增加投资和不可逆损失
- 2.→ 循环实际效率提高会打折扣
- 3.→ 是否采用复杂循环需**技术经济分析**后抉择

### 最佳增压比（考虑不可逆损失）

- 1.实际压气机效率 η_c 和燃气轮机效率 η_t 都 < 1
- 2.存在最佳增压比 π_opt 使循环**绝对内效率 η_i** 最高
- 3.课堂测验给出算例：η_c = η_t = **0.85**，τ = **3.8**，γ₀ = **1.4**
    - π_opt ≈ **10**，η_i,max ≈ **0.271**
- 4.升温比 τ 不高且 η_c、η_t 偏低时，增压比过高反而不利——甚至可能压气机消耗全部输出功率（净功为零）

> [!tip] 补充
> 最佳增压比 π_opt 的推导：令 ∂η_i/∂π = 0 求解。公式为 π_opt = (τ·η_c·η_t)^(γ₀/[2(γ₀−1)])。现代大型燃气轮机 π 可达 20～40，联合循环效率超 60%。

## 6-6 喷气发动机循环

**定义**：利用高温高压气体在喷管中加速时的**反作用力**推动飞行器。涡轮喷气发动机的两段压缩 + 两段膨胀最终构成布雷顿循环。

一·热力过程分解

- 1.1→5：扩压管中初步压缩（利用飞行速度的冲压效应）
- 2.5→2：压气机中继续压缩
- 3.2→3：燃烧室定压加热
- 4.3→6：燃气轮机中初步膨胀（做功供压气机用，w_T = w_C）
- 5.6→4：尾喷管中膨胀至大气压，高速喷出 → 产生推力

```
涡轮喷气发动机循环 T-s 图：

   T ↑
     |         3
     |        /|
     |       / |
     |      2  |
     |     /|  |6
     |    5 | /|
     |   /| |/ |
     |  / |/   4
     | /  |   /
     |1——————/
     +—————————→ s

   面积 d36cd = w_T（燃气轮机做功）
   面积 d25bd = w_C（压气机耗功）
   w_T = w_C
   面积 12341 = 循环净功 = 推力有效功
```

二·理论热效率

```
η_t,jet = 1 − 1 / π^((γ₀−1)/γ₀)    （6-16）
```

- 1.与简单布雷顿循环完全相同
- 2.π = p₂/p₁ = p₃/p₄（喷气发动机增压比）

三·推力产生原理

- 1.每 kg 气体在尾喷管获得的速度能 = 面积 c64ac
- 2.扩压管消耗的速度能 = 面积 01
- 3.二者之差 = 整个膨胀/压缩过程的面积差 = 循环功 = 推力有效功

> [!note] 核心理解
> 喷气发动机本质是**自带推进器的布雷顿循环**：燃烧室出来的高温燃气一部分做功驱动压气机（自持），剩余能量在尾喷管转为高速射流动能 → 反冲推力。整个循环可用 12341（定熵压缩→定压加热→定熵膨胀→定压冷却）统一描述，效率公式与地面燃气轮机一致。

## 6-7 活塞式热气发动机循环（斯特林循环）

**定义**：斯特林发动机（Stirling Engine）是**外燃式闭式循环**发动机，由 ==动力活塞 A + 配气活塞 B + 回热器 C== 组成。工质在封闭气缸内循环工作，不更换。

一·四个过程

- （1）1→2 定温压缩：活塞 B 停上止点，活塞 A 由下止点移向上止点，压缩腔有水冷 → 近似定温
- （2）2→3 定容加热：活塞 A 停上止点，活塞 B 下移，气体从压缩腔被赶到膨胀腔，流经回热器被加热 → 体积不变
- （3）3→4 定温膨胀：活塞 B 推活塞 A 下行，同时外界燃烧系统供热 → 定温膨胀做功（动力冲程）
- （4）4→1 定容冷却：活塞 A 停下止点，活塞 B 上移，气体从膨胀腔被赶回压缩腔，流经回热器时将热量传给回热器 → 体积不变

二·p-v 图与 T-s 图

```
斯特林循环 p-v 图：               斯特林循环 T-s 图：

   p ↑                              T ↑
     |  2                           |
     |  |\                          |  3——————4
     |  | \                         |  |      |
     |  |  \                        |  |      |
     |  |   \                       |  |      |
     | 1————3                       |  2——————1
     |  |   /                       |  |      |
     |  |  /                        +————————————→ s
     |  | /
     |  |/                         1→2 定温压缩
     |  4                           2→3 定容加热（回热器放热给气体）
     +————————→ v                  3→4 定温膨胀
                                    4→1 定容冷却（气体放热给回热器）
```

三·理论热效率

（1）斯特林循环 = 回热卡诺循环（两个定温 + 两个内部回热的定容）

（2）理论效率等于同温限下的卡诺效率

```
η_t,C = 1 − T₂ / T₃    （6-17）
```

- 1.==理论上达到该温度范围内最高循环热效率==
- 2.实际现代斯特林发动机 η ≈ **40%～50%**
- 3.优势：燃料品种不限（可用太阳能、生物质等外部热源）、工作稳定可靠
- 4.劣势：过程较慢 → 功率密度低，不可逆损失使实际 η 低于理论值

> [!note] 核心理解
> 斯特林循环的巧妙之处：回热器充当"内部蓄热器"——在定容冷却过程 4→1 中回收排气余热，在定容加热过程 2→3 中释放给新鲜工质。两个定容过程的吸/放热**内部抵消**，对外仅呈现两个定温过程 → 等效于卡诺循环。==这是唯一在实际中实现了"回热卡诺循环"的热机。==

> [!tip] 补充
> 斯特林发动机虽效率理论最高，但功率密度远低于内燃机。现代应用场景：太阳能热发电（碟式斯特林）、潜艇 AIP 系统（瑞典哥特兰级）、微型热电联产。近年因可再生能源和分布式能源需求，斯特林技术重新受到关注。

## 速记卡 / 一句话总结

- **1. 混合加热循环（萨巴特）**：η_t = 1 − (1/ε^(γ−1))·(λρ^γ−1)/[(λ−1)+γλ(ρ−1)]；ε ↑ → η ↑，λ ↑ → η ↑，ρ ↑ → η ↓
- **2. 奥托循环（定容加热，ρ=1）**：η_t,v = 1 − 1/ε^(γ−1)；仅取决于 ε；汽油机 ε=5～9（受爆燃限制）
- **3. 狄塞尔循环（定压加热，λ=1）**：η_t,p = 1 − (1/ε^(γ−1))·(ρ^γ−1)/[γ(ρ−1)]；柴油机 ε=13～20；增加负荷（ρ↑）→ η_t ↓
- **4. 三循环比较（相同 ε+q₁）**：η_t,v > η_t > η_t,p；**（相同 T_max+p_max）**：η_t,p > η_t > η_t,v；结论反转的原因——比较基准不同
- **5. 布雷顿循环（燃气轮机）**：η_t,p = 1 − 1/π^((γ−1)/γ)；理论效率仅与增压比 π=p₂/p₁ 有关，与升温比 τ=T₃/T₁ 无关
- **6. 回热布雷顿**：η_t,r = 1 − π^((γ−1)/γ)/τ；τ ↑ 或 π ↓ → η_t,r ↑（与简单循环相反）；回热逆转了 π 的影响方向
- **7. 中间冷却+中间再热+回热**：复杂循环在相同 π 和 τ 下效率更高，但增加设备和不可逆损失 → 需技术经济分析
- **8. 实际不可逆循环**存在最佳增压比 π_opt → 算例：η_c=η_t=0.85, τ=3.8 时，π_opt≈10, η_i,max≈0.271
- **9. 喷气发动机**：两段压缩+两段膨胀 = 布雷顿循环，η 公式同 (6-13)，尾喷管高速射流反冲 → 推力
- **10. 斯特林发动机**：外燃式闭式循环，回热卡诺循环，η_t,C = 1 − T₂/T₃（理论最高），实际 40%～50%；功率密度低但燃料不限
