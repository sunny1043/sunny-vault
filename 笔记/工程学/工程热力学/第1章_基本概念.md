## 1-1 热力系

### 热力系、外界与界面

一·**热力系的定义与选取**

**定义**：热力系是人为具体指定的热力学研究对象，是对各种热动设备的统一称谓。热力系可以是一群物体、一个物体或物体的某一部分；可大可小，但不能小到只包含少量分子以致不能遵守统计平均规律——热力学理论的正确性有赖于大量分子的统计平均。

- 1.**选取原则**：
  - （1）主要取决于**研究任务**（所要解决的问题）。例如：计算汽轮机功率 → 取蒸汽轮机或流过的蒸汽为热力系即可；研究全厂热效率 → 须将锅炉、汽轮机、水泵、凝汽器、回热加热器等全部纳入。
  - （2）与**分析方法**有关。同一问题，分析方法不同，热力系选取可能不同。例如分析工质吸热膨胀：可取封闭在气缸内的全部气体（界面变动），也可取固定空间内的气体（界面固定但质量减少）。
- 2.**外界**：热力系之外与热力系有**直接作用**的物体统称为外界。种类包括：热作用源、作功用源、物质作用源。热力学分析中，一般不考虑外界的变化。
- 3.**界面**：分隔热力系与外界的界面，控制好这个界面对分析至关重要。
  - （1）界面可以是**真实的**（气缸内壁、活塞内壁）或**假想的**（进口截面、出口截面）
  - （2）界面可以是**固定的**或**变动的**（活塞移动时界面变化）
  - （3）界面的核心功能：能质可传——能量和物质可以通过界面交换，不是隔绝而是**受控**

```
      ┌─────────────────┐
      │                  │
外界  │    热力系        │  外界
      │   (System)       │
      │                  │
      └──────边界────────┘
         ↕ 能量流  ↕ 质量流
```

二·**热力系的分类**

- 1.**按内部特征**分：
  - （1）**单元系**——由单一的化学成分组成
  - （2）**多元系**——由多种化学成分组成
  - （3）**单相系**——由单一的相（如气相或液相）组成
  - （4）**复相系**——由多种相（如气-液两相、气-液-固三相）组成
  - （5）**均匀系**——各部分性质均匀一致
  - （6）**非均匀系**——各部分性质不均匀
- 2.**按与外界相互作用**分：
  - （1）**闭口系**——与外界无物质交换（控制质量，Control Mass）
  - （2）**开口系**——与外界有物质交换（控制体积，Control Volume）
  - （3）**绝热系**——与外界无热量交换
  - （4）**孤立系**——与外界无**任何**相互作用（既无物质交换也无能量交换）

> [!note] 核心理解
> 物理上：热力系就像「画一个圈」——圈内是我要研究的，圈外不管。圈的画法取决于我想回答什么问题。数学上：热力系的分类本质是在界定系统的自由度——闭口系质量守恒减少一个自由度，绝热系再约束一个能量交换自由度，孤立系则完全封闭。

- 3.**特殊热力系**（补，课件新增）：
  - 热源（库）、冷源（库）——具有**无限大热容**的热力系，吸收或放出有限热量时自身温度不变。这是一个理想化模型。

**易错 / 边界**：
- 选取热力系时，一旦界面确定，内外必须「泾渭分明、内外有别」，不能混淆——这是分析不混乱的前提。
- 「孤立系」要求既无物质交换也无能量交换，现实中没有真正的孤立系（除非把整个宇宙取为热力系），是理论分析的极限情况。
- 热力系不能太小——小到只含少量分子时，统计平均规律失效，热力学不适用。

## 1-2 状态和状态参数

### 状态与状态参数总论

**定义**：**状态**是热力系在指定瞬间所呈现的全部宏观性质的总称。从不同方面描述这种宏观状态的物理量就是**状态参数**。工程热力学中常用的状态参数有 **6 个**：压力（$p$）、比体积（$v$）、温度（$T$）、热力学能（$U$）、焓（$H$）、熵（$S$）。其中压力、比体积、温度可直接测量，称为**基本状态参数**。

**状态参数的共有性质**（硬核）：
- ==状态参数是热力系状态的单值函数（点函数）==——值仅取决于给定状态，与达到这一状态的途径无关。
- 数学表现：微分是**全微分**，全微分的循环积分等于零。

$$\oint dp = 0,\quad \oint dv = 0,\quad \oint dT = 0,\quad \oint du = 0,\quad \oint dh = 0,\quad \oint ds = 0$$

> [!tip] 补充
> 循环积分为零 = 绕任意闭合路径回到原状态时，状态参数的净变化为零——这是「状态量」区别于「过程量」的数学判据。

---

### 状态参数一：压力 $p$

一·**定义与微观解释**

- 1.**宏观定义**：压力是单位面积上承受的**垂直作用力**：

$$p = \frac{F}{A}$$

  式中 $F$——垂直作用力（N）；$A$——面积（m²）。

- 2.**微观解释**：气体的压力是组成气体的大量分子在紊乱热运动中**对容器壁频繁碰撞**的总体效果。分子数越多、运动越剧烈 → 压力越大。

二·**压力的种类与换算关系**

```
  绝对压力 p ── 气体的真正压力
       │
       ├── p > p_b（大气压）时：
       │     表压力 p_g = p - p_b  →  p = p_b + p_g
       │
       └── p < p_b（大气压）时：
             真空度 p_v = p_b - p  →  p = p_b - p_v
```

- 1.**绝对压力 $p$**——气体的真正压力，是热力学计算中使用的压力。
- 2.**大气压力 $p_b$**——当时当地的大气压力，用气压计测定。
- 3.**表压力 $p_g$**——压力计读数，$p_g = p - p_b$（$p > p_b$ 时）。
- 4.**真空度 $p_v$**——真空计读数，$p_v = p_b - p$（$p < p_b$ 时）。

> [!note] 核心理解
> 物理上：测量仪表「看不见」绝对压力，只能测差值——因为仪表本身浸在大气环境中，大气压力同时作用在仪表两侧，差压才是净驱动力。数学上：p = p_b + p_g 或 p = p_b − p_v，关键是要记住「加表压、减真空」。如果大气压力 p_b 变化，即使绝对压力不变，表压力/真空度读数也会变。

三·**U 形管压力计换算**

- 用 U 形管通过液柱高度差测定表压力（或真空度）时：

$$p_g \;(\text{或 } p_v) = \rho g \Delta z$$

  式中 $\rho$——液体密度；$g$——重力加速度；$\Delta z$——液柱高度差。

**公式与推导**：
- （1）液柱底部与顶部的压力差来自液柱重力：$\Delta p = \rho g \Delta z$ ← 流体静力学基本方程
- （2）U 形管一端通被测气体、一端通大气 → 液柱高度差反映的就是 $p_g$ 或 $p_v$。

四·**压力单位**

- SI 单位：**Pa**（帕），$1\,\text{Pa} = 1\,\text{N/m}^2$。
- 工程常用：**MPa**（兆帕），$1\,\text{MPa} = 10^6\,\text{Pa}$。
- 关键换算（**数字必须记住**）：
  - $1\,\text{atm} = \mathbf{760}\,\text{mmHg} = \mathbf{1.03323}\,\text{at} = \mathbf{101325}\,\text{Pa} = \mathbf{0.101325}\,\text{MPa}$
  - $1\,\text{at} = 1\,\text{kgf/cm}^2 = \mathbf{98066.5}\,\text{Pa} = \mathbf{0.0980665}\,\text{MPa}$
  - $1\,\text{mmHg} = \mathbf{133.3224}\,\text{Pa}$
  - $1\,\text{mmH}_2\text{O} = \mathbf{9.80665}\,\text{Pa}$

### 状态参数二：比体积 $v$

**定义**：比体积是**单位质量的物质所占有的体积**：

$$v = \frac{V}{m}$$

式中 $v$——比体积（m³/kg）；$V$——体积（m³）；$m$——质量（kg）。

- 比体积的倒数称为**密度** $\rho$：$\rho = \dfrac{1}{v} = \dfrac{m}{V}$（单位：kg/m³）。
- 比体积和密度不是两个相互独立的状态参数——给出一个就意味着给出了另一个。

> [!note] 核心理解
> 物理上：比体积反映物质的「疏松程度」——同样质量占多大空间。数学上：v 和 ρ 是倒数关系，自由度只有 1 个，不能作为两个独立参数来定状态。

---

### 状态参数三：温度 $T$

一·**定义**

温度表示物体的**冷热程度**。对于气体，温度可以用**分子平均移动能**的大小来表示：

$$\frac{1}{2}m\bar{c}^2 = \frac{3}{2}kT$$

式中 $m$——分子质量；$\bar{c}$——分子的均方根移动速度；$k$——**玻尔兹曼常量**，$k = \mathbf{1.380658 \times 10^{-23}}\,\text{J/K}$；$T$——热力学温度（K）。

- 温度本质是分子热运动剧烈程度的**宏观度量**。温度越高 → 分子平均动能越大 → 运动越剧烈。

二·**温标**

- 1.**热力学温标**（开尔文温标 / 绝对温标）：用 $T$ 表示，单位 **K**（开）。
- 2.**摄氏温标**：用 $t$ 表示，单位 **℃**（摄氏度）。
- 3.**换算关系**：

$$t = T - T_0,\quad T_0 = \mathbf{273.15}\,\text{K}$$

  - 摄氏温标的每 **1℃** 和开尔文温标的每 **1K** 是相等的——只是零点不同。
  - 摄氏温标的零点比开尔文温标的零点高出 **273.15K**。

> [!note] 核心理解
> 物理上：温度表征微观粒子热运动的剧烈程度——T 直接正比于分子平均动能。绝对零度 T = 0 K 对应分子动能的理论零点。数学上：t 和 T 之间是线性平移变换，差值为常数 273.15，所以温差 ΔT = Δt 在两个温标下数值相同。

---

### 状态参数四：热力学能 $U$

**定义**：热力学能是组成热力系的大量微观粒子本身具有的能量，**不包括**热力系宏观运动的能量和外场（如重力场）作用的能量。热力学能包括：
- （1）**分子的动能**——分子的移动能、转动能和分子内部的振动能
- （2）**分子力所形成的位能**
- （3）化学能和原子能——在热能与机械能转换中一般不涉及化学变化和核反应，工程热力学中**通常只考虑前两者**

**比热力学能**：$u = \dfrac{U}{m}$，或 $U = mu$

- 单位：热力学能 $U$——**J**（焦耳）；比热力学能 $u$——**J/kg**。
- 工程单位制：$U$——kcal；$u$——kcal/kg。$1\,\text{kcal} = \mathbf{4186.8}\,\text{J} = \mathbf{4.1868}\,\text{kJ}$。

> [!note] 核心理解
> 物理上：热力学能是系统「内部」储存的所有微观能量——分子振转动的动能 + 分子间相互作用的位能。宏观静止的物体仍有热力学能。数学上：U 是状态函数，∮ dU = 0——不管经历了什么过程，只要回到原状态，热力学能恢复原值。

---

### 状态参数五：焓 $H$

**定义**：焓是一个**组合的状态参数**：

$$H = U + pV$$

式中 $H$——焓（J）；$U$——热力学能（J）；$p$——压力（Pa）；$V$——体积（m³）。

**比焓**：$h = \dfrac{H}{m} = u + pv$，或 $H = mh$

**公式推导与理解**：
- （1）定义式 $H = U + pV$ ← 由热力学能和 $pV$ 乘积组合而成
- （2）$pV$ 的物理意义：推动流体流动所需的**流动功**（后续章节详讲） ← 焓 = 内能 + 流动功
- （3）因为 $U$、$p$、$V$ 都是状态参数，所以 $H = U + pV$ 也是状态参数 → $\oint dH = 0$

> [!tip] 补充
> 焓在工程热力学中的重要性源于**开口系能量方程**——流动工质带入/带出系统的能量正是用焓来表示的。为什么不用内能？因为流体进出控制体时，除了携带自身的内能 U，还要克服界面压力做功 pV，焓恰好把两者打包为一个量。这是焓被「发明」出来的物理动机。

- 单位：与热力学能相同——**J**；比焓 **J/kg**。
- 工程单位制中 $pV$ 乘积的单位是功 $\text{kgf·m}$，与热量单位 kcal 之间换算：
  $$H = U + A\,pV,\quad h = u + A\,pv$$
  式中 $A = \dfrac{1}{426.936}\,\text{kcal/(kgf·m)}$ 称为**功的热当量**。（SI 制中 $A=1$，因为功和热量统一用 J 度量）

---

### 状态参数六：熵 $S$

**定义**：熵是一个**导出的状态参数**。对简单可压缩均匀系（只有两个独立变量的均匀热力系），它可以由其他状态参数导出。

$$dS = \frac{dU + p\,dV}{T}$$

**比熵**：$ds = \dfrac{du + p\,dv}{T}$

$$S = \int \frac{dU + p\,dV}{T},\quad s = \int \frac{du + p\,dv}{T}$$

**公式推导与理解**：
- （1）定义式 $dS = \dfrac{dU + p\,dV}{T}$ ← 可逆过程的熵变定义，来源于热力学第二定律（第 4 章详讲）
- （2）分子 $dU + p\,dV$ 是可逆过程中系统吸收的**微元热量** ← 热力学第一定律：$dQ = dU + p\,dV$
- （3）∴ $dS = \dfrac{dQ_{\text{rev}}}{T}$——可逆过程中熵变 = 吸热量除以温度
- （4）因为 $U$、$V$ 是状态参数，$p$、$T$ 也是状态参数，所以 $S$ 也是状态参数 → $\oint dS = 0$

> [!note] 核心理解
> 物理上：熵是系统「无序度」的度量，也是能量「品质」的度量——高温热量的品质高于低温热量。dS = dQ/T 意味着同样热量在低温下引起的熵增更大。数学上：1/T 是 dQ_rev 的积分因子，将非全微分的 dQ 转化为全微分的 dS——这是熵作为状态函数的数学保证。

- 单位：$S$——**J/K**；$s$——**J/(kg·K)**。
- 工程单位制：kcal/K；kcal/(kg·K)。

> [!tip] 补充
> 本章只给定义式，熵的完整物理意义和克劳修斯不等式等在第 4-3 节展开。第一章记住：熵是状态参数，∮ dS = 0，且对可逆过程 dS = dQ/T 即可。

---

### 状态参数小结

- 状态参数与相应比参数的关系 $X = m x$（$x = X/m$）**只对均匀系成立**——非均匀系各点参数不同，不能简单地用一个比参数代表整个系统。
- ==6 个状态参数中，$p$、$v$、$T$ 可直接测量（基本状态参数），$U$、$H$、$S$ 不能直接测量（导出状态参数）==。
- ==状态参数的核心判据：$\oint dX = 0$——绕任意闭合回路积分等于零==。

## 1-3 平衡状态

一·**平衡状态的定义**

**定义**：平衡状态是热力系在**没有外界作用**的情况下宏观性质**不随时间变化**的状态。外界作用指热力系与外界的能量交换和物质交换，不包括恒定的外场（如重力场）作用。

- 1.**必要条件**：
  - （1）**力平衡**——热力系内部压力处处均匀一致
  - （2）**热平衡**——热力系内部温度处处均匀一致
  - （3）**相平衡**（复相系）——各相之间达到动态平衡
- 2.**自发趋向**：任何热力系，如果原来处于非平衡状态（内部存在不平衡势：温差、压差等），内部必然会**自发地**进行一个变化过程。当不平衡势逐渐消失而内部不再发生变化时，就达到了平衡状态。

二·**平衡状态的关键辨析**（重点）

- 1.**平衡状态 vs 稳定状态**：
  - （1）**平衡状态**——宏观性质不随时间变化 + 没有外界作用 → 宏观**静态**
  - （2）**稳定状态**——宏观性质不随时间变化 + 定有外界作用 → 宏观**动态**
  - → ==平衡必定是稳定的，稳定未必平衡==（例如：稳定导热过程中，各处温度不随时间变化，但有外界持续供热，不是平衡态）
- 2.**平衡状态 vs 均匀状态**：
  - （1）**平衡状态**——强调宏观性质的**时间特性**：各时刻宏观性质一样
  - （2）**均匀状态**——强调宏观性质的**空间特性**：各空间位置宏观性质一样
  - → ==平衡状态未必是均匀的，均匀状态未必是平衡的==（例如：重力场中静止气体柱，力平衡、热平衡满足→平衡态；但密度、压力沿高度不均匀→非均匀）

三·**平衡状态的实例与意义**

- 1.**实例**：
  - （1）温差驱动传热直到温度一致 → 热平衡（图 1-4）
  - （2）压差驱动气体转移直到压力一致 → 力平衡（图 1-5）
  - （3）密闭容器中水与水蒸气共存达到动态平衡 → 气-液两相平衡（图 1-6，压力和温度均匀一致，但比体积等在两相间不连续）
- 2.**引入平衡状态的目的和意义**：
  - （1）整个热力系的平衡状态可以用**一个点**来代表 → 为用数学上的几何点代表热力系状态奠定了物理基础
  - （2）平衡状态的连续过渡可以运用**微积分**等高等数学工具进行分析计算

四·**热力学第零定律**

如果两个热力学系统分别与第三个热力学系统处于热平衡，则它们彼此也必定处于热平衡。→ 这是温度概念可以定义和温度测量可行的逻辑基础。

```
  A ←──热平衡──→ C
                    │
  B ←──热平衡──→ C
                    │
  ⇒ A 与 B 也处于热平衡
```

> [!note] 核心理解
> 物理上：平衡状态是热力学分析的「参照系」——只有平衡态才能用有限个参数完全描述。数学上：平衡态将无限维的微观状态压缩到有限维的宏观参数空间，使得微积分工具可用。这是热力学能从「蒸汽机经验」变成「严密科学」的关键一步。

**易错 / 边界**：
- 处于气-液两相平衡的流体，压力和温度均匀一致（满足力平衡、热平衡），但比体积在两相间不连续——平衡不等于均匀。
- 如果考虑重力影响，流体的压力及气体的密度和压力沿高度有所差别。但如果上下高度相差不大，这种差别可忽略（特别是气体）。

## 1-4 状态方程和状态参数坐标图

一·**独立状态参数与自由度**

- 1.对于一个和外界只有热能和机械能交换的**简单热力系**（两个自由度），只要给出**两个相互独立**的状态参数就可以规定它的平衡状态。
- 2.**相互独立**的含义：其中一个不能仅仅是另一个的函数。例如比体积和密度就不是相互独立的（$v = 1/\rho$）——给出 $v$ 就意味着给出了 $\rho$。
- 3.给出两个独立状态参数后，其他所有状态参数也随之完全确定 → 存在单值函数关系。

二·**状态方程**

**定义**：以压力和温度为独立变量时，比体积表达为 $p$ 和 $T$ 的函数 $v = f(p, T)$，这一关系称为**状态方程**。它建立了三个可直接测量的基本状态参数（$p$、$v$、$T$）之间的关系。

$$v = f(p, T)$$

也可以写为隐函数形式：

$$F(p, v, T) = 0$$

- 同理，其他状态参数也可表达为 $p$ 和 $T$ 的函数：

$$u = f_1(p, T),\quad h = f_2(p, T),\quad s = f_3(p, T)$$

> [!note] 核心理解
> 物理上：状态方程是对「物质热力行为」的数学编码——不同物质有不同的 f(p, T)，这正是理想气体 pv = RT 和实际气体复杂状态方程的区别所在。数学上：F(p, v, T) = 0 在 p-v-T 三维空间中定义了一个曲面，系统所有可能的平衡状态都落在这个曲面上。

三·**状态参数坐标图**

- 1.任意两个相互独立的状态参数构成的平面坐标系中，**任意一点**相应于热力系的某一**平衡状态**。

```
      p
      │
      │    ·状态1
      │
      │          ·状态2
      │
      └────────────── v
```

- 2.一个平衡状态 → 一个点；一个内平衡过程 → 一条连续曲线。
- 3.**不平衡状态无法表示**在这样的坐标系中——因为各部分状态参数不一致，无法用一个点代表。但如果各部分性质差异不悬殊，可以用各部分的**平均值**近似表示（此时不能反映实际的不平衡程度和分布状况）。

> [!tip] 补充
> 状态参数坐标图是热力学分析的核心可视化工具。最常用的 p-v 图和 T-s 图将在后续各章大量出现——过程曲线下的面积分别代表功和热量。

## 1-5 过程和循环

一·**过程的定义**

**定义**：过程是指热力系从**一个状态向另一个状态变化**时所经历的全部状态的总和。

二·**内平衡过程（准静态过程）**——本章最核心的过程概念

- 1.**定义**：热力系从一个平衡状态**连续经历一系列（无数个）平衡的中间状态**过渡到另一个平衡状态，这样的过程称为**内平衡过程**（也称准平衡过程、准静态过程）。
- 2.**核心矛盾与统一**：
  - （1）「平衡」意味着宏观静止
  - （2）「过程」意味着变化、意味着平衡被破坏
  - （3）→ ==内平衡过程把「平衡」和「过程」这对矛盾在特定条件下统一了起来==
- 3.**成立条件**：要求外界对热力系的作用**必须缓慢到足以使热力系内部能及时恢复**被不断破坏的平衡。→ 准静态 = 非常接近静态的过程。

```
       p
       │   ╭──状态1
       │  ╱
       │ ╱  连续曲线 = 内平衡过程
       │╱
       │──────状态2
       └────────────── v

  (内不平衡过程严格不能这样表示，
   初终状态平衡且中间不平衡程度
   较小时，可近似用虚线表示)
```

- 4.**图示特征**：内平衡过程在状态参数坐标图中表示为**一条连续的曲线**。内不平衡过程严格不能这样表示。

三·**循环**

**定义**：循环就是**封闭的过程**——热力系从某一状态开始，经过一系列中间状态后，**又回到原来状态**。作为过程的一种特例，循环也有内平衡循环和内不平衡循环的区分。

```
       p
       │    ╭────╮
       │   ╱      ╲
       │  │   循环  │
       │   ╲      ╱
       │    ╰────╯
       └────────────── v
```

> [!note] 核心理解
> 物理上：内平衡过程是一个理想化模型——现实过程都是不平衡的，但当过程进行得足够慢（弛豫时间 << 外界作用时间），偏差可忽略。数学上：内平衡过程在状态空间中是连续曲线，意味着每一步都可以做微积分——功 ∫ p dV 和热量 ∫ T dS 的计算才有意义。循环的封闭性意味着：绕一圈后所有状态参数恢复原值，∮ dX = 0。

**易错 / 边界**：
- 「内平衡」强调热力系**内部**一直保持平衡，不强调与外界是否平衡（外界可以有温差、压差）。
- 实际工程过程（如内燃机气缸中的燃烧膨胀）都是不平衡过程——但在很多情况下用内平衡过程近似，误差可接受。

## 1-6 功和热量

一·**功和热量的定义**

- 1.**功（$W$）**：热力系通过界面和外界进行的**机械能**的交换量，简称功。
- 2.**热量（$Q$）**：热力系通过界面和外界进行的**热能**的交换量，简称热量。

二·**过程量 vs 状态量**——本章最重要的概念区分

**定义与辨析**：

- 1.**功和热量都是过程量**——与热力系的**状态变化（即过程）**联系在一起。
  - （1）不能说「热力系在某一状态下有多少功、多少热量」
  - （2）只能说「热力系在某一过程中对外界做出了/从外界获得了多少功，从外界吸收了/向外界放出了多少热量」
  - （3）功和热量的数值大小**与所经历的具体过程路线有关**
- 2.**六个状态参数（$T, p, v, U, H, S$）都是状态量**——是储存量，仅取决于状态本身。

三·**符号规定**

- 1.**功 $W$**：热力系对外界做功为**正**（$W > 0$），外界对热力系做功为**负**（$W < 0$）
- 2.**热量 $Q$**：热力系从外界吸热为**正**（$Q > 0$），热力系向外界放热为**负**（$Q < 0$）
- 3.单位质量：功用 $w$ 表示，热量用 $q$ 表示

四·**单位**

- SI 单位制：
  - $W$、$Q$——**J** 或 **kJ**
  - $w$、$q$——**J/kg** 或 **kJ/kg**
- 工程单位制：
  - $W$——**kgf·m**；$Q$——**kcal**
  - $w$——kgf·m/kg；$q$——kcal/kg

五·**状态量与过程量的联系**

- ==过程量的传递必然引起状态量的变化，状态量的变化未必引起过程量的传递==。
- 例如：孤立系内部进行的过程（内部发生状态变化，但与外界无 $W$、$Q$ 交换）。

```
      状态量（储存量）              过程量（交换量）
  ┌─────────────────────┐    ┌─────────────────────┐
  │  T, p, v, U, H, S   │    │     W, Q            │
  │  · 是状态的函数      │    │     · 是过程的函数    │
  │  · ∮dX = 0          │◄──►│     · 与路径有关      │
  │  · 有「多少」        │    │     · 只有「传递了多少」│
  └─────────────────────┘    └─────────────────────┘
         ↕ 过程量传递 → 状态量变化
         ↕ 状态量变化不一定伴随与外界的过程量传递
```

> [!note] 核心理解
> 物理上：状态量是系统的「存款余额」——只看当下有多少；过程量是「转账记录」——只看某段时期转了多少钱，不能说账户在某时刻「有」一笔转账。功和热量是能量传递的两种形式：功 = 有序能量传递（宏观力 × 宏观位移），热量 = 无序能量传递（微观粒子碰撞 × 温差驱动）。数学上：状态量 = 全微分（dU），过程量 = 非全微分（δW、δQ，用 δ 区别于 d，后续章节详讲）。

**易错 / 边界**：
- 功和热量只在**越过界面**时才有意义——系统内部的能量转换（如摩擦使机械能变成热能）不是功也不是热量，而是内部能量形式的转化。
- 正负号规定是热力学的「惯例」——和某些力学教材的符号约定不同（力学常以外界对系统做功为正），务必统一使用热力学约定。

## 速记卡 / 一句话总结

- **热力系** = 人为画圈的研究对象；界面 = 圈线，管控能量/物质交换；选取取决于研究任务
- **6 个状态参数**：$p$（压力）、$v$（比体积）、$T$（温度）→ 可直接测量；$U$（热力学能）、$H = U + pV$（焓）、$S = \int dQ_{\text{rev}}/T$（熵）→ 导出参数
- ==状态参数是点函数，$\oint dX = 0$==——值与路径无关，循环一圈恢复原值
- **绝对压力** $p = p_b + p_g = p_b - p_v$；测量仪表只能读差值，必须加/减大气压力
- $1\,\text{atm} = \mathbf{101325}\,\text{Pa}$；$1\,\text{at} = \mathbf{98066.5}\,\text{Pa}$；$1\,\text{mmHg} = \mathbf{133.3224}\,\text{Pa}$
- $t = T - \mathbf{273.15}$——摄氏与开尔文温差数值相等
- **平衡状态** = 无外界作用下宏观性质不随时间变；内平衡过程 = 破坏平衡的速度足够慢 → 每一步都是平衡态 → 可用微积分
- ==平衡必定稳定，稳定未必平衡；平衡未必均匀，均匀未必平衡==
- **状态方程**：$F(p, v, T) = 0$——给定两个独立参数，所有状态参数全确定
- ==功 $W$ 和热量 $Q$ 是过程量，不是状态量==——只能说「传递了多少」，不能说「有多少」
- 符号约定：对外做功 $W > 0$（正），从外界吸热 $Q > 0$（正）
- $1\,\text{kcal} = \mathbf{4186.8}\,\text{J}$；玻尔兹曼常量 $k = \mathbf{1.380658 \times 10^{-23}}\,\text{J/K}$
