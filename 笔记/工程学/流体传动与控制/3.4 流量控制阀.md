## 一·节流阀

一·**定义**：通过改变节流口的通流截面积来调节液阻大小，从而控制流量，进而控制执行元件运动速度。

二·**流量公式（节流孔口方程）**——核心公式

  $$ Q = C_d \, A \, \sqrt{\frac{2\Delta p}{\rho}} $$

  其中：
  - $Q$ = 通过节流口的流量（m³/s）
  - $C_d$ = 流量系数（≈ **0.60~0.80**，取决于孔口形状和雷诺数）
  - $A$ = 节流口通流面积（m²）
  - $\Delta p$ = 节流口前后压差（Pa）
  - $\rho$ = 油液密度（kg/m³）

  > [!tip] 补充
  > **推导**（从伯努利方程出发）：
  > （1）取节流孔上游截面 1 和孔口最小收缩截面 2（缩脉）之间的流线：
  > $$ p_1 + \frac{1}{2}\rho v_1^2 = p_2 + \frac{1}{2}\rho v_2^2 $$
  > ← 理想伯努利（忽略位能差、忽略孔口段摩擦损失）
  > （2）因 $A_1 \gg A_2$，$v_1 \ll v_2$ → 忽略 $v_1$：
  > $$ v_2 = \sqrt{\frac{2(p_1-p_2)}{\rho}} = \sqrt{\frac{2\Delta p}{\rho}} $$
  > （3）理论流量 $Q_{th} = A_2 v_2$；实际因缩脉收缩（面积小于孔口 A）、局部损失 → 引入 $C_d$：
  > $$ Q = C_d \, A \, \sqrt{\frac{2\Delta p}{\rho}} $$

三·**节流口形式（按 $C_d$ 的雷诺数敏感性）**

  | 类型 | 特点 | $Q \propto \Delta p^n$ ~ n 值 | 适用 |
  |-----|------|----------------------------|------|
  | 薄壁孔（L/d < 0.5） | 流态：收缩–扩张，摩擦损失小；$C_d$ ≈ 常数（Re>10³后） | n=0.5，==流量与黏度无关==，稳定性好 | 调速阀中的节流口 |
  | 细长孔（L/d > 4） | 泊肃叶流公式 $Q=\pi d^4\Delta p/(128\mu L)$ | n=1，**流量与黏度 μ 成正比**⇒受温度影响大 | 阻尼孔 |
  | 缝隙 | $Q= \frac{b h^3 \Delta p}{12\mu L}$（平行平板缝隙流） | n=1，**∝ h³** 非常敏感 | 阀口泄漏分析 |

  > [!tip] 补充
  > **核心区别**：薄壁孔中节流效果由惯性力决定 → Cd ≈ const → Q 只受 Δp 和 A 影响、不受油温变化影响（μ 无关）；细长孔中液体摩擦主导 → Q ∝ 1/μ → 油温升高→μ降低→流量变大→速度不稳。
  > 所以**调速用的节流阀节流口应取薄壁孔形式**——保证速度对温度不敏感。

四·**节流阀调速的两个缺陷**：
  1.流量随负载变化——负载变大→Δp 变→Q 变→速度不稳
  2.发热——节流损失（Δp × Q）转化为热量→油温升高→劣化油品+泄漏

## 二·调速阀

一·**定义**：节流阀 + **定差减压阀**串联组合；核心思想是用减压阀来"稳压"——保持节流阀前后的压差 Δp_th 恒定，从而保持流量 Q 不随负载变化，实现**速度刚性**。

二·**结构**：定差减压阀的阀芯与节流阀的入口/出口之间有压力反馈通道。
  - 减压阀入口接系统压力 p₁
  - 减压阀出口（即节流阀入口）p₂
  - 节流阀出口 p₃（接执行元件，随负载变化）

三·**工作原理——压差恒定推导**：

  （1）定差减压阀阀芯受力平衡：
  $$ p_2 A + k x_0 = p_3 A + k x $$
  其中 A = 阀芯有效受压面积，k = 减压阀弹簧刚度，x₀ = 初始压缩量，x = 当前压缩量。

  （2）在正常工作范围内（减压阀阀芯开度适当），弹簧力变化 k(x-x₀) 很小 ≈ 0 → 近似有：
  $$ p_2 - p_3 = \frac{k(x_0 - x)}{A} \approx const $$
  ← 数学近似：弹簧变形量变化较小→弹簧力变化远小于液压推力

  （3）即：**节流阀前后压差 $\Delta p_{th} \approx 常数$**（与负载无关）。

  （4）代入节流阀流量公式：
  $$ Q = C_d A \sqrt{\frac{2\Delta p_{th}}{\rho}} = C_d A \sqrt{\frac{2\cdot const}{\rho}} \quad \Rightarrow \quad Q \propto A $$

  → ==流量 Q 只取决于节流口开度 A，与负载压力变化无关== → 无论负载怎么变，速度都稳定。

四·**优点**：
  - **速度刚性大**——负载变化几乎不影响速度
  - 适用于需要精确调速的场合（机床进给、注塑机注射缸等）

五·**缺点**：
  - 仍有节流损失 → 发热（尽管速度稳了但能量效率并不高）
  - 成本高于普通节流阀
  - 存在最小稳定流量（太低时会因弹簧迟滞导致不稳定）

## 三·分流集流阀

一·**定义**：将一路油液等分（或按比例）成多路输出（分流阀），或将多路油液汇集为一路（集流阀）。在流量分配/同步回路中起关键作用。

二·**结构原理**：
  基于固定液阻的压差平衡原理——在两个节流片之间靠压力补偿达到等量或比例分配。负载变化时阀芯自动微调各支路液阻，保持支路流量比例不变。

三·**应用**：
  - 多缸同步回路（如两个液压缸同步升降——分流集流阀保证两缸流量相等等 → 速度相等）
  - 按比例分配（如 2:1 比例给两个分支）

四·**局限性**：
  - 分流精度在低速/低负载时下降（摩擦力干扰偏大）
  - 不宜用于存在较大偏载的系统

## 速记卡

- 节流阀流量公式：$Q = C_d A \sqrt{2\Delta p/\rho}$ ——核心公式
- Δp 负载相关 → 普通节流调速速度不稳
- 薄壁孔（n=0.5，与 μ 无关）vs 细长孔（n=1，与 μ 相关）→ 调速用薄壁孔
- 调速阀 = 节流阀 + 定差减压阀 → 保持 Δp_th ≈ const → Q 只由 A 决定 → **速度刚性高**
- 分流集流阀：等分/比例分配流量 → 多缸同步
- ==调速阀解决的是节流阀"速度随负载变"的缺陷 —— 代价是成本 + 发热 ==
