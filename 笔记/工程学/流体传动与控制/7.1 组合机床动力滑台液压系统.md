## 一·系统概述

一·**设备**：YT4543 型组合机床动力滑台液压系统。组合机床用于钻孔、扩孔、铰孔、镗孔、铣削等加工工序，动力滑台带动主轴箱完成进给运动。

二·**典型工作循环**：==快进→一工进→二工进→死挡铁停留→快退→停止==（六工况）。

三·**系统主要元件**：
  - 动力源：**限压式变量叶片泵**（节能：快进大排量低压、工进小排量高压自动切换）
  - 执行元件：单杆双作用液压缸（差动连接快进）
  - 控制阀组：
    - 电液换向阀（三位四通 Y 型中位）
    - 两个调速阀（串接做两种工进速度——一工进/二工进）
    - 行程阀（机动阀，用于快进→一工进切换）
    - 液控顺序阀（用于快进→工进后压力升高→切换差动连接回路）
    - 溢流阀（安全阀）
    - 压力继电器（死挡铁停留→检测到升压→发信号切换快退）

## 二·工作循环与油路分析

  ![[图_组合机床动力滑台液压系统回路图.excalidraw]] <!-- 📌 待补图 -->
  > **图注**：YT4543 型组合机床动力滑台液压系统完整回路图。标注元件编号（泵1、电液换向阀2、行程阀3、调速阀4/5、液控顺序阀6、压力继电器7等）、各工况的进油/回油路径。

一·**快进（前进——差动连接快速）**
  - 电磁铁状态：1YA+、2YA-、3YA-
  - 进油路：变量泵 → 电液换向阀左位（P→B）→ 液压缸无杆腔
  - 回油路：液压缸有杆腔 → 电液换向阀左位（A→T？其实是经换向阀→行程阀→…）→ 重新进入无杆腔（差动连接）
  - 关键：快进时有杆腔回油不直接回油箱，而是经换向阀、行程阀回到无杆腔——形成差动快速。
  - 同时液控顺序阀（6）在低压时关闭→泵的压力低→变量泵处于大排量区→快进。

二·**一工进（第一次工作进给——慢速 1）**
  - 快进行程到位→压下滑块→切换行程阀（3）→进入一工进
  - 电磁铁状态：1YA+、2YA-、3YA-
  - 进油路：变量泵 → 电液换向阀左位（P→B）→ 行程阀 → 调速阀①（4）→ 液压缸无杆腔
  - 回油路：有杆腔 → 换向阀左位（A→T）→ 液控顺序阀（此时高压已打开）→ 回油箱
  - 关键：行程阀切换→回油路不再走差动路径→改为经顺序阀回油箱→差动回路被切断。压力升高→变量泵自动减小排量。调速阀设定低速→速度稳定。

三·**二工进（第二次工作进给——慢速 2）**
  - 一工进行程到底→挡块压到第二个行程开关→3YA+ 电磁铁得电
  - 电磁铁状态：1YA+、2YA-、3YA+
  - 进油路：变量泵 → 电液换向阀左位 → 行程阀 → 调速阀①→ 又经过调速阀②（5）（两阀串联）→ 液压缸无杆腔
  - 串入的第二个调速阀使速度进一步降低（两种工进速度适应粗→精加工转换）。

四·**死挡铁停留（保压延时）**
  - 工进到死挡铁后缸不能再动→压力急剧升高。
  - 压力继电器（7）检测到高压信号→发出电信号→开始**延时**（停留时间可调）。
  - 这时泵还在工进状态供油→但缸不动→泵出口压力升到变量泵拐点甚至溢流阀设定→能量消耗为维持压力和泄漏补偿。
  - 延时结束时→压力继电器信号传给电控→进入快退。

五·**快退**
  - 电磁铁状态：1YA-、2YA+、3YA-（电液换向阀切换到右位）
  - 进油路：变量泵 → 电液换向阀右位（P→A）→ 液压缸有杆腔
  - 回油路：液压缸无杆腔 → 行程阀（已复位为常通）→ 单向阀 → 电液换向阀右位（B→T）→ 回油箱
  - 关键：快退时泵卸荷（换向阀切换为卸荷状态？不——是变量泵自调节：快退时负载很小→压力降低→变量泵恢复大排量→快退速度快）

六·**原位停止**
  - 快退到原位→挡块压下终点行程开关→电磁铁全部失电
  - 电液换向阀回中位（Y 型机能：A/B通T、P堵）
  - 液压缸两腔回油经换向阀接回油箱→缸浮动（无油压锁定）
  - 泵出口 P 口堵住→泵压力升高→变量泵自动降到最小排量（维持泄漏），溢流阀尚未开启、泵相当于"憋泵"→泵有细小泄漏、压力维持在溢流阀设定值以下。

## 三·电磁铁动作表

| 工况 | 1YA | 2YA | 3YA | 行程阀位 | 顺序阀 |
|------|-----|-----|-----|---------|-------|
| 快进 | + | - | - | 右位（常通态） | 关 |
| 一工进 | + | - | - | 左位（压下行） | 开（高压通） |
| 二工进 | + | - | + | 左位 | 开 |
| 死挡铁停留 | + | - | + | 左位 | 开 |
| 快退 | - | + | - | 右位（复位） | 关 |
| 原位停止 | - | - | - | 右位 | 关 |

（+ 表示电磁铁通电；- 表示断电）

## 四·系统特点

  1.**调速回路**——采用**限压式变量泵 + 调速阀**的容积节流调速：无溢流损失、速度刚性高。
  2.**快速运动回路**——采用差动连接（快进速度只与活塞杆直径 d 有关，$v_{diff} = 4Q/\pi d^2$）。
  3.**换向回路**——电液换向阀 + 压力继电器 + 时间继电器的电控配合实现各工况自动切换。
  4.**快进→工进换接**——**行程阀（机动阀）** 实现（快速可靠不受负载干扰）+ 液控顺序阀（切断差动回油路径）。
  5.**二次工进换接**——**两调速阀串联**，通过 3YA 电磁阀切换是否串接第二个调速阀，实现两种工进速度选择。

  > [!note] 核心理解
  > YT4543 是教科书级的"机电液一体"典型案例：变量泵（节能）、差动连接（快进更省）、行程阀（自动换接、不受负载变）、电液换向阀（大流量控制）、压力继电器（检测到位）、调速阀（工进稳）——把本章讲过的多种回路集成为一台机床的液压系统。

## 速记卡

- ==YT4543 六工况：快进→一工进→二工进→死挡铁停留→快退→停止==
- 快进：差动连接 → 有杆腔回油引入无杆腔 → 快
- 工进：行程阀切断差动回路 + 调速阀稳速 + 变量泵自减排量
- 停留：压力继电器检测到位→发信号→延时
- 快退：电液换向阀切换油路→回油直通→活塞快退
- 5 项特点：容积节流调速 + 差动连接快进 + 电液换向 + 行程阀换接 + 串联双调速阀
- ==动力滑台 = 组合机床的"进给机构"，液压系统的集成是它的灵魂==
