## 一·换向回路

一·**定义**：改变执行元件运动方向（伸出↔缩回、正转↔反转），由控制油液流动方向实现。

二·**实现方式**：
  1.**手动/脚踏换向阀**：手动操作杆直接推动阀芯→换向。简单设备、小流量。
  2.**电磁换向阀**：电磁铁得电/失电→阀芯移动→换向。中小流量（≤ ~80 L/min）电控系统主流。
  3.**电液换向阀**：小电磁先导+大液动主阀换向。大流量（> ~80 L/min）系统必选。
  4.**行程阀（机动阀）**：工作台上的挡块推动阀芯（机械触发）→换向。不需要电信号、可靠、但位置固定后不易调。

三·**换向冲击问题**：
  - 换向阀切换瞬间——油路突然切换→执行元件快速启停→惯性→液压冲击（颤振）。
  - 解决办法：电液阀的主阀芯两端可加**单向节流阀**（减慢主阀芯切换速度→减缓油路切换速度→减小冲击）。

## 二·锁紧回路

一·**定义**：使执行元件（液压缸/马达）在任意位置停止并**长时间可靠保持位置不变**（即使泵卸荷、管路失压状态）。

二·**两种方式**：

  1.**用换向阀中位机能锁紧**（最简但**不可靠**）
    - 用 O 型（P-A-B 全闭）换向阀——中位时各口封闭 → 液压缸两腔油液被封住 → 活塞不动
    - **缺陷**：滑阀间隙存在 **~5~10 μm** 的内泄漏 → 长期使用（几小时至几天）油液会从内部泄漏通过间隙 → 活塞缓慢漂移
    - 适用：允许微速漂移的短期锁定（如机床进给暂停）

  2.**液控单向阀（液压锁）**（==最可靠的锁紧方案==）
    - 两个液控单向阀反向并联成液压锁——中位时控制口卸荷 → 两个单向阀反向截止 → 双向油路绝对封闭（阀座密封、零泄漏）
    - 换向阀切向某侧 → 供给控制油 + 主油 → 一个单向阀正向通、另一个反向被控通 → 缸可移动
    - 特点：**零内漏**——可在任意位置长时间锁紧
    - 应用：起重机支腿（保证放下后不会缩回）、矿用支架（支撑顶板不会下落）

  > [!note] 核心理解
  > 换向阀中位锁紧 ≈ 用木塞堵——有缝隙，油会慢慢渗走；液压锁锁紧 ≈ 用阀门关死——阀座接触密封，完全不漏。所以能承受长时间保压+零漂移要求的必须用液压锁。

## 三·制动回路

一·**定义**：对于液压马达驱动的旋转系统（如绞车、回转减速机），停止供油后马达因惯性会继续旋转 → 需要制动回路吸收惯性动能、平稳减速并锁定。

二·**常见制动方式**：

  1.**溢流阀制动（能耗制动）**：
    - 在马达两腔之前并联溢流阀（或双向溢流阀组）——停止供油后，马达依靠惯性继续旋转→变成"泵"→排油压力升高→溢流阀打开→排油经溢流阀回（产生背压）→吸收旋转动能→减速
    - 特点：制动快、能量转化为热（能耗制动）；需要散热

  2.**液压锁制动**：
    - 马达进出口各装一个液控单向阀——停止前先切换换向阀至中位→液控单向阀锁死→马达进出口油液被封住→瞬间强制制动
    - **问题**：油液不完全可压缩（含气）→制动后期可能有尖刺冲击
    - 必须配合溢流阀（吸收峰值冲击——称作"缓冲阀"）

  3.**制动缸 + 摩擦片**（机械制动辅助）：
    - 液压释放 + 弹簧制动（fail-safe）——停油时弹簧紧压摩擦片→摩擦制动→保持位置；通油时油压松开制动
    - 应用：车辆行走、回转机构（==起重机回转制动、挖掘机回转制动都用的这种==）

三·**典型回路**：起重机起升绞车制动回路——上升停止时，用溢流阀+液控单向阀+制动缸三重配合：溢流吸收旋转惯性动能、单向阀锁位防后溜、弹簧制动缸在停止油压后自动抱死。

## 速记卡

- 换向回路 = 换向阀（电磁/电液/行程）；大流量必须电液、换向冲击靠主阀芯节流调节
- 锁紧：换向阀中位（有内漏） < 液压锁（零内漏，可靠锁位）
- 制动三条路：溢流阀能耗制动（吸能量）、液控单向阀锁制动（配合缓冲）、弹簧制动缸（fail-safe 机械制动）
- ==起重机支腿必须用液压锁——用中位 O 型锁紧支腿会慢慢缩回去（有内漏），这就是设计事故==
