## 一·系统概述

一·**设备**：Q2-8 型汽车起重机（8 吨级），底盘为商用汽车底盘，上车为液压驱动起重机（转台、吊臂/伸缩臂、变幅、起升）。

二·**液压系统组成**：
  - 动力源：一台定量泵（由汽车发动机 PTO 取力器驱动）——因不变量，回路中溢流阀安全限定工作压力 **~21 MPa**。
  - 控制核心：一组手动弹簧复位**多路换向阀**（多个滑阀集成为一块，前端进油、后端回油集成）——每路控制一个子系统（支腿、转台、吊臂伸缩、变幅、起升）。
  - 执行元件：液压缸（支腿/变幅/伸缩）+ 液压马达（回转/起升吊重）。

  ```
  多路换向阀结构示意（M 型中位 → 中位卸荷）：
  定量泵 ───→ 多路阀总进油 P
              │
              总回油 T（回油箱）
              │
              各联滑阀（并联/串联）：
                ● 支腿控制联
                ● 回转控制联
                ● 伸缩控制联
                ● 变幅控制联
                ● 起升控制联
  ```

## 二·五大子系统逐工况分析

一·**支腿收放回路（液锁——保证安全停稳）**
  1.作用：起重机作业前将前后四个支腿从行车位置放下→支撑地面→使轮胎离地、底盘水平稳定。
  2.油路：
    - 支腿控制联切换到"放下"位 → 泵供油→液控单向阀组（液压锁）→ 支腿液压缸无杆腔→活塞伸出→支腿放下。
    - 支腿到位后，多路阀回到中位（卸荷）→ 液压锁控制口卸荷→ 两个液控单向阀反向截断→ 即使管路泄压、支腿因液锁不会缩回。
  3.收起时：支腿控制联切换到"收起"位 → 泵供油同时提供控制油→单向阀反向被控通→油经液控单向阀进入有杆腔→收回支腿。
  4.关键：==支腿回路用液压锁（液控单向阀）锁紧==，防止作业中支腿意外回缩。

二·**转台回转回路**
  1.作用：上车（吊臂/驾驶室）整体 360°回转——使起重机在不移动底盘的情况下扩大工作范围。
  2.执行元件：回转液压马达（双向——正反转实现顺时针/逆时针回转）。
  3.控制联换向阀切换左/右位 → 液压马达正/反转。
  4.特点：马达带缓冲阀（防止停止回转时的冲击）+ 制动缸（停止时弹簧制动锁定方位）。

三·**吊臂伸缩回路**
  1.作用：改变吊臂长度（伸/缩）以适应不同工作半径。
  2.执行元件：伸缩液压缸（多级单杆缸），活塞杆端固定于基本臂，缸筒端推动伸缩臂移动。
  3.油路分析：
    - 伸臂：换向阀→伸缩缸无杆腔→活塞伸出（推力大，带动臂伸出）
    - 缩臂：换向阀切换到反位→油进入有杆腔+同时控制口打开外控单向顺序阀（平衡阀）→回油通过平衡阀→臂回收
  4.重要：伸缩回路在外伸时（起重状态）由平衡阀防止缩回（如液压管路破裂→平衡阀反向截止→防止臂突然缩回造成起重事故）。

四·**吊臂变幅回路**
  1.作用：改变吊臂仰角（垂直于地面→水平展开）。
  2.执行元件：变幅液压缸（双缸并联，大负载）。
  3.油路分析类似伸缩缸——上升时无杆腔进油、下降时有杆腔进油+外控单向顺序阀（平衡阀）调节下降速度。
  4.关键：==变幅下降时回油路上设置外控单向顺序阀作平衡阀——在管路破裂时防止吊臂坠落==。下降速度由平衡阀的开度决定（不受负载重力影响），实现平稳下降。

五·**起升回路（重物提升与下降）**
  1.作用：吊钩升降（起吊/下放重物）——整个起重机最关键的功能回路。
  2.执行元件：起升液压马达（高速马达+减速器→卷扬/钢丝绳→吊钩升降）。
  3.油路：
    - 起升：换向阀→起升马达→马达旋转→卷扬机收紧→重物上升。**液压马达出油口通回油**（带背压）。
    - 下降：换向阀反向→马达反转→重物下降。回油路上同样有平衡阀（外控单向顺序阀）——控制下降速度且防止失速。
  4.特点：起升回路需**制动缸**（弹簧制动、通油松闸）——当系统停止/失压时制动缸自动抱死，防止重物滑落。
  5.注意：起升马达的正/反向换向作用外控顺序阀——控制下降的回油节流——走"进油"侧是平衡阀的通/断在反向时控制回油量。

## 三·系统特点

  1.**多路换向阀**（手操——M 型中位）——所有换向阀都采用 M 型中位机能(P/T 相通) → 在多路阀中位时**泵卸荷**（泵出口经所有联的 P→T 回油箱——中位卸荷**几乎零压**），节能。需要动某路时操杆→该联换向，卸荷停止。
  2.**平衡阀（外控单向顺序阀）** 广泛使用——起升下降、吊臂伸缩、变幅下降三条路的回油侧各装一个——在机械性重力作用下防止加速下降与管路破裂事故。==平衡阀是起重机的"安全锁"==。
  3.手动弹簧复位阀组的可靠性——机械操作简单直接、故障率低（适合户外/移动/恶劣环境）。驾驶员通过手感即可感知油路状态（阀口开度与工况相匹配）。
  4.定量泵 + 溢流阀安全限压（~21 MPa）：因不变量，系统功率略大（溢流损失发热），但对移动设备的可靠性而言简单比复杂好。

  > [!question] 为什么起重机多路阀用 M 型中位卸荷，而不是其他中位机能？
  > 因为 M 型 P-T 通时泵卸荷压力最低（约 0.2~0.5 MPa——管路背压），能耗最小。若用 O 型（P 堵），泵一直憋压至溢流阀设定压力→浪费燃油+发热。这在行走机械中燃油经济性差别巨大。

## 速记卡

- Q2-8 起重机五子系统回路：支腿、回转、伸缩、变幅、起升
- ==定量泵 + M 型多路换向阀组中位卸荷——节能的行走机械标准方案==
- 支腿用**液压锁**锁紧——防止作业中意外缩回
- 起升/变幅/伸缩 三条下降回路均用**平衡阀（外控单向顺序阀）**——防管路失速/坠落的"安全锁"
- 起升马达带弹簧制动缸——fail-safe 失压即刹停
- ==液压起重机安全链：支腿液压锁 + 平衡阀防掉臂 + 制动缸防吊重滑落——三重保护==
