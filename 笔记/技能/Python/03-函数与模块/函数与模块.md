# Python 函数与模块

#python #function #module

## 一·函数定义与调用

一·**定义**：用 `def` 关键字定义函数，**参数无类型声明**（动态类型），返回值用 `return`，缺省返回 `None`。

```python
def greet(name, greeting="你好"):
    """文档字符串（docstring），可被 help() 读到。"""
    return f"{greeting}，{name}！"

greet("Sunny")             # 你好，Sunny！
greet("Sunny", "早上好")   # 早上好，Sunny！
help(greet)                # 打印 docstring
```

二·**返回多值 = 返回元组**：
```python
def min_max(nums):
    return min(nums), max(nums)     # 隐式打包成 tuple
lo, hi = min_max([3, 1, 4, 1, 5])  # 解包
```

三·**核心理解**：
> [!note] 核心理解
> Python 函数是**一等公民**（first-class）：可以赋值给变量、作参数传入、作返回值返回、塞进数据结构里。这是「函数式风格」「高阶函数」「装饰器」一切特性的根。`def` 本质上就是把一个函数对象绑定到一个名字。

## 二·参数：四种传参方式

一·**位置参数**：按位置对应。
```python
def add(a, b): return a + b
add(2, 3)        # 5
```

二·**默认参数**：从右往左依次设默认值。
```python
def power(x, exp=2): return x ** exp
power(3)         # 9
power(3, 4)      # 81
```

> [!warning] 经典坑：可变默认参数
> ```python
> def f(xs=[]):   # ❌ 默认值在 def 时只创建一次，会跨调用共享！
>     xs.append(1)
>     return xs
> f(); f()        # [1, 1]——不是两次独立空表！
> ```
> 正确写法：默认值用 `None`，函数体内惰性建：
> ```python
> def f(xs=None):
>     if xs is None: xs = []
>     xs.append(1); return xs
> ```

三·**关键字参数**：调用时写 `名=值`，免记顺序，可读性强。
```python
def describe(name, age, city): ...
describe(name="Sunny", city="上海", age=20)   # 顺序无关
```

四·**可变参数**：
（1）`*args` 收集多余**位置参数**成元组：
```python
def sum_all(*nums):
    return sum(nums)
sum_all(1, 2, 3)    # 6
```
（2）`**kwargs` 收集多余**关键字参数**成字典：
```python
def info(**kw):
    for k, v in kw.items(): print(k, v)
info(name="Sunny", age=20)
```
（3）调用时反向**解包**：
```python
nums = [1, 2, 3]; sum_all(*nums)        # * 解包列表到 *args
cfg = {"name":"S", "age":20}; info(**cfg)   # ** 解包字典到 **kwargs
```

五·**参数顺序铁律**：`def f(pos, /, normal, *args, kw_only, **kwargs)`——**位置参数 → 普通参数 → *args → 仅关键字参数 → **kwargs**，不可乱序。

## 三·lambda 表达式

`lambda 参数: 表达式`：一行匿名函数，**只能一个表达式**，不能写语句。

```python
square = lambda x: x * x
square(5)            # 25

# 主要用途：作高阶函数的临时回调
sorted(students, key=lambda s: s["score"])        # 按分数排序
list(map(lambda x: x*2, [1,2,3]))                 # [2,4,6]（但推导式更 Pythonic）
```

> [!note] 核心理解
> lambda 的定位是「**临时、即用即弃**」——出现一次、就这一处用、不值得起名。一旦你在多处复用它，或者需要写多行逻辑，就别 lambda，老老实实 `def`。Python 社区推崇显式优于隐式，**`def` 永远是首选**。

## 四·作用域 LEGB

变量查找顺序：**L → E → G → B**。

| 层 | 含义 | 示例 |
|----|------|------|
| **L** Local | 函数内局部 | 函数体内的变量 |
| **E** Enclosing | 外层函数（嵌套函数） | 闭包的外层变量 |
| **G** Global | 模块级 | 文件顶部定义的 |
| **B** Built-in | 内置 | `print`, `len`, `int` ... |

```python
x = 10                     # G
def outer():
    y = 20                 # E（相对 inner）
    def inner():
        z = 30             # L
        print(x, y, z)     # 找 L 找不到 x → 找 E 的 y → 找 G 的 x
    inner()
```

> [!warning] `global` / `nonlocal` 易错
> 函数内**赋值**默认创建**新的局部变量**，不会改全局。要改全局用 `global`，要改外层函数变量用 `nonlocal`：
> ```python
> def counter():
>     n = 0
>     def step():
>         nonlocal n        # 不写就报错或建局部 n
>         n += 1
>         return n
>     return step
> ```
> 别滥用——大多数 `global` 是设计气味，能传参就别用全局。

## 五·模块与 import

一·**定义**：一个 `.py` 文件就是一个**模块**（module）；含 `__init__.py` 的文件夹是**包**（package）。模块用 `import` 复用。

二·**import 四种姿势**：
```python
import math                # 整模块，引用要带名 math.sqrt(2)
import math as m          # 起别名
from math import sqrt     # 只引入名字，调用 sqrt(2)
from math import *        # ⚠️ 全倒入命名空间，易冲突，避免用
```

三·**标准库速览**（「电池齐备」）：

| 模块 | 用途 |
|------|------|
| `math` | 数学函数 sqrt/sin/log |
| `random` | 随机数 randint/random.choice |
| `os` / `pathlib` | 路径、文件系统（新代码用 pathlib） |
| `sys` | 解释器参数、sys.path |
| `datetime` | 日期时间 |
| ` collections` | deque/Counter/defaultdict/namedtuple |
| `itertools` | 迭代器工具 chain/groupby/count |
| `json` | 序列化 |
| `re` | 正则 |
| `typing` | 类型注解（运行时无强制，便于 IDE/mypy） |

四·**自己写模块**：把可复用逻辑放 `myutil.py`，别处 `import myutil`。`__name__ == "__main__"` 套路：
```python
def main():
    print("运行主流程")

if __name__ == "__main__":   # 直接运行本文件时执行；
    main()                   # 被 import 时**不**自动执行
```

五·**核心理解**：
> [!note] 核心理解
> `if __name__ == "__main__"` 防护让一个文件「既能当脚本直接跑，又能当模块被 import」——这是 Python 工程最小的可复用单元，养成习惯写上它。

## 六·类型注解（typing，进阶但推荐）

```python
from typing import List, Dict, Optional

def avg(nums: List[float]) -> float:
    return sum(nums) / len(nums)

def lookup(d: Dict[str, int], key: str) -> Optional[int]:
    return d.get(key)        # 不存在返回 None，用 Optional 标注
```

一·**核心理解**：
> [!note] 核心理解
> 注解**不影响运行**（Python 不强制），它是给**人 / IDE / mypy** 看的文档。新代码加注解是「现代 Python」的标志，可读性 + 可维护性显著提升，又不增加运行时负担。

## 速记卡 / 一句话总结

- 函数 = 一等公民：可赋值、可传参、可返回、可存容器。`def` 本质是绑定名字。
- ==可变默认参数是经典坑==：`def f(xs=[])` 跨调用共享，默认值用 `None` + 函数体惰性建。
- 参数四种：位置 / 默认 / 关键字 / 可变（`*args` → 元组，`**kwargs` → 字典）；顺序铁律 `pos → normal → *args → kw_only → **kwargs`。
- `lambda` 只用一行回调，**多处复用就 `def`**。
- 作用域 LEGB：L 局部 → E 外层函数 → G 模块 → B 内置；改全局用 `global`，改闭包变量用 `nonlocal`，但都少用。
- 模块 = `.py`，包 = 含 `__init__.py` 的目录；`import` 四式，**避免 `from x import *`**。
- ==每个可执行文件加 `if __name__ == "__main__":`==，让脚本可复用。
- 类型注解纯文档、不强制，新代码都该加。
